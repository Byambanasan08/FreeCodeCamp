# Bash гэж юу вэ, мөн түүний зарим үндсэн командууд юу вэ?

Bash болон түүний зарим үндсэн командуудын талаар судалцгаая.

Bash нь Bourne Again SHell гэсэн утгатай бөгөөд Unix-тэй төстэй (Unix-like) орчинд хамгийн түгээмэл хэрэглэгддэг shell гэж хэлж болно. Иймээс энэ shell дотор хэрхэн navigation хийх, түгээмэл командуудыг хэрхэн ашиглахыг ойлгох нь маш их ашиг тустай.

Эхний команд бол pwd бөгөөд энэ нь одоогийн working directory-г terminal дээр хэвлэж (print) харуулдаг. "Working directory" гэдэг нь terminal одоогоор зааж байгаа directory-г хэлдэг бөгөөд дараагийн олон командуудыг ашиглахад энэ нь чухал ойлголт юм.

Таны terminal буруу directory руу заасан байх боломж бүрэн бий. cd команд нь directory солих боломжийг олгодог. Та absolute path (эхэнд нь forward slash (/) орсон зам), relative path (ямар нэг prefix-гүй зам), мөн double-dot (..) syntax ашиглан parent directory руу шилжиж болно.

Дараагийн команд бол ls бөгөөд энэ нь одоогийн working directory-ийн агуулгыг жагсааж харуулдаг. Энэ нь таны directory дотор файл эсвэл folder байгаа эсэхийг харахад тусалдаг. ls команд нь мөн hidden file-уудыг харуулах (-a flag), file permission-уудыг харуулах (-l flag) зэрэг нэмэлт боломжуудтай. Та command flag-уудын талаар дараагийн хичээл дээр илүү дэлгэрэнгүй сурах болно. Мөн та less эсвэл cat ашиглан файлын агуулгыг харж болно.

Хэрвээ directory дотор танд хэрэгтэй файл эсвэл folder байхгүй бол яах вэ? mkdir болон touch гэсэн командууд байдаг бөгөөд эдгээр нь шинэ directory болон файл үүсгэх боломж олгодог. mkdir нь шинэ directory буюу folder үүсгэнэ, харин touch нь шинэ файл үүсгэнэ. Эдгээр командууд нь үүсгэхийг хүссэн файл эсвэл folder-ийн нэрийг параметр болгон авдаг. Жишээлбэл, touch readme.md гэж бичвэл readme.md нэртэй шинэ Markdown файл үүсгэнэ.

Магадгүй таны шинэ файл readme.md биш, харин readthis.md гэж нэрлэгдэх ёстой байсан байж болно. Аз болоход, mv команд нь файлыг move хийх (эсвэл rename хийх) боломж олгодог. Энэ нь хуучин файлын нэр болон шинэ файлын нэрийг параметр болгон авдаг.

rm команд нь файлыг устгах (delete) боломж олгодог бөгөөд -r flag ашиглавал directory-г устгана. Заримдаа файл хамгаалагдсан (protected) байж болох бөгөөд энэ үед -f flag-ийг ашиглах шаардлагатай болно.

cp команд нь файл эсвэл directory-г шинэ байршил руу copy хийхэд ашиглагдана. mv командаас ялгаатай нь cp команд нь эх файлыг устгахгүй — мөн directory-г copy хийхийн тулд -r flag-ийг дамжуулах хэрэгтэй.

echo командыг Bash дахь console.log() эсвэл print() функцтэй адилтгаж ойлгож болно. echo нь хашилт ("") дотор бичигдсэн string аргументыг авч, terminal дээр хэвлэж харуулдаг. Энэ нь зүгээр л terminal дээр текст хэвлэх мэт харагдаж болох ч, та echo-г control symbol-уудтай холбож ашиглаж болно. > тэмдэг нь шинэ string-ээр файл үүсгэх эсвэл overwrite хийхэд ашиглагддаг, жишээлбэл echo "Naomi was here." > readme.md, харин >> тэмдэг нь файлын төгсгөлд нэмэх (append) үйлдэл хийдэг.

Танд ашиглах боломжтой Bash-ийн маш олон командууд байдаг, жишээлбэл head, tail, chown, болон chmod. Эдгээрийг бүгдийг нь энэ хичээлд багтаах боломжгүй. Гэхдээ та man командыг ашиглан бараг ямар ч командын manual буюу тусламжийн хуудсыг харах боломжтой.

Та хамгийн их ашигладаг командуудаа аяндаа тогтоож авна. Харин бусад командуудын хувьд, та үргэлж man page-ийг шалгаж болно.
