# Command Line дээр бүтээмжийг (productivity) нэмэгдүүлэхэд ашиглаж болох зарим shortcut-ууд юу вэ?

Command line shortcut-уудыг ашиглан бүтээмжээ хэрхэн нэмэгдүүлэх талаар судалцгаая.

Terminal болон command line shortcut-ууд нь таны workflow-ийг илүү үр ашигтай (streamline) болгож, бүтээмжийг дээд хэмжээнд хүргэхэд тусалдаг. Linux болон macOS нь хоёулаа Unix-ээс гаралтай тул эдгээр shortcut-уудын ихэнх нь ижил байдаг. Харин Windows-ийн хувьд зарим ялгаа бий.

Эхний shortcut бол дээш чиглэсэн сум (up arrow key) юм. Энэ товчийг дарснаар та command history-даа буцаж, өмнө ажиллуулсан командуудыг нэг нэгээр нь харах боломжтой. Харин доош чиглэсэн сум (down arrow key) нь command history-даа урагш шилжихэд ашиглагдана. Эдгээр хоёр товч нь өмнө ажиллуулсан командуудаас хэрэгтэйг нь хурдан олж дахин ажиллуулах боломжийг олгодог.

Олон shell-ууд auto-complete (автомат гүйцээх) боломжийг санал болгодог бөгөөд та бичиж эхэлсэн текст дээр үндэслэн командуудыг санал болгодог. Tab товчийг ашиглан тухайн санал болгосон командыг бүрэн бөглөж, command line-ийг хурдан гүйцээж болно. Эдгээр санал хэрхэн үүсэх нь shell бүрээс хамааран өөр өөр байдаг. Жишээлбэл, zsh дээр эдгээр санал нь таны хамгийн сүүлд ашигласан command history дээр тулгуурладаг. Харин PowerShell дээр, эдгээр нь боломжтой командууд, хувьсагч (variables), аргументууд дээр тулгуурладаг (сүүлийн үед ашигласан эсэх нь бага нөлөөтэй).

Заримдаа команд ажиллуулахад terminal дээр маш их output гарч ирдэг. Энэ илүүдэл мэдээлэл (noise) нь таны ажлын урсгалд саад болж болзошгүй. Linux болон macOS дээрх \*nix суурьтай terminal-уудад Control + L товчийг дарснаар дэлгэцийг бүхэлд нь цэвэрлэж, шинэ prompt-оос эхлэх боломжтой. PowerShell дээр та cls командыг ажиллуулах хэрэгтэй (үүнийг Control + L зэрэг товчны хослолд холбож болно).

Хэрвээ та ямар нэг командыг гүйцэтгэх явцыг зогсоох шаардлагатай бол Control + C ашиглан тухайн процессыг зогсоож болно. Энэ нь командыг terminate хийж, шинэ prompt үүсгэх бөгөөд ингэснээр та дараагийн ажлаа үргэлжлүүлэх боломжтой болно. PowerShell хэрэглэгчдийн хувьд Control + C нь мөн текст хуулбарлахад ашиглагддаг бөгөөд зөвхөн тодорхой нөхцөлд (жишээлбэл, хуулбарлах текст сонгогдоогүй үед) процессыг зогсоох үйлдэл ажиллана.

Дараагийн хэд хэдэн shortcut нь зөвхөн \*nix суурьтай terminal-уудад байдаг бөгөөд PowerShell дээр ижил хувилбар байхгүй.

Зарим үед та multitask хийх шаардлагатай байж болно, өөрөөр хэлбэл нэг процесс эсвэл командыг background дээр ажиллуулж байх үед өөр зүйл дээр ажиллах. Control + Z товчийг дарснаар одоогийн процессыг background task болгож, таныг command line руу буцаана. Ингэснээр та ажлаа үргэлжлүүлж болно. Background task руу буцаж шилжих шаардлагатай үед fg командыг ашиглан сэргээж болно.

Эцэст нь, зарим командуудыг дараалан хоёр удаа ажиллуулах шаардлага гарч болно. Та дээш сум товчийг дарж өмнөх командыг олж болно, мөн command line дээр хоёр анхаарлын тэмдэг (!!) бичиж enter дарахад хамгийн сүүлд ажиллуулсан командыг автоматаар дахин ажиллуулна.

Олон terminal-ууд үүнээс ч олон shortcut-уудыг санал болгодог бөгөөд өөрийн ашиглаж буй орчны documentation-ийг уншиж, өөрт хамгийн тохирох shortcut-уудыг олохыг зөвлөж байна. Гэхдээ эдгээр үндсэн, нийтлэг shortcut-ууд нь таны бүтээмжийг нэмэгдүүлэх, terminal дээр чадварлаг (wizard) болох эхлэлийн маш сайн суурь болно.
