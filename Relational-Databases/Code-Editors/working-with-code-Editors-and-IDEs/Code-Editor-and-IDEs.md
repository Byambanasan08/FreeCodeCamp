# Code Editor болон IDE гэж юу вэ?

Та эдгээрийг ижил зүйл гэж бодож магадгүй, гэхдээ яг адил биш. Code editor гэдэг нь кодын файлуудыг засварлах боломж олгодог аливаа програм юм. Харин IDE буюу Integrated Development Environment нь кодыг засварлахын зэрэгцээ compile хийх, ажиллуулах (run), мөн алдааг засах (debug) боломжийг олгодог бүрэн хэмжээний програм юм.

Жишээнүүдийг харах нь илүү ойлгомжтой байж магадгүй.

Эхлээд хэд хэдэн түгээмэл IDE-үүдийг авч үзье. Visual Studio нь Visual Studio Microsoft-ын боловсруулсан integrated development environment бөгөөд олон төрлийн платформ дээр application бүтээх, debug хийх, deploy хийхэд зориулсан иж бүрэн хэрэгслүүдийг санал болгодог.

Өөр нэг IDE бол Xcode юм. Xcode нь Apple-ийн боловсруулсан integrated development environment бөгөөд macOS, iOS, watchOS, tvOS-д зориулсан application үүсгэх, код бичих, debug хийхэд зориулагдсан.

Мөн өөр нэг IDE нь Android Studio юм. Android Studio нь Google-ийн боловсруулсан integrated development environment бөгөөд Android application-уудыг бүтээх, debug хийх, тестлэхэд тусгайлан зориулагдсан.

Эдгээр нь IDE-үүдийн хэдхэн жишээ юм. Одоо хэд хэдэн түгээмэл code editor-уудыг харцгаая.

Visual Studio Code нь Microsoft-ын гаргасан хөнгөн жинтэй (lightweight), нээлттэй эхтэй (open-source) code editor бөгөөд олон төрлийн програмчлалын хэл дэмждэг ба extension-уудаар дамжуулан debugging, syntax highlighting, version control зэрэг боломжуудыг олгодог.

Өөр нэг түгээмэл editor бол Sublime Text юм. Sublime Text нь хурдан, олон талт (versatile) text editor бөгөөд гоёмсог интерфэйс, хүчирхэг боломжууд, мөн customizable syntax highlighting болон plugin-уудаар дамжуулан олон төрлийн програмчлалын хэл дэмждгээрээ алдартай.

Мөн өөр нэг нь Notepad++ юм. Энэ нь Windows-д зориулсан үнэгүй (free), нээлттэй эхтэй (open-source) text болон source code editor бөгөөд syntax highlighting, code folding, мөн productivity болон customization-ийг нэмэгдүүлэх олон төрлийн plugin-уудыг санал болгодог.

Та code editor-ууд нь ихэвчлэн файлын текст агуулгад төвлөрдөг бол IDE-үүд нь кодыг удирдахад зориулсан олон төрлийн хэрэгслүүдийг ил гаргаж өгдөгийг анзаарсан байж магадгүй.

Эдгээр жишээнүүд бүгд таны компьютер дээр ажиллах local програмууд боловч, мөн cloud-based editor-ууд ч бас байдаг.

Cloud-based editor гэдэг нь хэрэглэгчдэд ямар нэг програм суулгах шаардлагагүйгээр шууд web browser-оор дамжуулан код эсвэл текст бичих, засварлах, удирдах боломж олгодог онлайн хэрэгсэл юм.

Одоо хэд хэдэн cloud-based editor-уудыг харцгаая.

Replit нь online платформ бөгөөд хэрэглэгчдэд хамтран ажиллах (collaborative) coding орчин санал болгож, web browser-оос шууд олон төрлийн програмчлалын хэл дээр код бичих, ажиллуулах, хуваалцах боломжийг олгодог.

Өөр нэг түгээмэл cloud-based editor бол GitHub Codespaces юм. Энэ нь cloud-д суурилсан хөгжүүлэлтийн орчин бөгөөд GitHub-аас шууд бүрэн тохируулсан code editor болон хөгжүүлэлтийн хэрэгслүүдэд шуурхай хандах боломж олгож, coding болон хамтын ажиллагааг саадгүй болгодог.

Мөн өөр нэг нь Ona юм. Ona нь cloud-based хөгжүүлэлтийн орчин бөгөөд GitHub болон GitLab-тай интеграцлагдаж, browser-оос шууд coding хийх, build хийх, debug хийхэд зориулсан шуурхай, customizable workspace-уудыг санал болгодог.

Мөн өөр олон сонголтууд байдаг. Зарим сонголтууд, жишээлбэл Visual Studio Code, нь маш өргөтгөх боломжтой (highly extensible) бөгөөд олон төрлийн project болон хэлүүдтэй ажиллах боломжтой. Харин зарим нь цөөн тооны хэл эсвэл тодорхой төрлийн project-д тусгайлан зориулагдсан байдаг.

Таны ашиглах програм нь тодорхой project-оос хамааран өөр байж болно. Та өөрийн хэрэгцээнд хамгийн тохиромжтойг олохын тулд боломжит сонголтуудыг судалж үзэх хэрэгтэй.
