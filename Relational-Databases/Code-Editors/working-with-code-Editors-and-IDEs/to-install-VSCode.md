# Visual Studio Code-ийг өөрийн компьютерт хэрхэн суулгах вэ

VS Code-ийг суулгахын тулд та өөрийн үйлдлийн системд (operating system) тохирох installer-ийг авахын тулд тэдний вэбсайтад зочлох хэрэгтэй. VS Code-ийн вэбсайтад зочилж байгаа эсэхээ баталгаажуулаарай, харин Visual Studio-ийн вэбсайт биш.

Windows-ийн хувьд, та EXE installer татаж авч болно. Та үүнийг татаж авсны дараа файлыг ажиллуулна, setup wizard нь таныг суулгалтын (installation) үйл явцаар дамжуулан чиглүүлэх болно.

Mac-ийн хувьд, та .app файлыг агуулсан zip архив татаж авч болно, үүнийг та ажиллуулж болно. Та мөн VS Code-ийг homebrew ашиглан суулгаж болно.

Mac дээрээ Terminal апп-ыг нээгээд дараах командыг ажиллуулна:

brew install --cask visual-studio-code

Linux дээр, та .deb эсвэл .rpm файлыг шууд татаж авч болно. Та үүнийг суулгасны дараа, тохирох executable эсвэл CLI команд (ихэвчлэн code) ашиглан application-ийг ажиллуулж болно. Ингээд одоо та код бичихэд бэлэн боллоо.
