# VS Code дээр Project үүсгэж, өөрийн кодыг локал орчинд (locally) хэрхэн ажиллуулах вэ?

Эхлэхийн тулд, VS Code нь project-уудыг "workspace" гэж үздэг гэдгийг та мэдэж байх хэрэгтэй. Workspace гэдэг нь VS Code дээр таны нээсэн directory юм.

Эхлэхийн тулд, та шинэ project-д зориулж шинэ directory үүсгэхийн тулд command line interface ашиглах хэрэгтэй болно. Directory гэдэг нь folder-ын өөр нэр юм.

Command-line interface буюу товчоор CLI нь текст дээр суурилсан командуудыг ашиглан таны үйлдлийн системтэй харилцах боломжийг олгодог. Та эдгээр хэрэгслүүдийн талаар дараагийн хичээлүүд дээр илүү ихийг сурах болно.

Хэрвээ та Windows ашиглаж байгаа бол Command Prompt эсвэл PowerShell ашиглаж болно. Хэрвээ Mac ашиглаж байгаа бол Terminal апп-аа нээж болно.

Command line нээгдсэний дараа, cd ~ командыг бичиж, гар дээрх enter товчийг дарснаар home directory руу очно. Дараа нь mkdir my-project командыг бичиж enter дарна. mkdir буюу "make directory" нь шинэ directory үүсгэхэд ашиглагддаг команд юм. Дараа нь та шинэ directory-гаа VS Code ашиглан нээх хэрэгтэй.

Шинэ directory үүсгэх өөр график аргууд ч бас байдаг.

Таны үйлдлийн системээс хамааран, file explorer-оос шууд VS Code нээх context menu (эсвэл right-click option) байж магадгүй. Та мөн VS Code application дээр double-click хийж нээж болно. Хэрвээ та command line-д дассан бол code /path/to/folder командыг ажиллуулж болно.

Нээсний дараа, та шинэ хоосон workspace харах ёстой.

Шинэ файл үүсгэхийн тулд, зүүн дээд буланд байгаа File цэс дээр дарж New File...-ийг сонгоно. Дараа нь файлаа index.html гэж нэрлээд Save дээр дарна.

Одоо та explorer tab дээр шинэ файлаа харах ёстой. index.html файлдаа HTML код нэмээд үзээрэй.

Одоо та HTML файлаа шууд browser дээр нээх хүсэл төрж магадгүй. Энэ нь ажиллаж болох ч, CSS зөв ачаалагдахгүй байх зэрэг олон төрлийн гэнэтийн алдаанууд (unexpected bugs)-д хүргэж болно.

Үүний оронд, та HTML файлаа зөв web server-ийн ард байрлуулах хэрэгтэй. Хэрвээ та coding-д шинээр орж байгаа бол энэ нь төвөгтэй мэт санагдаж магадгүй, гэхдээ үнэндээ VS Code-оос шууд хийх арга байдаг.

Та эхлээд Live Server extension-ийг татах хэрэгтэй, энэ нь marketplace дээр үнэгүй байдаг.

Зүүн талд байгаа extensions tab дээр дарж, search bar дээр Live Server гэж бичнэ.

Үүнийг суулгасны дараа, status bar дээр Go Live гэсэн товч гарч ирэх ёстой. HTML файлаа нээсэн байгаа эсэхээ шалгаад, дараа нь тэр товчийг дарна.

Таны шинэ хуудас browser-ийн шинэ tab дээр автоматаар нээгдэх ёстой. Хэрвээ нээгдэхгүй бол та http://localhost:5500/ хаяг руу шууд орж болно.

Баяр хүргэе! Та одоо VS Code дээр өөрийн project-ийг тохируулж, live server ашиглан өөрчлөлтүүдээ харах боломжтой боллоо.
