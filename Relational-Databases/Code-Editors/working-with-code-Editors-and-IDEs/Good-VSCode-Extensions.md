# VS Code дээр ашиглаж болох сайн extension-ууд юу вэ?

Better Comments нь тодорхой кодын comment-уудыг онцгойлон highlight хийдэг extension юм. Жишээлбэл, энэ нь "TODO" comment-уудыг тодруулж харуулдаг бөгөөд мөн асуулт эсвэл анхааруулга (warning) илэрхийлэх syntax-ийг ялгаж харуулдаг.

Code Spell Checker нь таны код дотор үг буруу бичигдсэн үед зааж өгдөг. Энэ нь кодын файлуудад зориулагдсан тул camel case зэрэг онцлогуудыг тооцож үздэг.

Error Lens нь таны код дахь highlight хийгдсэн алдаануудыг илрүүлэхэд тусалдаг. VS Code-ийн default-оор харуулдаг underline-д найдах шаардлагагүйгээр, энэ extension нь бүхэл мөрийг highlight хийж, алдааны мессежийг харуулдаг.

Indent Rainbow нь indentation-ийн (зай авалтын) өөр өөр түвшнүүдэд өнгө нэмдэг. VS Code-ийн native bracket pair colorization-той хамт ашиглахад, таны код аль scope-д байгааг тодорхойлоход маш их тусалдаг.

Эсвэл та илүү практик зүйл хайж байна уу? GitHub Copilot эсвэл Tabnine зэрэг AI assistant-уудыг ашиглаж үзээрэй, эдгээр нь код бичиж байх үед inline suggestion санал болгодог.

VS Code Great Icons зэрэг icon pack нь таны file tree-ийг илүү цэвэрхэн, нэг хараад ойлгомжтой болгоход тусалдаг. Мөн Colorize зэрэг extension нь таны CSS property-ууд дахь утгуудыг ойлгоход тусална.

Та мөн өөрийн project-д зориулсан хэл (language)-д суурилсан extension-уудыг ашиглах хэрэгтэй. Хэрвээ та JavaScript ашиглаж байгаа бол, ESLint болон Prettier-ийг кодоо lint хийх болон форматлахад ашиглах нь зүйтэй. Хэрвээ та TypeScript ашиглаж байгаа бол, илүү ойлгомжтой алдааны мессеж авахын тулд Pretty Typescript Errors ашиглаж болно.

Эцэст нь, та editor-оо бага зэрэг хөгжилтэй болгож болно. VS Code Pets нь configurable virtual pets санал болгож, та код дахь алдаануудыг засаж байх үед танд “хамт” байж өгнө.

Power Mode нь та илүү их код бичиж “combo” хангалттай өндөр болсон үед гялалзсан (flashy) эффектүүд үүсгэнэ.

Discord Presence нь та яг юу дээр ажиллаж байгаагаа найзууддаа харуулах боломж олгоно.

Мөн үүнээс гадна маш олон extension-ууд байдаг. Extension marketplace-ийг судалж, өөрт хамгийн тохирохыг нь сонгоорой.
