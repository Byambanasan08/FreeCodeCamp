# VS Code дээр бүтээмжээ (productivity) нэмэгдүүлэхэд хэрэгтэй хэд хэдэн keyboard shortcut-ууд юу вэ?

Та аль хэдийн зарим үндсэн shortcut-уудтай танил болсон байх магадлалтай, эдгээр нь таны үйлдлийн системээс (operating system) удамшсан байдаг. Ctrl + S (хадгалах), Ctrl + C (хуулбарлах), Ctrl + V (наах) зэрэг shortcut-ууд VS Code дээр бүгд ажиллана.

Гэхдээ application-д тусгайлан зориулагдсан хэд хэдэн shortcut-ууд байдаг бөгөөд эдгээр нь таны бүтээмжийг улам нэмэгдүүлж чадна. Эдгээр shortcut-уудын зарим нь үйлдлийн системээс хамааран ялгаатай байж болохыг анхаарах хэрэгтэй.

Жишээлбэл, Shift + Alt + F нь одоо нээгдсэн файл дээр таны тохируулсан formatter-ийг (жишээлбэл, JavaScript project-д prettier) ажиллуулна.

Эсвэл Ctrl + Shift + F (Windows), эсвэл Cmd + Shift + F (Mac) нь таны workspace доторх бүх файлуудын текст агуулгаас хайлт хийхэд ашиглагдана. Харин Ctrl + Shift + H (Windows), эсвэл Cmd + Shift + H (Mac) нь find-and-replace (хайж орлуулах) үйлдлийг ажиллуулна.

Хэрвээ та нэг мөрийг устгах хэрэгтэй бол Ctrl + Shift + K (Windows), эсвэл Cmd + Shift + K (Mac) нь тухайн мөрийг устгана.

Танд код бичихэд илүү зай хэрэгтэй байна уу? Ctrl + B (Windows), эсвэл Cmd + B (Mac) нь sidebar-ийг (файлын жагсаалт болон extensions цэс агуулсан хэсэг) нуух болно.

Эсвэл та кодоо сайн харж чадахгүй байна уу? Ctrl + plus (Windows), эсвэл Cmd + plus (Mac) нь editor-ийн масштабыг (scaling) нэмэгдүүлнэ, харин Ctrl + minus (Windows), эсвэл Cmd + minus (Mac) нь багасгана.

VS Code дээрх өөр нэг хэрэгтэй боломж бол multi-line editing бөгөөд энэ нь олон мөр дээр нэгэн зэрэг өөрчлөлт хийх боломжийг олгодог. Жишээлбэл, та Ctrl + Alt + down болон Ctrl + Alt + up (Windows/Linux), эсвэл Option + Command + down болон Option + Command + up (Mac) ашиглан cursor-ийг дараалсан мөрүүд дээр сунгаж олон cursor байрлуулж болно. Мөн Alt (Windows/Linux) эсвэл Option (Mac)-ийг дарж барин, хэрэгтэй газрууд дээрээ click хийж тус тусад нь cursor нэмэх боломжтой. Cursor-уудыг байрлуулсны дараа, таны бичсэн эсвэл устгасан бүх үйлдэл эдгээр бүх байрлалд нэгэн зэрэг үйлчлэх болно — энэ нь давтагддаг код дээр ажиллах үед маш их цаг хэмнэдэг.

Эцэст нь, хэрвээ та эдгээр shortcut-уудаас аль нэгийг мартвал, Ctrl + Shift + P (Windows), эсвэл Cmd + Shift + P (Mac) ашиглан command palette-ийг нээж, хэрэгтэй үйлдлээ сонгож болно.

Эдгээр мэдлэг болон бага зэрэг дадлагын тусламжтайгаар, та VS Code-ийн power user болох замдаа аль хэдийн оржээ.
