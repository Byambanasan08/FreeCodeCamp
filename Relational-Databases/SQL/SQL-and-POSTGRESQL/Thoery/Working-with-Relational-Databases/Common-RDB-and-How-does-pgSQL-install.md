# Нийтлэг ашиглагддаг харилцаа хамааралт өгөгдлийн сангууд (Relational Databases) юу вэ, мөн Postgres-ийг хэрхэн суулгаж, ашиглах вэ?

Харилцаа хамааралт өгөгдлийн сангууд **Relational databases** нь вэб хөгжүүлэлт, бараа материалын удирдлагын систем, эрүүл мэндийн систем, цахим худалдааны програмууд гэх мэт маш өргөн хүрээний хэрэглээтэй байдаг.
Хамгийн нийтлэг харилцаа хамааралт өгөгдлийн сангуудын зарим нь `MySQL`, `PostgreSQL`, `SQLite` болон `Microsoft SQL Server` юм. Тэдгээр тус бүрийн талаар бага зэрэг суралцацгаая.

1.  **MySQL** бол нээлттэй эхийн (open-source) харилцаа хамааралт өгөгдлийн сангийн удирдлагын систем юм. Энэ нь ашиглахад хялбар, найдвартай, бодит амьдрал дээрх програмуудыг бүтээхэд үр ашигтай байдаг тул вэб хөгжүүлэлтэд маш алдартай. Мөн үүнийг дэмжиж, хөгжүүлж байдаг идэвхтэй хөгжүүлэгчдийн нийгэмлэгтэй.
2.  **PostgreSQL**-ийг албан ёсоор `Дэлхийн хамгийн дэвшилтэт нээлттэй эхийн харилцаа хамааралт өгөгдлийн сан` гэж тодорхойлдог. Энэ нь найдвартай байдал, өгөгдлийн бүрэн бүтэн байдал (data integrity) болон өргөтгөх боломжоороо (extensibility) алдартай, үнэ төлбөргүй, нээлттэй эхийн объект-харилцаа хамааралт `object-relational` өгөгдлийн сангийн систем юм. Жишээлбэл, та өөрийн өгөгдлийн төрлүүд, тусгай функцүүдийг тодорхойлж, өгөгдлийн сангаа дахин хөрвүүлэх (re-compiling) шаардлагагүйгээр өөр өөр програмчлалын хэл дээр код бичих боломжтой.
3.  **SQLite** бол үнэ төлбөргүй, нээлттэй эх бүхий, хөнгөн жинтэй, файл дээр суурилсан SQL өгөгдлийн сангийн систем (database engine) юм. Энэ нь дэлхийд хамгийн өргөн хэрэглэгддэг өгөгдлийн сангийн системүүдийн нэг юм. SQLite нь бие даасан (self-contained) бөгөөд сервергүй (serverless) ажилладаг. Энэ нь тохиргоо шаарддаггүй (zero-configuration) өгөгдлийн сан бөгөөд энэ нь анхны тохиргооны процесс шаарддаггүй гэсэн үг бөгөөд хурдан ажиллуулж эхлэхэд маш тохиромжтой.
    Эдгээр гурван өгөгдлийн сан нь үнэ төлбөргүй бөгөөд нээлттэй эхтэй. Мөн Microsoft SQL Server болон Oracle database зэрэг бусад алдартай өмчлөлийн (proprietary) сонголтууд байдаг.
4.  **Microsoft SQL** server нь Microsoft-оос боловсруулсан, янз бүрийн програмуудад ашиглагддаг алдартай харилцаа хамааралт өгөгдлийн сангийн удирдлагын систем юм.
5.  **Oracle database** нь өндөр гүйцэтгэл шаарддаг аж ахуйн нэгжийн (enterprise) програмуудад ихэвчлэн ашиглагддаг, өргөтгөх боломжтой харилцаа хамааралт өгөгдлийн сангийн удирдлагын систем юм.

             Гайхалтай. Одоо та боломжит өөр өөр сонголтуудын талаар илүү ихийг мэддэг болсон тул PostgreSQL-ийг хэрхэн суулгахыг харцгаая.

        1. Эхлээд та албан ёсны вэбсайт руу орох хэрэгтэй: [https://www.postgresql.org/](https://www.postgresql.org/).
        2. Тэнд очсоныхоо дараа "Download" товчийг дарна уу.
        3. Энэ нь таныг үйлдлийн системээ (operating system) сонгох боломжтой хуудас руу хөтлөх болно. Хэрэв та аль нэг дээр нь дарвал тухайн үйлдлийн системд зориулсан зааварчилгаа бүхий хуудас руу шилжих болно.
        4. Бид Windows үйлдлийн систем дээр суулгах процессыг авч үзэх боловч энэ процесс нь бусад үйлдлийн системүүд дээр ижил төстэй байдаг.
        5. Энэ хуудаснаас та "Download the installer" холбоос дээр дарж суулгагчийг татаж авах боломжтой.
        Энэ нь таныг суулгахыг хүсэж буй PostgreSQL-ийн хувилбараа сонгох боломжтой хуудас руу хөтөлнө. Хэрэв та харгалзах дүрс дээр дарвал татан авалт эхлэх ёстой.
        6. Дараа нь таны "Downloads" хавтас эсвэл татан авалтаа анхдагчаар хадгалдаг хавтас дотор суулгагч харагдах ёстой.
        Хэрэв та түүн дээр хоёр товшвол тохиргооны (setup) процесс эхэлнэ. Next товчийг дарна уу.
        7. PostgreSQL-ийг суулгах лавлахыг (directory) сонгоно уу. Та анхдагч замыг харах боловч үүнийг өөрийнхөөрөө өөрчилж болно.
        8. Дараа нь суулгахыг хүсэж буй бүрэлдэхүүн хэсгээ (component) сонгоно уу. Дөрвөн боломжит бүрэлдэхүүн хэсэг байдаг:

            8.1 PostgreSQL Server, өгөгдлийн сангийн систем өөрөө.
            8.2 pgAdmin 4, өгөгдлийн санг удирдахад зориулсан хэрэглэгчид ээлтэй хэрэгсэл.
            8.3 Stack Builder, PostgreSQL-тэй холбоотой нэмэлт программ хангамж болон өргөтгөлүүдийг татаж авах, суулгах хэрэгсэл.
            8.4 Мөн Command Line Tools, өөрийн терминалаас PostgreSQL сервертэй шууд харилцахад зориулагдсан. Үүнд SQL асуулга гүйцэтгэхэд зориулсан интерактив терминал болох psql багтдаг.

        9. Бэлэн болмогц Next дарна уу.
        10. Дараа нь өгөгдлөө хадгалах лавлахыг сонгоно уу. Та анхдагч замыг харах боловч үүнийг өөрчилж болно. Бэлэн болмогц Next дарна уу.
        11. Одоо та бүх өгөгдлийн сан болон объектуудад хязгааргүй нэвтрэх эрхтэй хэрэглэгч болох өгөгдлийн сангийн супер хэрэглэгчийн (superuser) нууц үгийг оруулах шаардлагатай. Нууц үгээ бичээд, дахин давтаж бичээд Next дарна уу.
        12. Дараа нь серверийн сонсох портын (port number) дугаарыг сонгоно уу. Анхдагчаар энэ нь 5432 байна. Дараа нь Next дарна уу.
        13. Та шинэ өгөгдлийн сангийн кластерт зориулж байршлын тохиргоог (locale) сонгох хэрэгтэй болно. Энэ нь өгөгдлийн сан байршлаас хамааралтай өгөгдлийн төрлүүдийг тайлбарлахад чухал юм. Дараа нь Next дарна уу.
        14. Эцэст нь та суулгалтын өмнөх хураангуйг (summary) харах болно. Үүнийг хянаж үзээд Next дарна уу.

        15. Та суулгах процессыг эхлүүлэхийг хүсэж байгаагаа баталгаажуулах хэрэгтэй. Хэрэв та Next дарвал процесс эхэлнэ. Энэ нь ердөө хэдхэн минут болно.

        16. Процесс дууссаны дараа таны суулгасан бүрэлдэхүүн хэсгүүдээс хамааран нэмэлт хэрэгсэл, драйвер болон програмуудыг татаж авах, суулгахын тулд Stack Builder-ийг ажиллуулах эсэхээ сонгох шаардлагатай болж магадгүй.

        17. Хэрэв та Stack Builder-ийг ажиллуулахаар шийдсэн бол суулгалтыг эхлүүлэх боломжтой дэлгэц гарч ирнэ. Эсвэл үүнийг дараа хийхийн тулд Cancel дарж болно.

    Баяр хүргэе! Одоо таны компьютерт PostgreSQL суусан байна. Энэ процесс бусад үйлдлийн системүүд дээр ижил төстэй байх ёстой. Үүний дараа та өгөгдлийн санг ашиглаж эхлэх боломжтой.
    Програмууд руугаа орж pgAdmin-ийг хайгаарай, энэ нь PostgreSQL-д зориулсан нээлттэй эхийн удирдлага, хөгжүүлэлтийн платформ бөгөөд хэрэв та бүрэлдэхүүн хэсгүүдээс сонгосон бол суулгагдсан байх ёстой. pgAdmin 4-ийг нээхийн тулд түүн дээр дарна уу.
    Програм ажиллаж эхлэх үед та анхны ачааллын дэлгэцийг харах болно. Дараа нь та шинэ сервер эхлүүлэх, өгөгдлийн сангуудаа шалгах, pgAdmin-ийг тохируулах болон бусад зүйлийг хийх боломжтой хяналтын самбарыг **dashboard** харах болно. Та мөн баримт бичиг **documentation** рүү орох хэрэгтэй холбоосуудыг олох болно.
    Хэрэв та зүүн дээд талд байгаа "Servers" дээр дарвал суулгах явцад сонгосон нууц үгээ оруулах шаардлагатай болно.
    Нууц үгээ оруулсны дараа та өөрийн өгөгдлийн сангууд, түүний дотор тэдгээрийн хүснэгтүүд рүү нэвтрэх боломжтой дэлгэцийг харах болно.
    Та мөн SQL асуулгад зориулсан интерактив терминал болох өөр нэг маш хэрэгтэй хэрэгсэл psql-ийг олох болно. Үүнийг нээхийн тулд системээсээ psql-ийг хайж олоорой. Түүн дээр дарна уу.
    Энэ нь SQL Shell-ийг нээх бөгөөд тэнд та командууд ажиллуулж эхлэх боломжтой.
    Windows дээр PostgreSQL-ийг ингэж суулгаж болно. macOS болон Linux-ийн хувьд энэ процесс маш төстэй юм.
    Эсвэл, хэрэв та macOS дээр суулгаж байгаа бол багц удирдагч (package manager) болох Homebrew-ийг ашиглаж болно. Homebrew-ийг суулгасны дараа PostgreSQL-ийг суулгахын тулд энэ командыг ажиллуулна уу:

```bash
brew install postgresql@<version>

```

Та "at" (@) тэмдгийн ард дугаарыг бичиж суулгахыг хүсэж буй хувилбараа зааж өгч болно.
