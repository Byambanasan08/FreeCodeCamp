# SQL гэж юу вэ, хүснэгт бүхий өгөгдлийн санг хэрхэн үүсгэх вэ?

`SQL` гэдэг нь Бүтэцлэгдсэн асуулгын хэл **Structured Query Language** гэсэн үгний товчлол юм. Энэ нь `1970`-аад онд анх боловсруулагдсан бөгөөд тухайн үедээ бүтэцлэгдсэн англи хэлний асуулгын хэл **SEQUEL** гэж нэрлэгддэг байжээ. Энэ нэр томьёо нь хожим SQL болон богиноссон юм.
`SQL` бол харилцаа хамааралт өгөгдлийн санд өгөгдлийг хадгалах, удирдахад ашиглагддаг програмчлалын хэл юм.
Өмнөх хичээл дээр үзсэнчлэн харилцаа хамааралт өгөгдлийн сангууд нь өгөгдлийг хүснэгтүүдэд зохион байгуулдаг. `Хүснэгт` бүр нь өгөгдлийн `атрибутууд` болон тэдгээрийн харилцаа холбоог илэрхийлэх `мөр`,`багануудын` багцтай байна. `Мөр` бүр нь нэг бичлэгийг төлөөлдөг бол `багана` бүр нь тухайн бичлэгийн тодорхой `атрибут` эсвэл талбарыг төлөөлдөг.

`SQL`-ийн тусламжтайгаар та харилцаа хамааралт өгөгдлийн сангаас өгөгдөл `нэмэх`, `өөрчлөх`, `устгах`, `хайх`, `гаргаж авах` боломжтой. Энэ нь өгөгдлийг асуух (querying), шүүх (filtering), эрэмбэлэх (sorting) болон нэгтгэх (aggregating) иж бүрэн командуудыг агуулдаг.
SQL нь зөвхөн шаардлагатай өгөгдлийг асууж авснаар өгөгдлийн сангийн гүйцэтгэлийг оновчтой болгоход туслах бөгөөд олон төрлийн програмчлалын хэлтэй хялбархан уялдах боломжтой тул та өөрийн програмаас шууд өгөгдлийн сантай харилцахад ашиглаж болно.
Энэ нь мөн салбарын стандарт болж хүлээн зөвшөөрөгдсөн. `SQL` нь өгөгдлийн бүрэн бүтэн байдлыг (data integrity) хангаж, хэрэглэгчийг таних (user authentication) болон өгөгдөл шифрлэх (data encryption) зэрэг аюулгүй байдлын олон боломжуудыг агуулдаг.
Мөн энэ нь өргөжих боломжтой (scalable), зөөвөрлөгдөх чадвартай (portable) бөгөөд олон төрлийн өгөгдлийн сангийн удирдлагын системүүдтэй нийцтэй ажилладаг.
SQL нь ерөнхийдөө SQL мэдэгдэл **SQL statements** эсвэл SQL асуулга **SQL queries** гэж нэрлэгддэг командуудад суурилдаг.
Хөгжүүлэгчийн хувьд та эдгээр мэдэгдлүүдийг SQL хэлний тодорхой элементүүд эсвэл түлхүүр үгсийг (keywords) ашиглан бичих болно. Тэдгээр нь танд өгөгдлийн сандаа шаардлагатай үйлдлүүдийг хийх боломжийг олгоно.
SQL командуудыг оруулахаас өмнө та PostgreSQL өгөгдлийн сантай шууд харилцахын тулд psql shell-ийг нээх хэрэгтэй. Командын мөр (command prompt) эсвэл терминал дээр дараахыг оруулна уу:

```bash
psql -U <username> -d <database_name>

```

username-ийг өөрийн хэрэглэгчийн нэрээр, database_name-ийг холбогдохыг хүсэж буй өгөгдлийн сангийн нэрээр солиорой.
Хэрэв та өөрийн өгөгдлийн санг хараахан үүсгээгүй байгаа бол PostgreSQL суулгахад postgres нэртэй анхдагч өгөгдлийн санг автоматаар дагалдуулдаг. Та дагаж хийхдээ энэхүү postgres өгөгдлийн санг ашиглаж болно.
Нэгэнт холбогдсон бол командын мөр дараах байдлаар өөрчлөгдөхийг харах болно:

```sql
postgres=#

```

Энэ нь таны холбогдсон өгөгдлийн сангийн нэрийг харуулж, SQL командуудыг хүлээж байгааг илтгэнэ.
Таны мэдэх ёстой SQL-ийн хамгийн үндсэн зарим командуудыг авч үзэцгээе. SQL командуудыг тогтсон дүрмийн дагуу цэг таслалаар (;) төгсгөдөг болохыг та анзаарах болно.
Эхлээд та my_database нэртэй өгөгдлийн сан үүсгэхийн тулд энэ командыг ашиглаж болно:

```sql
CREATE DATABASE my_database;

```

Та `CREATE` гэж бичээд араас нь `DATABASE`, дараа нь үүсгэхийг хүсэж буй өгөгдлийн сангийнхаа нэрийг бичиж болно. Командын SQL түлхүүр үгсийг том үсгээр бичдэг болохыг анхаарна уу.
Дараа нь өгөгдлийн сантай болмогц түүн рүү холбогдох боломжтой.
Өгөгдлийн санд хэрхэн холбогдох нь таны ашиглаж буй орчин болон хэрэгсэлээс хамаарна.
Командын мөр эсвэл терминал дээр та өмнө үзүүлсэн аргыг ашиглаж болно. Интерактив psql shell дотор та `\c` товчлол командыг ашиглан араас нь өгөгдлийн сангийн нэрийг бичиж, өгөгдлийн санг ингэж сольж болно:

```sql
\c my_database

```

`\c` команд болон бусад товчлол командууд (урдаа налуу зураастай командууд) нь SQL хэлний хэсэг биш, харин psql shell-ийн хэсэг тул командыг дуусгахад цэг таслал шаардлагагүй гэдгийг анхаарна уу.
Одоо та шинэ өгөгдлийн сандаа холбогдсон тул командын мөр өөрчлөгдсөнийг харах болно:

```sql
my_database=#

```

Өгөгдлийн санг үүсгээд, холбогдсоны дараа та `CREATE TABLE` түлхүүр үгсийг ашиглан хүснэгт үүсгэж болно:

```sql
CREATE TABLE products (
  id SERIAL,
  name VARCHAR(255));

```

Командад та `CREATE TABLE` гэж бичээд араас нь үүсгэхийг хүсэж буй хүснэгтийнхээ нэрийг бичнэ. Энэ тохиолдолд `products` нэртэй хүснэгт үүсэх болно.
Дараа нь хаалтан дотор хүснэгтэд байх ёстой багануудын нэрийг бичиж, багана бүрт хадгалагдах утгуудын өгөгдлийн төрлийг **data type** зааж өгнө.
Энэ жишээнд хүснэгт нь бүтээгдэхүүний ID-д зориулсан багана, бүтээгдэхүүний нэрэнд зориулсан өөр нэг багана гэсэн хоёр баганатай байх болно.
Хүснэгт болон баганын нэр өгөх стандарт журам нь "snake case" буюу үгсийг жижиг үсгээр бичиж, доогуур зураасаар (\_) тусгаарлах явдал юм. Жишээлбэл: delivery_orders
Эдгээр нь таныг эхлэхэд хэрэг болох цөөн хэдэн чухал SQL командууд юм.
SQL-ийн тусламжтайгаар та өгөгдлийг хялбархан асууж, удирдаж, дүн шинжилгээ хийснээр үндэслэлтэй шийдвэр гаргах, бодит амьдрал дээрх асуудлуудыг шийдвэрлэх боломжтой.
