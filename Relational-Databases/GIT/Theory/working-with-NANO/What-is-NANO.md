# Nano гэж юу вэ?

`Terminal` дээр ажиллах үед стандарт `text editor`-ууд үргэлж бэлэн байдаггүй, харин terminal дотор шууд файл засварлах зориулалттай програмууд байдаг.

Linux орчинд ихэвчлэн text editor байдаг. `Vi` эсвэл `Vim` нь хамгийн түгээмэл байдаг. `Emacs` болон `Nano` нь гараар суулгах шаардлагатай байж болох ч, бараг бүх distribution дээр амархан олддог.

Vim-ийн гол асуудал нь сурахад нэлээд хэцүү (substantial learning curve) байдаг. Хэрвээ та өмнө нь ашиглаж байгаагүй бол application-ийг яаж хаахаа ч мэдэхгүй байж магадгүй! `Emacs` нь харьцангуй илүү user-friendly гэж хэлж болох ч мөн өөрийн гэсэн олон keyboard shortcut-уудтай.

`Nano` нь илүү энгийн (streamlined) бөгөөд хэрэглэгчдэд ээлтэй (user-friendly), ялангуяа шинээр сурч байгаа хүмүүст. Nano ашиглан файл засварлаж эхлэх нь илүү хурдан бөгөөд application нь үндсэн keyboard shortcut-уудын талаар дэлгэц дээр шууд тусламж (on-screen help) санал болгодог.

Гэхдээ үүний сул тал нь өргөтгөх боломж (extensibility) бага байдаг. Vim эсвэл Emacs шиг plugin-уудаар өргөтгөж бүрэн хөгжүүлэлтийн орчин (full development environment) болгох боломжгүй, Nano нь нэлээд энгийн (barebones) байдаг.

Nano юу хийж чаддгийг харцгаая. Файл нээхийн тулд terminal дээр nano <filename> гэж бичнэ.

Файл нээгдсэний дараа, дэлгэцийн дээд хэсэгт таны ашиглаж буй Nano-ийн хувилбар (version) болон нээсэн файлын нэр харагдана.

Доор нь файлын агуулга байрлана. Та arrow key-уудыг ашиглан cursor-оо хөдөлгөж, keyboard ашиглан текст нэмэх эсвэл устгах боломжтой. Файл шаардлагатай үед scroll хийнэ.

Дараа нь, доод хэсэгт боломжтой keyboard shortcut-уудын жагсаалт харагдана, энэ нь дараах байдалтай төстэй байна:

^G Help ^O Write Out ^F Where Is ^K Cut ^T Execute ^C Location M-U Undo M-A Set Mark M-] To Bracket M-B Previous
^X Exit ^R Read File ^\ Replace ^U Paste ^J Justify ^/ Go To Line M-E Redo M-6 Copy ^B Where Was M-F Next

Caret (^) тэмдгээр эхэлсэн shortcut-ууд нь Ctrl товчийг дарж байгаад тухайн үсгийг дарах шаардлагатайг илэрхийлнэ. Харин M- гэж эхэлсэн shortcut-ууд нь "meta key"-г дарж байгаад үсгийг дарах шаардлагатай бөгөөд ихэнх тохиолдолд энэ нь Alt товч байдаг.

Nano дээр файл засварлахын тулд танд цөөн хэдэн shortcut командуудыг мэдэхэд хангалттай. Файлд өөрчлөлт хийсний дараа Ctrl + O дарж "Write Out" буюу файлыг хадгалж болно. Энэ командыг оруулсны дараа доод цэс дараах байдлаар өөрчлөгдөнө:

File Name to write : <filename>
^G Get Help ^T To Files
^C Cancel TAB Complete

Enter дарж файлыг хадгалаад засварлалтаа үргэлжлүүлж болно, эсвэл Ctrl + C дарж цуцлаад засварлалтаа үргэлжлүүлж болно.

Editor дотор байх үед Ctrl + X дарж Nano-оос гарч terminal руу буцаж болно. Хэрвээ гарах үед хадгалагдаагүй өөрчлөлт байвал доод цэс дараах байдлаар харагдана:

Save modified buffer (ANSWERING "No" WILL DESTROY CHANGES) ?
Y Yes
^C Cancel N No

Y дарж хадгалагдаагүй өөрчлөлтүүдээ хадгална, эсвэл N дарж тэдгээрийг устгана.

Эдгээр мэдлэгийн тусламжтайгаар, та дараагийн удаа remote server дээр файл хурдан засварлах шаардлагатай үед, бүрэн график орчин (full graphical environment) байхгүй байсан ч илүү бэлтгэлтэй байх болно.
