# CSS-ийн гүйцэтгэлийг (Performance) сайжруулах ямар техникүүд байдаг вэ?

Төсөл бүр өөр өөр байдаг тул эдгээр техникүүдээс аль нь таны веб аппликейшнд хамгийн хэрэгтэйг сайтар бодож сонгох хэрэгтэй.

Юуны өмнө, бүх style-ууд parse хийгддэг тул CSS stylesheet-үүдээсээ шаардлагагүй style-уудыг устгах хэрэгтэй.

Мөн CSS style-уудаа олон файл болгон хуваах хэрэгтэй. Ингэснээр хэрэв тухайн файл тодорхой веб хуудсыг рэндэрлэхэд шаардлагагүй бол ачаалагдахгүй бөгөөд rendering процессыг блоклохгүй.

Анхдагчаар (by default), браузер бүх stylesheet-үүд rendering процессыг блоклох ёстой гэж үздэг. Гэхдээ та media attribute болон media query ашиглан stylesheet хэзээ ачаалагдахыг браузерт зааж өгч болно.

Энэ жишээнд print.css stylesheet нь зөвхөн баримт бичгийг хэвлэх үед (print) л хэрэгжинэ, учир нь media attribute нь print утгатай байна:

<link rel="stylesheet" href="print.css" media="print" />

Мөн та build процессын үед CSS файлуудаа автоматаар minify хийж хэмжээг нь багасгаж, веб аппликейшн байрлаж буй сервер дээр эдгээр файлуудыг шахах (compress) боломжтой.

CSS selector-уудаа аль болох энгийн (simple) байлгахыг хичээж, шаардлагагүй олон элемент сонгохоос (select) зайлсхийх хэрэгтэй.

CSS-ийн cascading шинж чанарыг (cascading nature) аль болох ашиглаарай. Зарим style-ууд удамшдаг (inherited).

rel="preload" ашиглан нөөцүүдийг урьдчилан ачаалах (preloading) нь бас нэг чухал техник юм. Чухал asset-ууд эхэлж ачаалагдах ёстой.

CSS animation-ууд гүйцэтгэлд нөлөөлж болохыг санах хэрэгтэй.

Хэмжээ (dimensions), байрлал (position), болон layout зэрэг тодорхой CSS property-уудыг animation хийхэд "reflow" гэж нэрлэгдэх процесс үүсдэг бөгөөд энэ үед браузер хуудсан дээрх элементүүдийн байрлал болон геометрийг дахин тооцоолдог. Энэ нь repaint шаарддаг бөгөөд тооцооллын хувьд (computationally) өндөр өртөгтэй.

Иймээс CSS animation-уудын тоог аль болох багасгах, эсвэл хэрэглэгчид тэдгээрийг асаах/унтраах (toggle) боломж олгохыг зөвлөдөг.

Мөн font-ууд веб аппликейшний визуал харагдах байдлыг сайжруулдаг ч гүйцэтгэлд нөлөөлж болно. Зарим font-ууд олон мегабайт хэмжээтэй том файлуудтай байдаг. Аль болох хоёр эсвэл гурван font ашиглаж, боломжтой бол web-safe font-уудыг хэрэглэ.

Мөн хамгийн чухал font-уудыг урьдчилан ачаалж (preload) хурдан ашиглахад бэлэн байлгахыг бодолцож болно.

Эдгээр нь веб хөгжүүлэлтэд CSS-ийг оновчтой болгоход хамгийн чухал, өргөн хэрэглэгддэг техникүүдийн зарим юм. Эдгээрийг хэрэгжүүлснээр та вебсайтынхаа гүйцэтгэлийг сайжруулж, жигд (smooth) хэрэглэгчийн туршлага бий болгож чадна.
