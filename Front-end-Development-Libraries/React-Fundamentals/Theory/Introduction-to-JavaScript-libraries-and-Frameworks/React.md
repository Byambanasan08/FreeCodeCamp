## React гэж юу вэ, мөн энэ нь ихэвчлэн юунд ашиглагддаг вэ?

React нь хэрэглэгчийн интерфэйс (user interfaces) болон вэб аппликейшн (web applications) бүтээхэд ашиглагддаг хамгийн алдартай JavaScript libraries (сангууд)-ын нэг юм. Анх Facebook-аас хөгжүүлж, арчилж (maintained) байсан React нь үр ашиг (efficiency), уян хатан байдал (flexibility), болон боломжууд (features)-ынхаа ачаар вэб хөгжүүлэлтийн салбарт маш их алдартай болсон.

React-ийн үндсэн ойлголтуудын нэг нь дахин ашиглагдах боломжтой UI components (хэрэглэгчийн интерфэйсийн компонентууд)-ыг үүсгэх явдал юм. Эдгээр компонентууд, жишээ нь buttons (товчлуурууд), cards, болон avatar components зэрэг нь таны аппликейшний туршид амархан дахин ашиглагдаж болно. Та эдгээр компонентуудыг хооронд нь давхарлан (nest) ашиглаж илүү төвөгтэй, динамик интерфэйсүүдийг бүтээж чадна.

React-ийн нэг гол давуу тал нь эдгээр custom components (захиалгат компонентууд) нь өгөгдөл өөрчлөгдөхөд бие даан (independently) шинэчлэгдэж (update) болон render (дүрслэгдэх) боломжтой байдаг.

Уламжлалт JavaScript-ээс ялгаатай нь, DOM (Document Object Model)-ийг шууд удирдах (direct manipulation) шаардлагатай байдаг бол React нь virtual DOM (виртуал DOM)-ийг ашигладаг бөгөөд энэ нь performance (гүйцэтгэл) болон efficiency (үр ашиг)-ийг сайжруулдаг. Та virtual DOM болон энэ хэрхэн ажилладаг талаар дараагийн хичээлүүдэд илүү дэлгэрэнгүй судлах болно.

Дахин ашиглагдах компонентуудаас гадна React нь аппликейшний state (төлөв)-ийг удирдах хүчирхэг арга замыг санал болгодог. State гэдэг нь компонент хэрхэн render (дүрслэгдэх) болон behave (ажиллах)-ыг тодорхойлдог өгөгдлийг хэлнэ.

React-ийн тусламжтайгаар та компонентуудын state-ийг амархан хянаж (track) болон шинэчилж (update), UI (хэрэглэгчийн интерфэйс) нь хамгийн сүүлийн өгөгдлийг зөв тусгаж байгааг баталгаажуулж чадна. Та state-тэй ажиллах талаар дараагийн хичээлүүдэд илүү ихийг сурах болно.

JavaScript ecosystem (экосистем)-д олон libraries байдаг ч freeCodeCamp нь React-ийг заахад голлон анхаардаг бөгөөд энэ нь салбарт (industry) өргөн хэрэглэгддэг, эрэлттэй (demand)-тэй холбоотой юм.

Дараагийн хэдэн хичээлүүдэд бид custom components бүтээх, state удирдах болон бусад сэдвүүдийг илүү гүнзгий судлах болно.
