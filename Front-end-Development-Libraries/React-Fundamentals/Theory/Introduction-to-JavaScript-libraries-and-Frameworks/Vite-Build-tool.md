## Vite гэж юу вэ, мөн шинэ React төсөл үүсгэхэд хэрхэн ашиглаж болох вэ?

Жижиг vanilla HTML, CSS, болон JavaScript төслүүд дээр ажиллахаас ялгаатай нь, шинэ React төсөл эхлүүлэхэд бүх зүйл зөв ажиллахын тулд хэд хэдэн нэмэлт алхмууд шаардлагатай байдаг. Бүх шаардлагатай тохиргоонуудыг (configurations) өөрөө хийхийн оронд, үүнийг таны өмнөөс хийдэг хэрэгслүүд (tools) байдаг.

Төслүүдийг үүсгэхэд ашиглагддаг хамгийн алдартай хэрэгслүүдийн нэг бол Vite юм. Vite гэдэг нь Франц хэлээр "хурдан" гэсэн утгатай бөгөөд орчин үеийн вэб төслүүдэд илүү хурдан хөгжүүлэлтийн туршлага (development experience) өгөх зорилготой build tool юм. Vite-ийг React-тэй ашиглаж болохоос гадна Vue.js эсвэл Svelte зэрэг бусад libraries болон frameworks-тэй хамт ашиглаж болно. Та бүр vanilla JavaScript төслүүдэд ч ашиглаж болно.

Vite ашиглан шинэ төсөл үүсгэхийн тулд та command line (командын мөр)-ийг ашиглах хэрэгтэй. Хэрэв та Windows ашиглаж байвал Command Prompt эсвэл Windows PowerShell ашиглаж болно. Харин Mac ашиглаж байвал Terminal app-ийг ашиглаж болно.

Хэрэв танд npm ажиллахад шаардлагатай Node.js суулгаагүй бол албан ёсны вэбсайтын холбоос энд байна: https://nodejs.org. Суулгах зааврыг дагаад дараа нь энд буцаж ирж React суулгалтаа үргэлжлүүлээрэй.

Та command line-аа нээсний дараа дараах командыг ашиглаж болно:

npm create vite@latest my-react-app -- --template react

Энэ команд нь Vite-ийн React template-ийг ашиглан my-react-app нэртэй шинэ React төсөл үүсгэнэ.

Дараа нь та шинэ төслөө нээгээд Vite-аас өгсөн React boilerplate кодыг харж болно.

Vite-ийн хамгийн сайн тал нь React төслийг эхлүүлэхэд зайлшгүй шаардлагатай файлуудыг л өгдөг явдал юм.

Төслийг яг одоогийн байдлаар ажиллуулахын тулд та дараах командуудыг command line дээр ашиглан dependencies-ийг суулгах хэрэгтэй:

cd my-react-app
npm install

cd my-react-app команд нь таныг төслийн байрлаж буй зөв директор (directory) руу шилжүүлнэ.

npm install команд нь таны React төслийг зөв ажиллуулахад шаардлагатай dependencies-ийг суулгахад ашиглагддаг. Dependencies гэдэг нь таны React төсөл зөв ажиллахын тулд шаардлагатай libraries эсвэл packages юм, жишээ нь React өөрөө, ReactDOM, эсвэл нэмэлт боломжууд өгдөг гуравдагч талын (third-party) packages.

npm install командыг ажиллуулахад таны төслийн директор доторх package.json файлыг уншиж, тэнд жагсаасан бүх dependencies-ийг суулгана. Ингэснээр та ямар нэг шаардлагатай компонент дутуугүйгээр React апп-аа бүтээж, ажиллуулах боломжтой болно.

package.json файл нь төслийн чухал тохиргооны файл (configuration file) бөгөөд төслийн нэр, хувилбар (version), dependencies зэрэг metadata (мета өгөгдөл)-ийг агуулдаг. Мөн scripts, лицензийн мэдээлэл, болон төслийг болон түүний dependencies-ийг удирдахад туслах бусад тохиргоонуудыг тодорхойлдог.

Dependencies суулгасны дараа таны төсөл дотор node_modules нэртэй шинэ хавтас (folder) үүссэнийг та анзаарах болно.

node_modules хавтас нь таны төслийн шаардлагатай бүх packages болон libraries хадгалагддаг газар юм. Энэ хавтас нь package.json файлд жагсаасан dependencies-ийн бодит кодыг агуулдаг бөгөөд таны төслийн шууд dependencies болон тэдгээрийн dependencies-ийг мөн багтаадаг.

Төслөө ажиллуулахын тулд npm run dev командыг ажиллуулаад http://localhost:5173/ хаягаар шинэ браузер tab нээнэ.

Та Vite-аас өгсөн starter template-ийг харах болно.

Хэрэв та local server-ийг зогсоох шаардлагатай бол command line дээр CTRL + C дарна.

Энэ starter template-ийн кодыг харахын тулд та төслийн src хавтас руу орж App.jsx файлыг харах боломжтой.

Тэндээс та файлыг өөрчилж, хадгалаад, браузер дээр шинэ өөрчлөлтүүдийг харах боломжтой.

Ингээд та React аппликейшн бүтээж эхлэхэд бэлэн боллоо!
