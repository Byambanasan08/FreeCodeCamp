**Яагаад анхдагч холбоосын (default link) стиль нь вэбийн ашиглахад хялбар байдалд чухал вэ?**

Анхдагч холбоосын стиль нь вэбийн **ашиглахад хялбар байдал (usability)** болон **хүртээмж (accessibility)**-ийг сайжруулахад маш чухал үүрэг гүйцэтгэдэг.

Ихэвчлэн:

* очоогүй холбоос цэнхэр,
* очсон холбоос ягаавтар (purple)

байдаг эдгээр стиль нь хэрэглэгчид олон жилийн турш дассан, навигаци хийхдээ найдаж ашигладаг **де-факто стандарт** болсон.

Анхдагч холбоосын стильний гол зорилго нь вэб хуудсан дээрх **дарж болдог (interactive)** болон **дарж болдоггүй (non-interactive)** элементүүдийг хооронд нь **тодорхой ялгаж харагдуулах** явдал юм.

Энэ ялгаа нь ойлгомжтой, хэрэглэгчдэд ээлтэй хөтчийн туршлага бий болгоход үндсэн ач холбогдолтой.

---

### Холбоосын анхдагч стильний жишээ

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: underline;
}

a:visited {
  color: purple;
}
```

Эдгээр стиль нь хэд хэдэн чухал үүрэг гүйцэтгэдэг.

---

### Очоогүй холбоосын өнгө (blue)

Очоогүй холбоосын цэнхэр өнгө нь ихэнх дэвсгэр болон текстийн өнгөнөөс **тод ялгарч**, холбоосыг амархан таних боломж олгодог.

Энэ ялгарал нь хэрэглэгчид хуудсыг хурдан гүйлгэн харах үед навигацийн элементүүд эсвэл чухал мэдээллийг **түргэн олж харахад** маш чухал.

---

### Доогуур зураас (underline)

Доогуур зураас нь тухайн текст **дарж болдог** гэдгийг илүү тодорхой харуулдаг нэмэлт харааны дохио юм.

Энэ нь ялангуяа:

* өнгө ялгаж харахад хүндрэлтэй хэрэглэгчид
* өнгөний харалгантай (colorblind) хүмүүс

дэд маш хэрэгтэй.

---

### Очсон холбоосын өнгө (purple)

Очсон холбоосын өнгө өөрчлөгдөх (ихэвчлэн ягаавтар болох) нь хэрэглэгчдэд **аль хуудсаар орж үзсэнээ санахад** тусалдаг.

Энэ нь:

* том вэбсайт
* судалгаа хийх
* олон холбоосоор дамжин явах

үед маш их хэрэгтэй бөгөөд хэрэглэгч нэг хуудсанд **давтан орж цаг алдах**-аас сэргийлдэг.

---

### Жишээ

```html
<p>
  Learn more about 
  <a href="https://www.example.com/cats">cats</a> 
  and 
  <a href="https://www.example.com/dogs">dogs</a>.
</p>
```

Ямар ч custom CSS бичээгүй тохиолдолд ихэнх браузер эдгээр холбоосыг:

* цэнхэр өнгөтэй
* доогуур зураастай

харуулна. Харин холбоос дээр дарсны дараа тухайн холбоос **ягаавтар өнгөтэй** болж, хэрэглэгчид шууд feedback өгнө.

---

### Custom стиль хийх үед анхаарах зарчим

Дизайнерууд вэбсайтын харагдах байдалд тохируулан анхдагч стильнийг өөрчлөх нь элбэг. Гэхдээ дараах үндсэн зарчмуудыг **заавал хадгалах** хэрэгтэй:

* Холбоос нь энгийн текстээс **тод ялгарч харагдах**
* Очсон болон очоогүй холбоосын хооронд **харагдах ялгаа байх**
* Ашигласан өнгөнүүд дэвсгэртэйгээ **хангалттай contrast**-тай байх

---

### Орчин үеийн харагдацтай custom стиль жишээ

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  color: purple;
  border-bottom: 1px solid purple;
}
```

Энэ жишээнд:

* уламжлалт цэнхэр/ягаавтар өнгө хадгалагдсан
* доогуур зураасыг доод хүрээгээр (border-bottom) орлуулж
* илүү орчин үеийн харагдац бий болгож байна

---

### Холбоосын бусад төлөвүүд (states)

Холбоос нь зөвхөн `link` болон `visited` төлөвтэй байдаггүй. Мөн дараах төлөвүүд байдаг:

* `:hover` — хулгана дээр очих үед
* `:active` — дарагдаж байх агшинд

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:hover {
  color: red;
}

a:active {
  color: darkorange;
}
```

Эдгээр төлөвүүд нь хэрэглэгч холбоостой харилцаж байх явцад **шууд, ойлгомжтой feedback** өгч, вэбсайтын ашиглахад хялбар байдлыг нэмэгдүүлдэг.

---

### Дүгнэлт

Хэдийгээр холбоосын стилийг өөрчлөх боломжтой ч, анхдагч стильний цаадах **үндсэн зарчмуудыг хадгалах нь маш чухал**.

Анхдагч холбоосын стиль нь:

* вэбийг ойлгомжтой болгодог
* хэрэглэгчийг чиглүүлдэг
* контенттой үр дүнтэй харилцах боломж олгодог

Иймээс вэбсайтын холбоосын стилийг зохиохдоо **гоо зүйгээс илүүтэй хэрэглэгчийн туршлага, тодорхой байдал, хүртээмжийг** үргэлж нэн тэргүүнд тавих хэрэгтэй.
