**`margin` эсвэл `line-height` ашиглан жагсаалтын элементүүдийн зайг хэрхэн тохируулах вэ?**

`margin` болон `line-height` нь жагсаалтын элементүүдийн хоорондох зайг тохируулж, уншихад илүү ойлгомжтой, харагдах байдлыг сайжруулахад чухал үүрэгтэй.

---

1. `margin` ашиглан жагсаалтын элементүүдийн зайг тохируулах

`margin`-ыг `li` элементүүд дээр өгснөөр жагсаалтын элементүүдийн **гаднах зайг** нэмэгдүүлж эсвэл багасгаж болно. Ингэснээр элемент бүрийн хоорондын зайг илүү нарийн удирдах боломжтой.

Жишээ нь, дараах гурван элементтэй unordered list байна гэж үзье:

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

Анхдагч байдлаар HTML нь жагсаалтын элементүүдийн хооронд төдийлөн их зай өгдөггүй.

Жагсаалтын элемент бүрийн доод талд зай нэмэхийн тулд `margin-bottom` property-ийг дараах байдлаар ашиглаж болно:

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

Энэ жишээнд unordered list доторх `li` элемент бүрийн доод талд **40px хэмжээтэй margin** нэмэгдэнэ.

---

2. `line-height` ашиглан жагсаалтын элементүүдийн зайг тохируулах

Жагсаалтын элементүүдийн зайг нэмэх өөр нэг арга нь `line-height` property ашиглах юм.

`line-height` нь **нэг `li` элементийн доторх текстийн мөр хоорондын босоо зайг** тохируулдаг.

Энэ property нь үндсэндээ нэг элемент доторх мөрүүдийн хоорондын зайд нөлөөлдөг боловч, хэрэв жагсаалтын элемент бүр **зөвхөн нэг мөр тексттэй** бол нийт харагдах босоо зайд шууд бус байдлаар нөлөөлж чадна.

Хэрэв жагсаалтын элементүүд олон мөр текст агуулж байвал:

* `line-height` нь тухайн элемент доторх мөрүүдийн хоорондын зайг өөрчилнө
* харин **өөр өөр `li` элементүүдийн хоорондох зайг шууд тохируулахгүй**

Тиймээс жагсаалтын элементүүдийн хоорондох зайг нарийн удирдах шаардлагатай бол `margin` эсвэл `padding` ашиглах нь зөв.

Доорх нь өмнөх unordered list дээр `line-height` ашигласан жишээ юм:

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

Энэ жишээнд `line-height: 2;` гэж тохируулснаар мөрийн өндөр нь тухайн текстийн үсгийн хэмжээнээс **хоёр дахин их** болж, `li` элемент бүрийн доторх босоо зай илүү нэмэгдэж байна.
