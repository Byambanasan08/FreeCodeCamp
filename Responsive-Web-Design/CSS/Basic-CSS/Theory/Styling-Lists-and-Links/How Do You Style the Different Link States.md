**Холбоосын (link) өөр өөр төлөвүүдийг хэрхэн стийлдэх вэ?**

Холбоос нь хэд хэдэн өөр **төлөв (state)**-тэй байдаг. Үүнд `link`, `visited`, `hover`, `focus`, `active` зэрэг орно. Эдгээр төлөвүүд нь хэрэглэгчид холбоосыг таних, харилцан үйлдэл хийсний дараах **тодорхой харааны feedback** авахад чухал үүрэгтэй тул **ашиглахад хялбар байдал (usability)** болон **хүртээмж (accessibility)**-ийг сайжруулдаг.

Эдгээр төлөвүүдийг зөв стийлдсэнээр:

* хэрэглэгч аль холбоосоор орж үзсэнээ ойлгоно
* аль холбоостой харилцаж байгаагаа мэдэрнэ
* дарахад юу болохыг урьдчилан таамаглана

Ялангуяа харааны бэрхшээлтэй эсвэл танин мэдэхүйн (cognitive) хүндрэлтэй хэрэглэгчдэд эдгээр ялгаатай стиль нь навигацийг илүү ойлгомжтой болгодог.

Мөн холбоосын төлөвүүд нь хэрэглэгчийн үйлдэл бүрт **шууд feedback** өгч, будлианыг багасган, сайтын навигацийг сайжруулдаг.

---

### Pseudo-class гэж юу вэ?

Эдгээр төлөвүүдийг CSS-д **pseudo-class** ашиглан стийлддэг.

Pseudo-class гэдэг нь selector-д нэмэгддэг **түлхүүр үг** бөгөөд тухайн элементийн **онцгой төлөв** эсвэл хэрэглэгчийн үйлдлийг илэрхийлдэг.

Жишээ нь:

* `:hover` — хэрэглэгчийн заагч (mouse) дээр очих үед
* `:visited` — өмнө нь орж үзсэн холбоос

Pseudo-class ашигласнаар HTML-д нэмэлт markup бичихгүйгээр, зөвхөн элементийн төлөв дээр үндэслэн стийл өгөх боломжтой.

Pseudo-class-ийн ерөнхий синтакс:

```css
A:B {
  property: value;
}
```

Энд:

* `A` — selector
* `:B` — pseudo-class

## Холбоосын төлөвүүдийг стийлдэх жишээнүүд

### `:link` — очоогүй (unvisited) холбоос

`:link` pseudo-class нь **очоогүй холбоосыг** стийлддэг бөгөөд тухайн текст дарж болдгийг илэрхийлнэ.

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Энгийн төлөв (очоогүй холбоос) */
a:link { 
  color: red;
}
```

Энэ жишээнд очоогүй холбоосын анхдагч цэнхэр өнгө улаан (`red`) болж өөрчлөгдөнө.

---

### `:visited` — очсон (visited) холбоос

`:visited` нь хэрэглэгч өмнө нь орж үзсэн холбоосыг стийлддэг. Энэ нь хэрэглэгчид аль холбоосоор орсон гэдгээ санахад тусалдаг.

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Очсон холбоос */
a:visited {
  color: green;
}
```

Энэ тохиолдолд холбоос дээр дарсны дараа түүний өнгө ногоон (`green`) болно.

---

### `:hover` — заагч дээр очих үед

`:hover` pseudo-class нь хэрэглэгчийн заагч холбоос дээр очих үед стийл өөрчилдөг. Энэ нь холбоос **interactive** гэдгийг илүү тодорхой харуулна.

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover төлөв */
a:hover {
  color: green;
}
```

Хулгана холбоос дээр очиход холбоос ногоон өнгөтэй болно.

---

### `:focus` — фокус авсан үед

`:focus` нь холбоос **фокус** авсан үед (жишээ нь keyboard-оор tab дарж шилжихэд) стийл өгдөг. Энэ нь хүртээмжийг сайжруулахад маш чухал.

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus төлөв */
a:focus {
  outline: 2px solid orange;
}
```

Энд холбоос фокус авмагц улбар шар (`orange`) хүрээ гарч ирнэ.

---

### `:active` — дарагдаж байх агшин

`:active` нь холбоос **дарж байгаа агшинд** хэрэгждэг. Энэ нь хэрэглэгчийн үйлдэл бүртгэгдэж байгааг илтгэх шууд feedback өгдөг.

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active төлөв */
a:active {
  color: pink;
}
```

Энэ жишээнд холбоос дээр дарах үед текстийн өнгө ягаан (`pink`) болж өөрчлөгдөнө.

---

### Дүгнэлт

Холбоосын өөр өөр төлөвүүдийг (`link`, `visited`, `hover`, `focus`, `active`) зөв стийлдэх нь:

* вэбсайтыг илүү ойлгомжтой болгоно
* хэрэглэгчийн харилцаанд шууд feedback өгнө
* хүртээмж болон ашиглахад хялбар байдлыг сайжруулна

Иймээс холбоосын стийл зохиохдоо **гоо зүйгээс гадна хэрэглэгчийн туршлага, тодорхой байдал, хүртээмжийг** заавал харгалзан үзэх хэрэгтэй.
