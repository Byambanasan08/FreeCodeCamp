**Cascade Algorithm (Каскадын алгоритм) ерөнхий түвшинд хэрхэн ажилладаг вэ?**

Cascade Algorithm гэдэг нь **нэг элементэд олон CSS дүрэм зэрэг үйлчлэх боломжтой үед, аль дүрмийг хэрэгжүүлэхийг браузер хэрхэн шийддэг процесс** юм. Энэ алгоритм нь тодорхой, дараалсан дүрмүүдэд тулгуурлан **хамгийн тохиромжтой стиль**-ийг сонгон хэрэглэдэг.

---
1. Хамаарал (Relevance)

Процесс хамгийн түрүүнд **хамааралтай эсэхийг** шалгахаас эхэлнэ. Браузер тухайн элементэд үнэхээр үйлчлэх боломжтой CSS дүрмүүдийг л шүүнэ.

Үүнд:

* selector тухайн элементтэй таарч байгаа эсэх
* тухайн үед идэвхтэй байгаа media query-д тохирч байгаа эсэх

зэргийг багтаана.

**Media query** гэдэг нь төхөөрөмж эсвэл viewport-ын шинж чанараас (өргөн, өндөр, чиглэл гэх мэт) хамаарч стиль хэрэгжүүлэх CSS арга юм.

---

2. Эх сурвалж ба чухалчлал (Origin and Importance)

Дараа нь браузер CSS-ийн **эх сурвалж** болон **чухалчлал**-ыг авч үзнэ.

CSS нь дараах эх сурвалжаас ирж болно:

* Браузерийн анхдагч стиль (user-agent styles and default browser styles)
* Хэрэглэгчийн тохируулсан стиль
* Хөгжүүлэгчийн бичсэн стиль (author styles)

Үүний дараа дүрэм бүр `!important` тэмдэглэгээтэй эсэхийг шалгана.
`!important`-тай дүрмүүд нь **эх сурвалжаас үл хамааран** бусад дүрмүүдийг дарж хэрэгжинэ.

---

3. Specificity

Эх сурвалж болон чухалчлал ижил түвшинд байгаа тохиолдолд браузер **specificity**-г харгалзан үзнэ.

Specificity гэдэг нь selector **ямар нарийвчлалтай** элементийг онилж байгааг илэрхийлэх хэмжүүр юм. Илүү нарийвчилсан selector нь ерөнхий selector-оос давуу эрхтэй.

---

4. Дараалал (Order of appearance)

Хэрэв:

* эх сурвалж ижил
* `!important` ижил
* specificity ижил

бол **CSS-д хамгийн сүүлд бичигдсэн дүрэм** хэрэгжинэ.

Иймээс CSS-ийг ямар дарааллаар бичсэн нь зарим тохиолдолд үр дүнд нөлөөлдөг.

---

### Жишээ

HTML файлд нэг `p` элемент байна гэж үзье:

```html
<link rel="stylesheet" href="styles.css">

<p>example paragraph</p>
```

CSS файлд дараах хоёр дүрэм байна:

```css
p {
  color: blue;
}

p {
  color: green; 
}
```

Эхний дүрэм нь бүх `p` элементүүдийн текстийг цэнхэр болгоно. Харин хоёр дахь дүрэм нь ногоон болгож байна.

Тэгвэл аль өнгө хэрэгжих вэ?

➡ **Ногоон (`green`) өнгө хэрэгжинэ.**

Учир нь хоёр дүрэм:

* ижил selector (`p`)
* ижил specificity
* ижил эх сурвалжтай

байгаа тул **хамгийн сүүлд бичигдсэн дүрэм** давуу эрх авна.

---

### Дүгнэлт

Cascade Algorithm нь:

* хамаарал (relevance)
* эх сурвалж ба чухалчлал (origin & importance)
* specificity
* бичигдсэн дараалал (order of appearance)

зэрэг хүчин зүйлсийг шат дараалан авч үзэж, CSS-ийн үйлдлийг **тогтвортой, урьдчилан таамаглах боломжтой** болгодог. Ингэснээр илүү нарийн, уян хатан загварчлалтай вэб хуудсуудыг бүтээх боломж бүрддэг.
