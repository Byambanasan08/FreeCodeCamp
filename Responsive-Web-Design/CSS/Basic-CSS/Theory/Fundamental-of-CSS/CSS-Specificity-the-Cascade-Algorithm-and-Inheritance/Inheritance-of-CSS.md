**CSS-ийн Inheritance (өвлөгдөх механизм) ерөнхий түвшинд хэрхэн ажилладаг вэ?**

Inheritance нь CSS-ийн чухал ойлголтуудын нэг бөгөөд **эцэг (parent) элемент дээр өгсөн зарим стиль нь түүний хүүхэд (child) элементүүдэд хэрхэн дамждагийг** тодорхойлдог.

Бодит амьдрал дээр хүүхдүүд эцэг эхээсээ зарим шинж чанарыг өвлөдөгтэй адил, CSS-д ч мөн **тодорхой property-ууд эцэг элементээс хүүхэд элемент рүү өвлөгдөж** дамждаг.

Энэ нь бүх баримт бичгийн хэмжээнд **нэгэн жигд загварчлал хийхийг илүү үр ашигтай** болгодог.

---

1. CSS-д өвлөгддөг property-ууд

CSS-д **бүх property анхнаасаа өвлөгддөггүй**.

Жишээ нь дараах property-ууд **анхдагчаар өвлөгдөнө**:

* `color`
* `font-family`
* `line-height`

Өөрөөр хэлбэл, хэрэв та эцэг элемент дээр текстийн өнгийг тохируулбал, тухайн элементийн бүх хүүхэд элементүүд **тусгайлан дарж бичээгүй л бол** тэр өнгийг өвлөж авна.

Жишээ:

```html
<div style="color: blue;">
  This is the parent element.
  <p>This is the child element inheriting the color.</p>
</div>
```

Энэ тохиолдолд:

* эцэг `div`
* хүүхэд `p`

хоёулаа **цэнхэр (`blue`) өнгөтэй** харагдана. Учир нь `color` property нь өвлөгддөг.

---

2. Анхдагчаар өвлөгддөггүй property-ууд

Харин дараах property-ууд **анхнаасаа өвлөгддөггүй**:

* `margin`
* `padding`
* `border`
* `background`

Хэрэв эдгээр стилийг хүүхэд элементэд өвлүүлэхийг хүсвэл:

* шууд тухайн хүүхэд элемент дээр тохируулах
* эсвэл `inherit` keyword ашиглах

шаардлагатай.

---

3. `inherit` keyword ашиглах

`inherit` keyword нь тухайн property нь **анхнаасаа өвлөгддөг эсэхээс үл хамааран**, эцэг элементээс нь **заавал өвлүүлж авах** зориулалттай.

Жишээ нь, хүүхэд элемент эцэгтэйгээ ижил `padding`-тай байхыг хүсвэл дараах байдлаар бичиж болно:

```html
<div style="padding: 20px;">
  This is the parent element with padding.
  <p style="padding: inherit;">This is the child element inheriting the padding.</p>
</div>
```

Энд:

* эцэг `div` нь `padding: 20px`
* хүүхэд `p` нь `padding: inherit`

гэсэн тул `p` элемент нь **эцэг `div`-ийн 20px padding-ийг өвлөж** авна.

---

### Inheritance-ийн давуу тал

Inheritance нь:

* стилийг **давхар давхар бичих шаардлагыг багасгана**
* CSS-ийг **товч, цэгцтэй, удирдахад хялбар** болгоно
* бүхэл баримт бичигт **нэгэн жигд харагдац** бий болгоно

Нэг property-г олон элемент дээр дахин дахин бичихийн оронд, **эцэг элемент дээр нэг удаа тодорхойлоод**, хүүхэд элементүүдээр өвлүүлэх боломжтой.

---

### Анхаарах чухал зүйл

Inheritance нь **зөвхөн нэг чиглэлтэй**:

➡ **эцэг → хүүхэд**

Хэрэв та хүүхэд элемент дээр тухайн стилийг дарж бичвэл:

* энэ нь **эцэг элементэд нөлөөлөхгүй**

Өөрөөр хэлбэл, өвлөгдөх нь **доошоо л дамжина**, дээшээ хэзээ ч нөлөөлөхгүй.
