**Meta viewport элемент юунд ашиглагддаг вэ?**

**Meta viewport элемент** нь responsive вэб дизайн дахь **маш чухал бүрэлдэхүүн хэсэг** юм.

Энэ нь HTML-ийн **тусгай meta элемент** бөгөөд браузерт хуудасны **хэмжээс (dimensions)** болон **масштаб (scaling)**-ыг янз бүрийн төхөөрөмж дээр, ялангуяа **гар утас** болон **таблет** дээр хэрхэн удирдахыг зааж өгдөг.

Meta viewport элементийн үндсэн синтаксыг харцгаая:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Энэ элемент нь ихэвчлэн HTML баримтын **`head` хэсэгт** байрладаг. Харин энэ элементийн хэсэг бүр юу гэсэн утгатай вэ?

`width=device-width` гэдэг хэсэг нь браузерт хуудсын өргөнийг тухайн төхөөрөмжийн **дэлгэцийн өргөнтэй тэнцүү** болгохыг зааж өгдөг. Энэ нь өөр өөр дэлгэцийн хэмжээнд дасан зохицох **responsive layout** үүсгэхэд зайлшгүй шаардлагатай.

`initial-scale=1.0` нь хуудас анх ачааллагдах үедх **эхний zoom түвшинг** тодорхойлдог. `1.0` гэсэн утга нь хуудсыг **ямар ч масштабгүйгээр**, өөрөөр хэлбэл **100% хэмжээгээр** харуулна гэсэн үг.

Meta viewport элементийг ашигласнаар таны вэб хуудсууд **гар утасны төхөөрөмжүүд дээр зөв харагдах** баталгааг бий болгодог.

Хэрэв үүнийг ашиглахгүй бол, мобайл браузерууд ихэнхдээ хуудсыг **desktop дэлгэцийн өргөнөөр render хийж**, дараа нь багасган харуулдаг. Үүний үр дүнд текст хэт жижиг болж, уншихад хэцүү, **муу хэрэглэгчийн туршлага** үүсдэг.

Meta viewport элемент нь мөн хэрэглэгчид хуудсыг **томруулж (zoom in)**, **жижигрүүлж (zoom out)** болох эсэхийг удирдах боломж олгодог.

`user-scalable=no` атрибут ашиглан zoom-ийг хаах боломжтой ч, **accessibility**-ийн үүднээс үүнийг **зайлсхийхийг зөвлөдөг**.

Олон хэрэглэгчид, ялангуяа **харааны бэрхшээлтэй хүмүүс**, илүү уншихад хялбар болгохын тулд zoom хийх боломжид тулгуурладаг.

Доорх нь **хийх ёсгүй** жишээ юм:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
```

Үүний оронд, вэб сайтаа **өөр өөр zoom түвшинд ч уншихад ойлгомжтой**, **responsive** байдлаар загварчлах нь илүү зөв шийдэл юм. Ингэснээр бүх хэрэглэгчид таны контентод **тав тухтай хандах** боломжтой болно.

Meta viewport элемент нь **mobile-friendly вэб сайт** бүтээхэд шийдвэрлэх үүрэг гүйцэтгэдэг.

Энэ нь таны нарийн төлөвлөж бүтээсэн responsive дизайнуудыг янз бүрийн төхөөрөмж дээр **яг төлөвлөсөн ёсоор нь харагдуулах**, улмаар бүх зочдод **илүү сайн хэрэглэгчийн туршлага** өгөх нөхцөлийг бүрдүүлдэг.
