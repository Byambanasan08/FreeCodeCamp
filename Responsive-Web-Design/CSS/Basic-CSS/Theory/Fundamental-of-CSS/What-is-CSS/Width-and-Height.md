**Width болон Height хэрхэн ажилладаг вэ?**

CSS-д **`width`** болон **`height`** шинж чанаруудыг вэб хуудсан дээрх элементүүдийн **хэмжээс (dimensions)**-ийг удирдахад ашигладаг.

Эдгээр шинж чанаруудыг:

* пиксел (`px`),
* хувь (`%`),
* viewport нэгжүүд (`vw`, `vh`),
* болон бусад нэгжүүдээр
  тодорхойлж болно.

---

### **`width` шинж чанар**

`width` нь элементийн **өргөнийг** заана. Хэрэв та `width`-ийг тодорхойлоогүй бол, анхдагч утга нь **`auto`** байна.

`width: auto` гэдэг нь браузер тухайн элементийн:

* доторх контент,
* эцэг элемент (parent),
* `display` төрөл
  зэргээс хамааран өргөнийг автоматаар тооцоолно гэсэн үг.

Жишээлбэл, `div` элемент дээр `width: auto` байвал тухайн `div` нь ихэвчлэн **эцэг контейнерийнхээ бүх өргөнийг дүүргэнэ**.

---

### **`height` шинж чанар**

`height` нь элементийн **өндрийг** заана. Үүнтэй адил, анхдагч утга нь мөн **`auto`** байдаг.

`height: auto` гэдэг нь элементийн өндөр нь доторх **контентийн хэмжээнд тааруулан** автоматаар өөрчлөгдөнө гэсэн үг.

---

### **`width` болон `height` ашигласан жишээ**

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

Энэ жишээнд `box` гэсэн класстай `div` элемент байна. Энэ элемент нь:

* **өргөн 100px**,
* **өндөр 100px**
  байх бөгөөд арын өнгө нь **skyblue** байна.

---

### **Pixels (`px`)**

Pixels нь CSS-д ашиглагддаг **тогтмол хэмжээтэй (fixed-size)** хэмжих нэгж бөгөөд элементүүдийн хэмжээг **маш нарийн удирдах** боломж олгодог.

---

### **`min-width` болон `min-height`**

**`min-width`** нь элементийн **хамгийн бага өргөнийг** заана. Доторх контент нь жижиг байсан ч элемент энэ утгаас **бага болж агшихгүй**.

**`min-height`** нь элементийн **хамгийн бага өндрийг** заана. Энэ нь элементийн өндөр тогтоосон утгаас **бага болохоос сэргийлнэ**.

Жишээ:

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

Энэ жишээ нь `min-width` болон `min-height` хэрхэн ажилладгийг харуулж байна.

Хэдийгээр `width` болон `height`-ийг **50px** гэж тохируулсан ч, бодит байдал дээр элемент нь **100px × 100px** хэмжээтэй харагдана. Учир нь `min-width` болон `min-height` нь **100px** гэж тохируулагдсан бөгөөд энэ нь `width` болон `height`-ээс **том** утга юм.

Санамж:

* Хэрэв `min-width` эсвэл `min-height` нь `width` эсвэл `height`-ээс **том** байвал,
* **min утгууд нь давуу эрхтэй (override)** болж, жижиг утгуудыг дарна.

Энэ нь элементүүд хэт жижиг болохоос сэргийлж, **ашиглахад эвтэйхэн, тогтвортой дизайн** хадгалахад чухал.

---

### **`max-width` болон `max-height`**

**`max-width`** нь элементийн **хамгийн их өргөнийг** заана. Хэдий хангалттай зай байсан ч элемент энэ утгаас **өргөн болохгүй**.

**`max-height`** нь элементийн **хамгийн их өндрийг** заана. Контент их байсан ч энэ утгаас **давж томрохгүй**.

Жишээ:

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

Энэ жишээнд `box` элемент нь:

* `width` болон `height`-ийг **200px** гэж тохируулсан боловч,
* бодит байдал дээр **150px × 150px** хэмжээтэй харагдана.

Учир нь `max-width` болон `max-height` нь **150px** гэж тохируулагдсан бөгөөд энэ нь `width` болон `height`-ээс **жижиг** утга юм.

Санамж:

* Хэрэв `max-width` эсвэл `max-height` нь `width` эсвэл `height`-ээс **бага** байвал,
* **max утгууд нь давуу эрхтэй** болж, илүү том утгуудыг хязгаарлана.

---

### **Дүгнэлт**

CSS-д:

* `min-width`, `min-height` нь `width`, `height`-ээс **давуу эрхтэй**,
* `max-width`, `max-height` нь хэмжээсүүдийг **дээд хязгаараар нь хязгаарладаг**.

Жишээлбэл, хэрэв:

* `width: 600px`
* `max-width: 500px`
  гэж тохируулбал, элемент нь **500px-ээс хэтрэхгүй**.

Энэ нь стандарт `width` болон `height` утгууд ямар байхаас үл хамааран элементүүдийг **хүссэн хэмжээний хүрээнд барих** боломж олгодог бөгөөд дизайн тогтвортой, удирдахад хялбар байх нөхцөлийг бүрдүүлдэг.
