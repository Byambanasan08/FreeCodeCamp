**HTML элементүүдэд хэрэглэгддэг браузерийн анхдагч загварууд (default browser styles) гэж юу вэ?**

HTML болон CSS-тэй ажиллаж эхлэх үед, та ямар ч CSS бичээгүй байхад ч таны вэб хуудсанд зарим загварууд аль хэдийн хэрэгжсэн байгааг анзаарах болно. Эдгээрийг **“анхдагч браузерийн загварууд” (default browser styles)** эсвэл **“user-agent styles”** гэж нэрлэдэг.

Эдгээр анхдагч загварууд нь HTML элементүүдийг **бүх браузер дээр уншихад ойлгомжтой** байдлаар харуулах үндсэн форматлалыг хангадаг.

Гэхдээ эдгээр загварууд нь браузер бүр дээр **бага зэрэг ялгаатай** байж болдог. Иймээс вэб сайтад **нэгэн жигд харагдах байдал (consistent look)** бий болгохдоо эдгээрийг ойлгож мэдэх нь чухал.

Одоо зарим түгээмэл HTML элементүүдэд хэрэглэгддэг анхдагч браузерийн загваруудыг авч үзье.

---

**Гарчгууд (Headings)** нь хамгийн өргөн хэрэглэгддэг HTML элементүүдийн нэг бөгөөд анхдагчаар **өөр өөр хэмжээ, зузаантай (font size, font weight)** байдлаар загварчлагдсан байдаг.

Жишээлбэл, `h1` (хамгийн дээд түвшний гарчиг) нь ихэвчлэн **бүдүүн (bold)** бөгөөд `h2`, `h3` гэх мэт доод түвшний гарчгуудаас **том хэмжээтэй** байдаг.

Эдгээр гарчгийн тагууд нь контентийн **hierarchy(шатлал), бүтцийг тодорхойлоход** тусалдаг.

Доорх жишээ нь `h1`-ээс `h6` хүртэлх бүх зургаан гарчгийн элементүүдийг харуулж байна:

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

---

Анхдагч загвартай өөр нэг элемент бол **`hr` элемент** юм. Энэ нь контентийн хэсгүүдийг харааны хувьд салгах зорилготой **хэвтээ зураас (horizontal rule)** үүсгэдэг. Браузерууд ихэвчлэн энэ элементэд энгийн шулуун шугамын загвар хэрэглэдэг.

Илүү сайн ойлгохын тулд дараах кодыг харцгаая:

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

Хэвтээ зураас нь текстийн дээр, доор **зайтай нимгэн шугам** хэлбэрээр харагдаж, контентийн хэсгүүдийг хооронд нь ялгаж өгнө.

---

Дараагийн элемент бол **`blockquote` элемент** юм.

Blockquote нь өөр эх сурвалжаас иш татсан текстийг илэрхийлэхэд ашиглагддаг. Браузерууд ихэвчлэн энэ элементэд **догол мөр (indentation)** нэмж, зарим тохиолдолд текстийг **налуу (italic)** болгодог.

Энэ догол мөр нь тухайн текст бусдаас иш татсан гэдгийг тодорхой харуулахад тусалдаг.

Дараах кодын жишээг харцгаая:

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

Дээрх жишээнд `blockquote` элемент нь браузерийн анхдагч загварын дагуу текстийг **доголтой** болгон харуулна.

---

Анхдагч загвартай өөр нэг элемент бол **anchor элемент (`<a>`)** юм. Энэ нь ихэвчлэн **цэнхэр өнгөтэй** бөгөөд **доогуураа зураастай (underline)** байдлаар харагддаг. Ингэснээр хэрэглэгчид үүнийг холбоос (link) гэдгийг шууд танина.

Дараах HTML жишээг харцгаая:

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://freecodecamp.org/">Visit the freeCodeCamp learn page</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

Дээрх кодод дөрвөн paragraph элемент байна, тэдгээрийн дунд нэг anchor элемент орсон. Anchor элемент нь **цэнхэр өнгөтэй, доогуураа зураастай** харагдаж, хэрэглэгчдэд freeCodeCamp-ийн суралцах хуудсанд очих боломжтойг илтгэнэ.

---

Эцэст нь **unordered list (`ul`)** болон **ordered list (`ol`)** элементүүдийг авч үзье.

Браузерууд жагсаалтуудад анхдагчаар **догол мөр**, мөн:

* unordered list (`ul`) дээр **bullet**
* ordered list (`ol`) дээр **дугаар (number)**
  хэрэглэдэг.

Жишээг харцгаая:

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

Дээрх кодын жишээнд unordered list болон ordered list хоёуланг нь ашигласан бөгөөд эдгээр жагсаалтууд анхдагчаар **баруун тийш бага зэрэг доголтой** харагдана.

---

Эдгээр бүх жишээнүүдээс харахад браузер нь HTML элементүүдэд:

* анхдагч зай (spacing),
* ялгаатай font хэмжээ,
* font-ийн зузаан
  зэргийг автоматаар хэрэглэж байгааг харлаа.

Дараагийн хичээлүүдэд та эдгээр анхдагч загваруудын заримыг **CSS reset** ашиглан хэрхэн тэгшлэх (reset хийх) талаар суралцах болно.
