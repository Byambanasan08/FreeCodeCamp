**CSS дүрмийн үндсэн бүтэц гэж юу вэ?**

CSS нь вэб хуудасны **загварыг (styles)** хариуцдаг. Эдгээр бүх загварууд нь төрөл бүрийн **CSS дүрэм (CSS rules)**-ээс бүрддэг.

Нэг CSS дүрэм нь **хоёр үндсэн хэсэгтэй**: **сонгогч (selector)** болон **мэдэгдлийн блок (declaration block)**.

Үндсэн синтаксыг харцгаая:

```css
selector {
  property: value;
}
```

**Сонгогч (selector)** гэдэг нь CSS-д ашиглагддаг загвар бөгөөд тодорхой HTML элементүүдийг тодорхойлж, тэдгээрт загвар хэрэглэхэд ашиглагдана.

Сонгогчийн жишээнд:

* төрлийн сонгогч (type selector),
* класс сонгогч (class selector),
* ID сонгогч (ID selector)
  зэрэг орно.

Үндсэн синтакст харагдаж буй **долгионт хаалт `{}`**-уудыг **мэдэгдлийн блок (declaration block)** гэж нэрлэнэ. Мэдэгдлийн блок нь өгөгдсөн нэг эсвэл хэд хэдэн сонгогчид хэрэгжих загваруудын багцыг агуулна.

Мэдэгдлийн блокийн дотор **олон мэдэгдэл (declarations)** байж болно. Нэг мэдэгдэл нь **шинж чанар (property)** болон **утга (value)**-аас бүрдэнэ.

**Шинж чанар (property)** нь ямар шинжийг загварчилж байгааг заадаг CSS танигч юм. Жишээ нь `background-color` нь нэг property юм.

**Утга (value)** нь тухайн шинж чанарт хэрэгжүүлэх тодорхой тохиргоо юм. Жишээлбэл, property нь `background-color` байвал, value нь `purple` байж болох бөгөөд энэ нь арын өнгийг нил ягаан болгоно.

Property нэрийн дараа **colon (`:`)** тавих ёстой бөгөөд value-ийн дараа **semicolon (`;`)** заавал байна.

Одоо CSS дүрмийн синтаксыг ойлгосон тул нэг жишээ харцгаая. Интерактив редакторыг идэвхжүүлээд `styles.css` таб дээр дарж CSS кодыг харна уу.

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

Энэ CSS дүрэмд **төрлийн сонгогч (type selector)** нь хуудсан дээрх бүх `p` (paragraph) элементүүдийг онилж байна.

Мэдэгдлийн блокийн дотор **нэг мэдэгдэл** байна.

Энэ мэдэгдэл нь `color` property бөгөөд value нь `blue` гэж тохируулагдсан. Ингэснээр бүх paragraph элементүүдийн текстийн өнгө цэнхэр болно.

Хэрэв та **нэг ижил загварыг олон сонгогчид** хэрэгжүүлэхийг хүсвэл, сонгогч бүрийг **таслалаар (comma)** тусгаарласан **сонгогчийн жагсаалт (selector list)** үүсгэж болно.

Олон сонгогчийг зэрэг загварчлах жишээг доор харуулав (жишээг туршихын тулд интерактив редакторыг идэвхжүүлнэ үү):

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

Энэ сонгогчийн жагсаалтад:

* `title` гэсэн `id` утгатай HTML элементийг онилсон **ID сонгогч** байна. Бүх ID сонгогчид **hash (`#`)** тэмдгээр эхэлдэг.
* Дараа нь таслал (`,`) тавиад, `subheading` гэсэн класс утгатай бүх HTML элементүүдийг онилсон **class сонгогч** байна. Бүх class сонгогчид **dot (`.`)** тэмдгээр эхэлдэг.

Эдгээр бүх сонгогчид **адилхан загвар** хэрэгжих бөгөөд текстийн өнгө `navy` гэж тохируулагдана.
