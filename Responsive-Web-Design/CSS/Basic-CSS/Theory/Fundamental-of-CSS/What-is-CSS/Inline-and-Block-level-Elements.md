**CSS-д Inline болон Block-level элементүүдийн ялгаа юу вэ?**

HTML болон CSS-д элементүүдийг **inline элемент** болон **block-level элемент** гэж ангилдаг бөгөөд энэ ангилал нь тухайн элементүүд **баримтын layout дээр хэрхэн аашлахыг** тодорхойлдог.

Энэ ялгааг ойлгох нь вэб хуудсан дээрх контентыг **яг хүссэнээрээ байрлуулах, харагдуулахад** маш чухал.

---

## 1. Block-level элементүүд

**Block-level элементүүд** нь анхдагчаараа өөрт боломжит **бүх өргөнийг** эзэлдэг. Өөрөөр хэлбэл, контейнерийнхээ өргөний дагуу **сунаж** байрладаг.

Эдгээр элементүүд:

* үргэлж **шинэ мөрөөс** эхэлдэг,
* өөрийн дараах контентыг дараагийн мөр рүү **шахаж** байрлуулдаг,
* ингэснээр нэгэн төрлийн **“block” контент** үүсгэнэ.

Block-level элементүүдэд анхдагчаар **`display: block;`** CSS шинж чанар хэрэглэгддэг. Энэ нь элементийг контейнерийн өргөнийг дүүргэж, шинэ мөрөнд харагдуулах үүрэгтэй.

Түгээмэл block-level элементүүдэд:

* `div`
* `p` (paragraph)
* гарчгууд (`h1`–`h6`)
* `ol` (ordered list)
* `ul` (unordered list)
* `section`
  зэрэг орно.

### 1.1 Block-level элементийн жишээ

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

Энэ жишээнд хоёр `p` элемент байна. Эхний paragraph нь улаан хүрээтэй.

Эдгээр хоёр paragraph нь:

* **нэг мөрөнд хамт харагдахгүй**,
* учир нь `p` элементүүд анхдагчаараа **block-level** элементүүд юм.

Иймээс:

* хоёулаа **контейнерийн (энд `body`) бүх өргөнийг** эзэлж,
* тус бүрдээ **шинэ мөрөөс** эхэлж харагдана.

Block-level элементүүд нь:

* контентыг **босоогоор давхарлах**,
* paragraph, section, том хэмжээний контент блок үүсгэх
  тохиолдолд хамгийн тохиромжтой.

Тэдгээрийг ихэвчлэн вэб хуудасны **layout болон бүтцийг** тодорхойлоход ашигладаг.

---

## 2. Inline элементүүд

**Inline элементүүд** нь block-level элементүүдээс ялгаатай нь:

* зөвхөн өөрт шаардлагатай **хэмжээг л** эзэлдэг,
* **шинэ мөрөөс эхлэхгүй**,
* тухайн мөрөн дотор урсаж (flow) байрладаг.

Inline элементүүдэд анхдагчаар **`display: inline;`** шинж чанар хэрэглэгддэг. Энэ нь элементийг контентийн урсгалд хадгалж, мөр таслахгүй байх нөхцөлийг бүрдүүлдэг.

Түгээмэл inline элементүүдэд:

* `span`
* `a` (anchor)
* `img`
  зэрэг орно.

### 2.1 Inline элементийн жишээ

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

Энэ жишээнд:

* `p` элементийн дотор `span` элемент байна,
* `span` нь текстийн өнгийг улаан болгож, ногоон хүрээтэй.

Эндээс харвал:

* `span` элемент нь зөвхөн **“red”** гэсэн үгийг хамрах хэмжээтэй,
* дараагийн контентыг **шинэ мөр рүү түлхэхгүй**,
* тухайн мөрөн дотороо хэвээр урсаж байна.

Inline элементүүд нь:

* нэг мөр доторх **жижиг хэсгүүдийг** загварчлах,
* үгийг тодруулах,
* hyperlink үүсгэх,
* paragraph дотор тодорхой хэсэгт тусгай загвар өгөх
  тохиолдолд хамгийн тохиромжтой.

---

## **Дүгнэлт**

* **Block-level элементүүд**
  → бүтэн өргөнийг эзэлж, шинэ мөрөөс эхэлдэг
  → layout, бүтэц, том контентод тохиромжтой

* **Inline элементүүд**
  → зөвхөн шаардлагатай хэмжээг эзэлж, мөр таслахгүй
  → текстийн доторх жижиг хэсгийг загварчлахад тохиромжтой

Энэ ялгааг зөв ойлгосноор та вэб хуудсынхаа layout-ыг илүү **нарийн хянах** боломжтой болно.
