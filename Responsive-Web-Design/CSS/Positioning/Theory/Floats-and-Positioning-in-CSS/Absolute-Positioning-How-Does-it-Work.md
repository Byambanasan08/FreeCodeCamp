**Absolute Positioning гэж юу вэ, мөн энэ хэрхэн ажилладаг вэ?**

Absolute positioning нь элементийг document-ийн хэвийн урсгалаас гаргаж, бусад элементүүдээс бүрэн тусгаарлагдсан байдлаар байрлуулах боломж олгодог. Элемент **absolute** байрлалтай болсон үед тэр нь layout дахь бусад элементүүдээс тусдаа давхаргад (layer) байрлана. Иймээс энэ нь **modal**, **tooltip**, **dropdown menu** зэрэг бусад элементүүдийн дээр давхарлаж харагдах UI элементүүдийг бүтээхэд тохиромжтой байдаг.

Анхдагчаар absolute байрлалтай элемент нь өөрт нь хамгийн ойр байрлах **positioned ancestor** элементтэй харьцангуйгаар байрладаг. Хэрэв positioned ancestor байхгүй бол элемент нь **initial containing block**-той харьцангуйгаар байрлана. Ихэнх тохиолдолд энэ нь browser-ийн **viewport** байдаг.

Та **top**, **bottom**, **left**, болон **right** property-уудыг ашиглан absolute байрлалтай элементийг хөдөлгөж болно. Эдгээр нь тухайн элемент container-ийнхээ ирмэгээс ямар зайтай байхыг тодорхойлдог.

Жишээлбэл **top: 30px** болон **left: 30px** гэж тохируулбал элемент container-ийнхээ дээд болон зүүн ирмэгээс **30px** зайтай байрлана. Доорх нь CSS дээр absolute positioning ашигласан жишээ юм:

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">Absolutely Positioned</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

Энэ код хэрэгжих үед элемент document-ийн хэвийн урсгалаас гарч, container-ийнхээ дээд болон зүүн талаас **30px** зайд байрлана.

Absolute positioning нь бусад контентын дээр давхарлаж харагдах элементүүдийг бүтээхэд маш хэрэгтэй. Жишээлбэл **overlay**, **modal window** зэрэгт ашиглагддаг. Гэхдээ элемент document flow-оос гардаг тул зөв зохицуулахгүй бол layout дотор **хоосон зай үүсэх** эсвэл бусад элементүүд **шахагдаж нийлж харагдах** асуудал үүсч болно.
