**calc() Function гэж юу вэ, мөн хэрхэн ажилладаг вэ?**

**calc()** function-ийг ашиглан та stylesheet дотор шууд тооцоолол хийж property-ийн утгыг динамик байдлаар тодорхойлж болно. Энэ нь viewport-ийн хэмжээ эсвэл бусад элементүүд дээр суурилж хэмжээсийг тооцоолон **уян хатан, responsive хэрэглэгчийн интерфэйс** бүтээх боломжийг олгодог.

**calc()** нь **CSS function** юм. Та JavaScript сурах үед function-уудын талаар илүү дэлгэрэнгүй судлах болно. Харин энэ хичээлд **calc() хэрхэн ажилладгийг ойлгоход шаардлагатай үндсэн ойлголтуудыг** авч үзнэ.

**Function** гэдэг нь тодорхой нэг ажлыг гүйцэтгэдэг кодын хэсэг юм. Зарим function-ууд CSS-д урьдчилан тодорхойлогдсон байдаг тул та тэдгээрийг шууд ашиглаж, шаардлагатай утгуудыг дамжуулан тухайн function-ийн ажиллагааг өөрчилж болно.

Програмчлалын ертөнцөд function-ийн гүйцэтгэдэг ажлыг ажиллуулахыг **function-ийг “call хийх”** гэж хэлдэг. Function-д дамжуулж буй утгуудыг **arguments** гэж нэрлэдэг.

Доорх кодоос харж болохоор function-ийг call хийхдээ **function-ийн нэрийг бичээд дараа нь хаалтанд arguments-ийг бичнэ**, arguments-уудыг таслалаар тусгаарлана. Function-ийн нэр болон нээлтийн хаалтын хооронд **зай байж болохгүй**:

```
function(argument1, argument2, argument3)
```

Зарим function-уудад зөвхөн **нэг утга** хангалттай байдаг. Тийм тохиолдолд function нь **зөвхөн нэг argument** авна. **calc() function** яг ийм байдлаар ажилладаг. Энэ нь **юуг тооцоолох ёстойг** мэдэхийн тулд **нэг argument** авдаг.

Үүний тулд та **expression** гэж нэрлэгддэг зүйлийг argument болгон дамжуулна. **Expression** гэдэг нь **утгууд болон операторуудын нийлбэр** бөгөөд үр дүн гаргадаг илэрхийлэл юм.

**calc() function**-ийг дараах байдлаар call хийж ашиглана. Та **calc** гэдэг нэрийг бичээд, хаалт дотор **expression**-ийг бичнэ:

```
calc(expression)
```

Expression нь үнэлэгдэж эцсийн үр дүнг тооцоолдог. **“Evaluated”** гэдэг нь утгууд болон операторууд дотооддоо тооцоологдон **нэг утга болж хувирна** гэсэн утгатай. Гарсан үр дүн нь тооцоолол хийсэн **CSS property-д оноогдоно**.

Та дараах төрлийн утгууд дээр тооцоолол хийж болно:

* **length**
* **angle**
* **time**
* **percentage**
* **numbers**
* **colors**

Мөн **pixel, percentage, em** зэрэг **өөр өөр нэгжүүдийг хооронд нь хослуулан** ашиглаж болно.

Тоон утгуудын хувьд expression доторх бүх утгууд (эдгээрийг **operands** гэж нэрлэдэг) өөрийн **нэгжтэй байх ёстой**, жишээлбэл **px, em, %**. Ашиглаж буй оператороос хамааран operands-ууд өөр өөр нэгжтэй байж болно.

Expression дотор та дараах операторуудыг ашиглаж болно:

* **addition (+)**
* **subtraction (-)**
* **multiplication (*)**
* **division (/)**

Хэрэв expression дотор олон operand болон оператор байвал **calc()** нь стандарт **operator precedence rule**-ийг дагана. Шаардлагатай бол үйлдлийн дарааллыг тодорхой болгохын тулд **хаалт (parentheses)** ашиглаж болно.

Доорх жишээнд **Hello, World!** гэсэн тексттэй **div** элемент байна.

**div**-ийг сонгохын тулд CSS **type selector** ашиглаж, түүнд **цагаан өнгийн текст** болон **хар хөх background** тохируулж болно.
--html---
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
---css---
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}

Энд шинэ зүйл нь **width** утга **динамикаар тооцоологдож** байгаа явдал юм. Бид **calc()** function-ийг call хийж, **expression**-ийг argument болгон дамжуулж байгааг анзаараарай. Энэ expression нь **өөр өөр нэгжтэй хоёр operand** болон **нэг оператор**, өөрөөр хэлбэл **subtraction оператор**-оос бүрдэж байна.

**Percentage** нь **relative unit** юм. **50%** гэсэн утга нь **parent container-ийн өргөнөөс** хамаарч тодорхойлогдоно. Дараа нь энэ утгаас **20px хасагдана**. Энэ expression-ийн үр дүн нь **div элементийн өргөнийг** тодорхойлно.

Ингэснээр **div** элементийн өргөн нь контейнерийнхээ нийт өргөний ойролцоогоор **тал хувьтай тэнцүү** болно. Хэрэв та parent container-ийн хэмжээг өөрчилбөл **width автоматаар дахин тооцоологдоно**.

Энэ нь **calc()** function ашиглах гол давуу тал юм. Та CSS property-ийн утгыг аппликейшний бусад хэсгүүд эсвэл viewport-ийн хэмжээнээс хамааруулан **динамикаар тодорхойлох боломжтой**.

Мөн expression дотор шаардлагатай бол **CSS function-ууд болон CSS variable-ууд** ашиглаж болно. Та дараагийн хичээлүүдэд **CSS variable**-уудын талаар илүү дэлгэрэнгүй судлах болно.

Одоо **calc() function-ийн үндсэн ойлголтыг** мэдсэн тул зарим **best practice**-уудыг авч үзье.

Нэгдүгээрт, **addition (+)** болон **subtraction (-)** операторуудын **хоёр талд заавал whitespace (зай)** байх ёстой.

Жишээлбэл дараах expression нь **зөв биш**. Учир нь subtraction оператор нь хоёр дахь operand-той шууд залгаатай байна.

```
calc(100% -30px)
```

Subtraction **(-)** операторын хоёр талд зай байх ёстой. Ингэснээр expression зөв болно.

```
calc(100% - 30px)
```

**Multiplication (*)** болон **division (/)** операторуудад энэ шаардлага заавал биш боловч зай тавихыг **ихэвчлэн зөвлөдөг**.

Хэрэв шаардлагатай бол **calc() function-ийг дотор нь давхарлан (nest)** ашиглаж, нэг тооцооллын үр дүнг өөр тооцоололд ашиглаж болно.

Мөн **calc() function** дотор **уртыг илэрхийлэхдээ 0 ашиглаж байгаа бол нэгжийг заавал бичих ёстой**. Жишээлбэл дараах expression **зөв биш**.

```
calc(100% - 0)
```

Та **нэгж нэмэх** хэрэгтэй.

```
calc(100% - 0px)
```

Одоогийн байдлаар **multiplication** болон **division** оператор ашиглах үед **operand-уудын нэг нь нэгжгүй (unitless) байх ёстой**. Ялангуяа **division операторын баруун талын operand unitless байх шаардлагатай**.

Доорх expression **зөв биш**, учир нь хоёр operand хоёулаа **pixel нэгжтэй** байна.

```
calc(5px * 50px)
```

Энд **5 эсвэл 50**-ийн аль нэг нь **unitless** байх ёстой. Дараах хоёр хувилбар хоёулаа **зөв**.

```
calc(5 * 50px)
calc(5px * 50)
```

Доорх нь **division операторын** жишээ юм. Энэ expression **зөв биш**, учир нь хоёр operand хоёулаа нэгжтэй байна.

```
calc(50% / 5%)
```

Division оператор ашиглах үед **баруун талын operand-ийн нэгжийг хасах** хэрэгтэй.

```
calc(50% / 5)
```

**calc() function** нь веб хөгжүүлэгчдэд маш хэрэгтэй хэрэгсэл юм. Энэ function-ийг ашигласнаар та **CSS property-ийн утгуудыг динамикаар тохируулж**, **уян хатан болон responsive дизайн** бүтээх боломжтой болно.
