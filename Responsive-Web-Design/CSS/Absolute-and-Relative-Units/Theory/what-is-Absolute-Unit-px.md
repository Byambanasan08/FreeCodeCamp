**CSS-д Absolute Units гэж юу вэ, мөн хэзээ ашиглах ёстой вэ?**

Вэб хуудсаа дизайн хийх явцад та width, height, padding, margin зэрэг олон төрлийн property-уудтай ажиллана. Эдгээр property-уудыг тодорхойлох үед та ямар **length unit** ашиглахаа зааж өгөх шаардлагатай.

Эдгээр property-уудыг тодорхойлоход ашиглаж болох **хоёр төрлийн unit** байдаг: **relative units** болон **absolute units**. Энэ хичээлд бид зөвхөн **absolute units** дээр анхаарна.

**Absolute length units** нь тогтмол урттай байдаг бөгөөд өөр ямар нэгэн зүйлээс хамаардаггүй. Харин **relative** гэдэг нь тухайн урт нь өөр зүйлтэй харьцангуй хэмжигдэж байгааг илэрхийлнэ. Жишээлбэл дэлгэцийн хэмжээ, эсвэл parent элементийн хэмжээтэй харьцуулах зэрэг. Relative units-ийн талаар дараагийн хэд хэдэн хичээл дээр илүү дэлгэрэнгүй ярилцах болно.

Хамгийн түгээмэл absolute unit бол **pixel (px)** юм. Pixel нь CSS-д ашиглагддаг тогтмол хэмжээтэй хэмжих нэгж бөгөөд хэмжээсийг маш нарийн хянах боломжийг олгодог. Энэ нь **1px үргэлж 1/96 inch-тэй тэнцүү** гэсэн үг юм.

Гэхдээ CSS layout-ийн зорилгоор **1px-ийг 1/96 inch** гэж стандартчилсан байдаг ч бодит физик хэмжээ нь тухайн дэлгэцээс хамааран өөр байж болохыг анхаарах хэрэгтэй.

Одоо **pixel ашиглан box-ийн width болон height-ийг хэрхэн тохируулах жишээг** авч үзье.

<link rel="stylesheet" href="styles.css" />
<div class="box"></div>

---css--
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}

Энэ жишээнд бид **width** болон **height** нь **100px** гэж тохируулагдсан улаан өнгийн хайрцагтай байна. Энэ хайрцгийн хэмжээс нь дэлгэцийн хэмжээ өөрчлөгдсөнөөс үл хамааран ижил хэвээр үлдэнэ.

Тиймээс хэрэв та жижиг дэлгэцтэй төхөөрөмж ашиглаж байгаа бол хайрцгийн өргөн **100px** хэвээр байна. Хэрэв том desktop дэлгэц ашиглаж байгаа бол мөн адил хайрцгийн өргөн **100px** хэвээр байна.

Тэгвэл **pixel** зэрэг absolute unit-ийг хэзээ ашиглах ёстой вэ? Энэ нь тухайн нөхцөл байдлаас шалтгаална. Гэхдээ ерөнхийдөө элементүүдийн хэмжээ, зай, болон layout-ийг маш нарийн хянах шаардлагатай үед pixel ашигладаг. Заримдаа **margin**, **padding**, болон **border** тохируулахдаа мөн pixel ашиглаж болно.

Доорх нь хоёр хайрцгийн **margin**-ийг pixel ашиглан тохируулсан жишээ юм.

<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
---css---
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}

**Margin** гэдэг нь хайрцгийн гаднах зайг хэлдэг гэдгийг санаарай. Тиймээс энэ жишээнд хайрцаг нь бүх талаараа **10px margin**-тай байна.

Бусад төрлийн **absolute unit**-ууд дараах байдалтай байна:

1. in (inches)** нэгж — **96px**-тэй тэнцүү
2. cm (centimeters)** нэгж — **1 inch-ийн 25.2/64**-тэй тэнцүү
3. mm (millimeters)** нэгж — **1 centimeter-ийн 1/10**-тэй тэнцүү
4. q (quarter-millimeters)** нэгж — **1 centimeter-ийн 1/40**-тэй тэнцүү
5. pc (picas)** нэгж — **1 inch-ийн 1/6**-тай тэнцүү
6. pt (points)** нэгж — **1 inch-ийн 1/72**-той тэнцүү

Эдгээр нэгжүүдийн ихэнх нь дэлгэцийн хэрэглээнээс илүү **хэвлэлийн (print)** зориулалтаар ашиглагддаг.

Хэдийгээр хамгийн түгээмэл ашиглагддаг нэгж нь **pixel** боловч бусад absolute unit-ууд мөн байдаг гэдгийг мэдэх нь чухал юм.
