**CSS-д em болон rem гэж юу вэ, мөн хэзээ ашиглах ёстой вэ?**

Өмнөх хичээлд бид **pixel** зэрэг **absolute length unit**-уудын талаар судалсан. Absolute unit-ууд зарим нөхцөлд хэрэгтэй байдаг боловч зарим үед **relative unit** ашиглах шаардлага гардаг.

Энэ хичээлд бид **хоёр төрлийн relative unit** болох **em** болон **rem**-ийн талаар судлах болно.

**em** нэгж нь тухайн **element-ийн font size-тэй харьцангуй** байдаг. Хэрэв та **font-size** property дээр **em** ашиглавал тухайн текстийн хэмжээ нь **parent element-ийн font size-тэй харьцуулж** тодорхойлогдоно.

Энэ нь хэрхэн ажилладгийг илүү сайн ойлгохын тулд нэг жишээг авч үзье.

<link rel="stylesheet" href="styles.css" />
<p class="para">I am a paragraph element</p>

<div class="blue-box"></div>
---css---
.para {
  font-size: 20px;
  margin-bottom: 1.5em;
  border: 2px solid red;
}

.blue-box {
  background-color: blue;
  color: white;
  padding: 10px;
  width: 100px;
  height: 100px;
}

HTML хэсэгт бид **paragraph** болон **div** элементүүдтэй байна. **Paragraph** элемент нь **para** гэсэн class-тай бөгөөд **div** элемент нь **blue-box** гэсэн class-тай.

**para** class дээр бид **font-size**-ийг **20px**, мөн **margin-bottom**-ийг **1.5em** гэж тохируулсан. Энэ нь margin нь paragraph элементийн **font size-ийн 1.5 дахин их** байна гэсэн үг юм. **1.5em** нь paragraph-ийн доод хэсэгт **30 пикселийн margin** үүсгэнэ. Мөн margin-ийг илүү тод харагдуулахын тулд **2px solid red border** тохируулсан.

**blue-box** class дээр бид **background color**-ийг **blue**, **text color**-ийг **white**, мөн бүх дөрвөн талд нь **10px padding** тохируулсан.

Энэ жишээнээс харахад paragraph элементийн доод хэсэг болон blue box-ийн хооронд тодорхой зай үүснэ.

Тэгвэл **para** class-аас **font-size** property-г хасвал юу болох вэ?

Энэ тохиолдолд **bottom margin** нь **parent элементийн font size-тэй** харьцангуй болно. Энэ жишээнд parent элемент нь **body** элемент бөгөөд анхдагч **font size нь 16px** байдаг.

**em** ашиглах тохиромжтой тохиолдлууд нь **button**, **card** зэрэг модуль хэлбэрийн компонентуудтай ажиллах үед юм. **em unit** ашигласнаар тухайн компонентын бүх хэсгүүд (жишээлбэл **padding**, **margin**, **border**) нь **font size-тэй пропорциональ байдлаар масштаблагдах** бөгөөд ингэснээр хэмжээсийн харьцаа тогтвортой хадгалагдана.

Одоог хүртэл бид элементүүдийн **font size**-ийг **pixel** ашиглан тохируулж байсан. Гэвч энэ нь хэрэглэгчдэд зарим хүндрэл үүсгэж болдог.

Browser-ийн тохиргоо дотор хэрэглэгч **анхдагч font size**-ийг өөрчлөх боломжтой байдаг.

Харааны бэрхшээлтэй хүмүүс текстийг илүү уншихад хялбар болгохын тулд font size-ийг нэмэгдүүлж болдог. Гэвч хэрэв вэб дизайнд **pixel** ашиглан font size тохируулсан бол текст нь бусад контенттойгоо **пропорциональ байдлаар масштаблагдахгүй**.

Энэ асуудлыг шийдэх нэг арга бол **typography-д rem unit ашиглах** явдал юм. **rem unit** нь **root элемент болох html элементийн font size-тэй** харьцангуй байдаг.

Анхдагчаар **html элементийн font size нь 16px** байдаг. Хэрэв хэрэглэгч browser-ийн тохиргоонд font size-ийг нэмэгдүүлбэл **html элементийн font size** мөн нэмэгдэнэ, ингэснээр **rem unit ашигласан бүх хэмжээсүүд пропорциональ байдлаар масштаблагдана**.

Доорх нь **pixel-ийн оронд rem unit ашиглан font size тохируулсан жишээ** юм.

<link rel="stylesheet" href="styles.css" />
<p>This is regular text.</p>
<p class="para">This text is slightly larger.</p>

---css---
.para {
  font-size: 1.2rem;
  margin-bottom: 1.5em;
  border: 2px solid red;
}

**font-size**-ийг **1.2rem** гэж тохируулснаар paragraph элементийн **font size** нь **root элементийн font size-ийн 1.2 дахин их** болно. Хэрэв хэрэглэгч анхдагч **font size**-ийг өөрчлөөгүй бол paragraph элементийн **font size нь 19.2px** байна, учир нь энэ нь **16px-ийн 1.2 дахин их** юм.

Тэгвэл **rem unit**-ийг хэзээ ашиглах ёстой вэ? **rem unit** нь typography-д **pixel-ээс илүү тохиромжтой** байдаг, учир нь энэ нь хэрэглэгчийн browser-ийн тохиргоотой **пропорциональ байдлаар масштаблагддаг**. Ингэснээр таны контент **харааны бэрхшээлтэй хэрэглэгчдэд илүү хүртээмжтэй** болдог.

Мөн **rem unit** нь өөр өөр элементүүдийн хооронд **spacing болон layout-ийг тогтвортой хадгалахад** тусалдаг.
