### `minmax()` функц гэж юу вэ, хэрхэн ажилладаг вэ?

**`minmax()` функц** нь CSS Grid дээр **grid track (мөр эсвэл багана)-ийн хэмжээний хүрээг** тодорхойлдог. Өөрөөр хэлбэл тухайн мөр эсвэл багана **хамгийн багадаа хэд байх**, **хамгийн ихдээ хэд хүртэл томрох** боломжтойг зааж өгдөг.

Та өмнө нь track-ийн хэмжээг дараах нэгжүүдээр тохируулж болдгийг мэднэ:

* `px` (pixel)
* `rem`
* `em`
* `fr` (fractional unit)

`minmax()` функц нь үүнээс илүү уян хатан байдлыг өгч, **хамгийн бага ба хамгийн их хэмжээг зэрэг тодорхойлох** боломж олгодог.

---

### `minmax()` синтакс

```css
minmax(min, max)
```

* **min** → grid track-ийн **хамгийн бага хэмжээ**
  (`px`, `%`, `auto` гэх мэт ашиглаж болно)

* **max** → grid track-ийн **хамгийн их хэмжээ**
  (мөн адил `px`, `%`, `fr`, `auto` гэх мэт ашиглаж болно)

---

### Хэрхэн ажилладаг вэ?

Эдгээр хоёр утга дараах байдлаар хамт ажиллана:

* **min утга** → track энэ хэмжээнээс **жижиг болохгүй**
* **max утга** → track энэ хэмжээнээс **том болохгүй**

Grid track-ийн хэмжээ нь **container-ийн хэмжээ болон контентоос хамааран** эдгээр хоёр утгын хооронд **динамикаар өөрчлөгдөнө**.

---

### Жишээ

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div>
    <h2>Item 1</h2>
  </div>
  <div>
    <h2>Item 2</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: minmax(150px, 300px) 1fr;
  gap: 20px;
}

.grid-container > div {
  background: crimson;
  padding: 20px;
  text-align: center;
}
```

---

### Энэ код дээр юу болж байна вэ?

* **Эхний багана** → `minmax(150px, 300px)`

  * хамгийн багадаа **150px**
  * хамгийн ихдээ **300px**

  available space-ээс хамаарч **150px – 300px** хооронд өөрчлөгдөнө.

* **Хоёр дахь багана** → `1fr`

  * grid container-д үлдсэн **бүх зайг эзэлнэ**
  * өөр багана байхгүй тул бүх сул зайг авна.

---

### `minmax()` ашиглахын давуу тал

`minmax()` нь:

* fixed хэмжээ (`px`)-ээс илүү **уян хатан**
* `fr` нэгжээс ч илүү **responsive control** өгдөг
* **adaptive layout** хийхэд тохиромжтой
* **responsive дизайн** хийхэд маш хэрэгтэй

---

### Дүгнэлт

`minmax()` функц нь CSS Grid дээр:

* track-ийн **хамгийн бага ба хамгийн их хэмжээг** тодорхойлох
* layout-ийг **уян хатан болгох**
* **responsive дизайн** хийх

маш чухал хэрэгсэл юм.
