**text-shadow Property гэж юу вэ, мөн энэ хэрхэн ажилладаг вэ?**

CSS нь анхдагчаар текст дээр ямар нэг сүүдэр (shadow) хэрэглэдэггүй. Энэ бол ямар ч сүүдэргүй paragraph-ийн жишээ юм. Харин хэрэв та сүүдэр нэмэх шаардлагатай бол text-shadow property-г ашиглаж болно.

```css
text-shadow: /* Values */
```

CSS-д та сүүдрийг X offset, Y offset, blur radius, болон color-ийн хослолоор тодорхойлж болно. Эхлээд та X болон Y offset-ийг зааж өгөх хэрэгтэй. Эдгээр нь тус тусдаа сүүдрийн текстээс хэвтээ болон босоо зайг илэрхийлдэг. Эдгээр утгууд заавал байх ёстой.

Доорх нь сүүдэрт X болон Y offset ашиглах жишээ юм. Бид text-shadow property-г x-offset 3px, y-offset 2px утгатайгаар ашиглаж байна:

```html id="x5lgc1"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="mptv7l"
p {
  text-shadow: 3px 2px;
}
```

Browser дээр текст болон сүүдэр төстэй харагдана. Гэхдээ бид offset-ийн өмнө эсвэл дараа өнгийг зааж өгснөөр сүүдрийн өнгийг өөрчилж болно. Одоо сүүдрийн өнгийг тохируулъя. Бид энд hexadecimal өнгө ашиглах боловч та ямар ч хүчинтэй өнгөний форматыг ашиглаж болно.

```html id="d0i8ct"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="36tfrp"
p {
  text-shadow: 3px 2px #00ffc3;
}
```

Бид offset-ийн дараа өнгийг бичиж байна. Гэхдээ хэрэв хүсвэл өнгийг offset-ийн өмнө бичиж болно. Энэ нь адил утгатай:

```html id="m9r4e1"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="w9qk4u"
p {
  text-shadow: #00ffc3 3px 2px;
}
```

Browser дээр сүүдэр тодорхой өнгөтэй харагдах тул текстээс амархан ялгагдана. Одоо бид сүүдрийг текстээс ялгаж харж байгаа тул эерэг болон сөрөг утгууд offset-д хэрхэн нөлөөлдгийг ойлгох нь чухал. X offset болон Y offset-ийн эерэг утгууд нь сүүдрийг баруун болон доош хөдөлгөнө. Харин сөрөг утгууд нь сүүдрийг зүүн болон дээш хөдөлгөнө. Доорх нь сөрөг утгууд ашигласан жишээ юм:

```html id="n0xt5u"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="t86q5o"
p {
  text-shadow: -3px -2px #00ffc3;
}
```

Одоо сүүдэр тексттэй харьцуулахад зүүн болон дээш байрлана. Гэхдээ энэ сүүдэр тийм ч сайхан харагдахгүй байна. Учир нь энэ нь яг эх тексттэй адилхан боловч ард нь байрласан мэт харагдаж байна. Үүнийг илүү гоё харагдуулахын тулд бид гурав дахь утга болох blur radius-г нэмэх хэрэгтэй. Энэ утга нь optional боловч сүүдрийг илүү зөөлөн, илүү жигд харагдуулдаг. Blur radius-ийн анхдагч утга нь zero байдаг. Утга нь их байх тусам blur ихсэж, сүүдэр илүү зөөлөн харагдана.

Доорх жишээнд бид blur radius-ийг 3px болгож, offset-д дахин эерэг утгууд ашиглаж байна:

```html id="nq4k8j"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="v2d6lo"
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

Одоо энэ нь илүү сүүдэр шиг харагдаж эхэлнэ. Илүү бүдгэрсэн, илүү зөөлөн болсон тул бид үндсэн текст дээр анхаарлаа төвлөрүүлэх боломжтой бөгөөд сүүдэр нь ар талд бага зэрэг стиль нэмж өгнө. Эндээс эхлэн бид offset, color, болон blur radius-ийн утгуудыг бага зэрэг өөрчилж, өөрт тохирох хослолыг олох боломжтой.

Мөн текст нь нэгээс олон сүүдэртэй байж болдгийг мэдэх нь ашигтай. Үүнийг хийхийн тулд text-shadow property дотор таслалаар тусгаарлан бичихэд хангалттай. Тэдгээр нь layer байдлаар, урдаас хойш дарааллаар хэрэглэгдэнэ. Эхний сүүдэр хамгийн дээр байрлана.

Доорх нь гурван сүүдэр layer байдлаар хэрэглэсэн paragraph-ийн жишээ юм. Эдгээр бүх сүүдрүүд нэг text-shadow property дотор тодорхойлогдож, таслалаар тусгаарлагдсан байгааг анзаараарай:

```html id="dsgn31"
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css id="8wq1si"
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```

text-shadow нь текстийг илүү тодруулж харагдуулах хүчирхэг CSS property юм. Мөн энэ нь таны вэб аппликэйшний ерөнхий дизайныг сайжруулах боломжтой. Offset, blur, болон өнгийг зөв сонгосноор та олон төрлийн сүүдрийн эффектүүдийг үүсгэж чадна.
