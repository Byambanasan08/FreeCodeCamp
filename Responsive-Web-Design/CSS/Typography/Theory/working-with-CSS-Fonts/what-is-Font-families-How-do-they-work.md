**Font Family гэж юу вэ, мөн тэд хэрхэн ажилладаг вэ?**

Font family нь ижил дизайнтай фонтуудын бүлэг юм. Нэг гэр бүлд хамаарах бүх фонтууд нь нэг үндсэн typeface дээр суурилдаг боловч тэдгээр нь хэв маяг, жин, болон өргөний хувьд өөр өөр хувилбаруудтай байдаг. Та тэднийг ижил төстэй шинж чанартай боловч тодорхой ялгаатай ах дүүс гэж ойлгож болно. Жишээлбэл, Arial нь Arial Bold болон Arial Italic зэрэг хувилбаруудыг агуулсан font family юм.

CSS дээр та font-family property ашиглан фонтын гэр бүлийг тохируулж болно. Доорх нь paragraph-д Arial фонт тохируулж буй жишээ юм:

```html
<link rel="stylesheet" href="styles.css">
<p id="arial-font">Example paragraph using Arial font.</p>
<p>Paragraph not using Arial font.</p>
```

```css
#arial-font {
  font-family: Arial;
}
```

Харин font family олдохгүй бол яах вэ? Та олон font family-г хамгийн өндөрөөс хамгийн бага хүртэлх давуу эрхийн дарааллаар, таслалаар (comma) тусгаарлан зааж өгч болно. Эдгээр өөр фонтууд нь fallback сонголт болж ажиллана. Доорх жишээнд Arial нь үндсэн фонт, харин Lato нь альтернатив фонт юм:

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato;
}
```

Хэрэв төхөөрөмж дээр эхний фонт (Arial) олдвол browser түүнийг харуулна. Хэрэв олдохгүй бол хоёр дахь фонт (Lato)-г харуулна.

Мөн энэ процесс цаанаа хэрхэн ажилладгийг мэдэх хэрэгтэй. Эхний фонт байгаа тохиолдолд сонголтын процесс зогсдоггүй. Font family нь тэмдэгт бүрийн түвшинд сонгогддог тул хэрэв тухайн фонт тодорхой тэмдэгтийг агуулаагүй бол browser дараагийн доогуур давуу эрхтэй фонтуудаас хайж олдог. Сонирхолтой, тийм үү?

Вэб хөгжүүлэлтийн хүрээнд та generic font families-тай ч мөн тааралдана. Эдгээр нь маш чухал. Generic font family нь илүү өндөр давуу эрхтэй фонтууд олдохгүй үед вэб browser-ууд харуулах default фонт юм. Контент уншигдах боломжтой хэвээр байлгахын тулд browser нь заасан generic font family дээр үндэслэн хамгийн тохиромжтой фонт руу эх фонтыг орлуулдаг. Хамгийн түгээмэл хэрэглэгддэг зарим нь:

* serif
* sans-serif
* monospace
* cursive
* fantasy

Гэхдээ сонгох өөр олон сонголт бий. Доорх кодод бид гурван альтернатив font family ашигласан байна:

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato, sans-serif;
}
```

Arial нь хамгийн өндөр давуу эрхтэй. Хэрэв Arial олдохгүй бол browser Lato-г ашиглахыг оролдоно. Хэрэв аль аль нь олдохгүй бол browser generic sans-serif font family-г ашиглаж, хэрэглэгчийн систем дээр суусан фонтуудаас энэ шинжтэй фонт сонгон харуулна.

Та font-family жагсаалтын төгсгөлд үргэлж generic font family оруулах хэрэгтэй. Generic фонт нь таны анх дизайнд төсөөлсөн фонттой өөр харагдаж болзошгүйг санаарай. Тиймээс эдгээр fallback фонтууд янз бүрийн browser дээр хэрхэн харагдаж байгааг шалгах нь үргэлж тустай байдаг.

Хэрэглэгчийн туршлагыг аль болох тогтвортой байлгахын тулд web-safe fonts ашиглахыг мөн зөвлөдөг. Эдгээр font family-ууд ихэнх төхөөрөмж дээр суусан байдаг тул ихэнх хэрэглэгчдэд зөв харагдах магадлал өндөр байдаг. Та дараагийн хичээлүүдээр тэдгээрийн талаар илүү ихийг сурах болно.

Font family-ууд нь вэб дизайны ертөнцөд маш чухал. font-family property ашигласнаар та текстийн харагдах байдал болон мэдрэмжийг шууд өөрчилж чадна.
