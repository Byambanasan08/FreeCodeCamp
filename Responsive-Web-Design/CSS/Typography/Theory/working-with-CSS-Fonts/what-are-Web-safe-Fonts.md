**Web Safe Fonts гэж юу вэ?**

Web-safe fonts нь компьютер эсвэл төхөөрөмж дээр суусан байх магадлал маш өндөр байдаг фонтуудын нэг дэд бүлэг юм. Эдгээр нь янз бүрийн үйлдлийн систем болон вэб browser-ууд дээр өргөн дэмжигддэг тул тогтвортой байдлаар харагдаж, ижил байдлаар дүрслэгдэх магадлал өндөр байдаг. Одоо энэ хэрхэн ажилладгийг харцгаая.

Вэбсайт дээр фонтыг тайлбарлан харуулах үүргийг browser-ууд гүйцэтгэдэг. Browser фонт дүрслэх шаардлагатай үед хэрэглэгчийн систем дээр тухайн фонтын файлыг хайдаг. Харин фонт олдохгүй бол ихэвчлэн системийн default фонт руу fallback хийдэг. Ингэснээр вэбсайт дээр харагдах ёстой тодорхой фонт байхгүй байсан ч контент уншигдах боломжтой хэвээр үлддэг.

Гэсэн хэдий ч browser сонгосон fallback фонт нь анх харагдах ёстой байсан фонтоос нэлээд өөр харагдаж болно. Энэ нь нийт дизайн болон хэрэглэгчийн туршлагад ноцтой нөлөө үзүүлж болзошгүй. Үүнээс зайлсхийхийн тулд боломжтой үедээ web-safe fonts ашиглах хэрэгтэй. Танд хоёр сонголт бий. Та тэдгээрийг үндсэн фонт болгон ашиглаж болно, эсвэл custom fonts ашиглаад web-safe фонт-ыг fallback сонголт болгон ашиглаж болно. Ингэснээр custom фонт олдохгүй үед вэбсайт хэрхэн харагдахыг та хянах боломжтой болно.

Одоо web-safe fonts-ийн зарим жишээг харцгаая. Өмнөх хичээлээр та sans-serif фонтууд нь тэмдэгтийн төгсгөл хэсэгт жижиг “хөл” буюу шугамгүй байдаг тул дэлгэц дээр уншихад хялбар учраас вэб хөгжүүлэлтэд түгээмэл ашиглагддаг гэдгийг сурсан. Web-safe sans-serif фонтуудын зарим жишээ нь:

* Arial
* Verdana
* Trebuchet MS

Үүний эсрэгээр serif фонтууд нь тэмдэгтийн төгсгөлд жижиг “хөл”-тэй байдаг тул уламжлалт хэвлэмэл материалд ихэвчлэн ашиглагддаг. Гэхдээ хэрэв та тэдгээрийг вэб хөгжүүлэлтэд ашиглах шаардлагатай бол web-safe serif фонтуудын жишээнд дараах фонтууд орно:

* Times New Roman
* Georgia

Web-safe fonts ашигласнаар таны дизайн төхөөрөмж болон платформуудын хооронд тогтвортой харагдахыг баталгаажуулж чадна. Мөн эдгээр фонтууд нь энгийн бөгөөд уншихад хялбар байдаг тул харааны бэрхшээлтэй хэрэглэгчдэд хүртээмжийг сайжруулж чадна. Мөн web-safe fonts нь аль хэдийн систем дээр суусан байдаг тул татаж авах шаардлагагүй, ингэснээр хуудасны ачааллах хугацааг багасгаж чадна.

Одоо та web-safe fonts гэж юу болохыг мэдсэн тул тэдгээрийг ашиглаж, browser болон платформуудын хооронд найдвартай хэрэглэгчийн туршлагыг бий болгохын тулд fallback сонголт болгон тохируулж эхлэх боломжтой.
