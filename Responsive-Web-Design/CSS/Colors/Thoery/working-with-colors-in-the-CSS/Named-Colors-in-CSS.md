**CSS дахь Named Colors гэж юу вэ, мөн хэзээ ашиглах вэ?**

CSS-д өнгө нь **вэб хуудасны дизайн хийхэд** маш чухал үүрэгтэй. Өнгө нь **уншихад хялбар байдлыг сайжруулах**, **уур амьсгалыг бүрдүүлэх**, мөн **хэрэглэгчийн туршлагыг сайжруулах**ад тусалдаг.

CSS-д өнгийг тодорхойлох хамгийн **энгийн аргуудын нэг** нь **named colors** ашиглах юм. **Named colors** гэдэг нь **browser-ууд урьдчилан таньж мэддэг өнгөний нэрүүд** юм.

Доорх нь **paragraph элементэд named color ашигласан** жишээ юм.

<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>

--css--
p {
  color: red;
}

Энэ жишээнд бид **paragraph** доторх текстийг стильдэхийн тулд **red** нэртэй өнгийг ашиглаж байна.

CSS дахь **named colors** нь **140 стандарт өнгөний нэрийн цуглуулга** юм. Жишээлбэл **red**, **blue**, **yellow**, **aqua**, **fuchsia**, **black** гэх мэт. Эдгээр нэрүүд нь **ашиглахад хялбар**, кодыг **илүү ойлгомжтой** болгодог бөгөөд **өөрсдөө утгаа илэрхийлсэн** байдаг.

**Named colors** нь **хурдан prototype хийх**, **энгийн дизайн** боловсруулах, мөн **кодын уншигдах байдлыг сайжруулах** үед хэрэгтэй байдаг.

Доорх нь **h1 selector** дээр named colors ашигласан өөр нэг жишээ юм.

<link rel="stylesheet" href="styles.css" />
<h1>This is a heading</h1>
--css--
h1 {
  color: navy;
  background-color: lightgray;
}

Энэ жишээнд **heading текстийг navy өнгөөр**, харин **background-ийг light gray өнгөөр** стильдэнэ. Named colors ашигласнаар кодын **уншигдах байдал сайжирч**, ямар өнгө хэрэглэх гэж байгааг **шууд ойлгох боломжтой** болдог.

CSS дахь **named colors** нь ашиглахад хялбар боловч **хязгаарлагдмал** бөгөөд нийтдээ **140 өнгөний сонголттой**. Илүү нарийн, тодорхой өнгө шаардсан **дэлгэрэнгүй дизайн** хийхэд шаардлагатай яг тохирох өнгийн сүүдэрийг үргэлж өгч чаддаггүй.

CSS дахь **named colors** нь өнгийг **хурдан бөгөөд ойлгомжтой байдлаар** хэрэглэх боломж олгодог. Эдгээр нь **энгийн дизайн**, **prototype хийх**, мөн **кодын уншигдах байдлыг сайжруулахад** маш тохиромжтой. Гэвч өнгөний **сонголт хязгаарлагдмал** тул **нарийн өнгөний хяналт шаардсан** илүү төвөгтэй дизайнуудад төдийлөн тохиромжтой биш.

Named colors-ийн **давуу болон сул талыг ойлгосноор** та тэдгээрийг **хэзээ ашиглах нь зөв** болохыг тодорхойлж чадна. Зарим тохиолдолд илүү **нарийн өнгөний удирдлага** өгдөг **RGB** эсвэл **HSL** зэрэг color model-уудыг ашиглах нь илүү тохиромжтой байдаг бөгөөд эдгээрийн талаар та **дараагийн хичээлүүдэд** суралцах болно.
