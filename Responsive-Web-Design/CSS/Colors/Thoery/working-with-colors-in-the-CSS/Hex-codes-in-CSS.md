**Hex Codes гэж юу вэ, мөн CSS-д хэрхэн ажилладаг вэ?**

Вэб хуудсыг CSS ашиглан стильдэх үед өнгө тодорхойлох хамгийн түгээмэл аргуудын нэг нь **hex color values** ашиглах юм. **Hexadecimal color values** буюу **hex codes** нь **RGB color model**-д өнгийг илэрхийлэх **товч бөгөөд нарийн арга** юм. Энэ хэсэгт бид **hex code гэж юу болох**, **CSS-д хэрхэн ажилладаг**, мөн **хэзээ ашиглах нь тохиромжтой** талаар үзнэ.

**Hex code** (hexadecimal code-ийн товчлол) нь **RGB color model**-д өнгийг илэрхийлэхэд ашиглагддаг **6 тэмдэгтээс бүрдсэн тэмдэгт мөр** юм. **Hex** гэдэг нь **16-т суурилсан тооллын системийг (base-16)** илэрхийлдэг бөгөөд энэ нь **0-9 хүртэлх цифрүүд болон A-F хүртэлх үсгүүдийг** ашигладаг.

Өнгөний хувьд hex code нь **улаан (red)**, **ногоон (green)**, **хөх (blue)** өнгө тус бүрийн хэмжээг тодорхойлдог. Hex code бүр **# тэмдгээр эхэлж**, дараа нь **6 тэмдэгт** бичигдэнэ. Эдгээр нь тухайн өнгөнд орж буй **red, green, blue intensity**-г тодорхойлдог.

Жишээлбэл:

* **Black** → `#000000`
* **White** → `#FFFFFF`

Энд:

* **эхний хоёр тэмдэгт → red**
* **дараагийн хоёр → green**
* **сүүлийн хоёр → blue**

CSS-д hex code ашиглан элементүүдэд өнгө өгөх үндсэн синтакс:

```css
element {
  color: #RRGGBB;
}
```

Энд **#RRGGBB** нь бодит hex кодын оронд бичигдсэн placeholder юм.

* **RR → red**
* **GG → green**
* **BB → blue**

Эдгээр утгууд **00-оос FF хүртэл** байж болно.

* **00 → хамгийн бага intensity**
* **FF → хамгийн их intensity**

Тоон утга **их байх тусам тухайн өнгө илүү хүчтэй** оролцоно.

Жишээ:

```css
h1 {
  color: #FF5733; /* улаан-улбар шар өнгө */
}

p {
  background-color: #4CAF50; /* ногоон өнгийн сүүдэр */
}
```

**#FF5733** өнгийн задлал:

* **FF → red хамгийн их**
* **57 → green дунд зэрэг**
* **33 → blue бага**

**#4CAF50** өнгийн задлал:

* **4C → red бага**
* **AF → green хүчтэй**
* **50 → blue дунд зэрэг**

Hex code нь **нарийвчлалтай**, **ашиглахад хялбар** учраас маш өргөн хэрэглэгддэг. Энэ нь өнгийг **маш нарийн тохируулах боломж** олгодог бөгөөд **дизайн дахь тогтвортой байдлыг** хангахад тохиромжтой.

Ихэнх **design software** (жишээ нь **Adobe Photoshop**, **Figma**) өнгийг сонгоход **hex code** өгдөг. Ингэснээр эдгээр утгыг **CSS файл руу шууд copy-paste хийх** боломжтой болдог.

Вэб хөгжүүлэлтэд hex code-ийг дараах зүйлсэд өргөн ашигладаг:

* текстийн өнгө
* background
* border
* button
* UI элементүүд

Жишээ:

```css
body {
  background-color: #F0F0F0; /* цайвар саарал background */
}

button {
  background-color: #007BFF; /* хөх өнгө */
  color: #FFFFFF; /* цагаан текст */
}
```

Энэ нь **вэбсайт эсвэл аппликейшний бүх хэсэгт өнгөний тогтвортой байдлыг** хадгалдаг.

Зарим тохиолдолд hex code-ийг **богино хэлбэрээр (shorthand)** бичиж болно. Энэ нь **RR, GG, BB бүрийн хоёр тэмдэгт ижил үед** боломжтой.

Жишээлбэл:

* **#FFFFFF → #FFF**
* **#000000 → #000**

Жишээ:

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF;
}
```

Энд:

* **#FFF = #FFFFFF**
* **#000 = #000000**

Hex code нь **нарийвчлал**, **browser compatibility** сайтай бөгөөд **16 сая гаруй өнгө** илэрхийлэх боломжтой.

Эдгээрийг ихэвчлэн дараах үед ашигладаг:

* **брэндийн яг тодорхой өнгө хэрэгтэй үед**
* **named colors эсвэл HSL-ээр тодорхойлоход хэцүү өнгө ашиглах үед**
* **design software-оос өнгө шууд хуулж авах үед**

Иймээс hex code нь **нарийн өнгөний удирдлага шаардсан дизайн** болон **вэб хөгжүүлэлтэд** маш өргөн хэрэглэгддэг арга юм.
