**RGB Color Model гэж юу вэ, мөн CSS дахь rgb() function хэрхэн ажилладаг вэ?**

CSS-д өнгөтэй ажиллах үед **RGB color model**-ийг ойлгох нь маш чухал. **RGB** гэдэг нь **Red (улаан)**, **Green (ногоон)**, **Blue (хөх)** гэсэн **гэрлийн үндсэн өнгөнүүдийг** илэрхийлдэг. Эдгээр гурван өнгийг **өөр өөр хүчтэй (intensity)** хослуулснаар маш олон төрлийн өнгө үүсгэдэг.

**RGB color model** нь **additive color model** юм. Энэ нь өнгө нь **гэрлийг өөр өөр хүчээр нэмж хослуулах замаар** үүсдэг гэсэн үг.

Өнгө бүрийн **intensity** нь **0-оос 255 хүртэл** утгатай байдаг.

* **0** нь тухайн өнгөний **гэрэл огт байхгүй** гэсэн утгатай
* **255** нь тухайн өнгө **хамгийн их гэрэлтэй** гэсэн утгатай

**Red**, **Green**, **Blue** өнгийн өөр өөр түвшинг холих замаар **дэлгэц дээр харагдах бараг бүх өнгийг** үүсгэж болно. Жишээлбэл:

* **Black** → `rgb(0, 0, 0)`
  (гурван өнгөнд аль алинд нь гэрэл байхгүй)

* **White** → `rgb(255, 255, 255)`
  (гурван өнгөнд бүгдэд нь хамгийн их гэрэл)

* **Red** → `rgb(255, 0, 0)`
  (улаан бүрэн, ногоон ба хөх байхгүй)

CSS-д **rgb() function** нь **RGB color model** ашиглан өнгө тодорхойлох боломжийг олгодог. Энэ функц нь **улаан, ногоон, хөх гэрлийн intensity**-г илэрхийлэх **3 утга** авдаг.

**Үндсэн синтакс:**

```css
element {
  color: rgb(red, green, blue);
}
```

**Red**, **Green**, **Blue** утгууд нь **0-оос 255 хүртэл** байж болно.

* **0** → тухайн өнгө байхгүй
* **255** → тухайн өнгө хамгийн их хүчтэй

Доорх нь CSS-д **rgb() function ашиглах жишээ** юм.

<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>

--css--

p {
  color: rgb(255, 0, 0);
}

Энэ код нь **paragraph текстийн өнгийг улаан** болгоно. Учир нь **red утга 255**, харин **green** болон **blue** утгууд **0**-д тохируулагдсан байна.

CSS-д мөн **rgba() function** байдаг. Энэ нь **дөрөв дэх утга — alpha**-г нэмдэг бөгөөд энэ нь өнгөний **тунгалаг байдал (transparency)**-ыг удирддаг.

**Alpha утга** нь **0-ээс 1 хүртэл** байна.

* **0** → бүрэн тунгалаг (completely transparent)
* **1** → бүрэн тод (completely opaque)

Доорх нь **rgba() ашигласан жишээ** юм.

<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>

div {
  background-color: rgba(0, 0, 255, 0.5);
}

Энэ код нь **div элементэд хагас тунгалаг (semi-transparent) хөх background** хэрэглэнэ. Учир нь **red болон green утгууд 0**, **blue утга 255**, харин **alpha утга 0.5** (50% тунгалаг байдал) гэж тохируулагдсан байна.

**RGB color model** нь ялангуяа **дижитал орчинд** маш хэрэгтэй байдаг. Учир нь энэ нь **дэлгэц дээр өнгө хэрхэн харагддагтай шууд холбоотой**. Monitor болон дэлгэцүүд нь **жижиг улаан (red), ногоон (green), хөх (blue) пикселүүдийг** ашиглан өнгийг үүсгэдэг. Эдгээр пикселүүдийн **intensity**-г RGB утгуудаар удирдсанаар та вэб дизайнд **маш олон төрлийн өнгө** гаргах боломжтой.

Мөн **RGB model** нь **dynamic дизайн** дээр ажиллах үед маш тохиромжтой. Жишээлбэл **animation** эсвэл **color effect** хийх үед өнгө **real-time байдлаар өөрчлөгдөх** шаардлагатай болдог. RGB function-ийн уян хатан байдал нь өнгийг **програмчлалын аргаар өөрчлөх** боломжийг олгодог.

**RGB color model** болон CSS дахь **rgb()**, **rgba()** function-уудыг ойлгосноор та вэб хуудсан дээрх **өнгөний харагдах байдлыг илүү нарийн хянах** боломжтой болно. Ингэснээр та **тод өнгөтэй дизайн**, **тунгалаг эффект**, мөн **динамик өнгөний өөрчлөлтүүд** бий болгож чадна.
