**HSL Color Model гэж юу вэ, мөн CSS дахь hsl() function хэрхэн ажилладаг вэ?**

Вэбсайтыг стильдэх үед өнгөтэй ажиллах хамгийн **уян хатан бөгөөд ойлгомжтой аргуудын нэг** нь **HSL color model** ашиглах юм. **HSL** гэдэг нь **Hue (өнгөний төрөл)**, **Saturation (ханалт)**, **Lightness (гэрэлтэл)** гэсэн өнгийг тодорхойлдог **гурван үндсэн бүрэлдэхүүн**-ийг илэрхийлдэг.

**HSL color model** нь өнгийг **хүн хэрхэн хүлээн авч ойлгодогтой илүү ойролцоо байдлаар** дүрслэн харуулдаг. Энэ хэсэгт бид **HSL color model** гэж юу болох болон CSS-д **hsl() function**-ийг хэрхэн ашигладаг талаар авч үзнэ.

**Hue** нь өнгөний төрөл бөгөөд **өнгөний тойрог дээрх өнцгөөр** илэрхийлэгддэг. Энэ нь **0-оос 360 градус** хүртэл утгатай байдаг.
Жишээлбэл:

* **0° → red (улаан)**
* **120° → green (ногоон)**
* **240° → blue (хөх)**

Hue утгыг өөрчилснөөр **өнгөний спектр дэх өөр өөр өнгөнүүд рүү шилжинэ**.

**Saturation** нь өнгөний **хүч буюу цэвэр байдлыг** илэрхийлдэг. Энэ нь **хувиар (%)** хэмжигддэг.

* **0% → бүрэн саарал (grayscale)**
* **100% → хамгийн тод, хүчтэй өнгө**

**100% saturation**-тай өнгө маш тод харагдана, харин **0% saturation**-тай өнгө **саарал** болж харагдана.

**Lightness** нь өнгө **хэр цайвар эсвэл бараан** болохыг илэрхийлдэг. Мөн **хувиар (%)** хэмжигддэг.

* **0% → black (хар)**
* **50% → өнгөний хэвийн өнгө**
* **100% → white (цагаан)**

CSS-д **hsl() function** нь **HSL color model** ашиглан өнгө тодорхойлоход хэрэглэгддэг.

Үндсэн синтакс:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Жишээ:

```css
body {
  background-color: hsla(0, 0%, 1%, 1.00);
}

p {
  color: hsl(120, 100%, 50%);
}
```

Энэ тохиолдолд:

* **Hue = 120° → green**
* **Saturation = 100% → маш тод ногоон**
* **Lightness = 50% → хэвийн өнгө**

Иймээс **paragraph текст тод ногоон өнгөтэй** харагдана.

**HSL color model**-ийн гол давуу тал нь **маш ойлгомжтой бүтэцтэй** байдагт оршино. Saturation болон Lightness утгыг өөрчилснөөр **өнгөний тод байдал болон гэрэлтлийг** амархан өөрчилж болно. Үүний тулд **үндсэн өнгө (hue)-г өөрчлөх шаардлагагүй**.

Жишээлбэл, нэг ижил өнгийн **өөр өөр сүүдэр (shade)** үүсгэхэд Lightness утгыг өөрчлөхөд хангалттай.

```css
div.light {
  background-color: hsl(240, 100%, 80%);
}

div.dark {
  background-color: hsl(240, 100%, 20%);
  color: hsl(0, 0%, 100%);
}
```

Энд хоёр **div** хоёулаа **ижил hue (240° → blue)** ашиглаж байгаа боловч:

* **80% lightness → цайвар хөх**
* **20% lightness → бараан хөх**

RGB model-д **rgba()** transparency ашигладагтай адил **HSL model**-д **hsla() function** байдаг.

Үндсэн синтакс:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Жишээ:

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

Энэ код:

* **Hue = 0° → red**
* **Saturation = 100%**
* **Lightness = 50%**
* **Alpha = 0.5 → 50% тунгалаг**

Ингэснээр **div элемент хагас тунгалаг улаан background**-тай болно.

Alpha утгыг **percentage** хэлбэрээр ч бичиж болно:

```
hsla(0, 100%, 50%, 50%)
```

энэ нь

```
hsla(0, 100%, 50%, 0.5)
```

тай адил утгатай.

**HSL color model** нь **color scheme үүсгэх**, мөн **shade болон tint** өөрчлөхөд маш тохиромжтой.

Hue, Saturation, Lightness-ийг тусад нь удирдах боломжтой учраас **өнгөний үндсэн theme-г хадгалж**, зөвхөн **гэрэлтэл эсвэл сүүдрийг өөрчлөх** боломжтой. Энэ уян хатан байдал нь **дизайнер болон хөгжүүлэгчдийн дунд маш их хэрэглэгддэг** шалтгаан болдог.

Мөн HSL нь **hex code** эсвэл **RGB**-тэй харьцуулахад **илүү ойлгомжтой** байдаг. Өнгөний зорилгыг илүү тодорхой харж, **дизайны туршилт хийхэд** илүү амар байдаг.

**HSL color model**-ийг ашигласнаар та өнгөний **тон**, **ханалт**, **гэрэлтлийг** илүү нарийн удирдах боломжтой болж, CSS-д **баялаг, динамик дизайн** бүтээхэд хүчирхэг хэрэгсэлтэй болно.
