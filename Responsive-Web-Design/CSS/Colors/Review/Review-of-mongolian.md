**CSS Colors тойм**

1. Color Theory

* **Color Theory тодорхойлолт:** Өнгөнүүд хоорондоо хэрхэн **харилцан үйлчилдэг** болон тэдгээр нь **хүний ойлголтод хэрхэн нөлөөлдөг** талаар судалдаг шинжлэх ухаан. Энэ нь **өнгөний харилцаа**, **өнгөний зохицол**, мөн **өнгөний сэтгэлзүйн нөлөө**-г хамардаг.

* **Primary Colors (үндсэн өнгөнүүд):** **Шар (yellow)**, **хөх (blue)**, **улаан (red)**. Эдгээр нь **бусад бүх өнгө үүсэх үндэс болдог** суурь өнгөнүүд.

* **Secondary Colors (хоёрдогч өнгөнүүд):** **Хоёр primary өнгийг тэнцүү хэмжээгээр холиход** үүсдэг. **Ногоон (green)**, **улбар шар (orange)**, **ягаан (purple)** нь жишээ юм.

* **Tertiary Colors (гуравдагч өнгөнүүд):** **Primary өнгө болон түүний хөрш secondary өнгийг** холиход үүсдэг. **Yellow-Green**, **Blue-Green**, **Blue-Violet** зэрэг нь жишээ юм.

* **Warm Colors (дулаан өнгөнүүд):** **Улаан**, **улбар шар**, **шар** зэрэг өнгөнүүд бөгөөд **дулаан, тав тух, тайван мэдрэмж** төрүүлдэг.

* **Cool Colors (сэрүүн өнгөнүүд):** **Хөх**, **ногоон**, **ягаан** зэрэг өнгөнүүд бөгөөд **тайван байдал, амгалан мэдрэмж, мэргэжлийн уур амьсгал** төрүүлдэг.

* **Color Wheel (өнгөний тойрог):** Өнгөнүүдийн **хоорондын харилцааг харуулсан тойрог диаграм**. Дизайнерууд **өнгөний хослол сонгоход** ашигладаг чухал хэрэгсэл.

* **Analogous Color Schemes:** **Өнгөний тойрог дээр бие биенийхээ хажууд байрлах** өнгөнүүдээс бүрдэх схем. **Нэгдмэл, тайван дизайн** үүсгэдэг.

* **Complementary Color Schemes:** **Өнгөний тойрог дээр эсрэг байрлалтай** өнгөнүүдээс бүрдэх схем. **Өндөр контраст** болон **хүчтэй харааны нөлөө** бий болгодог.

* **Triadic Color Scheme:** **Өнгөний тойрог дээр ойролцоогоор ижил зайтай** гурван өнгөөс бүрддэг схем. Эдгээрийг холбовол **тэнцүү талт гурвалжин** үүснэ.

* **Monochromatic Color Scheme:** **Нэг суурь өнгөнөөс** гарсан өнгөнүүд. Энэ нь **гэрэлтэл**, **бараан байдал**, **ханалт**-ыг өөрчилж үүсдэг бөгөөд **нэгдмэл байдал, зохицлыг** мэдрүүлдэг.

2. **CSS-д өнгөтэй ажиллах янз бүрийн аргууд**

* **Named Colors:** Эдгээр нь browser-ууд таньж мэддэг **урьдчилан тодорхойлогдсон өнгөний нэрүүд** юм. Жишээлбэл: **blue**, **darkred**, **lightgreen**.

* **rgb() Function:** **RGB** гэдэг нь **Red (улаан)**, **Green (ногоон)**, **Blue (хөх)** гэсэн **гэрлийн үндсэн өнгөнүүд**-ийг илэрхийлдэг. Эдгээрийг **өөр өөр intensity-ээр хослуулснаар** олон төрлийн өнгө үүсдэг. **rgb() function** нь **RGB color model** ашиглан өнгө тодорхойлох боломжийг олгодог.

```css
p {
  color: rgb(255, 0, 0);
}
```

---

* **rgba() Function:** Энэ функц нь **дөрөв дэх утга болох alpha**-г нэмдэг бөгөөд энэ нь **өнгөний тунгалаг байдал (transparency)**-ыг удирддаг. Хэрэв alpha утга өгөөгүй бол **default утга нь 1** байдаг.

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

---

* **hsl() Function:** **HSL** гэдэг нь **Hue (өнгөний төрөл)**, **Saturation (ханалт)**, **Lightness (гэрэлтэл)** гэсэн **гурван бүрэлдэхүүн**-ийг илэрхийлдэг.

```css
p {
  color: hsl(120, 100%, 50%);
}
```

---

* **hsla() Function:** Энэ функц нь **дөрөв дэх утга болох alpha**-г нэмдэг бөгөөд энэ нь **өнгөний тунгалаг байдал (opacity)**-ыг удирддаг.

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

---

* **Hexadecimal:** **Hex code** (hexadecimal code-ийн товчлол) нь **RGB color model**-д өнгийг илэрхийлэхэд ашиглагддаг **6 тэмдэгтээс бүрдэх код** юм. **Hex** гэдэг нь **16-т суурилсан тооллын систем**-ийг илэрхийлдэг бөгөөд **0–9 цифрүүд болон A–F үсгүүдийг** ашигладаг.

```css
h1 {
  color: #FF5733; /* улаан-улбар шар өнгө */
}

p {
  background-color: #4CAF50; /* ногоон өнгө */
}
```
3. **box-shadow Property**

* **Тодорхойлолт:** **box-shadow** property нь элементийн эргэн тойронд **нэг эсвэл хэд хэдэн сүүдэр (shadow)** нэмэхэд ашиглагддаг.

* **Offset Values:** Та **horizontal (offset-x)** болон **vertical (offset-y)** утгуудыг заавал тодорхойлох хэрэгтэй.

  * **Эерэг offset-x** → сүүдэр **баруун тийш** шилжинэ
  * **Сөрөг offset-x** → сүүдэр **зүүн тийш** шилжинэ
  * **Эерэг offset-y** → сүүдэр **доош** шилжинэ
  * **Сөрөг offset-y** → сүүдэр **дээш** шилжинэ
  * Хэрэв утга **0** бол **unit бичих шаардлагагүй**.

* **Blur Radius:** Энэ нь **заавал биш утга** бөгөөд сүүдэр **хэр бүдэг харагдахыг** тодорхойлдог.

  * Хэрэв заагаагүй бол **default = 0** (хурц ирмэгтэй сүүдэр)
  * **Утга их байх тусам** сүүдэр **илүү зөөлөн** харагдана.

* **Spread Radius:** Энэ нь **заавал биш утга** бөгөөд сүүдэр **хэр хэмжээгээр тэлэх эсвэл багасахыг** тодорхойлдог.

  * Хэрэв заагаагүй бол **default = 0**.

* **Shadow Color:** Сүүдрийн өнгийг дараах аргуудаар тодорхойлж болно:

  * **named colors**
  * **hex values**
  * **rgb()**
  * **rgba()**
  * **hsl()**
  * **hsla()**

* **inset Keyword:** **inset** keyword ашиглавал сүүдэр нь **элементийн гадна биш дотор** байрлана.

* **Олон box-shadow ашиглах:** Олон сүүдэр нэмэх боломжтой бөгөөд тэдгээрийг **таслалаар (comma)** тусгаарлана. Сүүдрүүд **урд талаас хойш** давхарлагдан байрлана.

---

### Syntax

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

---

### Жишээ

```css
.shadow-box {
  width: 200px;
  padding: 20px;
  background-color: lightblue;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

Энэ код нь:

* **баруун тийш 10px**
* **доош 10px**
* **20px blur**
* **хар өнгийн хагас тунгалаг сүүдэр**

үүсгэнэ.

4. **Linear болон Radial Gradient**

* **Linear Gradients:** Эдгээр gradient нь өнгөнүүдийн хооронд **шулуун шугамын дагуу аажмаар шилжилт (gradual blend)** үүсгэдэг. Та энэ шугамын **чиглэлийг** `to top`, `to right`, `to bottom right` зэрэг keyword-ууд эсвэл `45deg` зэрэг **өнцөг ашиглан** удирдаж болно. Мөн **аль ч хүчинтэй CSS өнгө** болон **хэдэн ч color stop** ашиглах боломжтой.

```css id="gts2n2"
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

---

* **Radial Gradients:** Эдгээр gradient нь **төв цэгээс гадагш тархах** хэлбэртэй бөгөөд **дугуй (circle)** эсвэл **эллипс (ellipse)** хэлбэртэй өнгөний шилжилт үүсгэдэг.

```css id="r7o5v9"
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
