# Progressive Enhancement гэж юу вэ?

**Progressive enhancement** гэдэг нь хэрэглэгч ямар браузер эсвэл төхөөрөмж ашиглаж байгаагаас үл хамааран **аппликейшний үндсэн контент болон үндсэн үйл ажиллагаанд хандах боломжтой байхыг хангах дизайн хандлага** юм.

Энэ хандлага нь **хүн бүрт ажиллах үндсэн туршлага (core experience)**-ийг хүргэхэд анхаардаг бөгөөд илүү дэвшилтэт браузер эсвэл илүү сайн интернет холболттой хэрэглэгчдэд **нэмэлт боломжууд болон сайжруулалтууд**-ыг санал болгодог.

**Progressive enhancement** хандлага дараах үндсэн зарчмууд дээр тулгуурладаг:

* Бүх **үндсэн контент болон суурь үйл ажиллагаа** бүх браузер дээр хүртээмжтэй байх ёстой
* Бүх **дэвшилтэт layout**-ууд **гадаад CSS stylesheet**-ээр хангагдах ёстой
* Бүх **дэвшилтэт үйл ажиллагаа** **гадаад JavaScript файлуудаар** хангагдах ёстой
* Хэрэглэгчийн **браузерын тохиргоо (preferences)**-г хүндэтгэх ёстой

**Progressive enhancement** хандлагыг ашиглах нь таны аппликейшнийг **илүү хүртээмжтэй** болгодог. Учир нь дэмжигдээгүй орчинд ч **үндсэн контент болон үндсэн үйл ажиллагаа хаагдах ёсгүй** байдаг.

Хурдны хувьд **progressive enhancement** нь таны аппликейшний **гүйцэтгэлийг сайжруулахад** тусалж чадна.

**Удаан интернет холболттой хэрэглэгчид** ч контентод хандах боломжтой хэвээр байх бөгөөд браузер эхлээд **шаардлагатай үндсэн нөөцүүдийг** татаж авдаг.

**SEO**-ийн хувьд мөн **progressive enhancement** нь таны аппликейшний **харагдах байдлыг (visibility)** сайжруулахад тусалж чадна.

Хайлтын системүүд таны аппликейшний контентыг **crawl хийх боломжтой** байдаг. Учир нь үндсэн контент нь **эхний HTML response** дотор байдаг.

Зарим хүмүүс энэ хандлагыг шүүмжилж, **JavaScript дээр ихээхэн тулгуурладаг аппликейшнүүдэд үргэлж бодитой хэрэгжих боломжгүй** гэж үздэг. Гэсэн хэдий ч аппликейшн хөгжүүлэх үед **дагаж мөрдөхөд тохиромжтой сайн практик** хэвээр байна.
