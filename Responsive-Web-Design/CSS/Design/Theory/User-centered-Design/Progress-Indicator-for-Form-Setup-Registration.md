# Форм, бүртгэл болон тохиргоонд Progress Indication дизайн хийхэд баримтлах шилдэг практикууд юу вэ?

**Тэмдэглэл:** Зарим интерактив жишээнүүдэд таны хараахан сураагүй байж болох CSS болон JavaScript код ашигласан байж болно. Бүх кодыг бүрэн ойлгох гэж оролдох шаардлагагүй. Эдгээр жишээнүүдийн зорилго нь эдгээр дизайн ойлголтууд хэрхэн ажилладгийг илүү сайн ойлгоход тань туслах **урьдчилсан харагдац (preview)** үзүүлэхэд оршино. Жишээнүүдийн preview-г харахын тулд **interactive editor**-ийг идэвхжүүлэх шаардлагатай.

**Progress indication** гэдэг нь хэрэглэгч **процессийн аль хэсэгт явж байгааг** харуулах арга юм. Үүнийг **форм (forms)**, **бүртгэл (registration)** болон **тохиргооны (setup)** процессууд дээр ашиглаж болно. Үүний зорилго нь хэрэглэгчид **процессийн хаана явж байгаагаа** болон **цаашид хэдэн алхам үлдсэнийг** ойлгоход туслах явдал юм.

Жишээлбэл, хэрэглэгч форм бөглөж байх үед **үлдсэн алхмуудыг харуулахын тулд progress indication bar** ашиглаж болно. Хэрэглэгч **урт форм бөглөж байгаа боловч хэдэн алхам үлдсэнийг мэдэхгүй** байх нөхцөл үүсгэхийг хүсэхгүй. **Ил тод байдал (transparency)** маш чухал бөгөөд ингэснээр хэрэглэгч **формыг сууж байгаад бүрэн бөглөх хангалттай цаг байгаа эсэхээ**, эсвэл **дараа нь эргэж ирж үргэлжлүүлэх шаардлагатай эсэхээ** ойлгож чадна.

**Progress indication** хэсгийг дизайн хийхдээ анхаарах хэд хэдэн шилдэг практик байдаг. Эхний анхаарах зүйл бол **энгийн байлгах** явдал юм. Хэрэглэгчийг **хэт их мэдээллээр ачаалж**, улмаар тэд **бухимдаж сайтаас гарах** нөхцөл үүсгэхийг хүсэхгүй.

Хоёр дахь анхаарах зүйл бол **өмнөх алхмууд руу буцах боломжтой** байлгах явдал юм. Энэ нь чухал, учир нь хэрэглэгчид **өмнөх хариултуудаа шалгах** эсвэл **засвар хийхийн тулд буцах** шаардлагатай байж болно.

Өөр нэг анхаарах зүйл бол **progress indication хэсгийг амархан олдохоор** байрлуулах явдал юм. Хэрэв хэрэглэгч үүнийг олж чадахгүй бол тэд **процессийн аль хэсэгт явж байгаагаа мэдэхгүй**.

Сүүлийн анхаарах зүйл бол **тодорхой хэсгийн гарчиг, хувь хэмжээ, эсвэл алхмууд** харуулах явдал юм. Хэрэв зөвхөн **progress bar** байгаад ямар ч тайлбар байхгүй бол хэрэглэгч **түүний утгыг ойлгохгүй**.

Эдгээр нь **форм, бүртгэл болон тохиргооны процессууд дээр progress indication дизайн хийхдээ** анхаарах зарим шилдэг практикууд юм. Томоохон вэбсайтуудын **жишээнүүдийг судалж**, тэд progress indication-ийг **хэрхэн хэрэгжүүлж байгааг** ажиглаарай. Дараа нь өөрийн дизайныг **жинхэнэ хэрэглэгчдээр туршиж**, хэр сайн ажиллаж байгааг нь шалгаарай.
