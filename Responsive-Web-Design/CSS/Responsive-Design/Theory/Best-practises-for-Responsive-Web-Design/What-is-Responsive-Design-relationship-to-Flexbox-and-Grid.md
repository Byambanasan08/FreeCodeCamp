**Responsive Web Design гэж юу вэ, мөн энэ нь CSS Grid болон Flexbox зэрэг хэрэгслүүдтэй ямар холбоотой вэ?**

Responsive Web Design нь вэб хөгжүүлэлтийн нэг аргачлал бөгөөд вэб сайтуудыг desktop компьютерээс эхлээд гар утас хүртэлх олон төрлийн төхөөрөмж дээр **хамгийн тохиромжтой харагдах ба ашиглах туршлага** өгөх зорилготой байдаг.

Responsive дизайны үндсэн зарчим нь **дасан зохицох чадвар (adaptability)** юм. Өөрөөр хэлбэл вэб сайт нь харагдаж буй төхөөрөмжийн **дэлгэцийн хэмжээ болон боломж**-оос хамааран layout болон контентоо өөрчлөн зохицуулах чадвартай байх ёстой.

Responsive дизайн нь ихэвчлэн дараах **гурван үндсэн бүрэлдэхүүн** дээр тулгуурладаг:

* **Fluid grids** нь пиксел зэрэг тогтмол хэмжээсийн оронд **percentage** зэрэг харьцангуй хэмжээс ашигладаг. Ингэснээр контент дэлгэцийн хэмжээнээс хамааран өөрчлөгдөж, дахин байрлана.

* **Flexible images** нь өөрсдийн container дотор хэмжээгээ өөрчилж чаддаг байдлаар тохируулагддаг. Ингэснээр жижиг дэлгэц дээр зураг container-оосоо хальж гарахгүй.

* **Media queries** нь төхөөрөмжийн шинж чанарт үндэслэн өөр CSS style хэрэглэх боломж олгодог. Ихэвчлэн **viewport-ийн өргөн** дээр үндэслэн ашиглагддаг. Media queries-ийн талаар та дараагийн хичээлүүдэд илүү дэлгэрэнгүй сурах болно.

Responsive Web Design болон **CSS Grid** болон **Flexbox** зэрэг хэрэгслүүдийн хоорондын холбоо нь **харилцан дэмжих (symbiotic)** шинжтэй байдаг. Responsive дизайн нь **концепц буюу аргачлал**, харин CSS Grid болон Flexbox нь энэ дизайныг хэрэгжүүлэх **практик хэрэгслүүд** юм.

Өмнөх хичээлүүдэд та **Flexbox** ашиглахыг сурсан. Харин дараагийн хичээлүүдэд **CSS Grid** ашиглахыг сурах болно. Одоогоор CSS Grid-ийн товч танилцуулгыг харцгаая.

**CSS Grid** нь **хоёр хэмжээст (two-dimensional) layout систем** бөгөөд илүү нарийн төвөгтэй layout зохион байгуулалтыг хийх боломж олгодог. Энэ нь бүхэл хуудасны layout болон жижиг компонент layout үүсгэхэд тохиромжтой.

Доорх нь CSS Grid-ийг responsive байдлаар ашиглах жишээ юм:

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

Энэ жишээнд:

* жижиг дэлгэц дээр grid **1 багана** байна
* дунд хэмжээний дэлгэц дээр **2 багана** болно
* том дэлгэц дээр **3 багана** болно

Энэ нь Grid ашиглан layout-ийг дэлгэцийн хэмжээнээс хамааран хэрхэн өөрчилж болдгийг харуулж байна.

Flexbox болон Grid нь **float** болон **table-based layout** зэрэг хуучин layout аргуудаас илүү давуу талтай. Тэд илүү уян хатан, илүү нарийн хяналттай, бага код шаарддаг бөгөөд responsiveness-д зориулагдсан байдаг.

Эдгээр хэрэгслүүд нь хөгжүүлэгчдэд **нарийн төвөгтэй responsive layout**-уудыг харьцангуй хялбар бүтээх боломж олгодог.

Мөн Flexbox болон Grid-ийг ихэвчлэн **хамтад нь** ашигладаг.

* **Flexbox** нь ихэвчлэн **нэг хэмжээст (one-dimensional)** layout буюу component layout-д ашиглагддаг.
* **Grid** нь ихэвчлэн **хоёр хэмжээст (two-dimensional)** layout буюу хуудасны үндсэн бүтэц дээр ашиглагддаг.

Аль хэрэгслийг ашиглах нь тухайн layout-ийн хэрэгцээнээс хамаардаг.

Flexbox болон Grid-ээс гадна CSS-ийн бусад боломжууд responsive дизайн дээр чухал үүрэгтэй.

Жишээлбэл **calc() function** нь өөр өөр хэмжээсүүдийг хольж тооцоолох боломж олгодог бөгөөд уян хатан layout бүтээхэд хэрэгтэй.

Мөн **responsive images** нь responsive web design-ийн чухал хэсэг юм. HTML5 дахь **srcset attribute** болон **picture element** нь төхөөрөмжийн боломж дээр үндэслэн өөр өөр зураг илгээх боломж олгодог. Ингэснээр жижиг дэлгэцтэй эсвэл бага нягтралтай төхөөрөмжүүд шаардлагагүй том хэмжээтэй зураг татахгүй.

Эцэст нь хэлэхэд **Responsive Web Design** нь вэб сайтуудыг ямар ч төхөөрөмж дээр сайн ажиллах боломжтой болгох зорилготой аргачлал юм.

Энэ нь тодорхой нэг технологи биш харин **дизайны философи** бөгөөд media queries зэрэг CSS боломжууд дээр тулгуурладаг. Мөн Flexbox болон Grid зэрэг орчин үеийн layout хэрэгслүүд responsive дизайныг хэрэгжүүлэхэд ихээхэн тусалдаг. Эдгээр хэрэгслүүд нь вэб сайтуудыг олон төрлийн төхөөрөмж дээр **тасралтгүй дасан зохицох боломжтой** болгодог.
