### CSS Box Model гэж юу вэ, хэрхэн ажилладаг вэ?

**CSS box model** нь веб хөгжүүлэлтийн хамгийн суурь ойлголтуудын нэг юм. Энэ нь HTML элементүүд хэрхэн бүтэцлэгдэж, хэрхэн байрладагийг тодорхойлдог. Хэрэв та энэ загварыг ойлговол веб сайтынхаа элементүүдийн **хэмжээ, зай, харагдах байдлыг** хянах боломжтой болно.

CSS box model-д **бүх элементүүд хайрцаг (box)**-аар хүрээлэгдсэн байдаг. Энэ хайрцаг нь дөрвөн хэсгээс бүрдэнэ: **content area**, **padding**, **border**, болон **margin**.

---

### Content Area

**Content area** нь хайрцгийн хамгийн дотоод хэсэг юм. Энэ нь тухайн элементийн бодит агуулга байрлах хэсэг бөгөөд жишээлбэл **текст эсвэл зураг** багтдаг.

---

### Padding

**Padding** нь content area-ийн дараах хэсэг юм. Энэ нь **content area болон элементийн border хоорондох зай** юм. Padding ашигласнаар агуулгын эргэн тойронд зай нэмэж, уншихад илүү ойлгомжтой болгож болно. `padding` property ашиглан **дээд, баруун, доод, зүүн** талуудад өөр өөр утга өгч болно.

Доорх нь `padding` shorthand property-ийн жишээ бөгөөд:

* дээд padding – **15px**
* баруун padding – **5px**
* доод padding – **2px**
* зүүн padding – **8px**

байхаар тохируулсан байна.

```css
padding: 15px 5px 2px 8px;
```

---

### Border

**Border** нь CSS box model дахь элементийн **гадаад хүрээ** буюу **контур** юм. Энэ нь элементийн харагдах хил хязгаарыг илэрхийлдэг. `border` property ашиглан хүрээний **style, width, color** болон бусад шинж чанаруудыг өөрчлөх боломжтой.

Доорх жишээнд border-ийн:

* өргөн — **5px**
* style — **solid**
* өнгө — **blue**

байхаар тохируулсан байна.

```css
border: 5px solid blue;
```

Хэрэв та зарим утгыг орхивол тухайн property-ийн **анхдагч утга (default value)** ашиглагдана. Үүнд:

* width — **medium**
* style — **none**
* color — **current color**

байна.

Хэрэв бүх талд ижил утга өгөхийг хүсвэл shorthand `border` property ашиглаж болно. Харин тал бүрт өөр утга өгөхийг хүсвэл `border-width`, `border-style`, болон `border-color` property-уудыг ашиглана.

```css
border-width: 2px 4px 7px 12px;
border-style: dashed solid solid dashed;
border-color: blue red green black;
```

Эдгээр property бүрт **дөрвөн хүртэл утга** бичиж болно. Эдгээр утгууд нь **дээд талаас эхлэн цагийн зүүний чиглэлээр** дарааллаар хэрэгжинэ. Хэрэв зөвхөн нэг утга бичвэл тэр утга **дөрвөн талд бүгдэд нь** хэрэглэгдэнэ.

---

### Margin

**Margin** нь элементийн border-ийн **гаднах зай** юм. Энэ нь тухайн элемент болон түүний эргэн тойрон дахь бусад элементүүдийн хоорондын зайг тодорхойлдог. `margin` property ашиглан **дээд, баруун, доод, зүүн** талуудад өөр өөр утга өгч болно.

Доорх жишээнд:

* дээд margin — **3px**
* баруун margin — **12px**
* доод margin — **9px**
* зүүн margin — **7px**

байхаар тохируулсан байна.

```css
margin: 3px 12px 9px 7px;
```

---

Эдгээр дөрвөн бүрэлдэхүүн хэсэг нь элементийн **нийт өргөн болон өндрийг** тооцоолоход маш чухал үүрэгтэй.

Дараагийн хэдэн хичээлд браузер энэ процессыг хэрхэн удирддаг болон та үүнийг хэрхэн өөрчлөн тохируулах боломжтой талаар илүү дэлгэрэнгүй үзэх болно. **CSS box model** нь веб хөгжүүлэлтийн үндсэн ойлголт юм.

Эдгээр бүрэлдэхүүн хэсгүүд хоорондоо хэрхэн харилцан ажиллаж, элементийн хэмжээнд хэрхэн нөлөөлдгийг ойлгох нь веб дизайныг хэрэгжүүлэхэд маш чухал юм.
