1. Өнөө үед түгээмэл ашиглагддаг вэб хөтчүүд (web browser) юу вэ, мөн тэдгээрийг хэрхэн суулгах вэ?

Вэб хөтчүүд (web browsers) гэдэг нь интернет дээрх вэб хуудсуудыг үзэхэд ашигладаг програмууд юм. Хэрэв та freeCodeCamp-ийн сургалтын хөтөлбөрийг үзэж байгаа бол, яг одоо **вэб хөтчөөр** дамжуулан ашиглаж байх магадлал маш өндөр.

**2024 оны байдлаар** хамгийн өргөн хэрэглэгддэг вэб хөтчүүд нь:

* Microsoft Edge
* Firefox
* Google Chrome
* Safari

Өөр олон төрлийн хөтчүүд байдаг бөгөөд зах зээлийн хувь хэмжээ (market share) байнга өөрчлөгдөж байдаг ч, одоогоор эдгээр **дөрөв** нь хамгийн түгээмэл хэрэглэгдэж байна.

---

### Анхнаасаа суусан (default) вэб хөтчүүд

Таны ашиглаж буй **үйлдлийн систем (operating system)** ихэнх тохиолдолд анхнаасаа нэг вэб хөтчийг суулгасан байдаг.

* **Windows** → Microsoft Edge
* **macOS** → Safari
* **Linux** → ихэвчлэн Firefox

---

### Өөр вэб хөтч ашиглахыг хүсвэл яах вэ?

Хэрэв та анхдагч (default) хөтчөөс өөр вэб хөтч ашиглахыг хүсвэл:

* Та аль хэдийн нэг вэб хөтчтэй байгаа тул
* Тэр хөтчөөрөө дамжуулан
* өөрийн хүссэн хөтчийн **албан ёсны вэб сайт** руу орж
* өөрийн үйлдлийн системд тохирсон **installer (суулгагч файл)**-ийг татаж авч болно

---

### Вэб хөтч сонгохдоо юуг анхаарах вэ?

Вэб хөтч сонгохдоо дараах хүчин зүйлсийг харгалзан үзэх нь зүйтэй:

* **Хурд (speed)**
* **Аюулгүй байдлын боломжууд (security features)**
* **Таны байнга ашигладаг вэб сайтуудтай нийцтэй байдал (compatibility)**

Зарим вэб хөтчүүд онцлог нэмэлт боломжуудтай байдаг. Жишээ нь:

* **Opera** – дотроо суурилсан VPN-тэй
* **Brave** – нууцлал (privacy) болон сурталчилгаа хаах (ad-blocking)-д анхаардаг

---

### Вэб хөтч суулгах процесс

Вэб хөтч суулгах нь ихэнхдээ маш энгийн байдаг.

* Installer файлыг татаж авна
* Файлыг ажиллуулна
* Дэлгэц дээр гарч ирэх зааврыг дагана

Ихэнх вэб хөтчүүд:

* одоо ашиглаж байгаа хөтчөөс тань
* **bookmark**, **тохиргоо (settings)** зэргийг
* автоматаар импортлох санал болгодог

Ингэснээр шинэ хөтч рүү шилжихэд илүү хялбар болдог.

---

### Package manager ашиглан суулгах

Мөн вэб хөтчийг **үйлдлийн системийн package manager**-аар дамжуулан суулгаж болно.

Жишээ нь:

* **macOS (Homebrew ашиглан)**

  ```
  brew install --cask google-chrome
  ```
* **Arch Linux**

  ```
  yay -S brave-bin
  ```

Анхаарах зүйл:

* Ашиглах боломжтой package manager
* болон package-ийн нэршил
  нь таны ашиглаж буй үйлдлийн системээс шалтгаалан өөр өөр байна.

---

### Нэгээс олон вэб хөтч ашиглах боломж

Та заавал **нэг л вэб хөтч** ашиглах албагүй.

Олон хүмүүс:

* нэгийг нь **ажлын зориулалтаар**
* нөгөөг нь **хувийн хэрэглээнд**
  ашигладаг.

Тиймээс:

* өөр өөр вэб хөтчүүдийг туршиж үзээд
* өөрт тань хамгийн тохиромжтойг сонгоорой.

---

Эргэлзэх зүйлгүй, янз бүрийн вэб хөтчийг туршиж үзэх нь таны ажлын үр ашиг, тав тухыг нэмэгдүүлэхэд тусална.
