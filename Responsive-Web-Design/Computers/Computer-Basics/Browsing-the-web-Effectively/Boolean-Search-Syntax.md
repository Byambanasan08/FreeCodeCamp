### Хайлтын системийг хэрхэн үр дүнтэй ашиглаж, хамгийн оновчтой үр дүнд хүрэх вэ?

Интернетээс мэдээлэл хайх нь заримдаа нэлээд **төвөгтэй санагдаж** магадгүй. Учир нь маш олон вэб сайт, асар их өгөгдөл байдаг тул яг хэрэгтэй мэдээллээ олоход амаргүй мэт харагддаг.

Гэхдээ **boolean search syntax** зэрэг аргуудын ачаар та хайлтаа нарийсгаж, **яг хэрэгтэй зүйлээ** олж чадна.

---

## Boolean search syntax гэж юу вэ?

**Boolean search syntax** гэдэг нь интернетээр хайх үедээ тусгай үг, тэмдэг ашиглан хайлтын үр дүнг илүү нарийвчлах арга юм. Энэ нь таны хайж буй мэдээлэлд илүү тохирох үр дүнг гаргахад тусалдаг.

---

## Энгийн хайлтын жишээ

Жишээ нь, та дараах хайлтыг хийе гэж бодъё:

```
freecodecamp curriculum
```

Ийм хайлт хийхэд:

* хайлтын систем нь **freecodecamp** эсвэл **curriculum** гэсэн үгсийн **аль нэгийг нь** агуулсан үр дүнг буцаана
* ихэвчлэн үр дүнг **хамааралтай байдлаар нь** (өөрөөр хэлбэл таны асуулгатай хэр ойр байгаагаар нь) эрэмбэлж харуулна

---

## Яг тухайн хэллэгийг бүхэлд нь тааруулах

Хэрвээ та **хоёр үг хамтдаа яг тэр хэлбэрээрээ** орсон үр дүнг л харахыг хүсвэл, асуулгаа **хашилтанд** оруулж болно:

```
"freecodecamp curriculum"
```

Гэхдээ анхаарах зүйл нь:

* энэ хайлт нь **freecodecamp curriculum** гэсэн **яг тэр хэллэгийг бүхэлд нь** агуулсан сайтуудыг л харуулна
* **freecodecamp** болон **curriculum** үгс тусдаа, өөр өөр газарт орсон сайтуудыг харуулахгүй

---

## Хоёр үг заавал орсон байхыг шаардах

Хэрвээ:

* тухайн хоёр үг **заавал хоёулаа** орсон байвал хангалттай
* гэхдээ **заавал залгаатай байх шаардлагагүй** бол

та үг бүрийн өмнө **plus (+)** тэмдэг ашиглаж болно:

```
+freecodecamp +curriculum
```

Энэ хайлт нь:

* freecodecamp
* curriculum
  гэсэн **хоёр үгийг хоёуланг нь** агуулсан сайтуудыг буцаана

---

## Зарим үгийг зориуд хасах

Заримдаа та:

* freeCodeCamp-ийн талаар хайж байгаа ч
* curriculum-тай холбоотой биш мэдээлэл сонирхож байж болно

Ийм тохиолдолд **minus (-)** тэмдэг ашиглана:

```
+freecodecamp -curriculum
```

Энэ хайлт нь:

* **freecodecamp** гэдэг үгийг агуулсан
* харин **curriculum** гэдэг үгийг **агуулаагүй**
  үр дүнг харуулна

---

## Тодорхой вэб сайтаас хайх

Хэрвээ та:

* зөвхөн **нэг тодорхой вэб сайт** дотроос хайхыг хүсвэл
  **site:** операторыг ашиглаж болно

Жишээ нь, freeCodeCamp-ийн news платформоос curriculum гэсэн үгтэй нийтлэл хайх бол:

```
site:freecodecamp.org/news curriculum
```

Энэ нь:

* **freecodecamp.org/news** домэйн дотор
* **curriculum** гэдэг үг орсон
  бүх үр дүнг буцаана

---

## Операторуудыг хослуулж ашиглах

Эдгээр аргуудыг хооронд нь **хослуулснаар** та маш нарийн хайлт хийж чадна.

Жишээ нь:

```
site:freecodecamp.org/news +python -curriculum
```

Энэ хайлт нь:

* freecodecamp.org/news сайт доторх
* **Python**-той холбоотой
* харин **curriculum**-ын талаар яриагүй
  нийтлэлүүдийг олж өгнө

---

## Анхаарах зүйл

* Хайлтын операторууд:

  * хайлтын систем бүр дээр
  * бага зэрэг **ялгаатай** байж болдог
* Тиймээс:

  * өөрийн ашиглаж буй хайлтын системийн
  * **албан ёсны баримт бичгийг (documentation)** шалгаж байх нь чухал

---

## Дүгнэлт

Эдгээр аргуудыг мэддэг болсноор та:

* вэб хайлтаа илүү **ухаалгаар**
* илүү **нарийвчлалтай**
* яг хэрэгтэй мэдээллээ **хурдан** олох боломжтой болно

Одоо та вэбээр хайхдаа хүссэн үр дүндээ хүрэхэд илүү **чадвартай** боллоо.
