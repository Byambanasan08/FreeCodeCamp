Компьютерийн суурь мэдлэг – Давтлага

1. Компьютер, интернет болон хөгжүүлэгчийн багаж хэрэгслийн суурь ойлголтууд**

---

**Motherboard (Эх хавтан):**
Компьютерийг ажиллуулахад шаардлагатай бүх **санах ой, холбогч портууд, хатуу дискүүд**-ийг агуулж, хооронд нь холбож өгдөг үндсэн электрон самбар юм. Өөрөөр хэлбэл, компьютерийн **гол хэлхээт самбар** болно.

**Central Processing Unit (CPU – Төв процессор):**
Заавруудыг гүйцэтгэж, тооцоолол хийх үүрэгтэй процессор юм. Компьютерийн “тархи” гэж ойлгож болно.

**Random Access Memory (RAM – Шууд хандалттай санах ой):**
CPU-д зориулсан **түр хадгалалтын санах ой**. Компьютер хурдан хандах шаардлагатай өгөгдлөө энд хадгалдаг.

**Hard Disk Drive (HDD – Хатуу диск):**
Компьютер унтарсан үед ч өгөгдлийг хадгалж үлддэг **байнгын хадгалалтын төхөөрөмж** юм.

**Solid State Drive (SSD – Хатуу төлөвт диск):**
**Дэгдэмхий биш (non-volatile)** флаш санах ой дээр суурилсан хадгалалтын төхөөрөмж бөгөөд HDD-ийн оронд ашиглаж болдог. Илүү хурдан, чимээгүй ажиллагаатай.

**Power Supply Unit (PSU – Цахилгаан тэжээлийн блок):**
Гэрийн залгуураас орж ирсэн цахилгааныг компьютер ашиглаж болох хэлбэрт **хувиргаж**, бүх эд ангиудад тараадаг төхөөрөмж.

**Graphics Processing Unit (GPU – График процессор):**
Дэлгэц дээр харагдах **дүрслэл, график, видео**-г боловсруулах үүрэгтэй процессор.

2. Интернет үйлчилгээ үзүүлэгчдийн төрөл

**Internet Service Provider (ISP):**
Интернетэд нэвтрэх боломж олгодог компани юм. ISP-ийн түгээмэл төрлүүдэд:

* Dial-up
* DSL
* Cable
* Fiber-optic
* Satellite
  багтана.

---

3. Компьютертээ аюулгүй нэвтрэх аргууд

Компьютертээ аюулгүй нэвтрэх жишээ аргуудад:

* **Хүчтэй нууц үг** ашиглах
* **Two-factor authentication (2FA)** идэвхжүүлэх
* **Password manager** ашиглах
  зэрэг орно.

---

4. Хөгжүүлэгчдийн ашигладаг багаж хэрэгслүүд

4.1 Integrated Development Environment (IDE):
Хөгжүүлэгчдэд код бичих, тест хийх, алдаа засах (debug) ажлыг **үр ашигтай** хийхэд тусалдаг цогц хэрэгсэл.

4.2 Code Editor (Код засварлагч):
Хөгжүүлэгчид код бичих, засварлах, алдаа шалгахад ашигладаг хэрэгсэл.

4.3 Git:
Хөгжүүлэгчдэд кодын өөрчлөлтийг хянах, түүхийг хадгалах, бусадтай хамтран ажиллах боломж олгодог **өргөн хэрэглэгддэг version control system**.

**Cloud-based Hosting Services for Repositories:**
**Repository** гэдэг нь төслийн файлууд болон хувилбарын түүхийг хадгалах сан юм.
Түгээмэл cloud суурьтай repository үйлчилгээ үзүүлэгчид:

* GitHub
* GitLab
* Bitbucket

4.4 Package Managers (Багцын менежерүүд):
Сангууд (libraries) болон төслийн хамаарлуудыг (dependencies) нэмэх, шинэчлэх, устгах үйл явцыг хялбарчилдаг хэрэгслүүд.
Жишээ нь:

* npm
* pip
* Maven

4.5 Testing Libraries and Frameworks (Тестийн сангууд ба framework-үүд):**
Программ хангамжид тест хийх нь код **зөв ажиллаж байгаа эсэхийг** шалгах зорилготой.
Жишээ тестийн сан, framework-үүд:

* Jest
* PHPUnit
* JUnit

---

Эдгээр ойлголтууд нь компьютерийн суурь мэдлэг, интернетийн хэрэглээ, мөн программ хангамж хөгжүүлэлтийн үндсийг ойлгоход чухал ач холбогдолтой.


2. Файлууд, файл систем болон медиа форматтай ажиллах

---

**Файлын нэршлийн шилдэг дадлууд (Best practices for naming files):**
Файлуудаа ойлгомжтой, арчлахад хялбар байдлаар нэрлэх хэрэгтэй. Жишээлбэл, **about-us.html** гэдэг нь **page1.html**-ээс илүү тодорхой, утгатай нэршил юм.

**Root directory (Үндсэн буюу дээд түвшний хавтас):**
Файл систем дэх хамгийн дээд түвшний хавтас юм. **Directory** гэдэг нь **folder**-ийн өөр нэршил болно.

**Markdown:**
Баримтжуулалт (documentation) болон **README** файлуудад түгээмэл ашиглагддаг markup хэл юм. **README** файл нь тухайн төслийн талаархи мэдээлэл, жишээлбэл хэрхэн суулгах, хэрхэн ашиглах тухай тайлбарыг агуулдаг файл юм.

**index.html:**
Хэрэглэгч вэб сайт руу ороход анхдагчаар (default) харагддаг хуудсыг илэрхийлдэг файл юм.

**Explorer / Finder ашиглан файл, хавтас үүсгэх, зөөх, устгах:**
**Explorer** нь Windows үйлдлийн системийн файл менежер, **Finder** нь macOS-ийн файл менежер юм. Эдгээр хэрэгслүүдийг ашиглан файл, хавтас үүсгэх, зөөх, устгах боломжтой.

**Файл болон хавтас хайх:**
Explorer эсвэл Finder доторх **search** (хайлтын) боломжийг ашиглан компьютер дээрх файл, хавтсуудыг хайж олж болно.

**HTML, CSS, JS файлын төрлүүд:**

* **.html** өргөтгөл — HTML файл
* **.css** өргөтгөл — CSS файл
* **.js** өргөтгөл — JavaScript файл

**Түгээмэл зураг болон график файлын форматууд:**

* **JPEG**, **PNG** — өргөн хэрэглэгддэг зургийн форматууд
* **GIF** — хөдөлгөөнт (animation) зургийг дэмждэг формат
* **SVG** — вектор графикийн файл формат

**Түгээмэл аудио болон видео форматууд:**

* **MP3** — аудио файлуудад түгээмэл ашиглагддаг формат
* **MP4** — видео файлуудад өргөн хэрэглэгддэг формат
* **MOV** — Apple-ийн хөгжүүлсэн, видео файлуудад түгээмэл хэрэглэгддэг формат

**Түгээмэл фонтын форматууд:**

* **TTF** — TrueType фонтын түгээмэл формат
* **WOFF** — вэб фонтод зориулсан түгээмэл формат
* **WOFF2** — WOFF-ийн дараагийн хувилбар бөгөөд илүү сайн шахалт (compression) өгдөг

**ZIP:**
Файл болон хавтсуудыг **шахах (compress)** зорилгоор ашигладаг файл формат юм.
3. ### Вэбийг үр дүнтэй ашиглан хөтлөх (Browsing the Web Effectively)

---

3. Вэб хөтөч (Web browser) гэж юу вэ?:**
Интернет дээрх вэб сайтуудад нэвтрэх, тэдгээрийг үзэх боломж олгодог **програм хангамж** юм.

**Хайлтын систем (Search Engine) гэж юу вэ?:**
Интернет дээрх мэдээллийг хайж олоход тусалдаг **хэрэгсэл** юм. Жишээлбэл:

* Google
* Bing
* Yahoo

**Түгээмэл вэб хөтчүүд (Common web browsers):**
Өргөн хэрэглэгддэг вэб хөтчүүдийн жишээнд:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
  орж байна.

---
4. Түгээмэл хайлтын стратегиуд (Common Search Strategies)

* **site:**
  Вэб сайтын URL-ийн араас **site:** гэж бичвэл тухайн вэб сайт доторх контентыг хайна.

* **filetype:**
  Файлын өргөтгөлийн өмнө **filetype:** гэж бичвэл тухайн төрлийн файлуудыг хайна.

* **minus (-) тэмдэг:**
  Хайлтын үгийн өмнө **-** тавибал тухайн үгийг агуулсан үр дүнг **хасна**.

* **plus (+) тэмдэг:**
  Хайлтын үгийн өмнө **+** тавибал тухайн үгийг заавал агуулсан үр дүнг **оруулна**.
