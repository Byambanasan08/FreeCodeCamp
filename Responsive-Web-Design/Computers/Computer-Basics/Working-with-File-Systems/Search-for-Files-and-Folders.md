### Компьютер дээрээ файл, хавтсыг хэрхэн хайх вэ?

Одоо **Windows** болон **macOS** дээр файл, хавтас хэрхэн хайхыг сурцгаая.

---

## Windows дээр файл, хавтас хайх

Windows дээр хайлт хийх **хоёр үндсэн арга** байдаг. Үүнд: **Taskbar**-аас хайх, мөн **File Explorer**-оос хайх.

### 1. Taskbar ашиглан хайх

* Taskbar дээрх хайлтын хэсэгт хайж буй **файл эсвэл хавтасны нэрийг** бичнэ
* Мөн **түлхүүр үг (keyword)**-ээр хайж болно

Ингэснээр:

* таны **PC**
* мөн **OneDrive** дээрх файлууд
  зэрэг бүх эх сурвалжаас үр дүн гарч ирнэ.

Хэрвээ:

* зөвхөн **баримтууд (documents)**-ыг харахыг хүсвэл
  → **“Documents”** tab дээр дарна

Мөн:

* **apps**
* **web results**
  гэсэн тусдаа tab-ууд ч бий.

---

### 2. File Explorer ашиглан хайх

Өөр нэг арга нь **File Explorer-ийн хайлтын хэсэг**-ийг ашиглах явдал юм.

* Explorer цонхны **баруун дээд хэсэгт**
* **“Search Documents”** гэсэн бичигтэй хайлтын талбар байдаг

Энд хайлтын **хамрах хүрээ (scope)**-г тодорхойлох **гурван сонголт** бий:

1. **Home**-оос хайх

   * PC болон cloud дээрх файлуудыг хурдан хайна

2. **Нэг фолдероос хайх**

   * тухайн фолдер дотор хадгалагдсан файлуудыг хайна

3. **This PC**-ээс хайх

   * илүү **удаан** боловч
   * **гүнзгий, нарийвчилсан** хайлт хийдэг

Ингэж Windows дээр файл, хавтсыг хайж болдог.

---

## macOS дээр файл, хавтас хайх

Одоо macOS дээр хэрхэн хайхыг үзье.

### 1. Spotlight ашиглах

macOS дээр файл хайх хамгийн түгээмэл хэрэгсэл бол **Spotlight** юм.

Spotlight-ийг нээхийн тулд:

* дэлгэцийн **баруун дээд хэсэгт** байрлах
* **томруулдаг шил (magnifying glass)** icon дээр дарна

Хэрвээ:

* энэ icon menu bar дээр харагдахгүй байвал
  → **Control Center settings**-ээс нэмж болно

Нэмэхийн тулд:

* **Settings** рүү орно
* хайлтаар **“Control Center”** гэж бичнэ
* уг сонголтыг дарна
* доош scroll хийж **“Spotlight”**-ийг олно

Мөн Spotlight-ийг нээх **keyboard shortcut** бий:

* **Command + Spacebar**

---

### Spotlight Search ашиглах

Spotlight нээгдэхэд:

* **Spotlight Search** цонх гарч ирнэ
* энд файл, хавтасны **нэрийг бичиж** хайна

Бичиж эхлэх үед:

* үр дүнг **төрлөөр нь ангилж**

  * suggestions
  * folders
  * presentations
  * photos
  * documents
  * related searches
    гэх мэтээр харуулна

Хэрвээ:

* аль нэг ангиллын илүү олон үр дүнг харахыг хүсвэл
  → **“Show More”** дээр дарж болно

---

### 2. Саяхан нээсэн файлуудыг Finder-оос харах

Саяхан нээсэн файлуудаа хайх өөр нэг арга бол:

* Dock дээрх **Finder icon** дээр дарж Finder-ийг нээнэ
* Finder-ийн sidebar-аас **“Recents”**-ийг сонгоно

Энд:

* таны **саяхан үзсэн бүх файлууд** жагсааж харагдана

---

### Дүгнэлт

Windows болон macOS дээрх:

* хайлтын хэрэгслүүдийг зөв ашигласнаар
* ажлын урсгалаа (workflow) илүү үр ашигтай болгож
* хэрэгтэй файл, хавтсыг **маш хурдан** олох боломжтой болно

Эдгээр хайлтын аргуудыг сайн эзэмшвэл таны өдөр тутмын компьютерийн хэрэглээ илүү хялбар, хурдан болно.
