### Компьютер дээрээ файл удирдах (File Management) програмуудыг хэрхэн ашиглах вэ?

**File management application** гэдэг нь компьютер дээрх файлуудаа **хадгалах, зохион байгуулах, засварлах** ажлыг хялбар болгох програм юм. Та Mac эсвэл Windows компьютер ашиглаж байгаагаас үл хамааран файлуудаа зохион байгуулахад зориулсан **суурилагдсан (built-in) програмууд** байдаг.

---

### Windows компьютер дээр

Хэрэв та Windows компьютер ашиглаж байгаа бол анхдагч файл удирдлагын програм нь **File Explorer** юм. File Explorer нь:

* файл, хавтаснуудыг **үзэх (browse)**
* **хайх (search)**
* **удирдах (manage)**
  боломж олгохоос гадна:
* хуулах (copy)
* зөөх (move)
* устгах (delete)
  зэрэг файлтай холбоотой үйлдлүүдийг хийхэд ашиглагддаг.

File Explorer-ийг нээхийн тулд:

* **Start menu** рүү орж нээх
* эсвэл keyboard дээрх **Windows logo товч**-ийг дарах
  боломжтой.

Хэрэв та хавтсыг **pin** хийхийг хүсвэл:

* mouse-оор **right click** хийнэ
* **“Pin”** гэсэн сонголтыг дарна

Файл эсвэл хавтсыг зөөхийн тулд:

1. тухайн файлыг эсвэл хавтсыг сонгоно
2. **“Cut”** командыг сонгоно
3. шинэ байршил руу очоод **paste** хийнэ

---

### Mac компьютер дээр

Хэрэв та Mac компьютер ашиглаж байгаа бол анхдагч файл удирдлагын програм нь **Finder** юм. Finder нь:

* локал болон алсын файлуудад (local & remote files) хандах
* файлын **урьдчилсан харагдац (preview)** үзүүлэх
* **tags**, **Smart Folders** зэрэг олон төрлийн зохион байгуулалтын боломжуудыг санал болгодог

Mac дээр Finder-ийг нээхийн тулд:

* дэлгэцийн доод хэсэгт байрлах **Dock** руу орж
* **Finder icon** дээр дарна

Finder-ийг нээхэд:

* зүүн талд байрлах **sidebar** дээр
* таны байнга ашигладаг (favorite) хавтаснууд харагдана

---

### Tags ашиглан файл зохион байгуулах

Файлуудаа зохион байгуулах нэг арга бол **tags** ашиглах юм.

Файл эсвэл хавтсанд tag өгөхийн тулд:

1. Finder дээр тухайн файлыг эсвэл хавтсыг сонгоно
2. mouse-оор **right click** хийнэ
3. гарч ирэх жагсаалтаас **“Tags…”** сонголтыг дарна
4. өөрийн хүссэн **өнгө бүхий tag**-ийг онооно

Tag-тай файлуудыг хайхын тулд:

* sidebar-ийн доод хэсэг рүү гүйлгээд тухайн **өнгөний tag** дээр дарна
* эсвэл хайлтын талбарт **өнгөний нэрээр** tag хайж болно

---

### Smart Folders ашиглах

Файлуудаа зохион байгуулах өөр нэг арга бол **Smart Folders** ашиглах явдал юм.

Smart Folder гэдэг нь:

* таны тогтоосон шалгуур (criteria)-ын дагуу
* файлуудыг **автоматаар цуглуулж, ангилдаг**
  хавтас юм.

Жишээ шалгуур:

* файлын төрөл (file type)
* үүсгэсэн огноо (date)
* түлхүүр үг (keywords)

---

### Дүгнэлт

Та Windows дээр **File Explorer**, Mac дээр **Finder** ашиглаж байгаагаас үл хамааран эдгээр файл удирдлагын програмууд нь:

* файл хайх
* tag ашиглах
* Smart Folder үүсгэх
  зэрэг боломжуудаар дамжуулан файлуудаа **уян хатан, үр дүнтэйгээр** удирдах боломж олгодог.
