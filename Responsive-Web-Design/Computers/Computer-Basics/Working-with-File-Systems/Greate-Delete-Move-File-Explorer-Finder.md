### Explorer / Finder ашиглан файл, хавтас үүсгэх, зөөх, устгах аргууд

#### Windows (File Explorer) дээр

Windows дээр **File Explorer** ашиглан файл үүсгэхийн тулд баруун дээд хэсэгт байрлах **“New”** сонголт дээр дарна. Ингэснээр та үүсгэж болох файлын төрлүүдийн жагсаалтыг харах болно.

Мөн файлуудыг тухайн файлыг үүсгэдэг **программуудаар** дамжуулан үүсгэж болно. Одоо жишээ болгож **text file** үүсгэх процессыг харцгаая.

“New” дээр дараад файлын төрлийг сонговол:

* тухайн фолдер дотор
* анхдагч нэртэй шинэ текст файл үүснэ
  Та хүсвэл нэрийг нь өөрчилж болно.

**Хавтас (folder) үүсгэх** процесс нь яг адилхан:

* Баруун дээд хэсгийн **“New”** товч дээр дарж **“Folder”**-ийг сонгоно
  эсвэл
* хоосон зай дээр **right-click** хийгээд **New → Folder**-ийг сонгоно

Ингэснээр одоогийн байршилд хоосон шинэ хавтас үүснэ.

---

#### Файл, хавтасны нэр солих (Rename)

Файл эсвэл хавтсыг сонгоод:

* дээд хэсгийн **“Rename”** товч дээр дарна
  Нэр нь тодорч харагдах үед:
* шинэ нэрийг бичээд
* **Enter** дарж баталгаажуулна

Өөр нэг арга нь:

* файл эсвэл хавтас дээр **right-click** хийгээд **“Rename”**-ийг сонгох
* эсвэл **F2** товчийг ашиглах

Дараа нь шинэ нэрийг бичээд **Enter** дарна.

---

#### Файл, хавтас зөөх (Move)

Файл эсвэл хавтсыг:

* зорьж буй хавтас руу **drag & drop** хийж зөөж болно
  (хэрвээ тухайн хавтас tab байдлаар нээгдсэн бол)

Мөн дараах аргыг ашиглаж болно:

* файл эсвэл хавтсыг сонгоод **“Cut”** товч дээр дарна
* дараа нь очих хавтсаа нээгээд **“Paste”** дээр дарна

Хэрвээ өмнө нь файл cut эсвэл copy хийсэн бол Paste товч идэвхтэй харагдана.

Мөн:

* хоосон зай дээр **right-click** хийгээд **Paste** дарж болно

Ингэснээр файл эсвэл хавтас анхны байршлаасаа устгагдаж, шинэ байршилд шилжинэ.

---

#### Файл, хавтас устгах (Delete)

Файл эсвэл хавтсыг устгахын тулд:

* сонгоод дээд хэсгийн **Trash icon** дээр дарна
  эсвэл
* **right-click** хийгээд **Delete**-ийг сонгоно

---

### macOS (Finder) дээр

Одоо macOS дээр **Finder** ашиглан үүнийг хэрхэн хийхийг харцгаая.

**Finder** нь macOS-ийн үндсэн file manager юм.

---

#### Файл үүсгэх (macOS)

macOS дээр:

* Finder-оос шууд файл үүсгэх **суурь боломж байхгүй**
* Файл үүсгэхийн тулд тухайн файлыг үүсгэж чаддаг **аппликейшн** нээх хэрэгтэй

Жишээ нь:

* энгийн текст файл үүсгэхийн тулд **TextEdit**-ийг нээнэ
* дараа нь файлаа хадгална

Энэ процесс нь бусад төрлийн файлуудад ч адилхан.

TextEdit-ийг нээх олон арга бий. Жишээлбэл:

* **Launchpad**-д орно
* хайлтаар **TextEdit** гэж бичээд
* аппликейшний icon дээр дарж нээнэ

Аппликейшн дотор орсны дараа:

* дээд хэсэгт байрлах **App menu**-гээс файлыг хадгалах сонголтыг олно
  Энэ menu нь:
* Apple icon-ын хажууд, дээд талд байрладаг

Ихэнхдээ:

* **File** цэс дотор **Save** команд байдаг

Түүний хажууд:

* файлыг хадгалах **keyboard shortcut** харагдана
  macOS дээр энэ нь ихэвчлэн **Command + S** байдаг

Анх удаа хадгалах үед:

* dialog window нээгдэж
* файлын нэр бичих
* хадгалах байршлыг сонгох боломж олдоно

Ихэнх тохиолдолд:

* файлын өргөтгөл (file extension)-ийг аппликейшн автоматаар нэмдэг
* гэхдээ хадгалахаас өмнө өөрчлөх боломжтой

Файлыг хадгалсны дараа:

* Dock дээрх **Finder icon** дээр дарж Finder-ийг нээнэ
* хадгалсан хавтас дотроо шинэ файлаа харна

---

#### Хавтас үүсгэх (macOS)

Шинэ хавтас үүсгэхийн тулд:

* хүссэн байршилдаа очно
* хоосон зай дээр **right-click** хийнэ
* хамгийн эхний сонголт болох **“New Folder”**-ийг дарна

Дараа нь:

* шинэ хавтасны нэрийг бичээд
* **Enter** дарж баталгаажуулна

---

#### Файл, хавтас зөөх (Finder)

Finder дээр файл, хавтас зөөх хэд хэдэн арга бий.

Хэрвээ:

* зорьж буй хавтас **Favorites** хэсэгт байвал
  → шууд **drag & drop** хийж болно

Өөр хавтас руу зөөх бол:

1. Зорьж буй хавтсыг **tab** хэлбэрээр нээгээд drag & drop хийх
   эсвэл
2. Keyboard shortcut ашиглах

Shortcut аргаар:

* файл эсвэл хавтсыг **Command + C** (copy)
* очих хавтас руу орно
* **Command + Option + V** дарна

Ингэснээр:

* файл шинэ байршилд очно
* анхны байршлаасаа устгагдана

---

#### Файл, хавтас устгах (macOS)

Файл эсвэл хавтсыг:

* **right-click** хийгээд **“Move to Trash”**-ийг сонгоно
  эсвэл
* Dock дээрх **Trash icon** руу drag & drop хийнэ

Эдгээр нь ижил үйлдэл юм.

Trash дотор орсны дараа:

* тухайн файл дээр right-click хийж **шууд устгаж**
* эсвэл Trash-ийг бүхэлд нь **empty** хийж
  бүх устгасан файлуудыг бүрмөсөн арилгаж болно

---

### Дүгнэлт

Windows болон macOS дээр:

* файл үүсгэх
* зөөх
* устгах
  аргуудыг сайн эзэмших нь маш чухал.

Эдгээр чадварыг эзэмшсэнээр:

* файлуудаа зөв зохион байгуулах
* хурдан олох
* хэрэггүй файлуудыг устгаж storage суллах
  боломжтой болно.
