### Вэб аппликейшнд түгээмэл ашиглагддаг файлын төрлүүд юу вэ?

Файлуудыг **агуулга (content)** болон **бүтцээр (structure)** нь ангилдаг. Файлын төрөл нь тухайн файлыг компьютерийн ямар програм хэрхэн нээж, боловсруулж ажиллахыг тодорхойлдог.

Файлын төрлийг бид **өргөтгөл (file extension)**-өөр нь таньж мэддэг. Файлын нэрийн төгсгөлд байдаг гурваас дөрвөн үсэгтэй хэсгийг файлын өргөтгөл гэнэ.

Вэб аппликейшнд хамгийн түгээмэл тааралддаг **гурван үндсэн файлын төрөл** нь:

* HTML
* CSS
* JavaScript
  юм.

---
1. HTML, CSS, JavaScript файлууд

* **HTML файл**
  Вэб аппликейшний **бүтэц болон агуулгыг** тодорхойлдог.
  `.html` өргөтгөлтэй.

* **CSS файл**
  Вэбийн **дизайн, харагдах байдал, стиль**-ийг тодорхойлдог.
  `.css` өргөтгөлтэй.

* **JavaScript файл**
  Вэб аппликейшнд **илүү нарийн логик, харилцан үйлдэл (interactivity)** нэмдэг.
  `.js` өргөтгөлтэй.

---

2. Зургийн файлын төрлүүд

Вэб аппликейшн хөгжүүлэх явцад та **зургийн файлууд**-тай зайлшгүй ажиллана. Дараах нь хамгийн өргөн хэрэглэгддэг зургийн форматууд юм.

* **JPEG (Joint Photographic Experts Group)**

  * Маш сайн **шахалт (compression)**-тай
  * Фото зураг, бодит зурагт тохиромжтой
  * **Lossy** формат тул файлын хэмжээ багасгахын тулд зарим чанар алдагддаг

* **PNG (Portable Network Graphics)**

  * Шахалтын дараа ч **зурагны чанарыг хадгалдаг**
  * Файлын хэмжээ харьцангуй **том**
  * **Transparency** (тунгалаг байдал) дэмждэг
  * Лого, icon зэрэг хурц ирмэгтэй зурагт тохиромжтой

* **GIF (Graphics Interchange Format)**

  * **Анимаци** болон **тунгалаг байдал** дэмждэг
  * Өнгөний палитр хязгаарлагдмал

* **SVG (Scalable Vector Graphics)**

  * **Vector** төрлийн зураг
  * Хэдий чинээ томруулсан ч **чанараа алдахгүй**
  * Лого, icon, диаграммд маш тохиромжтой

---
3. Видео болон аудио файлын төрлүүд

Одоо видео, аудио форматуудыг авч үзье.

* **MP3**

  * Аудио файл
  * Сайн шахалттай
  * **Lossy** формат тул зарим аудио мэдээлэл алдагддаг

* **MP4**

  * Хамгийн түгээмэл видео формат
  * Сайн шахалттай
  * Олон төрлийн аудио, видео кодек
  * Subtitle болон metadata дэмждэг

* **MOV**

  * Apple-ийн хөгжүүлсэн **QuickTime** мультимедиа формат
  * Ихэвчлэн QuickTime Player-тэй холбоотой

* **WAV**

  * **Lossless** аудио формат
  * Анхны дууны чанарыг бүрэн хадгална

* **WebM**

  * Өндөр чанартай
  * **Open-source** видео формат

---
4. Фонтын файлын төрлүүд

Хэрвээ та вэб аппликейшнийхээ **фонтыг өөрчлөх** шаардлагатай бол дараах форматуудтай ажиллана.

* **TTF (TrueType Font)**

  * Олон төрлийн үйлдлийн системд нийцдэг

* **WOFF (Web Open Font Format)**

  * Вэб хөгжүүлэлтэд зориулсан орчин үеийн(Modern) формат(Format)
  * Илүү сайн шахалттай → файлын хэмжээ бага
  * Лиценз зэрэг **metadata** хадгалах боломжтой

* **WOFF2**

  * WOFF-ын дараагийн үе
  * **Илүү үр ашигтай шахалт, илүү өндөр гүйцэтгэл**

---
5. Архив файлууд

Олон файл, хавтсыг нэгтгэх шаардлагатай үед **архив файл** ашигладаг.

* **ZIP**

  * Хамгийн түгээмэл архив формат
  * **Lossless** шахалттай (өгөгдөл алдагдахгүй)
  * Бүх үйлдлийн систем, програмууд өргөн дэмждэг

---

6. Баримтжуулалт (Documentation)

Вэб аппликейшнд ихэвчлэн **README** нэртэй файл байдаг. Энэ файлд:

* аппликейшн юу хийдэг
* хэрхэн ашиглах
* хэрхэн суулгах
* лиценз
* хэрхэн хувь нэмэр оруулах
  зэрэг мэдээллүүд багтдаг.

README файлуудыг ихэвчлэн **Markdown** форматаар бичдэг.

* Markdown ашиглан:

  * гарчиг, дэд гарчиг
  * холбоос
  * зураг
  * жагсаалт
    зэрэг бүтэцтэй баримт бичиг үүсгэж болдог

* Markdown файлууд:

  * `.md`
  * `.markdown`
    өргөтгөлтэй

Та карьерынхаа туршид:

* олон README файл уншиж
* олон README файл бичих болно

---

7. Дүгнэлт

HTML, CSS, JavaScript гэх үндсэн суурь файлуудаас эхлээд:

* зураг
* видео
* аудио
* фонт
* архив
* баримтжуулалт
  гэх мэт бүх файлын төрлүүд нь **интерактив вэб аппликейшн бүтээхэд чухал үүрэгтэй**.

Эдгээр түгээмэл файлын төрлүүдийг ойлгож мэдэх нь вэб хөгжүүлэлтийн зайлшгүй суурь мэдлэг юм.
