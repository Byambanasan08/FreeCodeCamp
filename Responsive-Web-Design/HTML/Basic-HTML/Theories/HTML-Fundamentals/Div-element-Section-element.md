Div элемент гэж юу вэ, хэзээ ашиглах ёстой вэ?

**div элемент** нь бусад элементүүдийг нэг дор бүлэглэхэд(group) ашиглагддаг **container** элемент юм.

Доорх нь div элементийн жишээ. Div элемент дотор **нэмэлт paragraph элемент** нэмж үзээд preview цонхонд гарах өөрчлөлтийг ажиглаарай. Preview харах бол interactive editor-ийг идэвхжүүлэх шаардлагатай.

```
<div>
  <p>Example paragraph element.</p>
</div>
```

Div элемент нь голчлон **нэг ижил CSS загвар (styles)** хуваалцах HTML элементүүдийг нэгтгэх үед ашиглагддаг.
CSS-ийн талаар та дараагийн хичээлүүд болон workshop-уудаар илүү их зүйл сурах болно.

Гэхдээ бодит код бичих орчинд div элементийг маш өргөн ашигладаг боловч **хэтрүүлэн ашиглахаас болгоомжлох хэрэгтэй**. Зарим тохиолдолд div биш илүү тохиромжтой семантик(semantic) элемент ашиглах нь зөв байдаг.

Жишээ нь, контентоо хэсгүүдэд хувааж байгаа нөхцөлд **section элемент** нь div-ээс илүү тохиромжтой.

---

Одоо эхнийх нь доор **нэг шинэ section элемент нэмнэ**.
Тэгээд шинээр нэмсэн section дотор **h2 болон p элементүүд** байрлуулна.
Та дурын текст бичиж болно — preview цонхонд шууд өөрчлөгдөн харагдана. Interactive editor-ийг идэвхжүүлсэн байх шаардлагатай.

Доорх нь section элементийн жишээ:

```
<section>
  <h2>Mammals</h2>
  <p>
    Mammals are warm-blooded animals with fur or hair. Most give birth to live
    young.
  </p>
  <ul>
    <li>Lion</li>
    <li>Elephant</li>
    <li>Dolphin</li>
  </ul>
</section>
```

---

**Section элемент** нь **семантик утгатай**, харин div элемент нь **семантик биш**.

**Semantics** гэдэг нь тухайн хэлний үг, өгүүлбэрийн *утга*, *учир холбогдлыг* хэлдэг.
HTML нь өөрөө нэг төрлийн хэл тул HTML элемент бүр өөрийн гэсэн семантик утгатай байдаг.

Тиймээс section элемент ашиглавал браузер тухайн элементийн семантик утгыг таньж, үүнийг **section буюу хуудасны нэгэн хэсэг** гэж ойлгоно — desktop, mobile, бүх төрлийн төхөөрөмжүүд дээр адилхан.
