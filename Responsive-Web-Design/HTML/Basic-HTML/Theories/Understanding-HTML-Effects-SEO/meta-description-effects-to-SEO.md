1. Meta Description гэж юу вэ, SEO-д ямар үүрэгтэй вэ?

SEO буюу *Search Engine Optimization* нь вэб хуудсуудыг хайлтын системд илүү харагдахуйц, илүү өндөр зэрэглэлтэй болгох зорилгоор оновчлох үйл явц юм. Таны сайтын SEO-г сайжруулах нэг арга бол тухайн вэб хуудсын богино тайлбарыг **meta элемент** ашиглан өгөх явдал юм. Жишээлбэл, дараах код нь цэцэрлэгжүүлэлтийн вэб хуудсын тайлбарыг meta ашиглан тодорхойлж байгаа жишээ:

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

`name` атрибутыг `description` гэж тохируулах нь браузерууд, хайлтын системүүд болон бусад вэб хэрэгслүүдэд энэ метадата-г зөв тайлбарлаж ойлгоход тусалдаг. Харин `content` атрибутод та өөрийн тайлбарыг бичих бөгөөд тайлбараа **богино, цэгцтэй** байлгахыг зөвлөдөг. Учир нь хайлтын системүүд үр дүнгийн хуудсын загвараас хамааран тайлбарыг тасалж харуулах нь элбэг.

Бусад meta элементүүдийн адил meta description нь өөрөө вэб хуудсан дээр харагддаггүй. Харин энэ тайлбар харагддаг нэг газар бол хайлтын системийн **search result snippet** юм. Доорх жишээнүүд нь freeCodeCamp-ийн subreddit болон GitHub repository-уудын үр дүнгийн snippet юм:

```
r\FreeCodeCamp: This is the official subreddit for the freeCodeCamp.org community. Learn to
code for free together with millions of other people...

Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

Эдгээр жишээнүүдэд тайлбар бүр нь тухайн сайтын линкний яг доор байрлаж байгаа харагдана. Ингэснээр хэрэглэгчид ердөө хэдхэн секундийн дотор тухайн хуудас юуг агуулж байгааг ойлгож, линк дээр дарах эсэхээ шийдэх боломжтой.

Meta description нь сайтын хайлтын систем дэх **эрэмбэд шууд нөлөөлдөггүй** ч, хүчтэй, сайн бичсэн тайлбар нь вэб хуудас руу илүү олон хэрэглэгчийг татах давуу талтай.
