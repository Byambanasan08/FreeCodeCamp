2. Open Graph Tag-уудын үүрэг юу вэ, SEO-д хэрхэн нөлөөлдөг вэ?

**Open Graph протокол** нь таны вэб сайтын контент Facebook, LinkedIn зэрэг олон төрлийн социал платформ дээр хэрхэн харагдахыг хянах боломжийг олгодог. Эдгээр OG property-уудыг зөв тохируулснаар хэрэглэгчдийг таны контент руу татах, даралт (click) болон engagement-ийг нэмэгдүүлэх боломжтой. OG property-уудыг HTML-ийн **head** хэсэг дотор meta элементүүд ашиглан тохируулна.

---

### **1. OG Title (og:title) – Заавал оруулах чухал property**

Доорх жишээ нь freeCodeCamp-ийн нүүр хуудсын OG title-ийг хэрхэн тохируулж буй жишээ:

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

`property` атрибутыг `og:title` гэж зааж өгнө. Харин `content` атрибутод социал платформ дээр харагдах гарчгийг бичнэ.

---

### **2. OG Type (og:type) – Контентийн төрлийг тодорхойлох property**

Жишээ:

```html
<meta property="og:type" content="website" />
```

`og:type` нь социал сүлжээнд хуваалцаж буй контентийн төрлийг илэрхийлнэ. Жишээлбэл:

* article (нийтлэл)
* website (вэбсайт)
* video (видео)
* music (음сик)

---

### **3. OG Image (og:image) – Хуваалцах үед харагдах зургийг тодорхойлох**

Жишээ:

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

Энд OG image нь freeCodeCamp-ийн лого руу зааж байна.

**OG image-ийн гол шаардлагууд:**

* Өндөр чанартай байх
* Зөв хэмжээ, харьцаатай байх
* Платформ бүр өөрийн image requirement-тэй байдаг

Facebook-ийн албан ёсны баримт бичигт дараахыг зөвлөсөн байдаг:

> "High resolution төхөөрөмжүүдэд хамгийн сайн харагдуулахын тулд 1200 × 630 пиксел хэмжээтэй зураг ашиглаарай.
> Хамгийн багадаа 600 × 315 пиксел байх ёстой."

---

### **4. OG URL (og:url) – Тухайн хуудсын үндсэн URL**

Жишээ:

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

---

## **Нэмэлт OG Property-ууд**

Дараах property-уудыг мөн тохируулах боломжтой:

* `og:description` — тайлбар
* `og:audio` — аудио контент
* `og:video` — видео контент
* `og:locale` — хэлний бүс

Гэхдээ хамгийн чухал дөрөв нь:

1. **og:title**
2. **og:type**
3. **og:image**
4. **og:url**

---

## **Open Graph SEO-д хэрхэн нөлөөлөх вэ?**

OG tag-ууд нь хайлтын систем дэх **эрэмбэд шууд нөлөөлдөггүй**.
Гэхдээ тэдгээр нь SEO-д **шууд бус** байдлаар чухал нөлөө үзүүлдэг:

* Социал сүлжээнд контент хуваалцах үед илүү сайхан харагдуулна
* Энэ нь **click-through rate (CTR)**-ийг нэмэгдүүлж чадна
* Илүү өндөр CTR нь хайлтын системд “энэ контент сонирхолтой, үнэтэй байна” гэсэн **signaling effect** үүсгэдэг
* Ингэснээр таны контент илүү сайн үнэлгээ авч, SEO-д эерэг нөлөө үзүүлж болно
