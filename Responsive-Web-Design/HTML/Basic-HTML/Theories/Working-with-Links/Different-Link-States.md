4. Линкүүдийн (link) өөр өөр state-үүд юу вэ, тэд яагаад чухал вэ?

Та веб хуудсан дээрх линкийг дарсны дараа ягаан(purple) өнгөтэй болсныг харж байсан байх. Энэ нь линк өөр state рүү шилжсэн тул өөр CSS стиль хэрэгжиж буй хэрэг юм. Линк нийтдээ **тав** төрлийн state-тэй байдаг.

---

### **1. `:link` — анхдагч (default) state**

`:link` нь хэрэглэгч тухайн линктэй ямар ч харилцаагүй, **дараагүй**, **зочлоогүй** үед үйлчилдэг анхны байдал юм.
Хуудасны бүх линкэд зориулагдсан үндсэн стиль эндээс эхэлдэг.

Доорх жишээ линк нь анхдагч байдлаар цэнхэр(blue) өнгөтэй харагдана. Дарвал ягаан болно.

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

---

### **2. `:visited` — хэрэглэгч өмнө нь дарж үзсэн линк**

Хэрэглэгч тухайн хуудас руу аль хэдийн орсон бол линк `:visited` state-д орно.
Анхдагчаар purple (ягаан) өнгөтэй болсон харагддаг.

Доорх жишээнд линк дарсны дараа бор өнгөтэй болно:

```css
a:visited {
  color: brown;
}
```

`:visited` нь хэрэглэгчид аль хуудас руу зочилж байсан гэдгийг ялгаж харуулахад хэрэгтэй — жишээ нь баримт бичиг уншиж буй үед.

---

### **3. `:hover` — хэрэглэгч хулганаа линк дээр авчирсан үед**

`:hover` нь хэрэглэгч хулганаа линк дээр байрлуулсан мөчид идэвхжинэ.
Энэ нь линк рүү анхаарал татах, хэрэглэгчийн интеракцийн санааг баталгаажуулахад тусалдаг.

```css
a:hover {
  color: red;
}
```

Хулганаа линк дээр аваачихад өнгө нь улаан болно.

---

### **4. `:focus` — линк keyboard фокус авсан үед**

Keyboard navigation ашиглах үед, жишээ нь Tab дарж шилжихэд линк `:focus` state-д орно.

```css
a:focus {
  color: green;
}
```

Энэ нь accessibility-д маш чухал бөгөөд keyboard хэрэглэгчдэд аль элемент одоо идэвхтэй байгааг илтгэнэ.

---

### **5. `:active` — линк яг дарж буй мөчид**

`:active` нь хэрэглэгч линкийг **даран**, primary mouse button идэвхтэй байх хугацаанд үйлчилнэ.
Энэ state маш богино хугацаанд харагдана.

```css
a:active {
  color: black;
}
```

Линк дээр дарсан агшинд өнгө нь хар болж үзэгдэнэ.

---

### **Эдгээр state-үүдийг CSS-д бичих зөв дараалал**

Линк state-үүдийг дарааллаар нь бичих нь чухал:

1. `:link`
2. `:visited`
3. `:hover`
4. `:focus`
5. `:active`

Энэ дарааллыг **LVHFA** гэж товчилж санах нь түгээмэл.

---

Эдгээр state-үүдийг ашигласнаар та веб хуудсан дээрх линкүүдийг илүү ойлгомжтой, хэрэглэгчдэд ээлтэй, интерактив болгох боломжтой.
