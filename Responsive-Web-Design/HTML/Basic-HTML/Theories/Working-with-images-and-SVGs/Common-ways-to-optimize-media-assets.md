1. **Медиа файлуудыг (зураг гэх мэт) оновчтой болгох нийтлэг аргууд юу вэ?**

Веб хуудсанд медиа — ялангуяа зураг — ашиглах үед анхаарах гурван гол зүйл байдаг:

1. **Хэмжээ (resolution & dimensions)**
2. **Файлын формат**
3. **Шахалт (compression)**

---

## **1. Зургийн хэмжээ**

Вебсайт хийхдээ зураг тодорхой хэмжээтэй харагдахаар CSS-ээр стиллэх нь түгээмэл.
Жишээ нь, та 640x480 хэмжээтэй зураг харуулах гэж буй гэж төсөөлье:

* **640** → өргөн (width)
* **480** → өндөр (height)

Хэрвээ та 640x480 пикселийн хэмжээнд харагдах зураг ашиглах гэж байгаа бол:

✔ Зургийг урьдчилан **640x480** болгон багасгаж хадгалах нь хамгийн зөв.

Харин стиллээр жижигрүүлж харагдуулах ч бодит файл нь **1920x1080** хэвээр байвал:

✘ Хэрэглэгч илүү их unnecessary data татаж авна
✘ Вебийн хурд удааширна

→ Бага resolution = Бага файл size = Хурдан ачаалах веб хуудас

---

## **2. Файлын формат**

Түгээмэл хоёр формат бол:

* **PNG**
* **JPG (JPEG)**

Гэвч эдгээр нь өнөө үед хамгийн оновчтой сонголт биш болсон.
Хэрвээ хуучин браузерүүдийг дэмжих шаардлагагүй бол илүү optimize форматуудыг сонгох хэрэгтэй:

### **✔ Шинэ үеийн илүү сайн форматууд:**

* **WEBP**
* **AVIF**

Эдгээр нь:

* Бага хэмжээтэй файл үүсгэдэг
* Илүү сайн шахалттай
* Өндөр чанарыг хадгалдаг
* Сайтын ачааллыг мэдэгдэхүйц хурдасгадаг

---

## **3. Зургийг шахах (Compression)**

Зурагийн файлыг жижигсгэхийн тулд шахалтын алгоритм ашиглаж болно.
Compression нь файл болон дата хэмжээ багасгах зорилготой.

### **Жишээ compression tool-ууд:**

* **pngcrush** (локал дээр PNG шахах)
* Онлайн compression сайтууд (TinyPNG, Squoosh гэх мэт)

### **Lossless ба Lossy шахалт**

* **Lossless compression**
  → Шахсан файлээс original data-г 100% буцаан сэргээж болно.
  PNG, WebP (lossless mode) зэрэгт боломжтой.

* **Lossy compression**
  → Шахах үед чанарын алдагдал зайлшгүй гарна.
  JPG/JPEG нь **lossy** формат.

Иймээс JPG файлыг дахин дахин шахвал чанар улам бүр муудна.

---

## **Дүгнэлт**

Веб хуудсандаа зураг сонгохдоо дараах гурвыг заавал анхаар:

1. **Зөв хэмжээ** → Хэрэглэгч unnecessary data татахгүй
2. **Зөв формат** → WebP/AVIF бол хамгийн оновчтой
3. **Зөв шахалт** → Чанар ба файл хэмжээний тэнцвэрийг сонгох

Эдгээрийг зөв хэрэглэх нь сайтын хурд, SEO, болон хэрэглэгчийн туршлагыг мэдэгдэхүйц сайжруулдаг.
