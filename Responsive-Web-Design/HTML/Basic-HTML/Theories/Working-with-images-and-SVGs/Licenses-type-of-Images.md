2. **Зураг ямар төрлийн лицензтэй байдаг вэ, тэд хэрхэн ажилладаг вэ?**

Зураг бол **оюуны өмч (intellectual property)** бөгөөд ихэнхдээ түүнийг бүтээсэн зохиогчийн **зохиогчийн эрхээр хамгаалагддаг**. Анхдагч байдлаараа зураг нь **all rights reserved** буюу бүх эрхээ зохиогч эсвэл нийтлэгч эзэмшдэг нөхцөлөөр гардаг.

Иймд та тухайн зургийг өөрийн веб хуудсанд ашиглахын тулд дараах гурван боломжийн аль нэгийг заавал биелүүлэх ёстой:

1. **Зохиогчийн эрх эзэмшигчээс бичгээр зөвшөөрөл авах**
2. **Зохиогчийн эрх эзэмшигчээс лиценз худалдан авах**
3. **Зургийг fair use (шударга хэрэглээ)-ийн хүрээнд ашиглах**

---

1. **Fair Use гэж юу вэ?**

Гурав дахь хувилбар нь хамгийн төвөгтэй.
Fair use нь зургийг ашиглахдаа **хязгаартай** байж, мөн **transformative** буюу шинэ утга агуулгатай болгож өөрчлөхийг шаарддаг.

Fair use-д тооцогдох зарим жишээ:

* Уран бүтээлийг **сэтгэгдэл тайлбар**, **шүүмжлэл** хэлбэрээр ашиглах
* Зургийн **пароди**, **элэглэл** хийх
* Боловсруулалт хийж шинэ утга бүхий бүтээл болгон хувиргах

Fair use нь улс орноосоо хамаараад хууль эрх зүйн нарийн ялгаатай тул анхаарах шаардлагатай.

---

2. **Permissive License — Илүү нээлттэй лицензтэй зургууд**

Зарим зураг илүү нээлттэй лицензтэй байдлаар гардаг. Үүнд:

* **Creative Commons (CC)** лицензүүд
* **BSD лиценз** (freeCodeCamp энэ лицензийг ашигладаг)

Эдгээр зургуудыг вебсайтдаа ашиглаж болох ч лицензийн заалтуудыг ягштал уншиж, мөрдөх шаардлагатай. Учир нь лицензээс хамааран дараах нөхцөлүүд тавигдаж болно:

* Зургийг ашиглавал **вебсайтаа open source болгох** шаардлагатай
* Зургийг **өөрчлөхийг хориглох**
* Зургийг ашиглахдаа **зохиогчийг заавал kreditleх**
* Ашиглах хүрээг хязгаарлах (commercial use not allowed гэх мэт)

Тиймээс лиценз бүрийн нөхцөлийг сайтар шалгах нь чухал.

---

3. **Public Domain — Хүмүүсийн нийтлэг эзэмшилд шилжсэн зураг**

Зарим зураг **public domain** буюу олон нийтийн өмчид гарсан байдаг.

Public domain зураг нь:

* Зохиогчийн эрхгүй
* Ямар нэгэн хязгаарлалтгүй
* Хэн ч, хүссэн үедээ, хүссэн зорилгоор ашиглаж болох

Creative Commons-ийн **CC0** лицензтэй зураг нь мөн адил public domain-тэй тэнцүү гэж үзэгддэг.
