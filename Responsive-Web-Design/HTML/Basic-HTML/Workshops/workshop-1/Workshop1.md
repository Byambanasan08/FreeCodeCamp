1## Step:1
HTML гэдэг нь **HyperText Markup Language** гэсэн үгийн товчлол юм. Энэ нь веб хуудсын бүтэц, контентыг тодорхойлдог код юм. Доорх нь таны HTML бичих **code editor** болно.

Editor-ийн **1-р мөрийг** олоод дараах текстийг бичнэ үү: Welcome to freeCodeCamp

## Step:2 
HTML нь element-үүдээс бүрддэг. Таны ашиглах хамгийн эхний элемент бол **h1 element** юм:

```
<h1>Welcome to freeCodeCamp</h1>
```

Энэ нь **opening tag** (`<h1>`)-аар эхэлж, **closing tag** (`</h1>`)-аар дуусдаг бөгөөд эдгээр tag-уудын хооронд харагдах текст байрлана.

Өөрийн *Welcome to freeCodeCamp* гэсэн текстийг **h1 element** болгохын тулд текстийнхээ урд **opening tag**-ийг, ард нь **closing tag**-ийг нэмж өгнө үү.

## Step:4 
An h1 element бол веб хуудасны үндсэн гарчиг бөгөөд нэг хуудас дээр зөвхөн нэгийг л ашиглах ёстой. **h2 elements** нь дэд гарчгуудыг илэрхийлдэг. Нэг хуудас дээр хэд хэдэн h2 ашиглаж болно. Тэд дараах хэлбэртэй харагдана:

```
<h2>This is a subheading.</h2>
```

**Full Stack Curriculum** гэсэн текстийг **opening** болон **closing h2 tags**-аар хүрээлж **h2 element** болгоно уу.
