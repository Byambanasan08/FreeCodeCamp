## Алхам 1

Өмнөх хичээлд та **svg элементүүд** болон тэдгээрийг төсөлд ихэвчлэн **дүрс (icon)** болгон ашигладаг талаар үзсэн. Бодит төслүүдийн кодын сангуудад ихэвчлэн icon library ашигладаг тул svg-ийг эхнээс нь өөрөө бичих шаардлагагүй байдаг. Харин энэ workshop-д та svg дотор ашиглагддаг үндсэн атрибутуудыг ойлгохын тулд зүрхэн дүрс (heart icon) үүсгэж үзэх болно.

Эхлээд хуудсан дээр **svg элемент** үүсгэнэ.

## Step 2
The next step is to set the width and height attributes for the svg element. As you are creating an icon, both values should be set small.

Set both values to 24.

## Алхам 3

Та улам дөхөж байна, одоо дараах жишээг харна уу:

**Жишээ код**

```html
<svg viewBox="0 0 50 50">
</svg>
```

`viewBox` атрибут нь SVG дотор ямар хэсэг харагдахыг удирддаг.

* Эхний хоёр тоо (**0 0**) нь viewBox-ийн эхлэх байрлал — зүүн дээд булангийн *x* болон *y* координатыг заана.
* Дараагийн хоёр тоо (**50 50**) нь viewBox-ийн өргөн ба өндрийг тодорхойлно.

`viewBox` атрибутыг **0 0 24 24** гэж тохируул.

## Алхам 4

Зургийг будож эхлэхээс өмнө дүрсэнд хэлбэр өгөхийн тулд `svg` элементийн дотор **нэг `path` элемент** үүсгэн байрлуулах хэрэгтэй.

`path` элемент үүсгэ.

## Алхам 5

`path` элемент дүрсээ тодорхойлохын тулд хэлбэрийн мэдээлэл шаардлагатай. Үүнийг **`d` атрибут** ашиглан өгдөг. Энэ атрибут нь дүрсийг зурахад хэрэглэгдэх командын үсгүүд болон координатын тоонуудын дарааллыг агуулдаг.

Командын үсгүүд нь *move to*, *line to*, *curve to*, *close path* зэрэг үйлдлүүдийг илэрхийлдэг бөгөөд тоонууд нь тэдгээрийн координатыг заана.

Зүрхэн дүрсний `d` атрибутыг дараах утгаар тохируул:

```
M12 21s-6-4.35-9.33-8.22C-.5 7.39 3.24 1 8.4 4.28 10.08 5.32 12 7.5 12 7.5s1.92-2.18 3.6-3.22C20.76 1 24.5 7.39 21.33 12.78 18 16.65 12 21 12 21z
```
