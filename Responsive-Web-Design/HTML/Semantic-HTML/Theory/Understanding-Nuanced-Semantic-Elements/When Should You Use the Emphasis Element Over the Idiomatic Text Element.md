Emphasis элемент (em)-ийг Idiomatic Text элемент (i)-ийн оронд хэзээ ашиглах ёстой вэ?

Эдгээр хоёр элемент нь presentational HTML болон semantic HTML гэсэн ойлголтуудтай нягт холбоотой.

Анх `i` элемент нь *текстийг ташуу* (italic) харагдуулах **зөвхөн харагдацын зориулалттай** байсан.
Гэвч орчин үед `i` элемент нь дараах утгыг илэрхийлэх **семантик** хэрэглээтэй болсон:

* өөр өнгө аястай, эсвэл өөр дуу хоолойгоор хэлж буй хэсэг
* өөр хэлний хэллэг, идиом
* техникийн нэр томъёо
* бодол, дотоод монолог гэх мэт

HTML-ийн албан ёсны спецификациас авсан жишээ:

```html
<p>There is a certain <i lang="fr">je ne sais quoi</i> in the air.</p>
```

Энд:

* `i` → тухайн текст орчин тойрноосоо өөр утгатай, өөр шинжтэй гэдгийг илэрхийлнэ
* `lang="fr"` → текст Франц хэл дээр гэдгийг зааж байна
* **`i` нь ямар ч "чухал байдал" илэрхийлдэггүй, зөвхөн *өөр* төрөлхөн утгатай гэдгийг харуулна

---

## **Тэгвэл `em` элементийг хэзээ ашиглах вэ?**

Хэрэв текстийн **ач холбогдлыг онцлох**, өгүүлбэрийн утгыг өөрчлөхүйц *тусгай ач холбогдолтой* хэсгийг тэмдэглэх шаардлагатай бол `em` элемент ашиглана.

`em` нь **семантик утга** дамжуулдаг:
→ энэ хэсэг *жигд бус*, *онцгой анхааралтай* уншигдах ёстой.

Жишээ:

```html
<p>
  Never give up on <em>your</em> dreams.
</p>
```

Энд:

* өгүүлбэрийн утга "таны" гэдэг үгийг онцолж, анхаарал татаж байна
* браузер үүнийг italic маягаар үзүүлэх боловч энэ нь гагцхүү харагдац биш
* screen reader болон бусад туслах технологи "онцолж уншина" → semantic ач холбогдол

Ийм семантик утга нь `i` элементэд байхгүй.

---

## **Гол ялгаа**

| Элемент | Юу илэрхийлдэг вэ?                                                        | Хэзээ ашиглах вэ?                              |
| ------- | ------------------------------------------------------------------------- | ---------------------------------------------- |
| **i**   | Текст орчноосоо өөр төрөлтэй, өөр өнгө аяс, өөр хэл, идиом, техникийн нэр | Текстийн утга өөр шинжтэй боловч чухал биш үед |
| **em**  | Текстийн **ач холбогдлыг**, онцгой анхаарлын утгыг илэрхийлнэ             | Утга зүйн хувьд онцолж, чухалчлах хэрэгтэй үед |

---

## **Хэзээ ч эдгээрийг зөвхөн харагдацын зорилгоор ашиглаж болохгүй**

Хэрэв:

* танд зүгээр л *italic* харагдуулах шаардлагатай
* харин текст нь ямар ч тусгай утга, онцгой агуулга агуулаагүй бол

### → Энэ үед HTML биш **CSS** ашиглах ёстой:

```css
.italic {
  font-style: italic;
}
```

```html
<p class="italic">This text is just visually italic.</p>
```

---

## **Дүгнэлт**

* `i` → утга зүйн хувьд *өөр* шинжтэй текст
* `em` → утга зүйн хувьд *чухал*, онцлох шаардлагатай текст
* Италигаар харагдуулах нь зөвхөн визуал тул CSS ашиглана

Товчоор:

**Хэрэв текстийн утга өөр бол `i`**,
**Хэрэв текстийн ач холбогдлыг онцлох бол `em`**.
