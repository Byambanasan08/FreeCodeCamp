**Strong элементийг Bring Attention To элемент (b)-ийн оронд хэзээ ашиглах ёстой вэ?**

`b` элемент буюу “bring attention to” элемент нь текстийн тодруулсан хэсэгт **анхаарал татах** зорилготой.
Энэ нь ихэвчлэн:

* товч тайлбар дахь түлхүүр үгс
* бүтээгдэхүүний нэрс
* тодорхой хэсгийг зүгээр л визуалаар тодруулах

зэрэгт ашиглагддаг.

Доорх жишээнд `b` элементээр бүтээгдэхүүний нэрсийг тодруулсан байна:

```html
<p>
  We tested several products, including the <b>SuperSound 3000</b> for audio
  quality, the <b>QuickCharge Pro</b> for fast charging, and the
  <b>EcoClean Vacuum</b> for cleaning. The first two performed well, but the
  <b>EcoClean Vacuum</b> did not meet expectations.
</p>
```

Браузер эдгээр `b` элементүүдийг **зузаан (bold)** текстээр харуулж, уншигчийн хараанд илүү өртөмтгий болгодог.
Гэхдээ энэ нь *утга зүйн чухал байдал* илэрхийлэхгүй, зөвхөн **анхаарал татах** зорилготой.

---

## **Харин текст үнэхээр чухал, яаралтай анхаарал шаардсан бол → `strong` ашиглана**

`strong` бол семантик HTML элемент бөгөөд:

* *маш чухал*
* *яаралтай анхаарах шаардлагатай*
* *эрсдэлтэй эсвэл онцгой утгатай*

контентыг илэрхийлдэг.

Жишээ:

```html
<p>
  <strong>Warning:</strong> This product may cause allergic reactions.
</p>
```

Энд:

* "Warning:" гэдэг нь уншигчийн анхаарлыг хүчтэй татах ёстой
* Screen reader болон бусад туслах технологи үүнийг “strong emphasis” гэж тод хэлж өгдөг
* Семантикаар илүү өндөр ач холбогдолтой

---

## **Визуал хувьд ижил мэт харагдавч, утга нь тэс өөр**

| Элемент    | Юуг илэрхийлдэг вэ?                                                | Юунд ашиглах вэ?                                         |
| ---------- | ------------------------------------------------------------------ | -------------------------------------------------------- |
| **b**      | Текстийг анхаарал татахуйц болгох (semantics бага)                 | Түлхүүр үг, бүтээгдэхүүний нэр, текстийн тусгай үзүүлбэр |
| **strong** | Текст *маш чухал, яаралтай*, утга зүйн хувьд өндөр ач холбогдолтой | Анхааруулга, чухал тэмдэглэл, ноцтой мэдээлэл            |

Иймээс:

* Хэрэв текстийн *утга онцгой биш* боловч *анхаарал татах хэрэгтэй* → **`b`**
* Хэрэв текст *яаралтай, чухал агуулгатай* → **`strong`**

---

## **Дүгнэлт**

**`b` = Зүгээр анхаарал татах (visual only)**
**`strong` = Утга зүйгээр чухал, яаралтай мэдээлэл (semantic importance)**

Тиймээс сонголт хийхдээ:

👉 Текстийн зорилго юу вэ?
👉 Энэ хэсэг бусдаас *илүү өндөр ач холбогдолтой* юу?

гэсэн зүйлсийг бодолцож сонгоно.
