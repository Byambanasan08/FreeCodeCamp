HTML-д компьютерын кодыг хэрхэн дүрслэх вэ?

Inline code элемент нь текст доторх богино хэмжээний кодын хэсгүүдийг илэрхийлэхэд ашиглагддаг. Code элементийн түгээмэл хэрэглээ нь техникийн нийтлэл болон баримтжуулалтын хуудсууд юм.

CSS кодын хэсгийг харуулахын тулд code элементийг ашигласан жишээг доор үзүүлэв:

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

Энэ жишээнд CSS-ийн `color` шинж чанарыг ашиглан текстийн өнгийг цэнхэр болгож байна. Уг кодын хэсгийг `<code>` таг дотор оруулснаар тухайн текст нь inline code буюу мөр доторх код гэдгийг браузерт ойлгуулж өгдөг.

Браузер нь code элементийн доторх контентод анхдагч (default) стиль хэрэглэдэг. Энэхүү анхдагч стиль нь ихэвчлэн monospace фонт байдаг.

Code элемент нь нэг мөр кодыг илэрхийлэхэд зориулагдсан. Хэрэв та олон мөр кодыг дүрслэхийг хүсвэл code элементийг preformatted text элемент дотор байрлуулах шаардлагатай.

Preformatted text элемент нь урьдчилан форматлагдсан текстийг илэрхийлэхэд ашиглагддаг. CSS declaration-ийг харуулахын тулд preformatted text элементийг ашигласан жишээг доор үзүүлэв:

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

Pre элементийг ашиглах үед зай авалт (spacing)-д анхаарах хэрэгтэй. Учир нь HTML баримт дотор хэрхэн бичигдсэн байна, браузер яг тэр хэвээр нь харуулна.

Браузер дээр харахад код нь баруун тийш хэд хэдэн зайгаар шургуулж (indent) байрласан байгааг анзаарах болно. Хэрэв та кодын жишээн дэх indent-ийг өөрчилбөл дэлгэц дээрх indent мөн адил өөрчлөгдөнө.

HTML баримт дотор кодын жишээ оруулахдаа богино inline жишээнүүдэд code элементийг ашиглах хэрэгтэй.

Харин урт кодын хэсгүүдийг харуулах шаардлагатай бол pre болон code элементүүдийг хамтад нь ашиглах хэрэгтэй.
