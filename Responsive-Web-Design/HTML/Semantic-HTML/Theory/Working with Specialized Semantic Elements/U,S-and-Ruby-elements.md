U, S болон Ruby элементүүд юунд зориулагдсан вэ, мөн тэд хэрхэн ажилладаг вэ?

Unarticulated annotation element буюу товчоор **u элемент** нь текстийн бус (non-textual) тайлбар, тэмдэглэгээ хэрэглэгдсэн inline текстийг илэрхийлэхэд ашиглагддаг.

Доорх нь үг үсгийн алдааг онцлох зорилгоор u элементийг ашигласан жишээ юм:

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

Энэ жишээнд *incorrect*, *spelling*, *issues* гэсэн үгсийг буруу бичсэн байна. U элементийн анхдагч (default) стиль нь текстийн доогуур хар зураас татах хэлбэртэй байдаг.

HTML4-д u элементийг зөвхөн стильд зориулан ашигладаг байсан. Харин HTML5-д u элементийг зөвхөн текстэд текстийн бус тайлбар, тэмдэглэгээ (non-textual annotation) хэрэглэгдсэнийг илэрхийлэхэд ашиглах ёстой.

Хэрэв та текстийг доогуур зураастай харагдуулахыг хүсвэл HTML биш, харин CSS ашиглах хэрэгтэй.

Strikethrough element буюу товчоор **s элемент** нь тухайн текст цаашид үнэн зөв эсвэл хүчинтэй байхаа больсон тохиолдлыг илэрхийлэхэд ашиглагддаг. Доорх нь ямар нэг үйл ажиллагаа цуцлагдсаныг харуулах жишээ юм:

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

Энэ жишээнд цаг агаарын шалтгаанаар аялал цуцлагдсан тул эхний өгүүлбэрийг зураасаар дарж (crossed out) харуулсан байна.

S элементийг баримт бичигт гарсан өөрчлөлтийг зүгээр л харуулах зорилгоор хэзээ ч ашиглах ёсгүй. Ийм тохиолдолд deleted text element болон inserted text element илүү тохиромжтой байдаг.

**Ruby элемент** нь үндсэн текстийн дээгүүр эсвэл доогуур байрлах жижиг текстийг илэрхийлдэг. Энэ нь ихэвчлэн Зүүн Азийн бичгийн системд тэмдэгтийн дуудлагыг харуулахад ашиглагддаг. Доорх нь MDN Web Docs дээрх ruby элементийн жишээ юм:

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

**Rp элемент** (ruby fallback parenthesis element) нь ruby annotation-ийг харуулах дэмжлэггүй браузеруудад зориулсан нөөц (fallback) хаалт болгон ашиглагддаг.

**Rt элемент** (ruby text element) нь ruby annotation-д ашиглагдах текстийг заадаг. Энэ текст нь ихэвчлэн дуудлага эсвэл орчуулгын мэдээлэл байдаг бөгөөд Зүүн Азийн typography-д өргөн хэрэглэгддэг.

Ruby элементийг өөр төрлийн тайлбар, тэмдэглэгээнд ашиглаж болох боловч хамгийн түгээмэл хэрэглээ нь Зүүн Азийн бичгийн хэвлэлзүй (typography) юм.
