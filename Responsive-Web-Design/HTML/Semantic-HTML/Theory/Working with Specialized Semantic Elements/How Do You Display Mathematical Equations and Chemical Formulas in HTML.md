**HTML дээр математик тэгшитгэл болон химийн томьёог хэрхэн харуулах вэ?**

`superscript` элемент (`sup`) нь текстийн нэг хэсгийг дээд индекс (superscript) хэлбэрээр харуулахад ашиглагддаг. Дээд индекс гэдэг нь энгийн текстийн мөрнөөс дээш байрласан тэмдэг эсвэл үсгийг хэлнэ.

Дараах нь зэргийг (exponent) харуулахын тулд superscript элемент ашигласан жишээ юм (preview-г харахын тулд interactive editor-ийг идэвхжүүлнэ үү):

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

Энэ жишээнд хоёр дахь **2** тоо нь paragraph дотор superscript байдлаар харагдахын тулд `sup` tag-аар ороогдсон байна. Preview цонхонд харахад хоёр дахь 2 нь эхний 2-оос жижиг бөгөөд арай дээш байрласан харагдана.

Superscript элементын түгээмэл хэрэглээ нь зэргүүд (exponents), superior lettering, мөн эрэмбийн тоонууд (ordinal numbers) юм. Дараах нь superior lettering-д superscript элемент ашигласан жишээ юм (preview-г харахын тулд interactive editor-ийг идэвхжүүлнэ үү):

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

Superior lettering гэдэг нь ихэвчлэн товчлолыг илэрхийлэх зорилгоор үсгийг superscript хэлбэрээр бичихийг хэлнэ. Энэ жишээнд **g** болон **r** үсгүүд нь товчилсон хэлбэрийг харуулахын тулд superscript tag дотор байрласан байна.

Superscript элемент нь **зөвхөн типографийн (typographical) зорилгоор** ашиглагдах ёстойг анхаарах нь чухал. Хэрэв та текстийн baseline-ийг дээш өргөж загварчлахыг хүсвэл superscript элемент биш, харин **CSS** ашиглах хэрэгтэй.

HTML дотор химийн тэгшитгэл, томьёо илэрхийлэхийн тулд `subscript` элемент (`sub`) ашигладаг. Энэ элемент нь текстийг жижигрүүлж, доош байрлуулсан baseline-тайгаар (subscript) харуулдаг.

Дараах нь нүүрстөрөгчийн давхар ислийн (carbon dioxide) химийн томьёог харуулахын тулд subscript элемент ашигласан жишээ юм (preview-г харахын тулд interactive editor-ийг идэвхжүүлнэ үү):

```html
<p>CO<sub>2</sub></p>
```

Энд **2** тоо нь тухайн тэмдэг subscript байх ёстойг илэрхийлэхийн тулд `sub` tag дотор байрласан байна.

Subscript элементын түгээмэл хэрэглээнд химийн томьёо, footnote, мөн хувьсагчийн доод индексүүд (variable subscripts) орно.
