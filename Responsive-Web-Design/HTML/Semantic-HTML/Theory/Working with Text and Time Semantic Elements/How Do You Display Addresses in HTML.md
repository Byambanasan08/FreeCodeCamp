**HTML дээр хаяг (Address) хэрхэн харуулах вэ?**

Холбоо барих хаягийн элемент (`address`) нь вэб хуудасны тодорхой нэг хэсгийн холбоо барих мэдээллийг илэрхийлэхэд ашиглагддаг. `address` элемент нь уян хатан хэрэглээтэй бөгөөд байгууллагын вэб хуудас, зохиогчийн хуудас, хувийн сайт зэрэг олон төрлийн тохиолдолд ашиглаж болдог.

Вэбсайтынхаа холбоо барих хэсгийг бүтээхдээ `div` зэрэг ердийн (generic) элементээс илүүтэйгээр семантик `address` элемент ашиглах хэрэгтэй.

Дараах нь байгууллагын холбоо барих хуудсанд `address` элемент ашигласан жишээ юм:

```html
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br />
    Springfield, IL 62701<br />
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
  <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
</address>
```

Энэ жишээнд байгууллагын нэр, байршлын хаяг, утасны дугаар, имэйл хаяг зэрэг мэдээллүүд багтсан байна. Байршлын хаягийн хувьд гудамжны нэр, хот, улс орныг хооронд нь ялгаж харуулахын тулд мөр шилжүүлэх элемент болох `br` ашигласан байна.

Утасны дугаарын хувьд `href` attribute-ийн утгыг утасны дугаарт зориулан тохируулсан anchor (`a`) элемент ашигласан байна. `href` attribute доторх `tel:+` утга нь тухайн боломжийг дэмждэг төхөөрөмжүүд дээр утасны дуудлага эхлүүлэх боломжтой, дарж болдог холбоос (clickable link) үүсгэдэг.

Имэйл хаягийн хувьд мөн anchor элемент ашигласан бөгөөд `href` attribute нь `mailto` холбоосоор тохируулагдсан байна. `mailto` холбоос нь HTML баримт бичигт хэрэглэгч өөрийн сонгосон имэйл client дотор шинэ имэйл нээх боломжийг олгодог.

`mailto` холбоос ашиглахын нэг сул тал нь хэрэглэгчид үүнийг spam гэж ойлгох нь элбэг байдаг. Харамсалтай нь олон spam илгээгчид энэ аргыг ашиглан хэрэглэгчдэд имэйл илгээдэг. Тиймээс үүнийг ашиглахдаа энэ эрсдэлийг анхаарч үзэх хэрэгтэй.
