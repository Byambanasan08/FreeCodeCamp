# Large Text буюу Braille keyboard гэж юу вэ, мөн хэн ашигладаг вэ?

**Large text keyboard** болон **braille keyboard** нь **харааны бэрхшээлтэй хэрэглэгчдэд** зориулан зохион бүтээгдсэн гарууд юм. **Large Text Keyboard** (эсвэл **Large Print Keyboard**) дээрх үсэг, тоо, тэмдэгтүүд нь энгийн keyboard-тэй харьцуулахад **илүү том** байдаг. Энэ загвар нь товчлуур дээрх жижиг бичвэрийг харахад хүндрэлтэй хүмүүст маш их тустай. Мөн ихэнх нь **өнгөний контраст(contrast)** болон **гэрэлтэлтийг(brightness)** сайжруулсан байдаг.

Жишээлбэл, **MaxiAids** брэндийн large print keyboard нь **шар товчлууртай**, дээр нь **хар өнгийн, том, тод үсэг, тоо, тэмдэгтүүдтэй** байдаг. Энэ нь хараа муутай хүмүүст харахад илүү хялбар болгодог.

Өөр нэг жишээ нь **хар өнгийн товчлууртай, цагаан бичигтэй large print keyboard** юм. Энэ keyboard нь мөн **backlit** буюу арын гэрэлтүүлэгтэй тул хэрэглэгч гэрлийн нөхцөлдөө тохируулан гэрэлтүүлгийн түвшинг өөрчлөх боломжтой.

Large print keyboard нь хараа муутай хүмүүст **харааны дохио** өгдөг бол **braille keyboard** нь харааны бэрхшээл илүү хүнд, тэр дундаа **хараагүй хүмүүст** зориулсан **бүрэн тактиль(tactile) (мэдрэхүйд суурилсан) туршлага** олгодог.

**Braille** нь **тэмтрэх замаар унших, бичих систем** юм. Энэ нь үсэг, тоо, цэг тэмдэгтийг илэрхийлэх **товгор цэгүүдийн тодорхой хээ (pattern)**-ээс бүрддэг.

Braille keyboard нь энэхүү системийг ашиглан хэрэглэгчид товчлуурыг **хуруугаараа тэмтэрч**, зөв товчийг олоход тусалдаг. Товчлуурууд дээр үсэг, тоо, тэмдэгтийг илэрхийлсэн **товгор цэгийн хээ** байрладаг.

Мөн зарим keyboard нь **хоёр аргыг хослуулсан** байдаг — том бичигтэйгээс гадна braille хээтэй. Энэ нь харааны бэрхшээлтэй хүмүүст төдийгүй **braille сурч байгаа хүмүүст** ч ашигтай.

Large text болон braille keyboard-ууд нь харааны бэрхшээлтэй хүмүүсийг **чадваржуулдаг хэрэгслүүд** юм. Өөр төрлийн оролтын аргуудыг санал болгосноор эдгээр **assistive technology**-ууд нь хүн бүр **дижитал ертөнцөд оролцох боломжтой** байхад тусалдаг.
