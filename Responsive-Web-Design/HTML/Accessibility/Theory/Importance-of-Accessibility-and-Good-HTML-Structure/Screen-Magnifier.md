Screen magnifier юунд ашиглагддаг вэ?

**Screen magnifier** нь хараа муутай болон бусад харааны бэрхшээлтэй хүмүүст дижитал контент, вэб орчныг илүү сайн ашиглахад тусалдаг **туслах хэрэгслүүд** юм.

Одоо эдгээр хэрэгслүүд гэж юу болох, мөн дижитал контентын хүртээмжид ямар үүрэгтэйг илүү дэлгэрэнгүй авч үзье.

Screen magnifier нь компьютер эсвэл гар утасны дэлгэц дээрх **текст, график болон бусад элементүүдийг томруулж** харуулдаг. Олон screen magnifier нь дэлгэцийг **200%-иас дээш** томруулах боломжтой байдаг. Хэрэглэгчид дараа нь заагч төхөөрөмж (mouse, trackball гэх мэт) эсвэл keyboard ашиглан хуудсаар навигац хийх боломжтой. Мөн ихэнх magnifier нь **томруулах хувь (zoom percentage)** болон бусад тохиргоог өөрчлөх боломжийг settings хэсэгтээ санал болгодог.

Screen magnifier нь голчлон **хараа муутай хүмүүст** текст уншихад тусалдаг. Баримт бичиг эсвэл аппликейшн доторх жижиг фонт нь тэдэнд уншихад хэцүү байж болдог. Харин текстийг томруулснаар тэд **нүдээ ядраалгүйгээр** имэйл, нийтлэл болон бусад контентыг уншиж чадна. Screen magnifier нь мөн **вэб хөтлөлтөд** ихээхэн тусалдаг. Тэд хэрэглэгчдэд харахад хэцүү байж болох **товч, холбоос, интерактив элементүүдийг** олох, дээр нь дарахад тусалдаг. Ийм байдлаар харагдах байдал сайжирснаар хэрэглэгчид вэб сайт үзэх, form бөглөх, онлайн үйл ажиллагаанд саадгүй оролцох боломжтой болдог.

Иймээс програм хангамж хөгжүүлэгчид өөрсдийн дижитал бүтээгдэхүүнийг **хараа муутай хүмүүст хүртээмжтэй** байхаар хийх шаардлагатай. Үүнд дараах зүйлсийг анхаарах хэрэгтэй:

* **Scalable font** ашиглах, ингэснээр хэрэглэгч хуудсыг томруулахад layout эвдрэхгүй байх
* **Responsive design** ашиглан хэрэглэгчийн интерфэйсийг өөр өөр дэлгэцийн хэмжээнд дасан зохицохуйц болгох
* **Өндөр контрасттай өнгөний схем** ашиглах, мөн өнгө тохируулах боломж олгох
* **Sticky биш, хэт жижиг биш navbar** ашиглах, ингэснээр magnifier ашиглах үед контент харагдсаар байх
* Текстийг зургаар биш, **жирийн HTML текст** хэлбэрээр ашиглах
* Үүссэн үйлдэлтэй нь холбоотой **feedback-ийг тухайн элементтэй ойр байрлуулах**, гэх мэт

Орчин үеийн бүх түгээмэл **үйлдлийн системүүд** үйлдвэрлэгчийн зүгээс дор хаяж нэг **суурилуулсан magnifier**-тай байдаг:

* **macOS** болон **iOS** системүүд нь **Zoom**-той. macOS дээр идэвхжүүлэхийн тулд **Settings** рүү орж, **Accessibility**-г шүүж, дараа нь **Zoom** дээр дарна. **“Use keyboard shortcuts to zoom”** сонголтыг асаана.
* **iPhone** дээр: **Settings > Accessibility > Zoom**
* **Android** төхөөрөмжүүд нь **Magnification**-той. Үүнийг асаахын тулд **Settings > Special Function > Accessibility > Magnification** руу орно. Төхөөрөмжөөс хамааран өөр байж болох тул settings-ийн хайлтад **“Magnification”** гэж бичиж олж болно.
* **Windows** нь **Magnifier**-тай. Үүнийг **Settings > Ease of Access > Magnifier** замаар ашиглана.
* **Linux** үйлдлийн системүүдийн хувьд magnifier нь түгээлтээсээ хамааран **Zoom** эсвэл **Magnifier** нэртэй байдаг.

Үйлдлийн системд суурилуулснаас гадна дараах **гуравдагч талын (third-party) screen magnifier** програмууд өргөн хэрэглэгддэг:

* **ZoomText** — Windows
* **ClaroView** — macOS болон Windows
* **iZoom** — Windows
* **Zoomify – Screen Magnifier** — macOS
* **LunarPluse** — Windows
* **Loupe** — macOS
