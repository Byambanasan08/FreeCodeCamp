ASCII гэж юу вэ, мөн charCodeAt() ба fromCharCode()-той хэрхэн ажилладаг вэ?

Програмчлалд тэмдэгтүүдийг тоогоор хэрхэн илэрхийлдгийг ойлгох нь чухал. Үүнд ASCII (American Standard Code for Information Interchange) ашиглагддаг. Энэ нь текстийг компьютерт ойлгогдох тоон утгаар илэрхийлэх стандарт юм.

ASCII нь тэмдэгт бүрт тодорхой тоон утга оноодог систем юм.

Жишээ нь:

* Том A үсэг → 65
* Жижиг a үсэг → 97

Ингэснээр компьютер текстийг хадгалах, боловсруулах боломжтой болдог.

ASCII стандарт нь 128 тэмдэгтийг агуулдаг:

* Том ба жижиг латин үсгүүд (A-Z, a-z)
* Тоонууд (0-9)
* Тэмдэгтүүд (!, @, # гэх мэт)
* Удирдлагын тэмдэгтүүд (newline, tab гэх мэт)

JavaScript дотор string нь Unicode (UTF-16) ашигладаг боловч эхний 128 тэмдэгт нь ASCII-тай таардаг тул ASCII жишээнүүд ажилладаг.

charCodeAt() method

JavaScript-д `charCodeAt()` method нь тухайн тэмдэгтийн тоон кодыг буцаадаг. Энэ нь UTF-16 код буцаадаг ч ASCII хүрээнд ижил байдаг.

Жишээ:

```id="2rl1r3"
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

Энд "A" тэмдэгтийн код 65 байна.

Өөр жишээ:

```id="q7t0s8"
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

"!" тэмдэгтийн код 33 байна.

fromCharCode() method

`String.fromCharCode()` method нь тоон кодоос тэмдэгт үүсгэдэг.

Жишээ:

```id="v23m7d"
let char = String.fromCharCode(65);
console.log(char);  // A
```

65 → "A" болж хувирч байна.

Өөр жишээ:

```id="g3l1s2"
let char = String.fromCharCode(97);
console.log(char);  // a
```

97 → "a"

Дүгнэж хэлбэл:

* ASCII нь тэмдэгтийг тоогоор илэрхийлдэг стандарт
* `charCodeAt()` нь тэмдэгтийг тоон код болгодог
* `fromCharCode()` нь тоог тэмдэгт болгодог

Эдгээр method-ууд нь тэмдэгтийг кодоор нь шалгах, харьцуулах, эсвэл динамикаар тэмдэгт үүсгэхэд ашиглагддаг.
