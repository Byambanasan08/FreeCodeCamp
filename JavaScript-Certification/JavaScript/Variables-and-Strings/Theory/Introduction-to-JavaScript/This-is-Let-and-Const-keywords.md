`let` болон `const` нь хувьсагч зарлах, утга оноох, дахин оноох үед хэрхэн ялгаатай ажилладаг вэ?

JavaScript дээр ажиллах үед та програмынхаа туршид ашиглах өгөгдлийг хадгалахын тулд хувьсагч зарлах шаардлагатай болдог.

Орчин үеийн JavaScript-д `let` болон `const` нь хувьсагч зарлах хамгийн түгээмэл арга бөгөөд эдгээр нь утга оноох болон дахин оноох дээр ялгаатай ажилладаг.

Энэ хичээлд бид `let` ба `const`-ийн ялгааг хувьсагч зарлах, утга оноох, дахин оноох талаас нь авч үзнэ.

`let` түлхүүр үг нь дараа нь шинэчилж (update) эсвэл дахин утга оноож (reassign) болох хувьсагч зарлах боломжийг олгодог. Та `let`-ийг уян хатан сав гэж ойлгож болно — нэг утга хадгалсны дараа түүнийг програмын явцад өөрчилж болно.

Жишээ:

```
let score = 10;
```

Энд `score` хувьсагч зарлагдаж, 10 гэсэн утга оноогдсон байна. Хэрэв дараа нь утгыг өөрчлөх шаардлагатай бол:

```
let score = 10;
console.log(score); // 10
score = 20;
console.log(score); // 20
```

Одоо `score` нь 20 гэсэн утгатай болсон. Энэ нь `let`-ийг програм ажиллах явцад утга нь өөрчлөгдөх хувьсагчид маш тохиромжтой болгодог.

Нөгөө талаас, `const` нь тогтмол (constant) хувьсагч зарлахад ашиглагддаг. `const`-оор зарласан хувьсагчид утга оноосны дараа дахин оноож болохгүй.

Иймээс `const` нь програм ажиллах явцад санамсаргүйгээр өөрчлөгдөх ёсгүй утгуудыг хадгалахад тохиромжтой.

Жишээ:

```
const maxScore = 100;
console.log(maxScore); // 100
```

`maxScore` хувьсагчид 100 гэсэн утга оноогдсоны дараа түүнийг өөрчилж болохгүй:

```
maxScore = 200; // Алдаа үүснэ
```

`const` хувьсагчид дахин утга оноохыг оролдвол JavaScript console дээр алдаа гарна, учир нь `const` хувьсагч нь нэгэнт утга оноогдсоны дараа өөрчлөгдөх боломжгүй.

Та `let` ашиглан хувьсагчийг шууд утга оноохгүйгээр зарлаж, дараа нь утга оноож болно:

```
let age;
console.log(age); // undefined
age = 25;
console.log(age); // 25
```

Харин `const` хувьсагчийг зарлах үед заавал утга оноох шаардлагатай. Хэрэв утга оноохгүй зарлавал алдаа гарна:

```
const age; // Алдаа: Missing initializer in const declaration
```

Хэрэв хувьсагчийн утга дараа нь өөрчлөгдөх шаардлагатай бол `let` ашиглана. Жишээ нь оноо хадгалах, хугацааны явцад өөрчлөгдөх утгууд.

Харин өөрчлөгдөх ёсгүй утгууд (жишээ нь тохиргоо, тогтмол утгууд) дээр `const` ашиглах нь зүйтэй.

Мөн `var` түлхүүр үгийг ашиглаж болно, гэхдээ одоо үед төдийлөн зөвлөдөггүй. `var` нь `let`-тэй төстэй боловч илүү өргөн хүрээ (scope)-тэй тул програмд асуудал үүсгэх магадлал өндөр байдаг.
