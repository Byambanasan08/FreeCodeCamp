Data Type гэж юу вэ, мөн JavaScript-д ямар ямар төрлийн өгөгдлийн төрлүүд байдаг вэ?

JavaScript-д data type (өгөгдлийн төрөл) гэдэг нь таны хадгалж буй утгын төрөл юм, жишээ нь тоо эсвэл текст.

Variable (хувьсагч) гэдэг нь тодорхой нэг өгөгдлийн төрлийн утгыг хадгалдаг нэртэй сав бөгөөд та кодын туршид түүнийг дуудаж, өөрчилж ашиглах боломжтой.

Та математикийн хичээл дээр дараах байдлаар хувьсагч ашиглаж байснаа санаж магадгүй:

x = 2
y = 4

x + y

Та x болон y гэх мэт хувьсагч үүсгээд, дараа нь тэдгээрийг ашиглан нэмэх зэрэг үйлдэл хийж байсан. Програмчлалд ч мөн адил ойлголттой. Та өөрийн хүссэн нэртэй хувьсагч үүсгэж, түүнд утга оноож болно. Эдгээр утгууд нь өөр өөр өгөгдлийн төрлүүд байна.

Өгөгдлийн төрлүүд нь програмд ямар төрлийн өгөгдөлтэй ажиллаж байгааг ойлгоход тусалдаг (тоо, текст гэх мэт).

JavaScript-д хэд хэдэн үндсэн өгөгдлийн төрлүүд байдаг. Дараагийн хичээлүүдэд бид эдгээрийг илүү дэлгэрэнгүй үзнэ. Одоохондоо товч танилцуулга:

Эхний өгөгдлийн төрөл: Number

Number нь бүхэл тоо болон бутархай (floating-point) тоог илэрхийлдэг. Бүхэл тооны жишээ: 7, 19, 90.

ТАЙЛБАР: console.log() нь мэдээллийг console (браузерын debug хийх хэсэг)-д хэвлэдэг функц юм. Үүнийг дараагийн хичээлүүдэд дэлгэрэнгүй үзнэ. Мөн // тэмдэг нь comment (тайлбар) бичихэд ашиглагддаг бөгөөд код ажиллах үед тооцогдохгүй.

// Бүхэл тооны жишээ
console.log(3);
console.log(5);
console.log(-67);

Floating point number нь аравтын бутархай тоо юм. Жишээ нь: 3.14, 5.2.

// Бутархай тооны жишээ
console.log(3.14);
console.log(7.2);
console.log(-14.5);

Дараагийн өгөгдлийн төрөл: String

String нь тэмдэгтүүдийн дараалал буюу текст бөгөөд хашилтад бичигддэг. Жишээ:

console.log("I love to code!");

Мөн нэг хашилт ашиглаж болно:

console.log('I love to code!');

Ихэнхдээ string-ийг нэр, шошго, анхааруулах мессеж зэрэгт ашигладаг.

Өөр нэг өгөгдлийн төрөл: Boolean

Boolean нь зөвхөн хоёр утгатай: true (үнэн) эсвэл false (худал). Жишээ нь програм хэрэглэгч нэвтэрсэн эсэхийг шалгаж болно. Хэрэв нэвтэрсэн бол (true) dashboard харуулна, эс бөгөөс (false) login хуудас харуулна.

Дараагийн хоёр төрөл: undefined ба null

undefined нь хувьсагч зарлагдсан боловч утга оноогдоогүй гэсэн утгатай. Үүнийг дараагийн хичээлд дэлгэрэнгүй үзнэ.

null нь хувьсагчийг зориуд “хоосон” гэж оноосон, ямар ч утга агуулаагүй гэсэн утгатай.

Сүүлийн гурван төрөл нь арай илүү төвөгтэй: Object, Symbol, BigInt

Object нь key-value (түлхүүр-утга) хосуудын цуглуулга юм.

```
{
  name: "Alice",
  age: 30
};
```

Object нь холбоотой мэдээллийг нэг дор бүлэглэхэд маш хэрэгтэй. Үүнийг дараагийн модульд илүү дэлгэрэнгүй үзнэ.

Symbol нь JavaScript-д үргэлж давтагдашгүй, өөрчлөгдөхгүй тусгай төрлийн утга юм. Ихэвчлэн өвөрмөц таних тэмдэг үүсгэхэд ашиглагддаг:

Symbol('mySymbol');

BigInt нь Number төрлийн хязгаараас давсан маш том тоонуудыг илэрхийлэхэд ашиглагддаг:

1234567890123456789012345678901234567890n;

Энд бид маш том тооны төгсгөлд n нэмснээр BigInt үүсгэж байна.

Symbol болон BigInt нь харьцангуй бага ашиглагддаг ч мэдэж байх нь чухал.

Эдгээр өгөгдлийн төрлүүдийг ойлгосноор та програмдаа янз бүрийн төрлийн өгөгдөлтэй зөв ажиллаж чадна, учир нь төрөл бүр өөрийн гэсэн онцлог, үйлдэлтэй байдаг.
