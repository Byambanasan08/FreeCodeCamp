JavaScript гэж юу вэ, мөн HTML ба CSS-тэй хэрхэн хамт ажилладаг вэ?

JavaScript бол вэбсайтад харилцан үйлчлэл (interactive байдал) болон динамик(Dynamic)  үйлдэл нэмдэг хүчирхэг програмчлалын хэл юм.

HTML ба CSS нь хуудсан дээрх контентыг бүтэцжүүлэх болон элементүүдийг загварчлахад ашиглагддаг тэмдэглэгээний хэлүүд бол JavaScript нь үүнээс давж, хэрэглэгчийн оролтыг боловсруулах, элементүүдийг хөдөлгөөнд оруулах, бүр бүтэн вэб аппликейшн бүтээх зэрэг илүү нарийн үйлдлүүдийг хэрэгжүүлэх боломжийг олгодог.

Жишээлбэл, та товч дээр дарах, форм илгээх, эсвэл цэс дээр хулганаа аваачих үед JavaScript нь тухайн хуудас хэрхэн ажиллахыг тодорхойлдог.

Эдгээр гурав хэрхэн хамт ажилладгийн жишээ:

```
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

Энэ жишээнд HTML нь контентыг тодорхойлоход ашиглагдсан: гарчиг (h1 элемент) болон товч (button элемент).

CSS нь гарчгийг загварчлахад, жишээлбэл текстийг ногоон өнгөтэй болгоход ашиглагдсан. JavaScript нь товч дээр дарахад анхааруулах (alert) мессеж харуулахад ашиглагдсан.

Дүгнэж хэлбэл, HTML нь бүтцийг хангаж, CSS нь загварчлалыг нэмж, JavaScript нь үйлдлийг боломжтой болгож, бүрэн бөгөөд интерактив (interactive) вэб туршлагыг бий болгодог.
