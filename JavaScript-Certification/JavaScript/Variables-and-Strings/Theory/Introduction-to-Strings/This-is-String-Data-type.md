# JavaScript-д String гэж юу вэ, мөн String-ийн immutability гэж юу вэ?

JavaScript-д string нь текст өгөгдлийг илэрхийлэхэд ашиглагддаг тэмдэгтүүдийн дараалал юм. String нь number, boolean, null, undefined зэрэгтэй адилхан primitive (анхан шатны) өгөгдлийн төрлүүдийн нэг юм.

JavaScript-д string үүсгэхийн тулд нэг хашилт (') эсвэл давхар хашилт (") ашиглаж болно.

Доорх нь string утга хадгалсан хоёр хувьсагчийн жишээ:

```
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

Та нэг эсвэл давхар хашилт ашиглаж болох ч нэг мөр баримтлах нь чухал. Хэрэв string нэг хашилтаар эхэлсэн бол мөн нэг хашилтаар дуусах ёстой.

Мөн адил, давхар хашилтаар эхэлсэн бол давхар хашилтаар дуусна. Дараах жишээ алдаа үүсгэнэ, учир нь давхар хашилтаар эхэлж нэг хашилтаар дууссан байна:

```
const improperStr = "Do not do this';
```

Өөр нэг анхаарах зүйл бол string нь immutable (өөрчлөгдөхгүй) байдаг.

Програмчлалд immutability гэдэг нь нэг зүйл үүссэний дараа шууд өөрчлөгдөх боломжгүй гэсэн утгатай. Тиймээс string үүссэний дараа түүний тэмдэгтүүдийг шууд өөрчилж болохгүй. Харин өөрчлөх шаардлагатай бол шинэ string үүсгэнэ.

Жишээ:

```
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

String нь immutable учраас эхний string-ийг шууд өөрчилж чадахгүй. Тиймээс бид developer хувьсагчид шинэ string оноож байна.

String нь програмчлалын маш чухал хэсэг бөгөөд дараагийн хичээлүүдэд та тэдгээрийг хэрхэн илүү нарийн аргаар ашиглах, өөрчлөх (manipulate) талаар суралцах болно.
