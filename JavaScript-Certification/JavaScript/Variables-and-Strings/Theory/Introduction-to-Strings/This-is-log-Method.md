# console.log гэж юуны тулд ашиглагддаг вэ, мөн хэрхэн ажилладаг вэ?

Өмнөх хичээлүүдэд та `console.log()`-той танилцсан бол энэ хичээлд түүний зорилго болон хэрэглээг илүү дэлгэрэнгүй авч үзнэ.

JavaScript-д `console.log()` нь браузерын console дээр мессеж эсвэл мэдээлэл харуулахад ашиглагддаг энгийн боловч хүчирхэг хэрэгсэл юм. Энэ нь ихэвчлэн хөгжүүлэгчид кодоо шалгах (debug хийх) болон ажиллаж байгаа байдлыг нь хянахад ашиглагддаг.

Та `console.log()`-ийг ашиглан текст эсвэл хувьсагчийн утгыг console дээр хэвлэж, код тань зөв ажиллаж байгаа эсэхийг шалгаж болно.

`console.log()` ашиглахдаа хаалт дотор нь гаргахыг хүссэн утга эсвэл мессежээ бичдэг. Жишээ нь:

```
console.log("Hello, world!");
```

```
let num = 5;
console.log(num); // 5
```

Эхний жишээнд "Hello, world!" текст console дээр хэвлэгдэнэ, харин хоёр дахь жишээнд 5 гэсэн утга гарна.

Өөр нэг жишээ:

```
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

Та мөн `console.log()`-д олон утгыг таслалаар (,) салгаж өгч болно. Жишээ нь:

```
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

Энэ нь олон төрлийн мэдээллийг нэг дор харуулахад хэрэгтэй.

`console.log()` нь програм ажиллах явцад юу болж байгааг хянахад тусалж, алдааг олоход болон програм хэрхэн ажиллаж байгааг ойлгоход ихээхэн дэмжлэг үзүүлдэг.
