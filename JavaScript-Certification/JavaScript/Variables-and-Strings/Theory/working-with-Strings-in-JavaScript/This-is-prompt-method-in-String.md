`prompt()` method гэж юу вэ, мөн хэрхэн ажилладаг вэ?

`prompt()` method нь JavaScript-д хэрэглэгчээс мэдээлэл авах энгийн аргуудын нэг юм. Энэ нь жижиг pop-up (харилцах) цонх нээж, хэрэглэгчээс утга оруулахыг хүсдэг.

Ихэнхдээ вэб хуудсанд хэрэглэгчээс нэр эсвэл текстэн мэдээлэл авах шаардлагатай үед ашиглагддаг.

`prompt()` method нь хэрэглэгчээс утга асууж, оруулсан утгыг string хэлбэрээр буцаадаг.

`prompt()` method нь хоёр аргумент авдаг:

* Эхнийх нь хэрэглэгчид харагдах мессеж
* Хоёр дахь нь (заавал биш) default утга

```
prompt(message, default);
```

```id="mjhw4l"
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

Энэ жишээнд хэрэглэгч товч дарсны дараа `prompt()` гарч ирнэ. "What is your name?" гэсэн мессеж харагдаж, input хэсэгт "Guest" гэсэн анхны утга байна.

Хэрэв хэрэглэгч нэрээ оруулаад "OK" дарвал тухайн утга `userName` хувьсагчид хадгалагдана.

Хэрэв "Cancel" дарвал `userName` нь `null` болно. `null` нь хэрэглэгч ямар ч утга оруулаагүй гэсэн утгатай.

Дараа нь `output` хэсэгт "Hello, ..." гэсэн мессеж гарна.

Анхаарах зүйл:

`prompt()` нь хэрэглэгч хариу өгөх хүртэл JavaScript кодын гүйцэтгэлийг түр зогсоодог. Өөрөөр хэлбэл хэрэглэгч "OK" эсвэл "Cancel" дарах хүртэл дараагийн код ажиллахгүй.

Мөн `prompt()` нь энгийн туршилт эсвэл жижиг програмд тохиромжтой боловч орчин үеийн том вэб аппликейшнуудад төдийлөн ашиглагддаггүй. Учир нь хэрэглэгчийн туршлагад саад учруулж болзошгүй бөгөөд браузер бүр дээр ялгаатай ажиллаж болно.

Дүгнэж хэлбэл, `prompt()` нь браузераар дамжуулан хэрэглэгчтэй шууд харилцах, мэдээлэл авах энгийн арга юм.
