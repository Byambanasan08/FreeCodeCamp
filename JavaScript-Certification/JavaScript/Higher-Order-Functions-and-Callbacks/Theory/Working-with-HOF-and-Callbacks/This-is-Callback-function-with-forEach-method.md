 Callback функц гэж юу вэ, тэр нь forEach аргатай хэрхэн ажилладаг вэ?

JavaScript-д **callback функц** гэдэг нь өөр нэг функц руу аргумент (параметр) болгон дамжуулагдсан функцийг хэлэх бөгөөд ингэснээр гадна талын функц нь түүнийг тодорхой нэг цэгт дуудаж ажиллуулах боломжтой болдог.

Энэхүү ойлголт нь JavaScript-ийн олон талыг, тэр дундаа `forEach` арга хэрхэн ажилладагийг ойлгоход маш чухал суурь болдог.

Callback функц гэж юу болохыг энгийн агуулгаар ойлгож эхэлцгээе. Танд нэг функц байгаа бөгөөд тэр функц дотроо ямар нэг зүйлийг олон удаа хийхийг хүсэж байна гэж төсөөлье. Бүх кодыг нэг том функц дотор бичихийн оронд та тэрхүү тодорхой үйлдлийг хийх шаардлагатай үед ажиллах хоёр дахь функцийг (callback) дамжуулж болно. Энэ нь кодыг илүү уян хатан, модульлаг (хэсэгчилсэн) болгодог.



Одоо `forEach` аргыг танилцуулъя. `forEach` бол JavaScript-ийн массивуудад (arrays) зориулсан бэлэн арга юм. Энэ нь массивын элемент бүрээр дамжиж (iterate), элемент бүр дээр тодорхой үйлдэл хийх боломжийг олгодог. Таны хийхийг хүсэж буй тэрхүү үйлдэл нь `forEach`-д өгч буй callback функц дотор тодорхойлогддог.

`forEach` нь callback функцтэй хэрхэн ажилладаг жишээг харна уу:

```javascript
let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});
```

Энэ жишээнд бидэнд тоонуудын массив байна. Бид энэ массив дээр `forEach` аргыг ашиглаж байгаа бөгөөд `forEach`-д callback функцийг аргумент болгон өгч байна.

Энэхүү callback функц нь массивын элемент бүрийг төлөөлөх нэг параметр хүлээн авдаг. `forEach` арга нь массивын элемент бүрийн хувьд энэхүү callback функцийг нэг удаа дуудаж ажиллуулах болно.



Энэ тохиолдолд callback функц нь тоо бүрийг 2-оор үржүүлж, үр дүнг нь консол дээр хэвлэж байна. Тиймээс энэ код ажиллах үед дараах үр дүн гарна:

```text
2
4
6
8
10
```

Callback функц нь массивын элемент бүрийн хувьд дэс дарааллаар нь нэг нэгээр дуудагддаг гэдгийг ойлгох нь чухал юм. `forEach` нь таны өмнөөс давталтыг (loop) хариуцан хийдэг тул та өөрөө `for` давталт бичих шаардлагагүй болно.

Та мөн callback болгон **arrow function** (сумтай функц) ашиглаж болох бөгөөд энэ нь таны кодыг илүү товчхон болгоно:

```javascript
let numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
```

Энэ нь өмнөх жишээтэй яг ижил зүйлийг хийх боловч илүү бага код ашиглаж байна.

`forEach` доторх callback функц нь үнэндээ гурав хүртэлх аргумент хүлээн авч чаддаг: **одоогийн элемент**, **одоогийн элементийн индекс**, болон **`forEach` дуудагдсан массив өөрөө**.

Гурван аргументыг ашигласан жишээг харна уу:

```javascript
let numbers = [1, 2, 3, 4, 5];
numbers.forEach((number, index, array) => {
  console.log(`Элемент ${number} нь ${array} массивын ${index} индекс дээр байна`);
});
```

Энэ код нь элемент бүр, түүний индекс болон анхны массивын талаарх мэдээллийг хэвлэх болно.

Callback функцууд болон `forEach` зэрэг аргуудыг ойлгох нь JavaScript-д ахихад маш чухал юм. Тэдгээр нь хэлний олон ахисан түвшний ойлголтуудын, ялангуяа ирээдүйн хичээлүүдээр үзэх асинхрон програмчлалын (asynchronous programming) суурь болдог.
