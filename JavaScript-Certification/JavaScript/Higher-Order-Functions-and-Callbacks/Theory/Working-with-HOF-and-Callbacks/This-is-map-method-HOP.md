 Map арга гэж юу вэ, тэр нь хэрхэн ажилладаг вэ?

`map` арга нь JavaScript-д массивууд дээр ажилладаг, маш хүчирхэг бөгөөд өргөн хэрэглэгддэг функц юм. Энэ нь анхны массивын элемент бүрт өгөгдсөн функцийг хэрэгжүүлэх замаар **шинэ массив** үүсгэх зориулалттай.

Энэхүү арга нь анхны массивыг огт өөрчилдөггүй бөгөөд үүний оронд элемент бүр дээр функцийг хэрэгжүүлсэн үр дүнг агуулсан шинэ массивыг буцаадаг.



Тоонуудын массив дээр `map` аргыг ашигласан жишээг харна уу:

```javascript
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);

console.log(numbers); // [1, 2, 3, 4, 5] (Хэвээрээ үлдсэн)
console.log(doubled); // [2, 4, 6, 8, 10] (Шинэ массив)
```

Элемент бүр нь хоёр дахин үржүүлэгдсэн шинэ массив үүсгэхийн тулд бид `map` аргыг ашиглаж байна. `map` арга нь массив дахь элемент бүр дээр дуудагдах callback функцийг хүлээн авдаг. 

Энэ тохиолдолд массив дахь тоо бүр 2-оор үржүүлэгдэх болно. Үр дүн нь 2, 4, 6, 8, 10 гэсэн тоонуудаас бүрдсэн шинэ массив байх болно.

`map` аргын callback функц нь гурав хүртэлх аргумент хүлээн авч болно. 

**1. Эхний аргумент нь боловсруулагдаж буй одоогийн элемент юм:**

```javascript
const doubledNumbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

**2. Хоёр дахь аргумент нь боловсруулагдаж буй одоогийн элементийн индекс (байршил) юм:**

```javascript
const numbersWithIndex = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

**3. Гурав дахь аргумент нь map арга дуудагдаж буй массив өөрөө юм:**

```javascript
const detailedNumbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```



`map` аргыг ойлгож, үр дүнтэй ашиглаж сурах нь JavaScript-д массивуудтай ажиллах таны чадварыг мэдэгдэхүйц сайжруулна. Ирээдүйн хичээлүүдээр бид `map`-ийн илүү ахисан түвшний хэрэглээг судалж, динамик бөгөөд үр ашигтай програмууд бүтээхэд энэ нь хэрхэн хүчирхэг хэрэгсэл болдог талаар үзэх болно.
