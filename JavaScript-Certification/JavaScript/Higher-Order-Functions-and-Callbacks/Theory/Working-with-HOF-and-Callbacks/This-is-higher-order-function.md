Higher-Order Functions гэж юу вэ?

Higher-order functions (дээд эрэмбийн функцууд) нь JavaScript-ийн маш хүчирхэг ойлголт бөгөөд таны код бичих чадварыг мэдэгдэхүйц нэмэгдүүлж, кодыг тань илүү уян хатан, дахин ашиглах боломжтой болгодог.

Үндсэндээ бол **higher-order function** гэдэг нь нэг буюу түүнээс дээш функцийг аргумент болгон авдаг, эсвэл функцийг утга болгон буцаадаг (эсвэл хоёуланг нь зэрэг хийдэг) функцийг хэлнэ.



Higher-order function-ийг ойлгохын тулд эхлээд JavaScript-д функцийг **"first-class citizens"** (дээд зэрэглэлийн иргэн) гэж үздэг болохыг санах хэрэгтэй. Энэ нь функцүүдтэй яг л бусад утгатай адил харьцаж болно гэсэн үг юм. Өөрөөр хэлбэл, тэднийг хувьсагчид оноож, өөр функц руу аргумент болгон дамжуулж, мөн функцээс утга болгон буцааж болдог. Энэхүү уян хатан чанар нь higher-order function-ийг үүсгэх болон ашиглах боломжийг олгодог юм.

Higher-order function-ийн нэг нийтлэг хэрэглээ бол нарийн төвөгтэй үйлдлүүдийг ерөнхийлөн тодорхойлох (abstract away) явдал юм. Жишээлбэл, танд массивын элемент бүр дээр тодорхой үйлдэл хийдэг функц байна гэж бодъё. Өөр өөр үйлдлүүдэд зориулж тусдаа функцүүд бичихийн оронд та тухайн үйлдлийг өөрийг нь аргумент болгон авдаг higher-order function үүсгэж болно. Энэ нь ижил функцийн бүтцийг өөр өөр үйлдлүүдэд дахин ашиглах боломжийг олгодог.

Энэхүү ойлголтыг тайлбарлах жишээг харна уу:

```javascript
function operateOnArray(arr, operation) {
  let result = [];
  for (let i = 0; i < arr.length; i++) {
    // operation гэдэг функц рүү утгыг дамжуулж, үр дүнг нь массив руу нэмж байна
    result.push(operation(arr[i]));
  }
  return result;
}

function double(x) {
    return x * 2;
}

let numbers = [1, 2, 3, 4, 5];
let doubledNumbers = operateOnArray(numbers, double);

console.log(doubledNumbers); // [2, 4, 6, 8, 10]
```

Энэ жишээнд `operateOnArray` бол higher-order function юм. Тэрээр массив болон функц (operation)-ийг аргумент болгон авч байна. Дараа нь тэрхүү үйлдлийг (operation) массивын элемент бүр дээр хэрэгжүүлж байна. `double` функцийг `operateOnArray` руу аргумент болгон дамжуулж байгаа нь функцуудыг хэрхэн утга (value) болгон ашиглаж болохыг харуулж байна.

Higher-order function нь мөн функцийг утга болгон буцааж чадна. Энэ нь ерөнхий функцээс илүү нарийн мэргэшсэн (specialized) функцүүдийг үүсгэхэд маш хэрэгтэй байдаг. Энэ ойлголтыг ихэвчлэн **"function factories"** (функцийн үйлдвэр) гэж нэрлэдэг. Жишээ нь:

```javascript
function multiplyBy(factor) {
  // Дотор талын функц гадна талын factor хувьсагчид хандаж байна (Closure)
  return function(number) {
    return number * factor;
  };
}

let double = multiplyBy(2); // double нь одоо 2-оор үржүүлдэг функц болсон
let triple = multiplyBy(3); // triple нь одоо 3-аар үржүүлдэг функц болсон

console.log(double(5)); // 10
console.log(triple(5)); // 15
```

Энэ тохиолдолд `multiplyBy` бол шинэ функцийг буцаадаг higher-order function юм. Энэхүү шинэ функц нь `multiplyBy` руу дамжуулсан "factor" дээр үндэслэн тусгайлан ажилладаг. Энэ нь бидэнд хүссэн үржвэрийн функцүүдээ хялбархан үүсгэх боломжийг олгож байна.



Higher-order function-ууд нь зөвхөн онолын ойлголт биш бөгөөд JavaScript-д маш өргөн ашиглагддаг. JavaScript-ийн массивын бэлэн аргууд болох `map()`, `filter()`, болон `reduce()` зэрэг нь бүгд higher-order function-ууд юм. Эдгээр аргууд нь функцийг аргумент болгон авч, массивын элементүүд дээр янз бүрийн байдлаар хэрэгжүүлдэг. Та эдгээр аргуудын талаар ирээдүйн хичээлүүдээр илүү дэлгэрэнгүй үзэх болно.

Higher-order function-ийг ашиглах нь кодыг илүү **declarative** (тунхагласан) бөгөөд ойлгоход хялбар болгодог. Даалгаврыг хэрхэн гүйцэтгэхийг алхам алхмаар дүрслэхийн оронд (imperative programming), higher-order function нь та юунд хүрэхийг хүсэж байгаагаа дүрслэх боломжийг олгодог (declarative programming). Энэ нь кодыг уншихад илүү хялбар, арчлахад илүү дөхөм болгодог.

Та JavaScript-ээр үргэлжлүүлэн ажиллах тусам higher-order function-уудтай байнга тааралдаж, ашиглах болно. Тэдгээр нь JavaScript-ийн функциональ програмчлалын гол хэсэг бөгөөд цэвэрхэн, үр ашигтай, уян хатан код бичихэд зайлшгүй шаардлагатай юм.

Higher-order function-уудыг ойлгож, ашиглаж сурах нь таны боловсронгуй бөгөөд тансаг (elegant) JavaScript програмууд бичих чадварыг мэдэгдэхүйц нэмэгдүүлэх болно.
