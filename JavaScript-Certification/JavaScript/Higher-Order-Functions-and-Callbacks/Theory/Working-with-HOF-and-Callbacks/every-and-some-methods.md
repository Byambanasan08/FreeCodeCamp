every() болон some() аргууд хэрхэн ажилладаг вэ?

JavaScript-д массивуудтай ажиллаж байх үед массивын бүх элементүүд тодорхой нэг нөхцөлийг хангаж байгаа эсэхийг, эсвэл ядаж нэг элемент тухайн нөхцөлийг хангаж байгаа эсэхийг шалгах шаардлага байнга гардаг.

Энэ тохиолдолд `every()` болон `some()` аргууд маш хэрэгцээтэй байдаг. Эдгээр аргууд нь таны кодыг хялбарчилж, илүү уншигдах боломжтой болгодог хүчирхэг хэрэгслүүд юм.

#### 1. every() арга

Эхлээд `every()` аргыг авч үзье. Энэ арга нь массивын **бүх элементүүд** өгөгдсөн функцээр хэрэгжүүлсэн шалгуурыг давж байгаа эсэхийг шалгадаг. Өөрөөр хэлбэл, таны массивын нэг бүрчлэн бүх зүйл таны зааж өгсөн нөхцөлийг хангаж байгаа эсэхийг шалгана гэсэн үг юм.

`every()` арга нь массивын бүх элементүүд дээр өгөгдсөн функц `true` утга буцааж байвал өөрөө `true` утга буцаадаг. Хэрэв **ямар нэгэн нэг л элемент** шалгуурт бүдэрвэл, уг арга шууд `false` утга буцааж, үлдсэн элементүүдийг шалгахаа зогсоодог.



`every()` хэрхэн ажилладгийг тайлбарлах жишээг харна уу:

```javascript
const numbers = [2, 4, 6, 8, 10];
const hasAllEvenNumbers = numbers.every((num) => num % 2 === 0);

console.log(hasAllEvenNumbers); // true
```

Энэ жишээнд бид массивын бүх тоо тэгш эсэхийг шалгаж байна. `every()`-д өгсөн функц маань тоо бүрийг 2-т үлдэгдэлгүй хуваагдаж байгаа эсэхийг шалгана. Массивын бүх тоонууд үнэхээр тэгш байгаа тул `hasAllEvenNumbers` хувьсагчийн утга `true` байна.

#### 2. some() арга

Харин одоо `some()` аргыг харцгаая. `every()` нь бүх элементийг шалгадаг бол `some()` нь **ядаж нэг элемент** шалгуурыг давж байгаа эсэхийг шалгадаг. `some()` арга нь шалгуурыг давах нэг л элемент олмогцоо шууд `true` утга буцаадаг. Хэрэв нэг ч элемент шалгуурыг давахгүй бол `false` утга буцаана.



`some()` хэрхэн ажилладаг жишээ:

```javascript
const numbers = [1, 3, 5, 7, 8, 9];
const hasSomeEvenNumbers = numbers.some((num) => num % 2 === 0);

console.log(hasSomeEvenNumbers); // true
```

Энэ жишээнд бид массивт ямар нэгэн тэгш тоо байгаа эсэхийг шалгаж байна. Массивын ихэнх тоонууд сондгой боловч ядаж нэг тэгш тоо (8) байгаа учраас `hasSomeEvenNumbers` утга нь `true` гарна.

#### Чухал тэмдэглэлүүд:

* **Өгөгдөл баталгаажуулалт:** `every()` болон `some()` нь массивын өгөгдлийг баталгаажуулах эсвэл тодорхой нөхцөлүүдийг шалгахад маш хэрэгтэй. Тэдгээр нь нүсэр их `for` давталт болон `if` нөхцөлүүдийг орлож, кодыг илүү цэвэрхэн, ойлгомжтой болгодог.
* **Гүйцэтгэл (Performance):** Хоёр арга хоёулаа үр дүнгээ тодорхойлмогцоо ажиллахаа шууд зогсоодог. `every()`-ийн хувьд нэг л `false` олбол, `some()`-ийн хувьд нэг л `true` олбол зогсоно. Энэ нь ялангуяа маш том массивууд дээр ажиллахад гүйцэтгэлийн хувьд маш ашигтай байдаг.
