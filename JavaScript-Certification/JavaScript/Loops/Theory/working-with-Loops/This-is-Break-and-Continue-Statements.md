Break болон Continue операторуудыг давталтад юунд ашигладаг вэ?

`break` оператор нь давталтаас хугацаанаас нь өмнө гарахад ашиглагддаг бол `continue` оператор нь давталтын тухайн итерацийг алгасаад дараагийнх руу шилжихэд ашиглагддаг.



#### 1. Break оператор
`for` давталтад `break` ашигласан жишээг энд харуулав:

```javascript
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

Дээрх жишээнд давталт 0-ээс эхлэн тоолж, `i` нь 10-аас бага байх хугацаанд үргэлжлэн ажиллана. Давталт дотор бид `i` нь 5-тай тэнцүү эсэхийг шалгаж байна. Хэрэв тийм бол бид `break` операторыг ашиглан давталтаас хугацаанаас нь өмнө гарна. Хэрэв үгүй бол `i`-ийн утгыг консол дээр хэвлэнэ. Тиймээс кодын үр дүнд 0, 1, 2, 3, 4 гэсэн тоонууд хэвлэгдэнэ.

`break` оператор нь тодорхой нэг нөхцөл дээр үндэслэн давталтаас эрт гарахыг хүссэн үед маш хэрэгтэй. Жишээлбэл, хэрэв та массиваас тодорхой нэг утгыг хайж байгаа бол тухайн утгыг олмогцоо давталтаас гарахын тулд `break`-ийг ашиглаж болно.

---

#### 2. Continue оператор
Заримдаа та давталтыг бүхэлд нь зогсоолгүйгээр зөвхөн тухайн нэг итерацийг алгасахыг хүсэж болно. Яг ийм үед `continue` оператор хэрэг болдог. `for` давталтад `continue` ашигласан жишээ:

```javascript
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

Өмнөхтэй адил бид `i`-ийг 0-ээр эхлүүлж, 10-аас бага байх нөхцөлтэй давталт ажиллуулж байна. Давталт дотор `i` нь 5-тай тэнцүү болох үед бид `continue` ашиглан тухайн итерацийг алгасаж, дараагийнх руу шилжинэ. Энэ кодын үр дүнд 0, 1, 2, 3, 4, 6, 7, 8, 9 гэсэн тоонууд хэвлэгдэнэ. `continue` операторын ачаар 5 гэсэн тоог алгассан байна.

---

#### 3. Шошго (Labels) ашиглах
`break` болон `continue` операторуудын тусламжтай хийж болох өөр нэг зүйл бол **шошго (labels)** ашиглан аль давталтыг зогсоох эсвэл үргэлжлүүлэхээ зааж өгөх явдал юм. Энэ нь үүрлэсэн (nested) давталтуудтай ажиллаж байх үед дотор талын давталтаас гадна талын давталтын урсгалыг хянахад маш хэрэгтэй байдаг.



`break` операторыг шошготой ашигласан жишээ:

```javascript
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

Энэ жишээнд бид `outerLoop` гэж шошголсон гадна талын `for` давталт, болон `innerLoop` гэж шошголсон дотор талын `for` давталттай байна. `i` нь 1, `j` нь 1-тэй тэнцүү болох үед бид `break outerLoop` гэж бичсэнээр гадна талын давталтаас шууд гарч байна. Энэ нь дотор болон гадна талын давталтыг хоёуланг нь зогсооно гэсэн үг юм.

Код консол дээр дараах үр дүнг гаргана:
```text
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

Ихэнх тохиолдолд танд `break` болон `continue`-тэй хамт шошго ашиглах шаардлага гарахгүй байх. Гэхдээ хэрэгцээ гарсан үед ийм боломж байдаг гэдгийг мэдэж байх нь зүйтэй.
