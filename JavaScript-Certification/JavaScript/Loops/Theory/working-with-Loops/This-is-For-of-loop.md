For...of давталт хэрхэн ажилладаг вэ, түүнийг хэзээ ашиглах ёстой вэ?

**For...of** давталтыг "iterable" буюу дараалсан бүтэцтэй өгөгдлөөс утгуудыг нь нэг бүрчлэн авч давтахад ашигладаг. "Iterable" өгөгдлийн жишээнд массив (array) болон тэмдэгт мөр (string) орно.

`for...of` давталтын үндсэн бичиглэл (syntax) энд байна:

```javascript
for (variable of iterable) {
  // ажиллах кодын блок
}
```



Энэхүү бичиглэл дэх `variable` (хувьсагч) нь тухайн үед давтагдаж буй өгөгдлийн "одоогийн утгыг" төлөөлнө.

Хэрэв танд тоонуудын массив байвал хувьсагч нь массив дахь одоогийн тоо байх болно. Хэрэв танд тэмдэгт мөр байвал хувьсагч нь тухайн мөр дэх одоогийн тэмдэгт байна. `for...of` давталт хэрхэн ажилладагийг илүү сайн ойлгохын тулд хэдэн жишээ харцгаая.

Эхний жишээнд бид тоонуудын массивтай бөгөөд тоо бүрийг нэг бүрчлэн консол дээр хэвлэхийг хүсэж байна:

```javascript
const numbers = [1, 2, 3, 4, 5];
for (const num of numbers) {
  console.log(num);
}
```

Бид массив дахь одоогийн тоог төлөөлөх `num` нэртэй хувьсагч үүсгэсэн. 1 дэх итерацид `num` нь 1, 2 дахь итерацид `num` нь 2 гэх мэтээр үргэлжилнэ. Давталт дотор бид одоогийн тоог консол руу хэвлэж байна.



Өөр нэг жишээ харцгаая. Энд бид тэмдэгт мөр (string) доторх тэмдэгт бүрийг консол дээр хэвлэхийг хүсэж байна:

```javascript
const str = 'freeCodeCamp';
for (let char of str) {
  console.log(char);
}
```

Энэ жишээнд бид тэмдэгт мөр дэх одоогийн тэмдэгтийг төлөөлөх `char` нэртэй хувьсагч үүсгэсэн. Итераци бүрт давталт нь одоогийн тэмдэгтийг консол дээр хэвлэх болно.

`for...of` давталтад хувьсагчийг зарлахдаа `let` эсвэл `const` ашиглаж болдог гэдгийг анхаарах нь чухал юм. Хэрэв та `const` ашиглах гэж байгаа бол давталт дотор тухайн хувьсагчийн утга өөрчлөгдөхгүй гэдэгт итгэлтэй байх хэрэгтэй. Хэрэв өөрчлөгдвөл алдаа заах болно.

`const` ашиглаад алдаа зааж буй жишээг энд харуулав:

```javascript
const numbers = [1, 2, 3, 4, 5];
for (const num of numbers) {
  console.log(num);
  num = num + 1; // Энэ нь алдаа заахад хүргэнэ
}
```



Энэ жишээнд бид давталт дотор `num`-ын утгыг өөрчлөхийг оролдож байна. Бид `num`-ыг `const`-оор зарласан тул алдаа заах болно. Тиймээс, хэрэв та давталт дотор хувьсагчийн утгыг өөрчлөх шаардлагатай бол `let` ашиглаарай.

Хамгийн сүүлийн жишээ болгон объектуудын массивтай хэрхэн ажиллахыг харцгаая:

```javascript
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

Энэ жишээнд бид `people` нэртэй объектуудын массивтай байна. Объект бүр `name` (нэр) болон `age` (нас) шинж чанартай. Массив дундуур давтахдаа бид массив дахь одоогийн объектыг төлөөлөх `person` хувьсагчийг үүсгэж байна.

Давталт дотор бид консол руу мессеж гаргаж байна. Эхний мессеж "John is 30 years old", хоёр дахь нь "Jane is 25 years old", гурав дахь нь "Jim is 40 years old" байх болно.

`for...of` давталт нь массив эсвэл тэмдэгт мөр гэх мэт "iterable" өгөгдлийн утгууд дээр давталт хийх шаардлагатай үед үнэхээр ашигтай байдаг. Тэдгээрийг уншихад маш хялбар бөгөөд таны кодыг илүү товчхон болгож чадна.
