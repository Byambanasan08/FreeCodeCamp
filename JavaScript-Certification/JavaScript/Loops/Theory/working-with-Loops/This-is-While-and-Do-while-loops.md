While давталт гэж юу вэ, тэр нь Do...while давталтаас юугаараа ялгаатай вэ?

Өмнөх хичээлүүдээр та `for`, `for...in`, болон `for...of` давталтуудтай хэрхэн ажиллахыг сурсан. Энэ хичээлээр та `while` болон `do...while` давталтын талаар сурах болно.

#### 1. While давталт
`while` давталт нь нөхцөл үнэн (true) байх хугацаанд кодын блокыг давтан ажиллуулдаг. `while` давталтын үндсэн бичиглэл (syntax) энд байна:

```javascript
while (condition) {
  // ажиллах кодын блок
}
```



Нөхцөлийг кодын блок ажиллахаас **өмнө** шалгадаг. Хэрэв нөхцөл худал (false) байвал кодын блок огт ажиллахгүй. `while` давталт нь тухайн кодын блокыг хэдэн удаа ажиллуулах шаардлагатайгаа урьдчилан мэдэхгүй байгаа тохиолдолд маш хэрэгтэй байдаг.

`while` давталт ашигласан жишээг энд харуулав:

```javascript
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

Энэ жишээнд бид `counter` нэртэй хувьсагчийг 0 утгаар эхлүүлсэн байна. `while` давталт нь `counter`-ийн утга 5-аас бага байх хугацаанд үргэлжлүүлэн ажиллана. Давталт дотор бид `counter`-ийн утгыг консол дээр хэвлээд, дараа нь `counter`-ийг 1-ээр нэмэгдүүлж байна.

---

#### 2. Do...while давталт
`while` давталттай төстэй өөр нэг давталт бол `do...while` юм. Үндсэн бичиглэл нь:

```javascript
do {
  // ажиллах кодын блок
} while (condition);
```



`do...while` болон `while` давталтын хоорондох гол ялгаа нь: `do...while` давталт нь нөхцөлийг шалгахаас өмнө кодын блокыг **хамгийн багадаа нэг удаа** заавал ажиллуулдагт оршино.

Хэрэв нөхцөл үнэн байвал кодын блок үргэлжлэн ажиллана. Хэрэв нөхцөл худал байвал кодын блок ажиллахаа зогсооно. `do...while` давталт ашигласан жишээ:

```javascript
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);
```

Энэ жишээнд бид `counter` хувьсагчийг 0-ээр эхлүүлсэн. `do...while` давталт нь эхлээд `counter`-ийн утгыг консол дээр хэвлэж, дараа нь 1-ээр нэмэгдүүлнэ. Кодын блокыг ажиллуулсны дараа л `counter` нь 5-аас бага байгаа эсэхийг шалгана. Хэрэв тийм бол давталт үргэлжилнэ, үгүй бол зогсоно.

---

### Дүгнэлт
Ихэнх тохиолдолд та `do...while`-аас илүү `while` давталтыг түгээмэл ашиглах байх. Гэсэн хэдий ч хоёр төрлийн давталтыг хоёуланг нь мэдэж, хэзээ ашиглахаа ойлгох нь зүйтэй юм.
