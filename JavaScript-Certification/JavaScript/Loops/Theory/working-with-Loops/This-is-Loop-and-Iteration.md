JavaScript дээр Давталт (Loops) болон Итераци (Iteration) хэрхэн ажилладаг вэ?

Програмчлал дахь давталтууд (loops) нь тодорхой нэг кодын блокыг олон удаа давтан ажиллуулахад ашиглагддаг. Давталтын нэг жишээ бол жагсаалт дахь зүйлсийг хэвлэх шаардлагатай програм зохиох үе юм. Та жагсаалтын зүйл бүрийг хэвлэхийн тулд давталт ашиглаж болно.

Өөр нэг жишээ бол тоглоом зохиож байхдаа дүрүүдийг дэлгэц дээгүүр хөдөлгөх үе юм. Та давталт ажиллах бүрт дүрийг тодорхой хэмжээний пикселээр хөдөлгөхөд давталтыг ашиглаж болно.

JavaScript-д таны ашиглаж болох хэд хэдэн төрлийн давталт байдаг. Энэ хичээлээр бид **for loop**-ийг үзэх болно. `for` давталтын үндсэн синтакс (бичиглэл) энд байна:

```javascript
for (initialization; condition; increment or decrement) {
  // ажиллах кодын блок
}
```



* **Initialization (Эхлүүлэх):** Энэ илэрхийлэл нь давталт эхлэхээс өмнө нэг удаа ажилладаг. Энэ нь ихэвчлэн тоолуур хувьсагчийг (counter variable) эхлүүлэхэд ашиглагддаг. Тоолуур хувьсагч гэдэг нь давталт хэдэн удаа ажилласныг хянахад ашигладаг хувьсагч юм.
* **Condition (Нөхцөл):** Энэ илэрхийлэл нь давталтын итераци (iteration) бүрийн өмнө шалгагддаг. Итераци гэдэг нь давталт доторх кодыг нэг удаа гүйцэтгэх процессыг хэлнэ.
* Хэрэв нөхцөл **үнэн (true)** байвал давталт доторх кодын блок ажиллана. Хэрэв нөхцөл **худал (false)** байвал давталт зогсож, дараагийн кодын блок руу шилжинэ.
* **Increment/Decrement (Нэмэгдүүлэх/Хасах):** Давталтын сүүлийн хэсэг болох энэ илэрхийлэл нь итераци бүрийн дараа ажилладаг. Энэ нь ихэвчлэн тоолуур хувьсагчийг нэмэгдүүлэх эсвэл хасахад ашиглагддаг.



```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

Дээрх жишээний эхний хэсэгт бид `i` тоолуур хувьсагчийг `0` утгаар эхлүүлж байна. `for` давталтад `i`-г тоолуур хувьсагчаар ашиглах нь түгээмэл байдаг.

Дараагийн хэсэг нь нөхцөлийг шалгах юм. Энэ тохиолдолд нөхцөл нь `i` нь 5-аас бага байгаа эсэхийг шалгаж байна. `i` нь 0 байгаа тул нөхцөл үнэн бөгөөд давталт доторх кодын блок ажиллана.

Давталт доторх кодын блок нь `i`-ийн утгыг консол дээр хэвлэх юм. `i`-ийн утга 0 байгаа тул консол дээр 0 гэсэн утга харагдана.

Үүний дараа нэмэгдүүлэх (increment) илэрхийлэл ажиллана. Энэ тохиолдолд бид `i`-ийг 1-ээр нэмэгдүүлж байна. Ингэснээр `i` одоо 1 болно.

Дараа нь бид нөхцөлийг (i нь 5-аас бага эсэхийг) дахин шалгана. `i` одоо 1 байгаа тул нөхцөл үнэн хэвээр байгаа бөгөөд давталтын доторх код дахин ажиллана.

Бид энэ процессыг нөхцөл худал болох хүртэл давтсаар байх болно. Энэ жишээнд `i` нь 5 болох үед нөхцөл худал болж, давталт зогсоно.



Та давталттай ажиллаж байхдаа нөхцөлийг ямагт үнэн байхаар тохируулахаас болгоомжлох хэрэгтэй. Хэрэв та ингэвэл давталт үүрд үргэлжлэх бөгөөд таны програм гацах болно. Үүнийг **хязгааргүй давталт (infinite loop)** гэж нэрлэдэг.

Мөн **үүрлэсэн (nested) for loops** үүсгэх боломжтой. Үүрлэсэн давталт гэдэг нь нэг давталтын дотор өөр нэг давталтыг байрлуулахыг хэлнэ. Үүнийг хэзээ ашиглах жишээг бид дараа үзэх болно.

Давталтууд нь тодорхой кодын блокыг олон удаа давтах шаардлагатай үед програмчлалд маш их тустай байдаг. Анх сурахад `for` давталттай ажиллах нь бага зэрэг төвөгтэй байж болох ч дадлага хийснээр та удахгүй сурах болно.
