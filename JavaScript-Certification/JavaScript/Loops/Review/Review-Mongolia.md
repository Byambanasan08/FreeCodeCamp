# JavaScript Давталтуудын Тойм (Loops Review)

### 1. Давталттай ажиллах

**1.1 for Loop:** Энэ төрлийн давталт нь кодын блокыг тодорхой тооны удаа давтахад ашиглагддаг. Энэ давталт нь гурван хэсэгт хуваагддаг: эхлүүлэх илэрхийлэл (initialization), нөхцөл (condition), болон нэмэгдүүлэх/хасах илэрхийлэл (increment/decrement).

- **Initialization:** Давталт эхлэхээс өмнө нэг удаа ажиллана. Ихэвчлэн тоолуур хувьсагчийг эхлүүлэхэд ашигладаг.
- **Condition:** Итераци бүрийн өмнө шалгагддаг. Итераци гэдэг нь давталтыг нэг удаа дүүргэхийг хэлнэ. Хэрэв нөхцөл үнэн бол кодын блок ажиллана. Хэрэв худал бол давталт зогсож, дараагийн код руу шилжинэ.
- **Increment/Decrement:** Итераци бүрийн дараа ажилладаг. Ихэвчлэн тоолуур хувьсагчийг нэмэх эсвэл хасахад ашиглана.

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

**1.2 for...of Loop:** Энэ төрлийн давталтыг "iterable" буюу дараалсан бүтэцтэй өгөгдлийн (массив, тэмдэгт мөр гэх мэт) утгууд дээгүүр давтахад ашигладаг.

```javascript
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

**1.3 for...in Loop:** Энэ давталт нь объектын шинж чанаруудыг (properties) дамжиж давтахад хамгийн тохиромжтой. Энэ нь объектын бүх тоологдох шинж чанарууд, түүний дотор удамшсан болон тоон бус шинж чанаруудыг дамждаг.

```javascript
const fruit = {
  name: "apple",
  color: "red",
  price: 0.99,
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

**1.4 while Loop:** Энэ төрлийн давталт нь нөхцөл үнэн байх хугацаанд кодын блокыг ажиллуулсаар байна.

```javascript
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

**1.5 do...while Loop:** Энэ төрлийн давталт нь нөхцөлийг шалгахаас өмнө кодын блокыг хамгийн багадаа нэг удаа заавал ажиллуулдаг.

```javascript
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

### 2. break болон continue операторууд

**2.1 Тодорхойлолт:** `break` оператор нь давталтаас хугацаанаас нь өмнө гарахад ашиглагддаг бол `continue` оператор нь давталтын тухайн итерацийг алгасаад дараагийнх руу шилжихэд ашиглагддаг.

**break-ийн жишээ:**

```javascript
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
// Үр дүн: 0, 1, 2, 3, 4
```

**continue-ийн жишээ:**

```javascript
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
// Үр дүн: 0, 1, 2, 3, 4, 6, 7, 8, 9
```
