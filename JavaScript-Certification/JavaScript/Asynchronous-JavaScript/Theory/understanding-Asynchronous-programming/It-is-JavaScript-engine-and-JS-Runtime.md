[ JavaScript Engine хэрхэн ажилладаг вэ, мөн JavaScript Runtime гэж юу вэ?

JavaScript engine нь таны кодыг унших, ойлгох, гүйцэтгэх чадвартай байдаг. Энэ нь таны кодыг авч, компьютер ойлгож гүйцэтгэх боломжтой заавар (instructions) болгон хувиргадаг хөрвүүлэгч (converter) шиг ажилладаг.

Хамгийн алдартай JavaScript engine-үүдийн нэг бол V8 JavaScript Engine бөгөөд энэ нь Google Chrome болон Node.js дээр ашиглагддаг. JavaScript engine нь хэд хэдэн алхмаар ажилладаг. Эхлээд таны кодыг parse (задлан унших) хийж, мөр бүрээр нь уншин JavaScript кодод алдаа байгаа эсэхийг шалгана. Дараа нь энэ кодыг bytecode (байткод) болгон хөрвүүлдэг бөгөөд энэ нь таны кодын илүү энгийн, завсрын хувилбар бөгөөд компьютер ойлгож гүйцэтгэхэд илүү хялбар байдаг. Эцэст нь энэ bytecode-ийг ажиллуулж таны програмын заавруудыг гүйцэтгэнэ. Доорх нь JavaScript кодын жишээ юм:

const greeting = "Hello, World!";
console.log(greeting);

Та энэ кодыг ажиллуулахад JavaScript engine эхлээд үүнийг parse хийж, ямар нэг syntax error (синтаксын алдаа) байгаа эсэхийг шалгана. Parsing (задлан унших) гэдэг нь engine кодыг уншиж, өөрт нь ойлгомжтой бүтэц болгон задалж, явцын дунд алдаа байгаа эсэхийг шалгах үйл явц юм.

Дараа нь энэ кодыг завсрын формат (ихэвчлэн bytecode эсвэл machine code, engine-ээс хамаарна) руу compile (хөрвүүлэх) хийнэ. Compiling (хөрвүүлэх) гэдэг нь хүний уншиж ойлгох кодыг компьютер илүү хурдан гүйцэтгэх боломжтой илүү үр ашигтай формат руу хувиргах үйл явц юм.

Эцэст нь engine кодыг execute (гүйцэтгэх) хийж, Hello, World! гэсэн текстийг console дээр хэвлэнэ.

Одоо JavaScript runtime-ийн талаар ярья. JavaScript runtime гэдэг нь таны JavaScript код гүйцэтгэгдэж буй орчин (environment) юм. Энэ нь JavaScript engine (жишээ нь V8 JavaScript Engine нь Chrome дээр, SpiderMonkey нь Mozilla Firefox дээр) болон кодыг боловсруулах, гүйцэтгэхэд ашиглагддаг хэсгүүдийг агуулдаг бөгөөд мөн орчноос өгөгдсөн нэмэлт боломжуудыг (жишээ нь web browser эсвэл Node.js — үүнийг та дараагийн хичээлүүдэд илүү дэлгэрэнгүй сурна) багтаадаг.

JavaScript хэлний үндсэн хэсэг нь variables (хувьсагч), loops (давталт), functions (функц) зэрэг зүйлсийг зохицуулдаг бол runtime нь JavaScript-д хэлний хүрээнээс гадуур зүйлстэй харилцах нэмэлт хэрэгслүүдийг өгдөг. Жишээ нь DOM (Document Object Model — вэб хуудсуудтай ажиллахад) эсвэл Fetch API (сүлжээний хүсэлт хийхэд) гэх мэт.

Товчхондоо, runtime нь JavaScript-д зөвхөн суурь програмчлалын ажлуудаас илүү зүйлсийг хийх боломжийг олгодог — жишээ нь вэб хуудсуудтай харилцах эсвэл хугацаатай холбоотой үйлдлүүдийг боловсруулах — эдгээр нэмэлт боломжуудыг хэлний өөрийнх нь хүрээнээс гадна өгснөөр.

Та JavaScript бичихийн тулд engine болон runtime-ийн бүх нарийн ширийн зүйлийг мэдэх шаардлагагүй ч гэсэн, суурь ойлголттой байх нь илүү үр ашигтай код бичих болон алдааг илүү үр дүнтэй засварлахад тусална. ]
