 Массивын дунд хэсгээс элемент хэрхэн нэмэх болон хасах вэ?

JavaScript дэх `splice()` арга нь массивыг өөрчлөх маш хүчирхэг хэрэгсэл юм. Энэ нь массивын дурын байрлалаас, тэр дундаа дунд хэсгээс элемент нэмэх эсвэл хасах боломжийг олгодог. `splice()` аргын буцаах утга нь массиваас хасагдсан элементүүдийг агуулсан массив байна. Хэрэв юу ч хасагдаагүй бол хоосон массив буцаана.

Энэ арга нь эх массивыг шууд өөрчилдөг (**mutate**) буюу шинэ массив үүсгэлгүйгээр тухайн массивыг байранд нь засаж байдаг гэдгийг анхаарах хэрэгтэй. Үндсэн бичиглэл (syntax) нь:

`массив.splice(startIndex, itemsToRemove, item1, item2, ...)`

* **startIndex:** Массивыг өөрчилж эхлэх индекс.
* **itemsToRemove:** Хэдэн элемент хасахыг заах параметр (заавал биш). Хэрэв үүнийг орхивол `startIndex`-ээс хойшх бүх элемент хасагдана.
* **item1, item2, ...:** Массивт нэмэх шинэ элементүүд (заавал биш).



#### 1. Массивын дундаас элемент хасах жишээ:

```javascript
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

Энд `splice(2, 2)` нь 2-р индексээс эхлэн 2 элементийг хасаж байна.

#### 2. Массивын дунд элемент нэмэх жишээ:

```javascript
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

Энд `splice(1, 0, "yellow", "purple")` нь 1-р индексээс эхлэн 0 элемент хасаад (юу ч хасахгүй), "yellow" болон "purple"-ийг нэмж байна.

#### 3. Нэгэн зэрэг нэмэх болон хасах:

```javascript
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

Энэ тохиолдолд 1-р индексээс эхлэн 2 элементийг (2 болон 3) хасаад, оронд нь 6, 7, 8-ыг нэмж байна.



#### Эх массивыг хадгалж үлдэх (Spread Operator)

Хэрэв та эх массивыг өөрчлөхгүйгээр хуулбар дээр ажиллахыг хүсвэл `spread` оператор (`...`) ашиглан хуулбар үүсгэж болно:

```javascript
let original = [1, 2, 3, 4, 5];
let copy = [...original]; // Хуулбар үүсгэх
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5] (Өөрчлөгдөөгүй)
console.log(copy);     // [1, 2, 6, 4, 5] (Шинэчлэгдсэн)
```

#### Түгээмэл хэрэглээ: Тодорхой элементийг хасах

Хэрэв та хасах гэж буй элементийнхээ утгыг мэдэж байгаа бол `indexOf()`-той хослуулан ашиглаж болно:

```javascript
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");

if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}
console.log(fruits); // ["apple", "banana", "mango"]
```

`splice()` нь маш хүчирхэг боловч маш том массивуудын хувьд (ялангуяа эхлэл хэсэгт өөрчлөлт хийхэд) бусад элементүүдийн байрлалыг шилжүүлэх шаардлагатай болдог тул гүйцэтгэлийн хувьд арай удаан байж болно. Зөвхөн төгсгөлд элемент нэмж хасах бол `push()` болон `pop()` нь илүү тохиромжтой.
