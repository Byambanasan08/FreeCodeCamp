 `indexOf` аргыг ашиглан массивын элементийн индексийг хэрхэн олж авах вэ?

JavaScript-д `indexOf()` арга нь массивын дотроос тодорхой нэг элементийн **анхны тохиолдлын индексийг** олоход маш хэрэгтэй байдаг. Хэрэв тухайн элемент массиваас олдохгүй бол энэ арга нь -1 гэсэн утгыг буцаадаг. Үндсэн бичиглэл (syntax) нь дараах байдалтай байна:

`массив.indexOf(элемент, эхлэх_индекс)`

* **элемент:** Массив дотроос хайхыг хүсэж буй утга.
* **эхлэх_индекс (fromIndex):** Хайлтыг аль байрлалаас эхлэхийг зааж өгөх параметр (заавал биш). Хэрэв үүнийг зааж өгөхгүй бол хайлт массивын эхнээс эхэлнэ.

Жишээг авч үзье:

```javascript
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");

console.log(index); // 1
```

Энэ жишээнд `fruits` массив дотроос "banana" гэсэн тэмдэгт мөрийн индексийг хайж байна. "banana" нь 1 болон 3 гэсэн индексүүд дээр байгаа боловч `indexOf()` нь хамгийн анхны тохиолдлыг буцаадаг тул үр дүн нь 1 байна.



Хэрэв таны хайж буй элемент массивт байхгүй бол `indexOf()` нь -1-ийг буцаана:

```javascript
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");

console.log(index); // -1
```

Энд "grape" гэсэн утга массивт байхгүй тул үр дүн нь -1 гарч байна.

Хэрэв та хайлтыг тодорхой нэг индексээс хойш эхлүүлэхийг хүсвэл хоёр дахь аргументыг дараах байдлаар дамжуулж болно:

```javascript
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);

console.log(index); // 4
```

Энэ жишээнд хайлт массивын эхнээс биш, харин 3-р индекс буюу "yellow" дээрээс эхэлж байна. Тиймээс эхний "green"-ийг (индекс 1) алгасаж, дараагийн "green"-ийг олсноор үр дүн нь 4 гарч байна.
