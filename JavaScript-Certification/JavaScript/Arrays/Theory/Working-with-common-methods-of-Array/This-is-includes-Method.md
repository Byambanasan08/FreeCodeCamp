 Массив тодорхой утгыг агуулж байгаа эсэхийг хэрхэн шалгах вэ?

JavaScript-д `includes()` арга нь массив дотор тодорхой нэг утга байгаа эсэхийг шалгах хамгийн энгийн бөгөөд үр дүнтэй арга юм. Энэ арга нь **boolean** утга буцаадаг: хэрэв заасан элемент массивт байвал `true`, байхгүй бол `false` гэж хариулна.



`includes()` арга нь тухайн элементийн байрлалыг (индекс) мэдэх шаардлагагүй, зөвхөн байгаа эсэхийг нь хурдан шалгахыг хүссэн үед маш тохиромжтой. Хэрхэн ашиглах жишээг доор харуулав:

```javascript
let fruits = ["apple", "banana", "orange", "mango"];
console.log(fruits.includes("banana")); // true
console.log(fruits.includes("grape"));  // false
```

Энэ жишээнд бид "banana" байгаа эсэхийг шалгахад `true` гэж гарч байна, учир нь энэ нь массивт байна. Харин "grape" байхгүй тул `false` гэж гарч байна.

#### Том, жижиг үсэг болон Өгөгдлийн төрөл

Тэмдэгт мөртэй (string) ажиллах үед `includes()` арга нь **case-sensitive** буюу том, жижиг үсгийг ялгадаг гэдгийг анхаарах хэрэгтэй. Жишээ нь:

```javascript
let fruits = ["apple", "banana", "orange"];
console.log(fruits.includes("banana")); // true
console.log(fruits.includes("Banana")); // false
```

Мөн энэ арга нь **strict equality (===)** буюу төрөл шалгадаг харьцуулалтыг ашигладаг. Иймээс тоо болон тэмдэгт мөрийг хооронд нь ялгана:

```javascript
let mixedArray = [1, "2", 3, "4", 5];
console.log(mixedArray.includes(2));   // false (Тоо хайж байна)
console.log(mixedArray.includes("2"));  // true  (Тэмдэгт мөр хайж байна)
```



#### Хайлтыг тодорхой индексээс эхлүүлэх

`includes()` аргад хайлтыг массивын аль хэсгээс эхлүүлэхийг зааж өгөх хоёр дахь параметр дамжуулж болно:

```javascript
let numbers = [10, 20, 30, 40, 50, 30, 60];
console.log(numbers.includes(30, 3)); // true
console.log(numbers.includes(30, 6)); // false
```

Эхний жишээнд бид 30 гэсэн тоог 3-р индексээс эхлэн хайж байна. 3-р индексээс хойш 30 байгаа тул `true` гарна. Харин хоёр дахь жишээнд 6-р индексээс хойш 30 байхгүй тул `false` гарч байна.

Дүгнэж хэлэхэд, `includes()` нь массив доторх өгөгдлийг шалгах маш хүчирхэг, ойлгомжтой хэрэгсэл юм. Энэ нь таныг илүү төвөгтэй давталт (loop) эсвэл нөхцөлт шалгалт бичихээс хамгаална.
