 String болон Array-ийн аргуудыг ашиглан тэмдэгт мөрийг (String) хэрхэн урвуулах вэ?

Тэмдэгт мөрийг урвуулах (reverse) нь програмчлалын түгээмэл даалгавруудын нэг бөгөөд JavaScript-д үүнийг string болон array-ийн аргуудыг хослуулан гурван үндсэн алхмаар гүйцэтгэж болно:

1.  Тэмдэгт мөрийг тэмдэгтүүдийн массив болгон хуваах.
2.  Массивыг урвуулах.
3.  Тэмдэгтүүдийг буцаагаад тэмдэгт мөр болгон нэгтгэх.



Эдгээр алхмуудыг `split()`, `reverse()`, болон `join()` аргуудыг ашиглан дэлгэрэнгүй авч үзэцгээе.

#### 1. split() аргаар массив руу хөрвүүлэх
Эхний алхам бол тэмдэгт мөрийг салангид тэмдэгтүүдийн массив болгон хөрвүүлэх явдал юм. Үүнийг бид `split()` аргаар хийж болно. `split()` арга нь тэмдэгт мөрийг заасан тусгаарлагчид (separator) үндэслэн дэд мөрүүдийн массив болгон хуваадаг. Түгээмэл тусгаарлагчуудын жишээ:

* **Хоосон тэмдэгт мөр (""):** Тэмдэгт мөрийг тэмдэгт бүрээр нь салгана.
* **Зай (" "):** Тэмдэгт мөрийг хоосон зай байгаа хэсгүүдээр салгана (үг бүрээр салгах).
* **Зураас ("-"):** Зураас бүр дээр тэмдэгт мөрийг хуваана.

Тэмдэгтүүдийн массив үүсгэх жишээ:
```javascript
let str = "hello";
let charArray = str.split("");
console.log(charArray); // ["h", "e", "l", "l", "o"]
```
Энд бид `split("")` ашиглан "hello" гэдэг үгийг тэмдэгт бүрээр нь салгаж массив болголоо.

#### 2. reverse() аргаар массивыг урвуулах
Массивтай болсон бол одоо `reverse()` аргыг ашиглан массивын элементүүдийн дарааллыг урвуулж болно. `reverse()` нь эх массивыг шууд өөрчилдөг (in place) гэдгийг санаарай.

```javascript
let charArray = ["h", "e", "l", "l", "o"];
charArray.reverse();
console.log(charArray); // ["o", "l", "l", "e", "h"]
```
Энэ алхамд `charArray`-ийн дараалал ["h", "e", "l", "l", "o"]-оос ["o", "l", "l", "e", "h"] болж өөрчлөгдөнө.

#### 3. join() аргаар буцаагаад тэмдэгт мөр болгох
Эцсийн алхам бол урвуулсан тэмдэгтүүдийн массивыг буцаагаад тэмдэгт мөр болгон хөрвүүлэх юм. Үүнийг `join()` аргаар гүйцэтгэнэ. `join()` арга нь массивын бүх элементийг заасан тусгаарлагчаар холбож, шинэ тэмдэгт мөр үүсгэдэг. Хэрэв ямар нэгэн тусгаарлагчгүйгээр холбохыг хүсвэл аргумент болгон хоосон тэмдэгт мөр ("") өгнө.

```javascript
let reversedArray = ["o", "l", "l", "e", "h"];
let reversedString = reversedArray.join("");
console.log(reversedString); // "olleh"
```

#### Нэг мөрөнд бичих (Method Chaining)
Та эдгээр бүх алхмыг нэгтгэн нэг мөрөнд бичиж болно:
```javascript
let str = "hello";
let reversed = str.split("").reverse().join("");
console.log(reversed); // "olleh"
```

JavaScript-д тэмдэгт мөрүүд нь **immutable** (хувиршгүй) байдаг тул та тэмдэгт мөрийг шууд өөрчлөх замаар урвуулах боломжгүй. Тиймээс бид заавал массив руу хөрвүүлж, массивыг урвуулаад, дараа нь буцааж тэмдэгт мөр болгох шаардлагатай байдаг.
