### Массивын элементэд хэрхэн хандах болон шинэчлэх вэ?

Өмнөх хичээлээр та массивын тухай болон түүний элементүүдэд хандах анхны ойлголттой болсон. Массивын хоёр дахь элементэд хэрхэн хандахыг дахин сануулъя:

```javascript
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

Массивын индекс тэгээс эхэлдэг тул эхний элемент 0 индекс дээр, хоёр дахь элемент 1 индекс дээр гэх мэтээр байрлана. Хэрэв та массивт байхгүй индекс рүү хандахыг оролдвол JavaScript `undefined` утга буцаана гэдгийг анхаарах нь чухал юм.



```javascript
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

Энэ жишээнд `fruits` массивт 3 гэсэн индекс байхгүй байна. Тиймээс консол дээр `undefined` гэж харагдана. Одоо массивын элементийг хэрхэн шинэчлэх талаар үзэцгээе. Та тодорхой нэг индекс дээр шинэ утга оноох замаар тухайн элементийг шинэчилж болно.

```javascript
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

Энэ жишээнд бид 1 индекс дээр байсан "banana"-г "blueberry"-ээр сольсон байна. Энэ арга нь тухайн элементийн индексийг мэдэж байгаа тохиолдолд массивын аль ч элементийг өөрчлөх боломжийг олгодог. Мөн та хараахан байхгүй байгаа индекс дээр утга оноож шинэ элемент нэмж болно:

```javascript
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "banana", "cherry", "date"]
```

Гэхдээ үүнийг хийхдээ анхааралтай байх хэрэгтэй. Хэрэв та массивын одоогийн уртаас (length) хамаагүй их индекст утга оноовол, дунд нь байгаа индексүүдэд `undefined` элементүүд үүсэх бөгөөд энэ нь гэнэтийн үр дүнд хүргэж болзошгүй.



JavaScript-тэй үргэлжлүүлэн ажиллах явцад массивын элементүүдэд хандах болон шинэчлэх эдгээр аргууд нь олон төрлийн програмчлалын даалгавруудын суурь болдог гэдгийг та анзаарах болно.
