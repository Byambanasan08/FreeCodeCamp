### JavaScript Array (Массив)-ийн гол онцлог шинж чанарууд юу вэ?

JavaScript дэх Array буюу массив нь тоон индексээр тодорхойлогддог, эрэмбэлэгдсэн утгуудын цуглуулга юм. JavaScript массивын утгууд нь тоо, тэмдэгт мөр (string), boolean, объект, бүр өөр массивууд гэх мэт өөр өөр өгөгдлийн төрлүүд байж болно.

JavaScript-д массив үүсгэхийн тулд дөрвөлжин хаалт `[]` ашиглаж, утгуудыг таслалаар тусгаарлана. Жишээ нь:

```javascript
let fruits = ["apple", "banana", "orange"];
```

Энэ жишээнд бид `fruits` нэртэй хувьсагч зарлаж, түүнд apple, banana, orange гэсэн гурван тэмдэгт мөрийн утга агуулсан массивыг оноосон байна.

Массивын нэг гол онцлог нь тэдгээр нь "zero-indexed" буюу тэгээс эхэлсэн индекстэй байдаг. Энэ нь массивын эхний элемент 0 индекс дээр, хоёр дахь элемент 1 индекс дээр гэх мэтээр байрлана гэсэн үг юм. Та индекс ашиглан массивын тодорхой элементүүдэд хандаж болно. Жишээ нь:



```javascript
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

Энэ жишээнд бид 0 индексийг ашиглан эхний элементэд (apple), 2 индексийг ашиглан гурав дахь элементэд (orange) хандаж байна.

JavaScript дэх массивууд нь массивт байгаа элементийн тоог буцаадаг тусгай `length` (урт) шинж чанартай байдаг. Та `length` түлхүүр үгийг ашиглан энэ шинж чанарт хандаж болно. Жишээ нь:

```javascript
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

JavaScript массивын өөр нэг гол онцлог нь тэдгээр нь динамик шинжтэй байдаг бөгөөд энэ нь үүсгэгдсэнийхээ дараа хэмжээ нь өөрчлөгдөх боломжтой гэсэн үг юм. Та `push()`, `pop()`, `shift()`, `unshift()`, `splice()` гэх мэт массивын төрөл бүрийн аргуудыг ашиглан массиваас элемент нэмэх эсвэл хасах боломжтой. Эдгээр аргуудыг дараагийн хичээлүүдээр үзэх болно.



JavaScript массивууд нь таны програм дотор өгөгдөл хадгалахад маш уян хатан бөгөөд хэрэгцээтэй байдаг. Энэхүү модулийн туршид та массивтай ажиллах нь өгөгдлийн цуглуулгыг үр ашигтайгаар удирдах, боловсруулахад хэрхэн тусалдаг болохыг биечлэн харах болно.
