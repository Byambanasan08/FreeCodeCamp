What Are Some Examples of Common JavaScript Errors?

JavaScript-д тохиолддог түгээмэл алдаануудын жишээнүүд юу вэ?

---

Та JavaScript дээр код бичиж байхдаа зайлшгүй алдааны мессежүүдтэй тулгарч байсан. Түгээмэл алдааны мессежүүдийг ойлгох нь алдаа засах (debug хийх)-д илүү үр дүнтэй болж, илүү чадварлаг программист болоход тусална. Хамгийн түгээмэл дөрвөн төрлийн алдаа нь SyntaxError, ReferenceError, TypeError, болон RangeError юм.

SyntaxError нь кодондоо ямар нэг зүйлийг буруу бичих үед гардаг, жишээ нь хаалт (parenthesis) эсвэл дөрвөлжин хаалт (bracket) дутуу бичих. Үүнийг өгүүлбэрийн дүрмийн алдаатай адилтгаж ойлгож болно. Дараах нь array үүсгэх үед хөгжүүлэгчид ихэвчлэн гаргадаг алдааны жишээ юм:

```js
const arr = ["Beau", "Quincy" "Tom"]
```

Array-ийн элемент бүрийг таслалаар (comma) тусгаарлах шаардлагатай, эс бөгөөс алдаа үүснэ.

Хоёр дахь түгээмэл JavaScript алдаа нь ReferenceError юм. ReferenceError нь олон янзаар үүсч болно. Эхний жишээ нь тодорхойлогдоогүй хувьсагч (variable) ашиглах явдал юм.

```js
console.log(price);
```

Энэ жишээнд бид price хувьсагчийг console дээр хэвлэх гэж оролдож байгаа боловч энэ хувьсагч тодорхойлогдоогүй байна. Энэ нь ReferenceError үүсгэнэ.

ReferenceError-ийн өөр нэг жишээ бол let эсвэл const-оор зарласан хувьсагчийг тодорхойлогдохоос нь өмнө ашиглах явдал юм:

```js
console.log(b);
const b = 50;
```

Дээрх жишээ нь Cannot access 'b' before initialization гэсэн алдаа үүсгэнэ.

Гурав дахь түгээмэл алдаа бол TypeError юм. Энэ төрлийн алдаа нь буруу төрлийн (type) өгөгдөл дээр үйлдэл хийхийг оролдох үед гардаг. Дараах нь object дээр map method ашиглах гэж оролдсон жишээ юм:

```js
const developerObj = {
  name: "Jessica",
  country: "USA",
  isEmployed: true
};

developerObj.map()
```

Энэ жишээ нь developerObj.map is not a function гэсэн алдаа үүсгэнэ, учир нь map method нь array-д зориулагдсан бөгөөд object-д хэрэглэгддэггүй.

Сүүлчийн авч үзэх алдаа бол RangeError юм.

RangeError нь таны код JavaScript-ийн зөвшөөрөгдсөн хүрээнээс (range) гадуур утга ашиглахыг оролдсон үед гардаг. Дараах нь array-ийн уртад буруу утга оноож буй жишээ юм:

```js
const arr = [];
arr.length = -1; 
```

Array-ийн урт нь 0 эсвэл түүнээс дээш бүхэл тоо байх ёстой тул -1 гэж оноох нь RangeError үүсгэнэ.

Та JavaScript дээр үргэлжлүүлэн код бичих явцдаа эдгээр төрлийн алдаануудтай тулгарах магадлалтай тул ямар үед, яагаад үүсдэгийг ойлгож байх нь чухал юм.
