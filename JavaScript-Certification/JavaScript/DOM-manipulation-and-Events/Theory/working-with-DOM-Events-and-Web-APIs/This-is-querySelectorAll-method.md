querySelectorAll() арга гэж юу вэ, түүнийг хэзээ ашиглах вэ?

`querySelectorAll()` аргын талаар суралцацгаая. Та энэхүү аргыг ашиглан тодорхой нэг CSS селекторт тохирох DOM-ийн **бүх элементүүдийн жагсаалтыг** авч чадна.

`querySelectorAll()` аргыг дараах байдлаар дуудаж ашиглана:

```javascript
document.querySelectorAll(selectors);
```

Үүнийг `document` объект дээр дуудах бөгөөд CSS селекторуудыг агуулсан тэмдэгт мөрийг (string) аргумент болгон дамжуулна. Аргумент нь заавал хүчинтэй CSS селектор байх ёстой. Үгүй бол `SyntaxError` алдаа заах болно.

`querySelectorAll()` нь заасан CSS селекторт тохирох зангилаануудын цуглуулга болох **NodeList** объектыг буцаадаг.

Энэхүү жагсаалт нь CSS селекторт тохирох элемент бүрт харгалзах нэг Элемент (Element) объектыг агуулна. Хэрэв тохирох элемент олдохгүй бол жагсаалт хоосон байна.

Элементүүд нь HTML баримт бичигт гарч ирсэн дарааллаараа жагсаалтад орно. Жишээлбэл, та тодорхой төрлийн бүх элементүүдийг сонгож болно:

```javascript
document.querySelectorAll("div");
```

Тодорхой класс бүхий бүх элементүүдийг:

```javascript
document.querySelectorAll(".rounded");
```

Тодорхой ID бүхий бүх элементүүдийг:

```javascript
document.querySelectorAll("#logo");
```

Эсвэл тодорхой атрибут бүхий бүх элементүүдийг, жишээ нь хэрэглэгчийг тухайн URL руу хөтлөх бүх холбоосуудыг:

```javascript
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Та мөн илүү нарийн төвөгтэй CSS селекторуудыг ашиглаж болно. Жишээ нь, `ingredients` класс бүхий эрэмбэгүй жагсаалт (`ul`) доторх бүх жагсаалтын элементүүдийг (`li`) ингэж сонгоно:

```javascript
document.querySelectorAll("ul.ingredients li");
```

Энэ арга нь зангилаануудын цуглуулга буцаадаг тул та энэхүү цуглуулгыг хувьсагчид оноож, програмдаа элемент бүр дээр ажиллахад ашиглаж болно. Ерөнхий синтакс нь:

```javascript
const matches = document.querySelectorAll(selectors); 
```

Жишээлбэл, хэрэв танд гурван жагсаалтын элемент бүхий ийм эрэмбэгүй жагсаалт байвал:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Та `ingredients` класстай `ul` доторх бүх `li` элементүүдийг сонгохын тулд `querySelectorAll()` аргыг дуудаж, буцаах утгыг нь `matches` тогтмолд оноож болно:

```javascript
const matches = document.querySelectorAll("ul.ingredients li");
```



Энэ нь гурван зангилаанаас бүрдсэн цуглуулгыг буцаана. Зангилаа бүр нь нэг жагсаалтын элементийг төлөөлнө:

```javascript
// NodeList(3){
//   0: <li>Flour</li>,
//   1: <li>Cheese</li>,
//   2: <li>Water</li>,
//   length: 3,
// }
```

Та энэхүү цуглуулгатай яг л JavaScript-ийн бусад массивтай ажилладаг шиг ажиллаж болно.
Жишээлбэл, та үүнийг `console.log()` ашиглан консол дээр харуулж болно:

```javascript
console.log(matches);
```

Мөн `length` шинж чанарыг (property) ашиглан цуглуулгын уртыг шалгаж болно:

```javascript
console.log(matches.length);
```

Үр дүн нь цуглуулга дахь зангилааны тоог илэрхийлэх бүхэл тоо байна:
`3`



Мөн цуглуулгын элемент бүрт харгалзах индексээр нь хандаж болно. Хамгийн эхний элемент 0 индекс дээр байна:

```javascript
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Гаралт нь ийм байх болно:
`<li>Flour</li>`
`<li>Cheese</li>`
`<li>Water</li>`

Элементүүд консол дээр нэг нэгээрээ хэвлэгдэнэ.
Та мөн `for` давталт ашиглан элементүүд дээгүүр гүйж (iterate) болно:

```javascript
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);
}
```

Энэ тохиолдолд үр дүн нь яг ижилхэн байх болно. Гэхдээ энэ арга нь цуглуулгын бүх элементүүдтэй ажиллахад маш хүчирхэг хэрэгсэл болдог.

JavaScript-ийн `querySelectorAll()` арга нь CSS селекторт үндэслэн олон элементийг нэгэн зэрэг сонгох маш хүчирхэг хэрэгсэл юм. Энэхүү аргын тусламжтайгаар та вэб хуудсан дээрх тодорхой элементүүдийг сонгож, тэдгээрийг хэрэгцээнийхээ дагуу удирдаж чадна.

---

**`querySelector` (Ганц элемент) болон `querySelectorAll` (Олон элемент) хоёрын ялгаа тодорхой байна уу? NodeList-ийг массив шиг ашиглаж болдог ч зарим нэг массив функцууд дээр ялгаа гардаг тал бий. Хэрэв та сонирхвол энэ талаар дэлгэрүүлж болох юм.**
