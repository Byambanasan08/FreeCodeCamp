 Event объект гэж юу вэ?

**Event объект** гэдэг нь хэрэглэгч таны вэб хуудастай ямар нэг байдлаар харилцах үед өдөөгддөг өгөгдлийн багц (payload) юм. Эдгээр харилцан үйлдэл нь товчлуур дээр дарах, оруулах талбарт (input) төвлөрөх (focus), эсвэл бүр гар утсаа сэгсрэх зэрэг юу ч байж болно.



Бусад бүх JavaScript объектуудын адил Event объект нь хэрэгцээтэй олон шинж чанаруудыг (properties) агуулдаг. Ямар шинж чанарууд ашиглах боломжтой байгаа нь тухайн өгөгдлийг өдөөсөн "event"-ээс хамаарна.

#### Үндсэн шинж чанарууд:

* **type:** Бүх Event объектууд `type` шинж чанартай байна. Энэ нь тухайн үйлдлийг өдөөсөн event-ийн төрлийг илчилдэг (жишээ нь "keydown" эсвэл "click"). Эдгээр утгууд нь таны `addEventListener()`-д дамжуулдаг утгуудтай ижил байх бөгөөд тэнд та Event объектыг барьж авч, ашиглах боломжтой.
* **target:** Event объектууд үргэлж `target` шинж чанартай байна. Энэ нь тухайн event-ийг өдөөсөн объект руу заасан холбоос (reference) юм. Ихэнх тохиолдолд энэ нь ямар нэгэн `HTMLElement` объект, эсвэл `Document`, `Window` объектууд байдаг. Гэхдээ `AudioContext` гэх мэт илүү нарийн зүйлс ч байж болно.



#### Event-ийн аргууд (Methods):

Event-үүд нь мөн объект дээрх шинж чанар хэлбэрээр илэрхийлэгдсэн функцүүд буюу аргуудтай байдаг. 

* **preventDefault():** Түгээмэл хэрэглэгддэг нэг арга бол `preventDefault()` юм. Үүнийг дуудсанаар тухайн event-ийн анхдагч үйлдлийг (default behavior) зогсоодог. 
    > *Жишээ нь: Хэрэв та "form"-ын илгээлтийг (submission) өөрөө удирдахыг хүсвэл, хөтчийг тухайн өгөгдлийг POST хүсэлт болгон илгээхээс нь сэргийлж `preventDefault()`-ийг дуудаж болно. Та POST хүсэлтийн талаар ирээдүйн хичээлүүдээр илүү ихийг үзэх болно.*
* **stopPropagation():** Та мөн `stopPropagation()` аргатай тааралдах байх. Энэ арга нь тухайн event-ийг эцэг элементүүд рүү "хөөсрөх" (bubbling) буюу дамжихаас сэргийлдэг. Энэ нь ямар утгатай болохыг та ирээдүйн хичээлээр сурах болно.

#### Тусгай шинж чанарууд:

Мөн Event объектын тодорхой төрлүүдэд зориулсан маш олон тооны шинж чанарууд байдаг. Жишээлбэл, `FetchEvent` нь тухайн event-ийг өдөөсөн хүсэлтийг агуулсан `request` шинж чанартай байх жишээтэй.

Хэрэв та ямар шинж чанарууд ашиглах боломжтойг мэдэхгүй эргэлзэж байвал, тухайн Event объектыг консол дээр хэвлэж үзэх (log) эсвэл албан ёсны баримт бичгийг (documentation) шалгаж болно.
