API болон Вэб API гэж юу вэ?

API болон тэдгээр нь вэб хөгжүүлэлтэд яагаад зайлшгүй шаардлагатай болох талаар суралцацгаая. 

**API** гэдэг нь **Application Programming Interface** (Хэрэглээний програмын интерфейс) гэсэн үгний товчлол юм. API нь програм хангамжууд хоорондоо харилцах, өгөгдөл үр ашигтайгаар солилцох боломжийг олгодог багц дүрэм болон протоколуудыг тогтоож өгдөг.

Та үүнийг аль хэдийн хэрэгжсэн энгийн "барилгын блокууд" дээр үндэслэн илүү нарийн төвөгтэй функцийг бүтээх боломжийг хөгжүүлэгчдэд олгодог бүтэц гэж ойлгож болно.



### Вэб API (Web APIs)

API-ийн олон төрөл байдаг бөгөөд **Вэб API** нь тусгайлан вэб аппликейшнд зориулагдсан байдаг. Клиент талын (client-side) JavaScript хөгжүүлэлтэд зориулсан өөр өөр төрлийн вэб API-ууд байх ба тэдгээр нь JavaScript хэлний өөрийнх нь хэсэг биш юм.

Эдгээр төрлийн API-уудыг үндсэн хоёр ангилалд хуваадаг: **Browser APIs** (Хөтчийн API) болон **Third-party APIs** (Гуравдагч талын API).

---

### 1. Browser APIs (Хөтчийн API)

Хөтчийн API-ууд нь вэб хөтчөөс (Chrome, Firefox г.м) өгөгдлийг ил гаргаж өгдөг. Вэб хөгжүүлэгчийн хувьд та JavaScript ашиглан эдгээр өгөгдөлд хандах, удирдах боломжтой.

Тэд мөн вэбсайтын бүтцийг өөрчлөх, үйл явдлыг (events) боловсруулах, дата хадгалах, сүлжээгээр харилцах зэрэг олон төрлийн функцүүдэд хандах боломжийг олгодог. 

Түгээмэл хэрэглэгддэг хөтчийн API-уудын жишээ:
* **DOM API:** HTML элементүүд, тэдгээрийн загвар (style) болон атрибутуудыг удирдахад ашигладаг. Энэ бол вэб хөгжүүлэлтийн гол цөм ойлголт юм.
* **Storage API:** Хэрэглэгчийн төхөөрөмж дээр өгөгдлийг локал байдлаар хадгалахад ашиглана.
* **Geolocation API:** Хэрэглэгчийн одоогийн байршлын мэдээллийг авахад ашиглана.

Хөтчийн API-ууд нь хүчирхэг вэб аппликейшн бүтээхэд туслах хэрэгсэл бөгөөд тэдгээр нь хөтөч дотор аль хэдийн суурилагдсан байдаг.



---

### 2. Third-party APIs (Гуравдагч талын API)

Гуравдагч талын API-ууд нь хөтөч дээр анхнаасаа суудаггүй. Тэдгээрийг ашиглахын тулд та ямар нэгэн байдлаар тэдний кодыг татаж авах эсвэл холбогдох шаардлагатай. Ихэвчлэн тэдгээр нь үйлчилгээгээ хэрхэн ашиглахыг тайлбарласан нарийн бичиг баримт (documentation)-тай ирдэг.

Жишээ нь:
* **Google Maps API:** Вэбсайт дээрээ интерактив газрын зураг харуулахад ашигладаг.
* Бусад төрлүүд: Цаг агаарын API, сошиал медиа API, төлбөрийн API (Stripe г.м), орчуулгын API гэх мэт. Та зөвхөн өөрийн хэрэгцээнд тохирох API-г олоход л хангалттай.



### Дүгнэлт

Вэб API-ууд нь янз бүрийн үйлчилгээ болон өгөгдлийн эх сурвалжтай харилцаж чадах орчин үеийн вэб аппликейшн бүтээхэд зайлшгүй шаардлагатай хэрэгсэл юм. Тэдгээрийн хүчийг хэрхэн ашиглах талаар суралцсанаар та динамик, сонирхолтой вэбсайтуудыг бүтээх чадвараа эрс өргөжүүлэх болно.
