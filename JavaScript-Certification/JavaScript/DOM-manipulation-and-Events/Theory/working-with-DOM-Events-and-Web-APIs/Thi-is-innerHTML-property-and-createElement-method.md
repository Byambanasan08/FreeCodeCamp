innerHTML болон createElement() ашиглан шинэ зангилаа хэрхэн үүсгэх вэ?

`innerHTML` болон `createElement()` ашиглан зангилааг хэрхэн үүсгэхийг үзэцгээе. Эхлээд `innerHTML`-ээс эхэлье.

#### 1. innerHTML ашиглах

`innerHTML` нь Element объектуудын нэг шинж чанар (property) бөгөөд үүнийг ашиглан тухайн элементийн HTML бүтцийг тохируулж болдог. `innerHTML`-ийн тусламжтайгаар та бэлэн байгаа элементийн HTML бүтцийг тэмдэгт мөр (string) ашиглан зааж өгч, шаардлагатай бүх зангилаануудыг нэгэн зэрэг үүсгэх боломжтой.

Жишээг харна уу. Бидэнд нэг хоосон `div` байгаа бөгөөд бид `innerHTML` ашиглан түүний дотор зарим элементүүдийг нэмэх болно:

```html
<div id="container">
  </div>
```

Эхлээд бид тухайн элементийг ID-аар нь сонгох хэрэгтэй:

```javascript
const container = document.getElementById("container");
```

Дараа нь бид `container`-ийн `innerHTML` шинж чанарт тэмдэгт мөр оноож өгнө. Энэхүү тэмдэгт мөр нь таны үүсгэхийг хүсэж буй бүтэц болон зангилаануудыг илэрхийлэхэд шаардлагатай бүх HTML бичиглэлийг агуулсан байх ёстой. Үүнийг тэмдэгт мөр дотор HTML бичиж байна гэж ойлгож болно.

```javascript
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

Бид энэ элементийн `innerHTML`-ийг пиццаны орцуудын (бяслаг болон улаан лооль) эрэмбэгүй жагсаалт болгон тохирууллаа. Хэрэв HTML бүтэц нь урт бол илүү ойлгомжтой болгох үүднээс тэмдэгт мөрийг олон мөр дамнуулан бичиж болно. Хамгийн гол нь бичиглэл таны үүсгэхийг хүссэн HTML бүтцийг үнэн зөв илэрхийлэх ёстой.



Энэ кодыг ажиллуулсны дараа та HTML бүтцээ шалгаж үзвэл дараах байдалтай харагдах болно:

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Тэмдэгт мөрийг хөрвүүлсний (parsed) дараа шинэ зангилаанууд үүсэж, DOM-д динамикаар нэмэгдсэн байна.

**Аюулгүй байдлын санамж:**
`innerHTML` нь тодорхой тохиолдолд маш хэрэгтэй боловч аюулгүй байдлын эрсдэл дагуулдаг гэдгийг мэдэх хэрэгтэй. Хэрэв та тухайн тэмдэгт мөрийг бүрэн хянаж чадахгүй бол үүнийг ашиглах ёсгүй. Жишээ нь, хэрэв тэмдэгт мөр нь хэрэглэгчийн оруулсан өгөгдөл байвал `innerHTML`-ийг бүү ашиглаарай. Учир нь хэрэглэгч таны вэбсайт руу хортой код (malicious content) оруулж болзошгүй. Тийм учраас зөвхөн энгийн текст оруулахдаа үүний оронд `textContent`-ийг ашиглахыг зөвлөдөг.

#### 2. createElement() ашиглах

Шинэ зангилаа үүсгэх өөр нэг арга бол `createElement()` аргыг ашиглах юм. Энэхүү аргын тусламжтайгаар та элементийн "tag" нэрийг зааж өгөх замаар шинэ элемент үүсгэж чадна.

Жишээлбэл, хэрэв та зураг (`img`) элемент үүсгэхийг хүсвэл энэ аргыг дуудахдаа "img" гэсэн тэмдэгт мөр дамжуулна:

```javascript
document.createElement("img");
```

Мөн та энэхүү шинэ объектыг хувьсагчид оноож болно:

```javascript
const img = document.createElement("img");
```



`createElement()` арга нь хэрэв баримт бичиг нь HTML баримт бичиг бол шинэ `HTMLElement` объектыг, үгүй бол `Element` объектыг буцаадаг.

Та шинэ элементээ бэлэн болгосны дараа `appendChild()` аргыг ашиглан өөр нэг бэлэн байгаа элементийн "хүүхэд" болгож DOM-д нэмж болно, эсвэл бусад аргуудыг ашиглан тодорхой байршилд оруулж болно.

```html
<div id="container"></div>
<script src="index.js"></script>
```

```javascript
const container = document.getElementById("container");
const img = document.createElement("img");

img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";

// Шинэ үүсгэсэн img элементийг container дотор нэмж байна
container.appendChild(img);
```

`innerHTML` болон `createElement()` ашиглан зангилаа үүсгэх нь вэбсайтын бүтэц болон агуулгыг динамикаар удирдах боломжийг танд олгоно. Эдгээр техникүүдийг хослуулснаар та интерактив вэб аппликейшнүүдийг бүтээж чадна.
