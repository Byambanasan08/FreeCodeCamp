Үүрлэсэн (Nested) объект болон массив дахь объектын шинж чанарт хэрхэн хандах вэ?

JavaScript-тэй ажиллах явцад та объектын дотор өөр объектууд болон массивууд агуулсан нарийн төвөгтэй өгөгдлийн бүтэцтэй байнга тулгарах болно. Эдгээр бүтэц нь баялаг, шатлалтай өгөгдлийг илэрхийлж чаддаг ч тэдгээрийн доторх өгөгдөлд хэрхэн хандах, удирдах талаар тодорхой ойлголттой байхыг шаарддаг. Эдгээр үүрлэсэн бүтцүүдийг хэрхэн үр дүнтэй удирдах талаар судалцгаая.



Үүрлэсэн объектын шинж чанарт хандахдаа энгийн объектын нэгэн адил цэгэн бичиглэл (dot notation) эсвэл дөрвөлжин хаалтын бичиглэлийг (bracket notation) ашигладаг. Гэхдээ та үүрлэсэн бүтэц рүү гүн рүү нь орохын тулд эдгээр хандагчдыг (accessors) хэлхээ (chain) болгон ашиглах хэрэгтэй болно.

Жишээлбэл, холбоо барих мэдээллийг агуулсан хүнийг илэрхийлэх үүрлэсэн объектыг авч үзье:

```javascript
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Элис (Alice)-ийн ажлын утасны дугаарт хандахын тулд та шинж чанарын хандагчдыг дараах байдлаар хэлхэж бичнэ:

```javascript
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

Та мөн дөрвөлжин хаалтын бичиглэлийг ашиглаж болно. Энэ нь шинж чанарын нэр нь хоосон зай эсвэл тусгай тэмдэгт агуулсан үед, эсвэл шинж чанарт хандахдаа хувьсагч ашиглаж байгаа үед онцгой хэрэгтэй байдаг:

```javascript
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

Одоо объектын шинж чанаруудын нэг нь массивын утгатай байх үед өгөгдөлд хэрхэн хандахыг харцгаая. Хаягуудын массивыг агуулсан, шинэчлэгдсэн `person` объектыг энд харуулав:

```javascript
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Элис-ийн ажлын хаяг байгаа хотын нэрийг (city) хэрхэн олж авах жишээг энд харуулав:



```javascript
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

Энэ жишээнд `person.addresses` нь хаягуудын массивыг зааж байна. Тухайн массив дахь хоёр дахь хаяг руу хандахын тулд бид дөрвөлжин хаалтын бичиглэл болон индекс 1-ийг ашиглаж байна. Дараа нь тухайн хаягийн объектоос хотыг (city) олж авахын тулд цэгэн бичиглэлийг ашиглалаа.

Нарийн төвөгтэй өгөгдлийн бүтэцтэй ажиллах үед үүрлэсэн объект болон массив дахь шинж чанаруудад хэрхэн хандахыг ойлгох нь маш чухал юм. Ирээдүйн сургалт болон лабораторийн ажилд та ийм төрлийн өгөгдлийн бүтэцтэй ажиллах дадлага хийх боломжтой болно.
