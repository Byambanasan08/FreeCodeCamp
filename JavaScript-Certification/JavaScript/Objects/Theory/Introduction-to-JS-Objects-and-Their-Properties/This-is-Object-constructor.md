Object() байгуулагч (Constructor) гэж юу вэ, түүнийг хэрхэн ашиглах вэ?

JavaScript-д байгуулагч (constructor) гэдэг нь объект үүсгэх болон түүнийг эхлүүлэхэд (initialize) ашиглагддаг тусгай төрлийн функц юм. Энэ нь `new` түлхүүр үгний тусламжтайгаар дуудагддаг бөгөөд шинээр үүсгэгдсэн объектын шинж чанар (properties) болон аргуудыг (methods) тодорхойлж өгдөг.

Энэ хичээлээр бид `Object()` байгуулагчтай хэрхэн ажиллах талаар үзэх болно. `Object()` байгуулагч нь шинэ хоосон объект үүсгэдэг. Жишээ нь:

`new Object()`

Та `new Object()`-ийг дуудах үед энэ нь утга хадгалахад ашиглаж болох шинэ объектыг буцаадаг.



`Object()` байгуулагчийг `new` түлхүүр үгтэй эсвэл үггүйгээр ашиглаж болно. `new` үггүйгээр функц хэлбэрээр дуудахад дамжуулсан утгын төрлөөс хамааран өөр өөрөөр ажилладаг. `Object()` байгуулагчийг `new` түлхүүр үггүйгээр ашигласан жишээг энд харуулав:

```javascript
const num = 42;
const numObj = Object(num); // Тоонд зориулсан объектын бүрхүүл (wrapper) үүсгэж байна

console.log(numObj);
console.log(typeof numObj); // "object"
```

Хоёр дахь `console.log` дээр харж байгаачлан `numObj` нь объект байна. Энэ нь бид `Object()` байгуулагчийг ашиглан орж ирсэн тоон утгыг объект болгон хувиргасантай холбоотой юм.

Хэрэв бид `Object()` байгуулагчид `null` эсвэл `undefined` утга дамжуулбал юу болох вэ?

```javascript
const newObj = new Object(undefined);
console.log(newObj); // {}
```

Үр дүн нь хоосон объект байх болно. `Object()` байгуулагчийг ашиглах өөр нэг тохиолдол бол та ямар нэгэн үл мэдэгдэх төрлийн утгатай ажиллаж байгаа бөгөөд түүнийг объект мөн эсэхийг баталгаажуулах шаардлагатай үе юм. Дараах жишээг харцгаая:

```javascript
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));    // {}
console.log(toObject(true));    // [Boolean: true]
console.log(toObject([1, 2, 3])); // [1, 2, 3]
```

Энэ жишээнд бид `toObject` нэртэй функцтэй байна. Хоёр дахь нөхцөл нь тухайн утга объект төрлийнх эсэхийг шалгаад, хэрэв нөхцөл биелж байвал утгыг буцааж байна. Энэ нөхцөл нь объектууд болон массивуудыг хоёуланг нь шалгана, учир нь массивууд бол объектын тусгай төрөл юм.



Хэрэв нөхцөлүүдийн аль нь ч биелэхгүй бол функц `Object(value)`-ийг буцаах бөгөөд энэ нь орж ирсэн утгыг объект болгон хувиргадаг. Энэ нь тоо, тэмдэгт мөр, болон boolean утгуудын хувьд ажилладаг.

Ихэнх тохиолдолд та шинэ объект үүсгэхдээ `Object()` байгуулагчийг ашиглахгүй, харин үүний оронд объектын литерал бичиглэлийг (object literal syntax) ашиглах болно (жишээ нь: `const objectLiteral = { name: "Beau" }`). Гэхдээ `Object` байгуулагчтай ажиллах үндсийг ойлгох нь хэрэгтэй хэвээр байна.
