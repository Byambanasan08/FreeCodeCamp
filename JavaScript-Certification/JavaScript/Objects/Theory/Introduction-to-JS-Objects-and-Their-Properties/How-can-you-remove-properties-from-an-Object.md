Объектоос шинж чанарыг (Properties) хэрхэн хасах вэ?

Объектоос шинж чанарыг хасах хэд хэдэн арга байдаг бөгөөд үүнээс `delete` оператор нь хамгийн ойлгомжтой бөгөөд түгээмэл хэрэглэгддэг арга юм.

Та `delete`-ийг ашиглах үед энэ нь тухайн объектоос сонгосон шинж чанарыг устгадаг. `delete` операторыг хэрхэн ашиглах жишээг энд харуулав:



```javascript
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

Энэ жишээнд бид `name`, `age`, болон `job` гэсэн гурван шинж чанартай `person` объектаос эхэлж байна. Дараа нь бид `delete` операторыг ашиглан `job` шинж чанарыг устгалаа. Устгасны дараа `person` объект нь `job` шинж чанаргүй болсон байна.

Шинж чанарыг хасах өөр нэг арга бол **rest** параметр бүхий **деструктурчлал** (destructuring assignment) ашиглах явдал юм. Энэ арга нь шинж чанарыг шууд устгахгүй боловч заасан шинж чанаруудыг агуулаагүй шинэ объект үүсгэдэг:



```javascript
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

console.log(remainingProperties); // { name: "Bob", age: 25 }
```

Энэ жишээнд бид деструктурчлал ашиглан `person` объектаос `job` болон `city`-г салган авч, үлдсэн шинж чанаруудыг `remainingProperties` нэртэй шинэ массивт цуглуулж байна. Энэ нь `job` болон `city` шинж чанаруудыг агуулаагүй шинэ объектыг бий болгож байгаа юм.

Объектоос шинж чанарыг хэрхэн хасахыг ойлгох нь JavaScript програмчлалын чухал ур чадвар юм. Энэ нь танд объектуудыг динамикаар удирдах, шаардлагагүй өгөгдлүүдийг цэвэрлэх боломжийг олгодог.
