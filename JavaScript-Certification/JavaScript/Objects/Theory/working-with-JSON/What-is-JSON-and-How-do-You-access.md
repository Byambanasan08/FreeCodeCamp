JSON гэж юу вэ, утгуудад нь Bracket болон Dot notation ашиглан хэрхэн хандах вэ?

JSON гэж юу вэ, түүний утгуудад дөрвөлжин хаалт (bracket) болон цэгэн (dot) бичиглэл ашиглан хэрхэн хандах вэ? JSON гэдэг нь **JavaScript Object Notation** гэсэн үгний товчлол юм. Энэ нь сервер болон вэб програмын хооронд өгөгдөл солилцоход түгээмэл хэрэглэгддэг, хөнгөн жинтэй, текстэд суурилсан өгөгдлийн формат юм.



JSON вэб хөгжүүлэлтэд ийм алдартай байгаагийн нэг шалтгаан нь энэ нь машин уншиж/боловсруулах боломжтой (machine-parseable) төдийгүй хүн уншиж ойлгоход хялбар (human-readable) байдагт оршино. JSON нь ямар нэгэн програмчлалын хэлээс хамааралгүй (language-independent) учраас та JSON өгөгдлийг Java програм бүхий серверээс Python програм руу, эсвэл JavaScript-ээс C# програм руу хялбархан илгээж болно.

JSON нь объект, массив, тэмдэгт мөр (string), boolean, null болон тоо зэрэг олон төрлийн өгөгдлийг дэмждэг. JSON объектын жишээг энд харуулав:

```json
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Таны харж байгаагаар JSON нь мэдээллийг хадгалахдаа "түлхүүр-утга" (key-value) хослолыг ашигладаг бөгөөд хослол бүр нь таслалаар тусгаарлагддаг. Түлхүүр (key) бүр нь заавал **давхар хашилтад (" ")** бичигдсэн байх ёстой, эсвэл та алдаа авах болно.

JSON объектаос өгөгдөлд хандахын тулд та цэгэн (dot) эсвэл дөрвөлжин хаалтын (bracket) бичиглэлийг ашиглаж болно. Энэ жишээнд бид JSON объектаос `age` (нас) утгад хандахын тулд цэгэн бичиглэлийг ашиглаж байна:

```javascript
import data from "./example.json" with { type: "json" };

console.log(data.age); // 30
```

Энэхүү тодорхой жишээнд "import statement" гэж нэрлэгддэг зүйлийг ашиглаж байгаа бөгөөд энэ нь JSON объектыг тухайн файл руу импортолж оруулснаар бидэнд хандах эрх олгож байгаа юм. Та `import` бичиглэлийн талаар ирээдүйн хичээлүүд дээр илүү ихийг сурах болно.

Та мөн JSON объектаос мэдээлэл авахад дөрвөлжин хаалтын бичиглэлийг ашиглаж болно. Хичээлүүдийн жагсаалт (list of courses) бүхий массив руу хандах жишээг энд харуулав:



```javascript
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Дөрвөлжин хаалтын бичиглэлийг ашиглах нь энэ тохиолдолд онцгой ач холбогдолтой, учир нь түлхүүр (key) нь хоосон зайгаар тусгаарлагдсан олон үгнээс бүрдэж байна. Хэрэв бид цэгэн бичиглэл ашиглахыг оролдвол энэ нь алдаа гарахад хүргэх болно.

Дүгнэж хэлэхэд, JSON нь массив болон үүрлэсэн (nested) объектууд зэрэг олон төрлийн өгөгдлийг хадгалах чадвартай олон талын ач холбогдолтой формат юм. Цэгэн бичиглэл эсвэл дөрвөлжин хаалтын бичиглэлийг ашигласнаар та JSON объект дотор хадгалагдсан утгуудад хялбархан хандаж чадна.
