# JavaScript функцын тойм

1. JavaScript функц

Функцүүд нь тодорхой даалгавар гүйцэтгэх зориулалттай, дахин ашиглах боломжтой кодны блокууд юм.

Функцийг `function` түлхүүр үг, түүний араас нэр, параметрийн жагсаалт болон тухайн даалгаврыг гүйцэтгэх кодны блокоор тодорхойлж болно.

```javascript
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Гаралт: 16
```

- **Аргументууд** гэдэг нь функцийг дуудах үед дамжуулж буй утгууд юм.
- **Функц дуудах (Function call)** гэдэг нь програмын дотор функцийн нэрийг бичиж, араас нь дугуй хаалт (шаардлагатай бол дотор нь аргумент оруулж) ашиглан кодыг ажиллуулах процесс юм.
- Функц ажиллаж дуусахдаа үргэлж ямар нэг утга буцаадаг.
- Өгөгдмөлөөр функцийн буцах утга нь `undefined` байна.
- `return` түлхүүр үг нь функцээс буцах утгыг зааж өгөхөд ашиглагддаг бөгөөд функцийн гүйцэтгэлийг зогсоодог.
- **Өгөгдмөл параметрүүд (Default parameters)** нь функцийг дуудах үед аргумент дамжуулаагүй тохиолдолд ашиглагдах урьдчилан тодорхойлсон утгыг тохируулах боломжийг олгодог. Энэ нь функцийг илүү уян хатан болгож, зарим аргументыг орхигдуулсан үед алдаа гарахаас сэргийлдэг.

```javascript
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + amount * taxRate;
};

console.log(calculateTotal(100)); // Гаралт: 105
```

- **Аноним функц (Anonymous functions)** гэдэг нь нэргүй функц бөгөөд тэдгээрийг хувьсагчид оноож болно. Ингэж хувьсагчид оноосноор тухайн хувьсагч хандах боломжтой хаана ч функцийг дахин ашиглах боломжтой болно.

```javascript
const multiplyNumbers = function (firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Гаралт: 20
```

2. Arrow Functions (Сумт функц)

Arrow функц нь JavaScript дээр функц бичих илүү товчхон арга юм.

```javascript
const calculateArea = (length, width) => {
  const area = length * width;
  return `Тэгш өнцөгтийн талбай бол ${area} квадрат нэгж юм.`;
};

console.log(calculateArea(5, 10)); // Гаралт: "Тэгш өнцөгтийн талбай бол 50 квадрат нэгж юм."
```

- Arrow функцийг тодорхойлоход `function` түлхүүр үг шаардлагагүй.
- Хэрэв та ганцхан параметр ашиглаж байгаа бол параметрийн жагсаалтын гадуурх дугуй хаалтыг орхиж болно.

```javascript
const cube = (x) => {
  return x * x * x;
};

console.log(cube(3)); // Гаралт: 27
```

- Хэрэв функцийн их бие нь ганцхан илэрхийллээс бүрдэж байвал гоёлын хаалт болон `return` түлхүүр үгийг орхиж болно.

```javascript
const square = (number) => number * number;

console.log(square(5)); // Гаралт: 25
```

3. Програмчлалын Scope (Хүрээ)

- **Global scope (Глобал хүрээ):** Энэ нь JavaScript-ийн хамгийн гадна талын хүрээ юм. Глобал хүрээнд зарлагдсан хувьсагчдад кодын хаанаас ч хандах боломжтой бөгөөд тэдгээрийг глобал хувьсагч гэж нэрлэдэг.
- **Local scope (Локал хүрээ):** Энэ нь функц дотор зарлагдсан хувьсагчдыг хэлнэ. Эдгээр хувьсагчдад зөвхөн тухайн зарлагдсан функц дотроос хандах боломжтой бөгөөд тэдгээрийг локал хувьсагч гэж нэрлэдэг.
- **Block scope (Блок хүрээ):** Блок гэдэг нь `if` нөхцөл эсвэл давталт (loops) зэрэг гоёлын хаалтаар `{}` хүрээлэгдсэн багц операторууд юм.
- `let` болон `const` ашиглан блок хүрээ (block scoping) үүсгэх нь хувьсагчийн хандалтад илүү нарийн хяналт тавьж, алдаанаас сэргийлэх болон кодын ажиллагааг илүү тодорхой болгоход тусалдаг.
