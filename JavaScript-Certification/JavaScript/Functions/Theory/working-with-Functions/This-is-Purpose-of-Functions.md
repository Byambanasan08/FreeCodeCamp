### Функцын зорилго юу вэ, тэдгээр нь хэрхэн ажилладаг вэ?

Функцүүд гэдэг нь тодорхой даалгавар гүйцэтгэх эсвэл утга тооцоолоход зориулагдсан, дахин ашиглах боломжтой кодны хэсгүүд юм. Функцийг ямар нэгэн оролт (input) аваад, түүн дээр тодорхой үйлдлүүд хийж, дараа нь гаралт (output) гаргадаг машин гэж төсөөлж болно. Функц зарлах нэг жишээ энд байна:

```javascript
function greet() {
  console.log("Sain uu, Jessica!");
}
```

Энэ жишээнд бид `greet` нэртэй функцийг зарласан байна. Функц дотор "Sain uu, Jessica!" гэсэн зурвасыг консол дээр хэвлэх `console.log` байгаа. Хэрэв бид энэ кодыг ажиллуулахыг оролдвол консол дээр ямар нэгэн зурвас харагдахгүй. Учир нь бид уг функцийг дуудах (call) шаардлагатай.



Функц дуудах буюу ажиллуулах (invocation) гэдэг нь функцийг бодитоор ашиглах эсвэл гүйцэтгэх үйлдэл юм. Функцийг дуудахын тулд функцийн нэрийг бичээд ард нь дугуй хаалт ашиглана:

```javascript
function greet() {
  console.log("Sain uu, Jessica!");
}
greet(); // "Sain uu, Jessica!"
```

Одоо "Sain uu, Jessica!" гэсэн зурвас консол дээр хэвлэгдэх болно. Гэхдээ бид зурвасаа "Sain uu, Nick!" эсвэл "Sain uu, Anna!" гэж хэлүүлэхийг хүсвэл яах вэ? Өөр өөр хэрэглэгчтэй мэндлэх бүрдээ шинэ функц бичихийг бид хүсэхгүй. Үүний оронд бид функцийн параметрүүд (parameters) болон аргументуудыг (arguments) ашиглан дахин ашиглах боломжтой функц үүсгэж болно.

Параметрүүд нь функцийг дуудах үед дамжуулагдах утгуудыг түр орлох байрлал (placeholders) болдог. Тэд функцэд оролт хүлээн авч, тэрхүү оролттой ажиллах боломжийг олгодог. Аргументууд нь функцийг дуудах үед дамжуулж буй бодит утгууд юм. Параметр болон аргумент ашигласан `greet` функцийн шинэчилсэн хувилбар энд байна:

```javascript
function greet(name) {
  console.log("Sain uu, " + name + "!");
}
greet("Alice"); // Sain uu, Alice!
greet("Nick"); // Sain uu, Nick!
```



Энд `name` нь параметр болж байхад "Alice" болон "Nick" гэсэн стринүүд (strings) нь аргумент болж байна. Одоо бид кодынхоо хаана ч өөр өөр аргументтайгаар олон удаа ашиглаж болох дахин ашиглагдах функцтэй боллоо.

Функц ажиллаж дуусахдаа үргэлж ямар нэг утга буцаадаг. Өгөгдмөлөөр (by default) буцах утга нь `undefined` байдаг. Жишээ нь:

```javascript
function doSomething() {
  console.log("Doing something...");
}
let result = doSomething();
console.log(result); // undefined
```

Хэрэв та функцээ тодорхой нэг утга буцаалгахыг хүсэж байвал `return` операторыг ашиглах шаардлагатай. Хоёр утгын нийлбэрийг буцаахад `return` оператор ашигласан жишээ энд байна:

```javascript
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

Функцээс гарсан тэрхүү утгыг кодынхоо дараагийн хэсэгт ашиглах боломжтой байдаг тул та ихэнх тохиолдолд `return` операторыг ашиглах болно.

Бид одоог хүртэл нэртэй функцүүдтэй ажиллаж байсан бол та мөн аноним функц (anonymous function) гэж нэрлэгддэг зүйлийг үүсгэж болно. Аноним функц нь нэргүй функц бөгөөд түүнийг дараах байдлаар хувьсагчид оноож болно:

```javascript
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

Энэ жишээнд бид `sum` нэртэй `const` хувьсагч зарлаад, түүнд `num1` болон `num2`-ын нийлбэрийг буцаадаг аноним функцийг оноосон байна. Ингэснээр бид `sum`-ыг дуудаж, 3 болон 4 гэсэн тоонуудыг дамжуулан 7 гэсэн үр дүнг авах боломжтой болж байна.

Функцүүд нь өгөгдмөл параметрүүдийг (default parameters) дэмждэг бөгөөд энэ нь параметрүүдэд анхны утга оноох боломжийг олгодог. Хэрэв функцийг дуудах үед тухайн параметрт аргумент дамжуулаагүй бол эдгээр өгөгдмөл утгууд ашиглагддаг. Жишээ нь:

```javascript
function greetings(name = "Zochin") {
  console.log("Sain uu, " + name + "!");
}
greetings(); // Sain uu, Zochin!
greetings("Anna"); // Sain uu, Anna!
```

Энэ жишээнд, хэрэв `name`-д ямар нэг аргумент өгөөгүй бол тэр нь өгөгдмөлөөр "Zochin" болж байна.

Дүгнэж хэлэхэд, функцүүд нь танд дахин ашиглах боломжтой, зохион байгуулалттай код бичих боломжийг олгодог. Тэд оролт (параметр) авч, үйлдэл гүйцэтгэж, гаралт (үр дүн) буцааж чадна.
