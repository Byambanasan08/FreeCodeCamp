### Програмчлалд Scope (Хүрээ) гэж юу вэ, Global, Local болон Block Scope хэрхэн ажилладаг вэ?

Програмчлалд "Scope" (хүрээ) гэдэг нь таны кодын өөр өөр хэсгүүд дэх хувьсагчдын харагдах байдал болон тэдгээрт хандах боломжийг хэлнэ. Энэ нь хувьсагчдыг хаанаас дуудаж ашиглах эсвэл өөрчилж болохыг тодорхойлдог. JavaScript-д scope-ийг ойлгох нь цэгцтэй, үр ашигтай, алдаагүй код бичихэд нэн чухал юм. Үндсэн гурван төрлийн scope байдаг: global scope, local scope болон block scope.

Global scope (глобал хүрээ) нь JavaScript програмын хамгийн гадна талын хүрээ юм. Глобал хүрээнд зарлагдсан хувьсагчдад функц болон блок доторх хэсгүүдийг оролцуулан таны кодын хаанаас ч хандах боломжтой. Эдгээр хувьсагчдыг ихэвчлэн глобал хувьсагч гэж нэрлэдэг. Глобал хувьсагч нь ашиглахад тохиромжтой байж болох ч тэдгээр нь нэрний давхцал үүсгэж, кодыг арчлахад хэцүү болгодог тул маш хязгаарлагдмал ашиглах хэрэгтэй. Глобал хувьсагчийн жишээ энд байна:

```javascript
let globalVar = "Би бол глобал хувьсагч";

function printGlobalVar() {
    console.log(globalVar);
}

printGlobalVar(); // "Би бол глобал хувьсагч"
```

Энэ жишээнд `globalVar` нь глобал хүрээнд зарлагдсан тул `printGlobalVar` функц дотроос түүнд хандаж болж байна.

Нөгөө талаас, Local scope (локал хүрээ) гэдэг нь зөвхөн тухайн функц дотроо хандах боломжтой хувьсагчдыг хэлнэ. Локал хүрээний жишээ энд байна:

```javascript
function greet() {
    let message = "Сайн уу, локал хүрээ!";
    console.log(message);
}

greet(); // "Сайн uu, локал хүрээ!"
// console.log(message); // Энэ нь алдаа заах болно
```

Энэ кодонд `message` нь `greet` функц доторх локал хувьсагч юм. Үүнийг функц дотор ашиглаж болох боловч функцээс гадна хандахыг оролдвол алдаа гарна.



Block scope (блок хүрээ) нь ES6 хувилбарт `let` болон `const` түлхүүр үгстэй хамт нэвтэрсэн ойлголт юм. "Блок" гэдэг нь `if` нөхцөл, `for` давталт эсвэл `while` давталт зэрэг гоёлын хаалтаар `{}` хүрээлэгдсэн кодын дурын хэсэг юм. Давталтын тухай ойлголтыг дараагийн хичээлээр үзэх болно.

Блок дотор `let` эсвэл `const` ашиглан зарласан хувьсагчдад зөвхөн тухайн блок дотор л хандах боломжтой. Блок хүрээний жишээ энд байна:

```javascript
if (true) {
    let blockVar = "Би блок дотор байна";
    console.log(blockVar); // "Би блок дотор байна"
}

console.log(blockVar); // Энэ нь алдаа заах болно
```

Энэ жишээнд `blockVar`-д зөвхөн `if` блок дотор хандах боломжтой. Блокноос гадна хандахыг оролдвол алдаа гарна. Эдгээр өөр өөр төрлийн scope-уудыг ойлгох нь хувьсагчийн хандалтыг удирдах болон кодонд гэнэтийн гаж нөлөө (side effects) гарахаас зайлсхийхэд зайлшгүй шаардлагатай.



Глобал хувьсагчийг маш бага ашиглах хэрэгтэй, учир нь тэдгээр нь нэрний давхцал үүсгэж, кодыг арчлахад хэцүү болгодог. Локал хувьсагчид нь кодын өөр өөр хэсгүүдийг тусгаарлахад тусалдаг бөгөөд энэ нь ялангуяа том хэмжээний програмууд дээр ажиллахад хэрэгтэй байдаг. `let` болон `const` ашиглан блок хүрээ үүсгэх нь хувьсагчийн хандалтад илүү нарийн хяналт тавьж, алдаанаас сэргийлэх болон кодын ажиллагааг илүү тодорхой болгоход тусалдаг. Глобал, локал болон блок хүрээний эдгээр үндсэн ойлголтуудыг эзэмших нь илүү ахисан түвшний сэдвүүдийг ойлгох бат бөх суурь болж өгнө.
