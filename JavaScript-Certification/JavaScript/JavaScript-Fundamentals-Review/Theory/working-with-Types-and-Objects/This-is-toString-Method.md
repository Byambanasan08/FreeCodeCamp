 toString() арга гэж юу вэ, тэр нь хэрхэн ажилладаг вэ?

`toString()` арга нь утгыг тэмдэгт мөрт (string) хэлбэрт хөрвүүлдэг JavaScript-ийн үндсэн функц юм. Энэ аргыг та тоо, boolean, массив болон объектууд дээр ашиглаж болно. `toString()`-ийн хамгийн түгээмэл хэрэглээний нэг бол тоог тэмдэгт мөрт хэлбэрт хөрвүүлэх явдал юм. Жишээ нь:

```javascript
const num = 10;
console.log(num.toString()); // "10"
```

Энэ арга нь 2-оос 36 хүртэлх тоо байж болох нэмэлт **radix** (суурь) параметрийг хүлээн авдаг. Энэхүү radix нь тооллын системийг илэрхийлдэг бөгөөд жишээ нь хоёртын системд 2, наймтын системд 8 гэх мэт байна. Хэрэв radix-ыг зааж өгөхгүй бол анхдагч утга нь аравтын систем буюу 10 байна. `toString()` аргад 2-ыг аргумент болгон дамжуулсан жишээг энд харуулав:

```javascript
const num = 10;
console.log(num.toString(2)); // "1010"
```



Үр дүн нь `1010` байх бөгөөд энэ нь аравтын системийн 10 гэсэн тооны хоёртын систем дэх илэрхийлэл юм.

Та мөн `toString()` аргыг ашиглан массив болон объектуудыг тэмдэгт мөр рүү хөрвүүлж болно. Массивууд нь элемент бүрийг тэмдэгт мөр рүү хөрвүүлж, тэдгээрийг таслалаар зааглан нэгтгэдэг өөрийн гэсэн `toString()` үйлдлийн заавартай байдаг. Жишээ нь:

```javascript
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

Энэ жишээнд массивын бүх элементүүд хоорондоо нэгдэж "1,2,3" гэсэн тэмдэгт мөрийг үүсгэж байна.

Харин `toString()` аргыг объектууд дээр ашиглах үед үр дүн нь объектын шинж чанаруудын текстэн хувилбар (stringified version) байхгүй.

```javascript
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```



Энэ жишээнд үр дүн нь объектын анхдагч тэмдэгт мөрт илэрхийлэл болох `[object Object]` байх болно. `person` объектын шинж чанаруудын текстэн хувилбарыг авахын тулд та `JSON.stringify()`-ийг ашиглах шаардлагатай бөгөөд үүнийг та ирээдүйн хичээлүүдээр илүү дэлгэрэнгүй үзэх болно.

Дүгнэж хэлэхэд, `toString()` арга нь утгуудыг тэмдэгт мөр рүү хөрвүүлэхэд ашиглагддаг. Энэ нь өгөгдлийн өөр өөр төрлүүд дээр хэрхэн ажилладагийг ойлгож, өөрийн объектууд дээрээ хэрхэн тохируулан ашиглахыг сурах нь таны JavaScript програм дээр өгөгдөл удирдах, харуулах чадварыг ихээхэн ахиулах болно.
