Number() байгуулагч гэж юу вэ, тэр нь төрөл хувиргахад (Type Coercion) хэрхэн ажилладаг вэ?

`Number()` байгуулагч нь тоон объект (number object) үүсгэхэд ашиглагддаг. Тоон объект нь `isNaN` болон `toFixed` зэрэг хэд хэдэн хэрэгцээт шинж чанар, аргуудыг агуулдаг. `new` түлхүүр үгийг ашиглан `Number()` байгуулагчийг ашигласан жишээг энд харуулав:

```javascript
const myNum = new Number("34");
console.log(typeof myNum); // "object" 
```

Энэ жишээнд бид `Number()` байгуулагч руу тэмдэгт мөрт утга (string literal) дамжуулсан бөгөөд буцах төрөл нь "string" биш "object" байна.



`Number()` байгуулагчийг `new` түлхүүр үггүйгээр функц хэлбэрээр дуудах үед буцах утга нь "primitive number" (энгийн тоон төрөл) байх болно. Ихэнх тохиолдолд та `Number()` байгуулагчийг өгөгдлийн бусад төрлүүдийг тоон төрөл рүү хөрвүүлэхэд ашиглах болно. Тэмдэгт мөрийг тоо руу хөрвүүлэх жишээг энд харуулав:

```javascript
const myNum = Number("100");
console.log(myNum); // 100
console.log(typeof myNum); // "number"
```

Энэ нь хэрэглэгчээс оролт авах үед тухайн тэмдэгт мөрийг тоо руу хөрвүүлж, математик тооцоолол хийхэд маш хэрэгтэй байдаг.

#### Төрөл бүрийн утгуудыг тоо руу хөрвүүлэх нь:

**1. Хоосон тэмдэгт мөр (Empty String):**
Хэрэв та `Number()` байгуулагчийг хоосон тэмдэгт мөрөөр дуудвал үр дүн нь 0 байх болно:
```javascript
const num = Number("");
console.log(num); // 0
```
Учир нь JavaScript тухайн мөрийг задлан уншихыг оролдох бөгөөд ямар нэгэн цифр агуулаагүй тул үр дүн нь тэг болно.

**2. Текст агуулсан тэмдэгт мөр:**
Хэрэв та санамсаргүй тэмдэгтүүд бүхий мөр дамжуулбал үр дүн нь `NaN` буюу "Not a Number" (Тоо биш) байх болно.
```javascript
const num = Number("random");
console.log(num); // NaN
```

**3. Boolean утгууд:**
Boolean утгуудтай ажиллах үед `true` нь 1-ийг буцаадаг (учир нь `true` нэг рүү хөрвөдөг), харин `false` нь 0-ийг буцаадаг.
```javascript
const boolTrue = Number(true);   // 1
const boolFalse = Number(false); // 0
```



**4. Null болон Undefined:**
Хэрэв та `null` дамжуулбал үр дүн нь 0 байх ба `undefined` дамжуулбал үр дүн нь `NaN` байх болно.
```javascript
const undefinedNum = Number(undefined); // NaN
const nullNum = Number(null);           // 0
```

**5. Массив (Arrays):**
Массивуудтай ажиллахдаа хэд хэдэн зүйлийг анхаарах хэрэгтэй:
* Хоосон массив 0-ийг буцаана.
* Ганц тоо агуулсан массив тухайн тоог буцаана.
* Олон тоо агуулсан массив `NaN` буцаана.
* Тэмдэгт мөр (string) агуулсан массив мөн `NaN` буцаана.

```javascript
const emptyArr = Number([]);           // 0
const arrOneNum = Number([7]);         // 7
const arrMultiNum = Number([7, 36]);   // NaN
const arrStr = Number(["str1"]);       // NaN
```

**6. Объект (Objects):**
Объектуудтай ажиллах үед үр дүн нь үргэлж `NaN` байдаг.
```javascript
const obj1 = Number({});          // NaN
const obj2 = Number({key: "val"}); // NaN
```



 Дүгнэлт
Дүгнэж хэлэхэд, та `Number()` байгуулагчийг тоо эсвэл тоон объект үүсгэхээс илүүтэйгээр өгөгдлийн төрөл хөрвүүлэхэд голчлон ашиглах болно.
