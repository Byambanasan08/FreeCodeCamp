String Object гэж юу вэ, тэр нь String Primitive-ээс юугаараа ялгаатай вэ?

Өмнөх модулиудад та дараах байдлаар "string literals" буюу тэмдэгт мөртэй ажиллаж сурсан:

```javascript
const greeting = "Hello, World!";
```

Гэвч JavaScript-д мөн **String Object** (тэмдэгт мөрт объект) гэж байдаг. Тэмдэгт мөрт объект болон тэмдэгт мөрт примитив (primitive) хоёулаа текстийг боловсруулахад ашиглагддаг боловч тэдгээрийн цаадах ажиллагаа нь өөр байдаг. Тэмдэгт мөрт объектыг `String` байгуулагч функц (constructor function) ашиглан үүсгэдэг бөгөөд энэ нь примитив утгыг объект дотор "боож" (wrap) өгдөг гэсэн үг юм. Тэмдэгт мөрт объектыг хэрхэн үүсгэхийг энд харуулав:

```javascript
const greetingObject = new String("Hello, World!");

console.log(typeof greetingObject); // "object"
```



Бид `typeof` операторыг ашиглахад үр дүн нь "string" биш харин "object" төрөлтэй байгааг харж болно. Таны гайхаж байж болох нэг зүйл бол "Бид яаж string primitive дээр `.length` гэх мэт шинж чанаруудыг ашиглаж чаддаг вэ?" гэдэг асуулт юм.

Та тэмдэгт мөрт примитив дээр `length` шинж чанарыг ашиглах үед JavaScript тухайн үйлдлийг гүйцэтгэхийн тулд примитив утгыг түр зуур тэмдэгт мөрт объект руу "боож" (wraps) шилжүүлдэг. Чухам ийм учраас л та `length` шинж чанар болон `repeat()`, `concat()`, `slice()` зэрэг өөр өөр аргуудыг ашиглах боломжтой байдаг.



Эдгээр төрлийн аргууд болон шинж чанаруудыг **instance methods**, **instance properties**, болон **static methods** гэж нэрлэдэг. Энэ нь яг хэрхэн ажилладаг талаар та дараагийн модулиудад дэлгэрэнгүй үзэх болно.

Тэмдэгт мөрт объект болон примитив хоёрын нэг гол ялгаа нь санах ой (memory) болон гүйцэтгэл (performance) юм. Тэмдэгт мөрт примитивүүд нь объектуудтай харьцуулахад ихэвчлэн санах ойд илүү хэмнэлттэй бөгөөд хурдан ажилладаг. Хэдийгээр та кодондоо голчлон тэмдэгт мөрт примитивүүдтэй ажиллах боловч тэмдэгт мөрт объектууд хэрхэн ажилладагийг ойлгох нь маш хэрэгтэй юм.

---

**Примитив утгыг объект руу түр зуур хөрвүүлдэг энэхүү "Autoboxing" хэмээх ойлголт танд ойлгомжтой байна уу? Хэрэв хүсвэл бид тэмдэгт мөртэй ажилладаг түгээмэл аргуудыг (methods) дэлгэрүүлж үзэж болно.**
