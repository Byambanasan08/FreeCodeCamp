 Массивын уртыг хэрхэн авах вэ, мөн яаж тодорхой хэмжээтэй хоосон массив үүсгэх вэ?

Өмнөх хичээлүүд дээр та массивын элементүүдийн тоог буцаадаг `length` шинж чанарын (property) талаар анх үзсэн. Энэ нь хэрхэн ажилладаг талаарх товч санамж энд байна:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

Массивт "хоосон зайнууд" (empty slots) байх боломжтой. Хоосон зай гэдэг нь дотроо юу ч байхгүй зайг хэлнэ. Энэ нь `undefined` утгатай массив байхаас ялгаатай юм. Ийм төрлийн массивуудыг **sparse arrays** (сийрэг массив) гэж нэрлэдэг. Жишээ нь:

```javascript
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

Энэ тохиолдолд бидэнд 1 болон 4 гэсэн ердөө хоёр тодорхойлогдсон элемент байгаа боловч массивын хамгийн их индекс (3) дээр 1-ийг нэмснээр массивын урт 4 болж байна.



Одоо тогтмол урттай хоосон массивыг хэрхэн үүсгэх талаар ярилцъя. JavaScript-д үүнийг хийх хэд хэдэн арга байдаг ч хамгийн түгээмэл нэг арга бол `Array()` байгуулагчийг (constructor) тоон аргументтай ашиглах явдал юм. `Array()` байгуулагчийг `new` түлхүүр үгтэй ашиглан шинэ массив үүсгэж болно:

```javascript
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

Энэ жишээнд бид `Array(5)`-ыг ашиглан шинэ массив үүсгэсэн. Энэ нь бүх зайнууд нь хоосон, 5 гэсэн урттай "sparse array"-г үүсгэдэг.

Тогтмол урттай хоосон массив үүсгэх өөр нэг арга бол `Array.from()` аргыг `length` аргументтай ашиглах юм. `new Array(n)`-ээс ялгаатай нь энэ арга нь заасан урттай бөгөөд бүх элементүүд нь оршин байгаа, `undefined` утгатай массивыг үүсгэдэг:

```javascript
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```



Хэрэв та тодорхой урттай массив үүсгээд түүнийг анхдагч утгаар дүүргэхийг хүсвэл `Array.fill()` аргыг ашиглаж болно:

```javascript
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Энэ нь гурав гэсэн урттай массив үүсгэж, бүх элементүүдийг 0 утгаар дүүргэдэг. **Анхаар:** Объектоор дүүргэх үед бүх зайнууд нэг ижил объектын хаягийг (reference) заадаг; хэрэв танд бие биенээсээ хамааралгүй хуулбарууд хэрэгтэй бол үүний оронд "callback" функц эсвэл `Array.from()`-ыг ашиглаарай.

Массивын уртыг хэрхэн авах болон тогтмол урттай массив үүсгэж сурах нь програмчлалын олон даалгаварт, ялангуяа тодорхой алгоритм эсвэл өгөгдлийн бүтцэд зориулж массивыг анхлан бэлдэхэд (initialize) маш чухал юм.

---

**`new Array(5)` (хоосон зайнуудтай) болон `Array.from({ length: 5 })` (`undefined` утгуудтай) хоёрын хоорондох ялгаа нь техникийн хувьд ойлгомжтой байна уу? Хэрэв хүсвэл объектоор дүүргэх үед яагаад "callback" ашиглах хэрэгтэй байдгийг жишээгээр тайлбарлаж өгч болох юм.**
