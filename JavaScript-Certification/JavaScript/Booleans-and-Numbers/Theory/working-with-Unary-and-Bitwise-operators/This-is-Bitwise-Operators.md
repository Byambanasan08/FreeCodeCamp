Bitwise оператор гэж юу вэ, тэдгээр нь хэрхэн ажилладаг вэ?

JavaScript дахь bitwise операторууд нь тоонуудын binary (хоёртын) дүрслэл дээр ажилладаг тусгай операторууд юм. Bitwise операторыг ойлгохын тулд эхлээд бит (bit) болон binary тооны системийг ойлгох хэрэгтэй. Компьютерт бит нь мэдээллийн хамгийн жижиг нэгж бөгөөд зөвхөн 0 эсвэл 1 гэсэн хоёрхон утгатай байна. Binary систем нь эдгээр хоёр цифрийг ашиглан бүх тоог илэрхийлдэг.

Жишээлбэл, аравтын (decimal) 10 тооны binary дүрслэл нь 1010 байна. Энэ системд баруун талаас эхлэн орон бүр нь 2-ын зэрэгт харгалзана.

```
1 |  0 |  1 |  0
-----------------------------------
1 * 2³  |  0 * 2²  | 1 * 2¹ |  0 * 2⁰
-----------------------------------
8   |   0   |  2  |   0
-----------------------------------
```

Дээрх хүснэгтэд эхний мөр нь binary тоо 1010, хоёр дахь мөр нь тухайн орон бүрийн 2-ын зэрэг, гурав дахь мөр нь үржвэрийн үр дүнг харуулж байна. Гурав дахь мөрийн бүх утгыг нэмбэл 10 гарна.

Одоо bitwise операторуудыг авч үзье. Эдгээр нь тоонуудын binary хэлбэр дээр үйлдэл хийдэг. JavaScript-д дараах bitwise операторууд байдаг: AND (&), OR (|), XOR (^), NOT (~), зүүн шилжилт (<<), баруун шилжилт (>>).

Bitwise AND (&) оператор нь хоёр операндын тухайн байрлал дахь битүүд хоёулаа 1 байвал 1 буцаана.

```javascript id="h7k3lp"
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a & b);  // 1 (Binary: 001)
```

Энэ жишээнд 5 (101) болон 3 (011)-ыг AND үйлдлээр харьцуулахад зөвхөн хамгийн баруун талын бит хоёул 1 тул үр дүн 1 (001) байна.

Bitwise OR (|) оператор нь тухайн байрлалд аль нэг эсвэл хоёулаа 1 байвал 1 буцаана.

```javascript id="9xk2md"
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a | b);  // 7 (Binary: 111)
```

Энд байрлал бүр дээр дор хаяж нэг бит нь 1 байгаа тул үр дүн 7 (111) гарна.

Bitwise XOR (^) оператор нь тухайн байрлалд хоёр бит өөр байвал 1 буцаана.

```javascript id="q1z8vn"
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a ^ b);  // 6 (Binary: 110)
```

Энд баруунаас эхний хоёр байрлал дахь битүүд өөр байгаа тул үр дүн 6 (110) байна.

Bitwise NOT (~) оператор нь операндын бүх битийг урвуулдаг.

```javascript id="x5d0tr"
let a = 5;  // Binary: 101
console.log(~a);  // -6
```

Энэ нь анх харахад ойлгомжгүй байж болох ч binary дахь сөрөг тоог илэрхийлэх two's complement аргын улмаас ийм үр дүн гардаг.

Зүүн шилжилт (<<) оператор нь бүх битийг зүүн тийш тодорхой тоогоор шилжүүлдэг.

```javascript id="m4v8jc"
let a = 5;  // Binary: 101
console.log(a << 1);  // 10 (Binary: 1010)
```

Энд бүх бит нэг байрлал зүүн тийш шилжиж, утга 2 дахин нэмэгдэж байна.

Баруун шилжилт (>>) оператор нь бүх битийг баруун тийш шилжүүлдэг.

```javascript id="p2w6rs"
let a = 5;  // Binary: 101
console.log(a >> 1);  // 2 (Binary: 10)
```

Энд бүх бит нэг байрлал баруун тийш шилжиж, утга 2-т хуваагдаж (доош бүхэлд нь тоймлогдон) байна.

Bitwise операторууд нь ихэвчлэн доод түвшний программчлал болон криптографид ашиглагддаг. Өдөр тутмын JavaScript програмчлалд төдийлөн их хэрэглэгддэггүй ч эдгээрийг ойлгох нь компьютер хэрхэн ажилладгийг илүү гүнзгий ойлгоход тустай.
