Increment ба Decrement операторууд хэрхэн ажилладаг вэ?

JavaScript-д тоон утгыг 1-ээр нэмэх эсвэл хасах шаардлагатай үед increment (`++`) болон decrement (`--`) операторуудыг ашигладаг.

* `++` → утгыг 1-ээр нэмнэ
* `--` → утгыг 1-ээр хасна

Жишээ нь:

```id="a1x2y3"
let x = 5;
x++; // x = 6

let y = 5;
y--; // y = 4
```

Энэ нь `x = x + 1` эсвэл `y = y - 1` бичихээс илүү товч, ойлгомжтой.

Prefix ба Postfix ялгаа

Эдгээр операторууд хоёр хэлбэртэй:

* Prefix → `++x`, `--x`
* Postfix → `x++`, `x--`

Ялгаа нь утга хэзээ өөрчлөгдөхөд оршино.

Prefix (++x)

Эхлээд утгыг өөрчилнө, дараа нь буцаана.

```id="b4c5d6"
let x = 5;

console.log(++x); // 6
console.log(x);   // 6
```

Энд x эхлээд 6 болж, дараа нь ашиглагдаж байна.

Postfix (x++)

Эхлээд одоогийн утгыг ашиглана, дараа нь өөрчилнө.

```id="e7f8g9"
let y = 5;

console.log(y++); // 5
console.log(y);   // 6
```

Энд эхлээд 5 хэвлэгдэж, дараа нь y = 6 болсон.

Decrement (--)

Ижил зарчмаар ажиллана:

Prefix:

```id="h1i2j3"
let x = 5;
console.log(--x); // 4
console.log(x);   // 4
```

Postfix:

```id="k4l5m6"
let y = 5;
console.log(y--); // 5
console.log(y);   // 4
```

Хэзээ аль хэлбэрийг ашиглах вэ?

Жишээ:

```id="n7o8p9"
let a = 5;
let b = ++a;
console.log(b); // 6
```

Энд a эхлээд нэмэгдэж, дараа нь b-д оноогдож байна.

```id="q1r2s3"
let c = 5;
let d = c++;
console.log(d); // 5
```

Энд c-ийн хуучин утга d-д оноогдож, дараа нь c нэмэгдэж байна.

Дүгнэж хэлбэл:

* `++` → 1-ээр нэмнэ
* `--` → 1-ээр хасна
* Prefix → эхлээд өөрчлөөд дараа нь ашиглана
* Postfix → эхлээд ашиглаад дараа нь өөрчилнө

Хэрэв шууд шинэ утга хэрэгтэй бол prefix, харин одоогийн утгыг эхэлж ашиглах бол postfix ашиглана.
