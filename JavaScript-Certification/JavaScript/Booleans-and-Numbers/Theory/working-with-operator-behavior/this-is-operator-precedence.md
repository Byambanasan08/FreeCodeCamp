Operator precedence хэрхэн ажилладаг вэ?

JavaScript-д олон оператортой илэрхийлэл бичихэд аль үйлдлийг эхэлж гүйцэтгэхийг operator precedence (үйлдлийн эрэмбэ) тодорхойлдог.

Operator precedence гэдэг нь илэрхийлэл доторх үйлдлүүдийг ямар дарааллаар бодохыг заадаг дүрэм юм. Өндөр precedence-тэй операторууд эхэлж ажиллана.

Жишээ:

```id="a1b2c3"
const result = 2 + 3 * 4;

console.log(result); // 14
```

Хэрэв зүүнээс баруун руу бодвол:
2 + 3 = 5 → 5 * 4 = 20 болох ёстой мэт санагдана.

Гэхдээ `*` (үржих) нь `+`-аас өндөр precedence-тэй тул:
3 * 4 = 12 → 2 + 12 = 14

Тиймээс зөв хариу нь 14.

Parentheses ()

Хэрэв та тодорхой хэсгийг эхэлж бодохыг хүсвэл хаалт ашиглана:

```id="d4e5f6"
const result = (2 + 3) * 4;

console.log(result); // 20
```

Энд:
2 + 3 = 5 → 5 * 4 = 20

Division (/) мөн өндөр precedence-тэй:

```id="g7h8i9"
const result = 2 + 6 / 3;

console.log(result); // 4
```

Энд:
6 / 3 = 2 → 2 + 2 = 4

Associativity (ижил precedence үед)

Хэрэв операторууд ижил precedence-тэй бол associativity (чиглэл) ашиглана.

Ихэнх операторууд зүүнээс баруун (left-to-right):

```id="j1k2l3"
const result = 10 - 2 + 3;

console.log(result); // 11
```

Энд:
10 - 2 = 8 → 8 + 3 = 11

Зарим операторууд баруунаас зүүн (right-to-left):

Assignment (=):

```id="m4n5o6"
let a, b;
a = b = 5;

console.log(a); // 5
console.log(b); // 5
```

Энд:
эхлээд b = 5 → дараа нь a = b

Exponent (**):

```id="p7q8r9"
const result = 2 ** 3 ** 2;

console.log(result); // 512
```

Энд:
3 ** 2 = 9 → 2 ** 9 = 512

Хэрэв зүүнээс баруун байсан бол:
2 ** 3 = 8 → 8 ** 2 = 64 болох байсан.

Дүгнэж хэлбэл:

* Operator precedence нь аль үйлдлийг эхэлж хийхийг шийддэг
* `*`, `/` нь `+`, `-`-аас түрүүнд хийгддэг
* `()` ашиглан дарааллыг өөрчилж болно
* Ижил precedence үед associativity ашиглана

  * Ихэнх нь → left-to-right
  * Зарим нь → right-to-left

Эдгээр дүрмийг ойлгосноор та илэрхийлэл хэрхэн бодогдохыг зөв таамаглаж чадна.
