parseFloat() болон parseInt() хэрхэн ажилладаг вэ?

parseFloat() болон parseInt() нь JavaScript-д string утгыг number төрөлд хөрвүүлэхэд ашиглагддаг чухал методууд юм. Эдгээр нь хэрэглэгчийн оруулсан утга эсвэл string хэлбэртэй өгөгдлийг тоон утга болгон ашиглах шаардлагатай үед маш хэрэгтэй.

Эхлээд parseFloat()-ийг авч үзье. Энэ метод нь string-ийг задлан уншиж, бутархай (floating-point) тоо буцаадаг. Энэ нь string-ийн эхнээс эхлэн тоон утгыг уншиж, дараа нь тоонд хамаарахгүй тэмдэгт таарах хүртэл үргэлжилдэг.

```javascript id="p4k8xn"
console.log(parseFloat("3.14"));     // 3.14
console.log(parseFloat("3.14 abc")); // 3.14
console.log(parseFloat("3.14.5"));   // 3.14
console.log(parseFloat("abc 3.14")); // NaN
```

Эндээс харахад parseFloat() нь string-ийн эхнээс эхэлж уншдаг. Хэрэв эхэнд нь хүчинтэй тоо байхгүй бол NaN буцаана.

parseInt() нь string-ийг задлан уншиж бүхэл (integer) тоо буцаадаг. parseFloat()-тай адил эхнээс нь уншдаг боловч эхний тоон биш тэмдэгт дээр зогсдог.

```javascript id="z7v3ms"
console.log(parseInt("42"));       // 42
console.log(parseInt("42px"));     // 42
console.log(parseInt("3.14"));     // 3
console.log(parseInt("abc123"));   // NaN
```

parseInt() нь бутархай тооны зөвхөн бүхэл хэсгийг буцаана. Хэрэв эхэнд нь хүчинтэй бүхэл тоо байхгүй бол NaN буцаана.

Эдгээр хоёр методын зарим онцлог:

Эхний хоосон зайг үл тооно:

```javascript id="k2x9dw"
console.log(parseFloat("  3.14"));  // 3.14
console.log(parseInt("  42"));      // 42
```

"+" болон "-" тэмдэгтийг зөв танина:

```javascript id="n5q1rt"
console.log(parseFloat("+3.14"));  // 3.14
console.log(parseInt("-42"));      // -42
```

Гэхдээ эдгээр методууд нь зарим хязгаарлалттай. Жишээлбэл, шинжлэх ухааны бичиглэл (scientific notation) зэрэг бүх төрлийн тоон форматыг бүрэн дэмждэггүй. Илүү төвөгтэй нөхцөлд нэмэлт арга эсвэл сан (library) хэрэглэх шаардлагатай байж болно.

Дүгнэвэл, parseFloat() болон parseInt() нь string-ийг number болгон хөрвүүлэхэд маш хэрэгтэй хэрэгслүүд юм. Эдгээрийн хэрхэн ажилладгийг ойлгосноор та хэрэглэгчийн оролт болон гадаад өгөгдлийг илүү зөв, үр дүнтэй боловсруулах боломжтой болно.
