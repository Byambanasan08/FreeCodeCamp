.toFixed() метод гэж юу вэ, мөн хэрхэн ажилладаг вэ?

.toFixed() метод нь JavaScript-д байдаг дотоод (built-in) функц бөгөөд тоог fixed-point (тогтмол бутархай оронтой) хэлбэрт оруулж форматлахад ашиглагддаг. Энэ нь ялангуяа мөнгөн дүн харуулах, эсвэл нарийвчлал шаардсан хэмжилтүүд дээр бутархайн орны тоог хянах үед маш хэрэгтэй.

.toFixed() нь тоон дээр дуудагддаг бөгөөд нэг optional параметр авдаг. Энэ параметр нь таслалаас хойш хэдэн орон харуулахыг заана. Үр дүнд нь string (тэмдэгт мөр) буцаадаг.

```javascript id="k9w2pz"
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

Энд бид бутархайн орныг 2 болгон хязгаарласан тул 3.14159 → 3.14 болж байна. Анхаарах зүйл нь .toFixed() нь number биш, string буцаадаг.

.toFixed() нь утгыг хамгийн ойрын боломжит утга руу тоймлодог:

```javascript id="m3x7qd"
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

Энд дараагийн орон 5 ба түүнээс их байвал дээш, бага байвал доош тоймлогдоно.

Хэрэв параметр өгөхгүй бол автоматаар 0 орон авна:

```javascript id="t8n4vs"
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

.toFixed() нь санхүүгийн тооцоолол болон үнийн дүн харуулахад маш хэрэгтэй:

```javascript id="p2r6ky"
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

Энд .toFixed(2) ашигласнаар нийт дүнг үргэлж 2 орны нарийвчлалтай харуулж байна. Энэ нь ихэнх улс оронд мөнгөн дүнг илэрхийлэх стандарт хэлбэр юм.

Дүгнэвэл, .toFixed() нь тоог форматлахад маш хэрэгтэй метод бөгөөд бутархайн орны тоог хянах боломжийг олгодог. Гэхдээ энэ нь string буцаадаг гэдгийг анхаарч, тооцоололд ашиглах шаардлагатай бол дахин number төрөлд хөрвүүлэх хэрэгтэй.
