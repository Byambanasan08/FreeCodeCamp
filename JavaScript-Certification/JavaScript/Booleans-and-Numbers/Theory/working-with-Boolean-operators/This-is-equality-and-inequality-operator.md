Boolean гэж юу вэ, мөн тэдгээр нь тэнцүү болон тэнцүү биш операторуудтай хэрхэн ажилладаг вэ?

Өмнөх хичээлд та boolean ойлголттой анх танилцсан, харин энэ хичээлээр бид boolean хэрхэн ажилладаг болон тэнцүү (equality) ба тэнцүү биш (inequality) операторууд хэрхэн ажилладгийг илүү гүнзгий судлах болно.

Boolean нь зөвхөн true болон false гэсэн утгатай өгөгдлийн төрөл юм. Эдгээр нь тодорхой нөхцөлөөс хамаарч ямар нэг үйлдэл хийх боломжийг олгодог тул маш хэрэгтэй. Жишээлбэл, таны апп дээр хэрэглэгч тодорхой боломжид хандаж болох эсэхийг шийдэхэд boolean чухал үүрэгтэй. Доорх жишээнд isOldEnoughToDrive нэртэй хувьсагчид true утга оноож байна:

```javascript
let isOldEnoughToDrive = true;

console.log(isOldEnoughToDrive); // true
```

Та энэ хувьсагчийг дараах байдлаар нөхцөлт шалгалтад ашиглаж болно:

```javascript
let isOldEnoughToDrive = true;

if (isOldEnoughToDrive) {
 console.log("You're old enough to drive"); // You're old enough to drive
} else {
 console.log("Sorry, you are not old enough to drive");
}
```

Нөхцөлт бүтэц (conditional) нь тодорхой нөхцөл дээр үндэслэн код дотор шийдвэр гаргахад тусалдаг. Энэ жишээнд if/else нөхцөл ашигласан байна.

Хэрэв isOldEnoughToDrive нь true байвал "You're old enough to drive" гэсэн өгүүлбэр консолд хэвлэгдэнэ. Харин false байвал "Sorry, you are not old enough to drive" гэж хэвлэгдэнэ. Манай жишээнд хувьсагч true тул эхний өгүүлбэр хэвлэгдэнэ. Та if/else-ийн талаар дараагийн хичээлүүдэд илүү дэлгэрэнгүй судлах болно.

Хоёр утгыг харьцуулахын тулд та тэнцүү (regular equality) эсвэл хатуу тэнцүү (strict equality) оператор ашиглаж болно. Харьцуулалтын үр дүн нь true эсвэл false boolean утга байна. Доорх жишээнд тэнцүү оператор (==) ашиглан string болон number-ийг харьцуулж байна:

```javascript
console.log(5 == "5"); // true
```

Энэ тохиолдолд JavaScript нь "5" string-ийг 5 тоо болгон хөрвүүлээд дараа нь харьцуулдаг. Хоёр утга ижил болсон тул үр дүн нь true байна. Тэнцүү оператор нь харьцуулахаас өмнө төрөл хувиргалт (type coercion) хийдэг.

Харин хатуу тэнцүү оператор (===) нь төрөл хувиргалт хийдэггүй. Энэ оператор нь өгөгдлийн төрөл болон утга хоёул ижил эсэхийг шалгана. Доорх жишээг харна уу:

```javascript
console.log(5 === '5'); // false
```

Энд үр дүн false гарна. Учир нь string төрөл нь number төрөлтэй ижил биш.

Хэрэв та хоёр утга тэнцүү биш эсэхийг шалгахыг хүсвэл тэнцүү биш (inequality) эсвэл хатуу тэнцүү биш (strict inequality) оператор ашиглаж болно. Доорх жишээнд (!=) оператор ашиглаж байна:

```javascript
console.log(5 != "5"); // false
```

Энэ тохиолдолд string утгыг number болгон хөрвүүлсний дараа харьцуулдаг тул утгууд ижил болж false буцаана.

Харин хатуу тэнцүү биш оператор (!==) ашиглавал өөр үр дүн гарна:

```javascript
console.log(5 !== "5"); // true
```

Энд төрөл хувиргалт хийгдэхгүй тул number 5 нь string "5"-тай тэнцүү биш гэж үзэж true үр дүн гарна.

Ихэвчлэн хатуу тэнцүү (===) болон хатуу тэнцүү биш (!==) операторуудыг ашиглах нь зөв дадал гэж үздэг. Учир нь эдгээр нь төрөл хувиргалт хийдэггүй. Мэргэжлийн төслүүд дээр ихэнхдээ эдгээр операторуудыг илүүд үзэн ашигладаг.
