# JavaScript Математикийн Тойм

1. Number өгөгдлийн төрөлтэй ажиллах

   1.1 Тодорхойлолт: JavaScript-ийн Number төрөлд бүхэл тоо, хөвөгч цэгтэй тоо (floating-point), Infinity болон NaN орно. Хөвөгч цэгтэй тоо гэдэг нь аравтын бутархайтай тоонууд юм. Эерэг Infinity нь бусад бүх тооноос их утга бөгөөд -Infinity нь бусад бүх тооноос бага утга юм. NaN (Not a Number) нь "Jessica" гэх мэт буруу тоон утгыг илэрхийлнэ.

2. Түгээмэл арифметик үйлдлүүд

   2.1 Нэмэх оператор: (+) оператор нь хоёр ба түүнээс олон тооны нийлбэрийг тооцоолоход ашиглагдана.
   2.2 Хасах оператор: (-) оператор нь хоёр тооны ялгаврыг тооцоолоход ашиглагдана.
   2.3 Үржих оператор: (\*) оператор нь хоёр ба түүнээс олон тооны үржвэрийг тооцоолоход ашиглагдана.
   2.4 Хуваах оператор: (/) оператор нь хоёр тооны ноогдворыг тооцоолоход ашиглагдана.
   2.5 Тэгээр хуваах: Хэрэв та тэгээр хуваавал JavaScript Infinity буцаана.
   2.6 Үлдэгдэл оператор: (%) оператор нь хуваалтын үлдэгдлийг буцаана.
   2.7 Зэрэгт дэвшүүлэх оператор: (\*\*) оператор нь нэг тоог нөгөө тооны зэрэгт дэвшүүлнэ.

3. Тоо ба мөртэй тооцоолол хийх

   3.1 Тайлбар: Та + операторыг тоо болон мөр (string)-тэй хамт ашиглахад JavaScript нь тоог мөр болгон хувиргаж хоёр утгыг нийлүүлнэ. Харин -, \*, / операторуудыг мөр болон тоотой ашиглахад JavaScript нь мөрийг тоо болгон хувиргаж үр дүн нь тоо болно. null болон undefined-ийн хувьд JavaScript нь null-ийг 0, undefined-ийг NaN гэж үзнэ.

const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' \* 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number

4. Операторын эрэмбэ (давуу байдал)

   4.1 Тодорхойлолт: Операторын эрэмбэ нь илэрхийлэл доторх үйлдлүүдийг ямар дарааллаар гүйцэтгэхийг тодорхойлно. Илүү өндөр эрэмбэтэй операторууд эхэлж гүйцэтгэгдэнэ. Хаалт доторх утгууд эхэлж бодогдож, үржих/хуваах нь нэмэх/хасахаас илүү давуу байна. Хэрэв операторууд ижил эрэмбэтэй бол JavaScript нь ассоциатив чанарыг ашиглана.

const result = (2 + 3) \* 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512

```
4.2 Тодорхойлолт: Ассоциатив чанар нь ижил төрлийн олон оператор байгаа үед илэрхийллийг ямар чиглэлээр бодохыг заана. Өөрөөр хэлбэл зүүнээс баруун (left-to-right) эсвэл баруунаас зүүн (right-to-left) чиглэлтэй байна. Жишээлбэл, зэрэгт дэвшүүлэх оператор нь баруунаас зүүн чиглэлтэй:
```

const result4 = 5 ** 4 ** 1;

console.log(result4); // 625

5. Нэмэх ба хасах (Increment/Decrement) операторууд

   5.1 Increment оператор: Энэ оператор нь утгыг нэгээр нэмэгдүүлнэ. Prefix хэлбэр (++num) нь эхлээд утгыг нэмэгдүүлээд дараа нь буцаана. Postfix хэлбэр (num++) нь эхлээд одоогийн утгыг буцаагаад дараа нь нэмэгдүүлнэ.

let x = 5;

console.log(++x); // 6
console.log(x); // 6

let y = 5;

console.log(y++); // 5
console.log(y); // 6

```
5.2 Decrement оператор: Энэ оператор нь утгыг нэгээр багасгана. Prefix болон postfix хэлбэр нь дээрхтэй адил ажиллана.
```

let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3

6. Нийлмэл оноох операторууд

   6.1 Нэмэх оноох (+=) оператор: Утгуудыг нэмээд үр дүнг хувьсагчид онооно.
   6.2 Хасах оноох (-=) оператор: Утгуудыг хасаад үр дүнг хувьсагчид онооно.
   6.3 Үржих оноох (\*=) оператор: Утгуудыг үржээд үр дүнг хувьсагчид онооно.
   6.4 Хуваах оноох (/=) оператор: Утгуудыг хуваагаад үр дүнг хувьсагчид онооно.
   6.5 Үлдэгдэл оноох (%=) оператор: Хувьсагчийг өгөгдсөн тоонд хувааж үлдэгдлийг онооно.
   6.6 Зэрэгт дэвшүүлэх оноох (\*\*=) оператор: Хувьсагчийг өгөгдсөн тооны зэрэгт дэвшүүлж үр дүнг дахин онооно.

7. Boolean ба тэнцүү байдлын операторууд

   7.1 Boolean тодорхойлолт: Boolean нь зөвхөн хоёр утгатай өгөгдлийн төрөл: true эсвэл false.
   7.2 Тэнцүү (==) оператор: Энэ оператор нь утгуудыг харьцуулахаас өмнө төрлийг хувиргана.

console.log(5 == '5'); // true

7.3 Хатуу тэнцүү (===) оператор: Энэ оператор нь төрөл хувиргалт хийхгүй бөгөөд төрөл болон утгыг хоёуланг нь шалгана.

console.log(5 === '5'); // false

7.4 Тэнцүү биш (!=) оператор: Энэ оператор нь төрөл хувиргалт хийсний дараа тэнцүү биш эсэхийг шалгана.
7.5 Хатуу тэнцүү биш (!==) оператор: Энэ оператор нь төрөл хувиргалт хийхгүй бөгөөд төрөл болон утгыг хоёуланг нь шалгана.

8. Харьцуулах операторууд

   8.1 Их (>) оператор: Зүүн талын утга баруун талаасаа их эсэхийг шалгана.
   8.2 Их эсвэл тэнцүү (>=) оператор: Зүүн талын утга баруун талаасаа их эсвэл тэнцүү эсэхийг шалгана.
   8.3 Бага (<) оператор: Зүүн талын утга баруун талаасаа бага эсэхийг шалгана.
   8.4 Бага эсвэл тэнцүү (<=) оператор: Зүүн талын утга баруун талаасаа бага эсвэл тэнцүү эсэхийг шалгана.

9. Unary операторууд

   9.1 Unary plus оператор: Энэ оператор нь операндыг тоо болгон хувиргана. Хэрэв аль хэдийн тоо бол өөрчлөгдөхгүй.

const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number

```
9.2 Unary negation (-) оператор: Энэ оператор нь операндыг эсрэг тэмдэгтэй болгоно.
```

const num = 4;
console.log(-num); // -4

```
9.3 Logical NOT (!) оператор: Энэ оператор нь boolean утгыг урвуулна. true бол false, false бол true болно.
```

10. Bitwise операторууд

10.1 Bitwise AND (&) оператор: Хоёр операндын ижил байрлал дахь битүүд хоёулаа 1 бол 1 буцаана.
10.2 Bitwise AND Assignment (&=) оператор: Bitwise AND үйлдэл хийж үр дүнг дахин онооно.
10.3 Bitwise OR (|) оператор: Аль нэг эсвэл хоёулаа 1 бол тухайн байрлалд 1 буцаана.
10.4 Bitwise OR Assignment (|=) оператор: Bitwise OR үйлдэл хийж үр дүнг дахин онооно.
10.5 Bitwise XOR (^) оператор: Аль нэг нь 1, нөгөө нь 0 байвал 1 буцаана.
10.6 Bitwise NOT (~) оператор: Тооны хоёртын дүрслэлийг урвуулна.
10.7 Зүүн тийш шилжүүлэх (<<) оператор: Бүх битүүдийг зүүн тийш шилжүүлнэ.
10.8 Баруун тийш шилжүүлэх (>>) оператор: Бүх битүүдийг баруун тийш шилжүүлнэ.

11. Нөхцөлт өгүүлбэр, truthy/falsy утгууд ба ternary оператор

    11.1 if/else if/else: if өгүүлбэр нь нөхцөл авч, нөхцөл truthy байвал код ажиллуулна. Хэрэв false бол else if хэсэг рүү шилжинэ. Аль нь ч үнэн биш бол else хэсгийг гүйцэтгэнэ. Truthy утгууд нь Boolean нөхцөлд true болж үнэлэгддэг утгууд юм. Falsy утгууд нь false болж үнэлэгддэг.

const score = 87;

if (score >= 90) {
console.log('You got an A');
} else if (score >= 80) {
console.log('You got a B'); // You got a B
} else if (score >= 70) {
console.log('You got a C');
} else {
console.log('You failed! You need to study more!');
}

```
11.2 Ternary оператор: if else-г богино хэлбэрээр бичихэд ашиглагддаг.
```

const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!

12. Хоёртын логик операторууд

12.1 Logical AND (&&) оператор: Хоёр операнд хоёулаа true эсэхийг шалгана. Хэрэв хоёулаа true бол хоёр дахь утгыг буцаана. Хэрэв аль нэг falsy бол тэр утгыг буцаана. Хэрэв хоёулаа falsy бол эхний falsy утгыг буцаана.

const result = true && 'hello';

console.log(result); // hello

12.2 Logical OR (||) оператор: Аль нэг операнд truthy байвал true гэж үзнэ.
12.3 Nullish coalescing (??) оператор: Эхний утга null эсвэл undefined байвал дараагийн утгыг буцаана.

const userSettings = {
theme: null,
volume: 0,
notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light

14. Түгээмэл Number методууд

14.1 isNaN(): NaN нь "Not-a-Number" гэсэн утгатай. Энэ нь тодорхойгүй эсвэл илэрхийлэгдэх боломжгүй тоон үр дүнг илэрхийлнэ. isNaN() функц нь утга NaN эсэхийг шалгана. Number.isNaN() нь илүү найдвартай шалгах арга юм, ялангуяа төрөл хувиргалтаас шалтгаалсан алдаанаас сэргийлнэ.

console.log(isNaN(NaN)); // true
console.log(isNaN(undefined)); // true
console.log(isNaN({})); // true

console.log(isNaN(true)); // false
console.log(isNaN(null)); // false
console.log(isNaN(37)); // false

console.log(Number.isNaN(NaN)); // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0)); // true

console.log(Number.isNaN("NaN")); // false
console.log(Number.isNaN(undefined)); // false

```
14.2 parseFloat() метод: Тэмдэгтээс бутархай тоо гарган авна.
14.3 parseInt() метод: Тэмдэгтээс бүхэл тоо гарган авна. Эхний тоон бус тэмдэг дээр зогсоно. Хэрэв эхэнд нь зөв бүхэл тоо байхгүй бол NaN буцаана.
14.4 toFixed() метод: Тоон дээр дуудагдаж, аравтын дараах оронгийн тоог зааж өгнө. Үр дүнг мөр хэлбэрээр буцаана.
```
