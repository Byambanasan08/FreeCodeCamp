# JavaScript-ийн харьцуулалт болон нөхцөлт шалгах операторуудын тойм

1. Харьцуулалт болон null, undefined өгөгдлийн төрлүүд

**1.1 Харьцуулалт болон undefined:** Хувьсагчийг зарласан боловч утга оноогоогүй тохиолдолд тухайн хувьсагч нь `undefined` байна. Энэ нь анхны утга оноогоогүй хувьсагчид болон аргумент дамжуулаагүй функцийн параметрийн өгөгдмөл утга юм. `undefined` нь тоон контекстэд `NaN` (Not-a-Number) рүү хөрвүүлэгддэг бөгөөд энэ нь `undefined`-ийг тоотой харьцуулсан бүх тоон харьцуулалтыг `false` утгатай болгодог.

```javascript
console.log(undefined < 0); // false (NaN < 0 нь false байна)
console.log(undefined >= 0); // false (NaN >= 0 нь false байна)
```

**1.2 Харьцуулалт болон null:** `null` төрөл нь зориудаар "утга байхгүй" гэдгийг илэрхийлдэг. `null` нь тоон контекстэд `0` рүү хөрвүүлэгддэг бөгөөд энэ нь тоон харьцуулалт хийх үед гэнэтийн үр дүнд хүргэж болзошгүй:

```javascript
console.log(null < 0); // false (0 < 0 нь false байна)
console.log(null >= 0); // true (0 >= 0 нь true байна)
```

Тэнцүүгийн оператор (`==`) ашиглах үед `null` болон `undefined` нь зөвхөн бие биетэйгээ болон өөрсөдтэйгөө тэнцүү байдаг:

```javascript
console.log(null == undefined); // true
console.log(null == 0); // false
console.log(undefined == NaN); // false
```

Гэсэн хэдий ч, утга болон төрлийг хоёуланг нь шалгадаг, төрөл хувиргалт хийдэггүй нарийн тэнцүүгийн операторыг (`===`) ашиглах үед `null` болон `undefined` нь тэнцүү биш байна:

```javascript
console.log(null === undefined); // false
```

#### 2. Switch оператор

**2.1 Тодорхойлолт:** `switch` оператор нь илэрхийллийг тооцоолж, түүний утгыг хэд хэдэн `case` заалтуудтай тулгадаг. Тохирох утга олдоход тухайн `case`-тэй холбоотой кодны блок ажиллана. `case` бүрийн төгсгөлд түүний ажиллагааг зогсоож, дараагийн хэсэг рүү шилжихийн тулд `break` операторыг байрлуулах ёстой. `default` тохиолдол нь заавал байх албагүй бөгөөд өөр ямар ч `case` тохирохгүй үед л ажилладаг. `default` тохиолдлыг `switch` операторын төгсгөлд байрлуулдаг.

```javascript
const dayOfWeek = 3;

switch (dayOfWeek) {
  case 1:
    console.log("Даваа гараг! Долоо хоногоо эрч хүчтэй эхэлцгээе.");
    break;
  case 2:
    console.log("Мягмар гараг! Эрчээ алдалгүй үргэлжлүүлээрэй.");
    break;
  case 3:
    console.log("Лхагва гараг! Бид замынхаа хагаст ирлээ.");
    break;
  case 4:
    console.log("Пүрэв гараг! Амралтын өдрүүд дөхөж байна.");
    break;
  case 5:
    console.log("Баасан гараг! Амралтын өдрүүд ойрхон байна.");
    break;
  case 6:
    console.log("Бямба гараг! Амралтын өдрөө сайхан өнгөрүүлээрэй.");
    break;
  case 7:
    console.log("Ням гараг! Амарч, цэнэглэгдээрэй.");
    break;
  default:
    console.log("Буруу өдөр! 1-ээс 7-ын хооронд тоо оруулна уу.");
}
```
